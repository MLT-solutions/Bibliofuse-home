# Как конвертировать ZIP в CBZ онлайн: переименуйте архивы комиксов за секунды

У вас есть ZIP-файл с изображениями комиксов, но ридер его не распознаёт. Решение занимает три секунды: CBZ-файлы — это ZIP-архивы с другим расширением. Переименуйте файл, и ридер увидит его как комикс. [Веб-инструмент BiblioFuse](https://bibliofuse.com/ru/webapp/) делает это прямо в браузере — без загрузки на сервер, без ПО, без регистрации.

## Почему ридер игнорирует ZIP-файлы

Большинство ридеров определяют тип файла по расширению, а не по содержимому. Файл с именем `chapter-01.zip` выглядит как обычный архив. Тот же файл, переименованный в `chapter-01.cbz`, сразу распознаётся как контейнер комикса. Байты не изменились — только имя.

CBZ расшифровывается как Comic Book ZIP. Формат CBZ создан исключительно для того, чтобы дать ридерам распознаваемое расширение. Внутри всегда ZIP с изображениями, отсортированными в нужном порядке чтения. CBR — тот же принцип, но с RAR-архивом.

## Способ 1 — Переименовать файл напрямую

Самый быстрый способ — просто переименовать. На Mac или Windows измените расширение с `.zip` на `.cbz` в Finder или Проводнике. На iOS:

1. Откройте приложение **Файлы** и найдите ZIP-файл.
2. Нажмите и удерживайте файл → **Переименовать**.
3. Измените `.zip` на `.cbz` и нажмите **Готово**.

iOS может предупредить об изменении расширения. Подтвердите. Файл теперь CBZ, который откроет любой ридер комиксов — включая [BiblioFuse для iPhone](https://bibliofuse.com/ru/).

## Способ 2 — Использовать веб-инструмент BiblioFuse

[Веб-инструмент BiblioFuse](https://bibliofuse.com/ru/webapp/) работает полностью в браузере через WebAssembly. Ваши файлы никуда не отправляются — всё обрабатывается локально.

Для конвертации ZIP в CBZ:
1. Откройте **[bibliofuse.com/ru/webapp](https://bibliofuse.com/ru/webapp/)** в любом современном браузере.
2. Выберите инструмент **Конвертировать**.
3. Перетащите ZIP-файл или нажмите для выбора.
4. Выберите **ZIP → CBZ** как формат вывода.
5. Нажмите **Конвертировать** и скачайте результат.

Полученный CBZ содержит точно те же изображения, что и исходный ZIP, в том же порядке, без перекодирования и потери качества.

## Пакетная конвертация

Если у вас десятки ZIP-файлов, веб-инструмент поддерживает множественный ввод: перетащите несколько ZIPs сразу и получите CBZ для каждого.

## Что внутри CBZ

CBZ — это ZIP с изображениями, отсортированными по имени файла в алфавитном порядке. Ридер отображает их в этом порядке. Рекомендуемое именование: `0001.jpg`, `0002.jpg`... (с ведущими нулями). Без нулей `10.jpg` встанет раньше `2.jpg` в алфавитной сортировке.

## Что насчёт CBR?

CBR использует RAR вместо ZIP. Конвертация CBR в CBZ требует распаковки RAR и перепаковки в ZIP. Веб-инструмент BiblioFuse обрабатывает и это: выберите **CBR → CBZ**, и он перепакует изображения в ZIP с расширением CBZ.

## Главный вывод

CBZ — это замаскированный ZIP. В большинстве случаев достаточно переименовать расширение. Когда нужна браузерная конвертация без загрузки файлов, [веб-инструмент BiblioFuse](https://bibliofuse.com/ru/webapp/) конвертирует ZIP в CBZ за секунды — полностью конфиденциально, без ПО.
