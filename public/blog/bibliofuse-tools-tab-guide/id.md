# Menggunakan Tab Alat BiblioFuse: Kompres, Konversi, dan Gabung di iPhone

Anda sedang istirahat makan siang, memegang ponsel, dan menyadari koleksi komik menghabiskan terlalu banyak ruang penyimpanan iPhone. File CBZ dari seri manga masing-masing berukuran 80–120 MB. Anda ingin mengompresnya, tetapi laptop ada di rumah. Atau mungkin Anda punya beberapa file bab yang ingin digabung menjadi satu volume sebelum sesi membaca yang panjang.

[BiblioFuse](/id/) menangani semua ini tanpa komputer. Tab Alat — yang terintegrasi dalam aplikasi iOS — memungkinkan Anda mengompres, mengonversi, dan menggabungkan komik serta ebook langsung di iPhone atau iPad. Semua pemrosesan berjalan di perangkat. Tidak ada file yang diunggah ke mana pun.

## Apa yang Dilakukan Tab Alat

[BiblioFuse](/id/) mengintegrasikan pemrosesan file langsung ke dalam pembaca. Tab Alat menawarkan tiga fungsi utama:

- **Kompres** — kurangi ukuran file dengan mengode ulang gambar di dalam CBZ atau EPUB
- **Konversi** — ubah file dari satu format ke format lain (misalnya CBZ ke EPUB, PDF ke CBZ)
- **Gabung** — kombinasikan beberapa file menjadi satu arsip

Setiap operasi bersifat non-destruktif secara default: file baru dibuat, asli tidak berubah.

## Kompres: Perkecil File Tanpa Merusak Kualitas

1. Buka tab **Alat** (ikon kunci pas)
2. Ketuk **Kompres**
3. Tambahkan file CBZ, CBR, atau EPUB
4. Pilih pengaturan: kualitas (75–85% biasanya tidak terlihat di layar), format (JPEG atau WebP), dimensi maksimum
5. Ketuk **Kompres**

File yang dikompres muncul di perpustakaan. Aslinya tidak berubah.

## Konversi: Ubah Format File di iPhone

**PDF ke CBZ**: memisahkan setiap halaman menjadi file gambar individual. **RAR ke CBZ**: CBZ didukung lebih luas. Cara penggunaan: Alat → Konversi → pilih file → pilih format target → Konversi.

## Gabung: Kombinasikan Beberapa File Menjadi Satu Volume

1. Alat → **Gabung**
2. Tambahkan file dan atur urutannya sesuai urutan baca
3. Beri nama file output
4. Ketuk **Gabung**

Hasilnya adalah arsip CBZ tunggal. File asli tidak dihapus.

## Pemrosesan di Perangkat

Ketiga alat ini berjalan sepenuhnya di iPhone atau iPad tanpa koneksi internet. BiblioFuse memproses file di latar belakang, sehingga Anda bisa terus membaca selagi kompresi berjalan.

[BiblioFuse](/id/) tersedia di App Store untuk iPhone dan iPad.
