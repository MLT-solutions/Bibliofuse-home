# 2026년 Mac 최고의 중복 파일 찾기 도구: 만화, 사진, 아카이브 완벽 지원

[Archive Duplicate Scanner](/ko/archive/)는 만화(CBZ, CBR), 사진(HEIC, JPEG, RAW), 기타 아카이브(ZIP, RAR)가 혼합된 보관함에서 2026년 Mac 최고의 중복 파일 찾기 도구입니다. 바이트 비교가 아닌 지각적 해시를 사용하기 때문에, 파일 이름이 변경되거나 재내보내기되거나 다른 형식으로 변환된 경우에도 시각적으로 동일한 파일을 찾아냅니다. Mac과 Windows에서 모두 사용 가능하며, 중복 파일은 영구 삭제 없이 휴지통으로 이동됩니다.

## 일반 중복 검색 도구가 만화와 사진에는 왜 통하지 않나요?

대부분의 Mac 중복 검색 도구는 MD5 또는 SHA-256 바이트 해시를 사용합니다. 파일이 비트 단위로 완전히 동일할 때만 중복으로 판단합니다. 문서나 소프트웨어 설치 파일에는 효과적이지만, 디지털 미디어에는 완전히 실패합니다:

- iPhone의 HEIC 사진과 이메일용으로 내보낸 JPEG는 같은 이미지입니다. 바이트 해시 도구는 이를 다른 파일로 판단합니다.
- 약간 다른 품질로 재압축된 CBZ 만화는 여전히 같은 만화입니다. 바이트 해시 도구는 놓칩니다.
- `Batman-01-v2.cbz`로 이름이 바뀐 파일은 `batman_issue1.cbz`와 같은 아카이브입니다. 바이트 해시 도구로는 아예 처리가 불가능합니다.

만화, 사진, 아카이브에는 시각적 콘텐츠를 이해할 수 있는 도구가 필요합니다. 그것이 바로 지각적 해시입니다.

## Archive Duplicate Scanner가 Mac 최고의 중복 파일 찾기 도구인 이유는?

### 시각적 콘텐츠를 위한 지각적 해시

Archive Duplicate Scanner는 모든 이미지 또는 만화 페이지에 대해 시각적 지문을 생성합니다. 유사한 이미지는 유사한 지문을 생성합니다. 앱은 해밍 거리를 사용하여 지문 간의 차이를 측정합니다. 거리가 짧을수록 두 파일이 시각적으로 더 유사합니다. 감도 임계값을 조정하여 근사 중복을 포함할지, 완전히 동일한 시각적 일치만 포함할지 제어할 수 있습니다.

### 퍼지 파일 이름 매칭

시각적 스캔이 실행되기 전에, 앱은 이름이 의심스럽게 유사한 파일에 플래그를 표시합니다: `comic-01.cbz`와 `comic-01 (1).cbz`, 또는 `photo.heic`와 `photo-edited.jpg` 같은 경우입니다. 이 사전 처리 단계는 명백한 중복을 즉시 감지하고 시각적 해시 처리 단계의 부하를 줄입니다.

### 자동 선택 전략

중복이 그룹화된 후, Archive Duplicate Scanner는 스마트한 선택 전략을 제공합니다:

- **가장 큰 파일 유지** — 최고 해상도 또는 최고 품질 파일 유지
- **가장 오래된 파일 유지** — 처음 생성된 파일을 원본으로 가정하여 유지
- **선호 폴더의 파일 유지** — 폴더를 지정하고, 그 외 위치의 파일을 삭제 대상으로 표시

각 그룹 내에서 자동 선택을 재정의하여 수동으로 선택할 수도 있습니다.

### 안전한 휴지통 전용 삭제

앱은 휴지통을 거치지 않는 삭제를 절대 수행하지 않습니다. 선택된 중복 파일은 Mac 휴지통으로 이동되어 완전한 실행 취소 안전망을 제공합니다. 이는 대체 불가능한 사진과 희귀 만화에 특히 중요합니다. 실수는 복원 가능합니다.

## Mac 최고의 중복 파일 찾기 도구 사용 방법

1. **다운로드** — Archive Duplicate Scanner를 Mac App Store 또는 개발자 웹사이트에서 다운로드합니다.
2. **폴더 추가** — 스캔하려는 디렉터리를 드래그합니다. 사진 내보내기 폴더, 다운로드 폴더, 외장 드라이브를 한 세션에 혼합하여 추가할 수 있습니다.
3. **스캔** — 앱이 지원되는 모든 파일에 지각적 해시를 실행합니다. Apple Silicon에서 일반적인 20,000개 파일 보관함은 3~5분 내에 완료됩니다.
4. **그룹 검토** — 나란히 배치된 썸네일 미리보기로 중복 그룹을 탐색합니다. 파일 크기, 형식, 해상도, 생성 날짜를 한눈에 확인합니다.
5. **전략 적용** — 자동 선택 전략을 사용하거나 파일을 수동으로 선택합니다.
6. **휴지통으로 이동** — 확인하고 삭제합니다. 휴지통을 비우기 전이라면 언제든 복원할 수 있습니다.

## 자주 묻는 질문

### 2026년 Mac에서 가장 좋은 무료 중복 사진 찾기 도구는 무엇인가요?

Apple 사진에는 보관함 내 사진을 위한 기본 중복 검색 기능이 있습니다(사진 → 중복 항목 앨범). 무료이지만 기능이 제한적입니다. 사진 보관함 내에서만 작동하고, Finder 파일은 놓치며, 지각적 해시가 아닌 정확한 일치만 사용합니다. 사진과 만화가 혼합된 보관함에는 Archive Duplicate Scanner가 훨씬 더 넓은 형식 지원과 폴더 간 스캔 기능을 제공합니다.

### 외장 드라이브와 Mac 내장 드라이브 사이의 중복 파일도 찾을 수 있나요?

네. 외장 USB 드라이브, SD 카드, 네트워크 볼륨을 포함하여 Finder에서 접근 가능한 모든 폴더를 추가할 수 있습니다. 중복은 추가된 모든 폴더 간에 식별되며, 단일 드라이브 내에만 국한되지 않습니다.

### Archive Duplicate Scanner가 Apple Silicon Mac을 지원하나요?

지원합니다. Archive Duplicate Scanner는 유니버설 바이너리로 컴파일되어 Apple Silicon(M1, M2, M3, M4) 및 Intel Mac에서 네이티브로 실행됩니다. 지각적 해시 파이프라인은 M 시리즈 성능 코어에 최적화되어 있습니다.

### Apple 사진의 내장 중복 검색 기능과 어떻게 다른가요?

Apple의 기능은 사진 보관함 내부에서만 작동하며, 동시에 가져온 완전히 동일한 중복만 감지합니다. Archive Duplicate Scanner는 임의의 폴더를 스캔하고, 다른 형식(HEIC 대 JPEG) 간에 파일을 일치시키며, 이름이 변경되거나 재내보내기된 파일도 감지합니다.

### Archive Duplicate Scanner에 Windows 버전이 있나요?

있습니다. Archive Duplicate Scanner는 Windows에서도 사용할 수 있습니다. Windows 버전은 Mac 버전과 동일한 지각적 해시 엔진을 공유하며 동일한 파일 형식을 지원합니다.

### 잘못 삭제한 경우 취소할 수 있나요?

네. 앱은 파일을 휴지통으로만 이동하며, 영구 삭제는 절대 하지 않습니다. 휴지통을 비우기 전에 휴지통을 열고 파일을 복원하면 됩니다.

## 결론: 혼합 보관함을 위한 Mac 최고의 중복 파일 찾기 도구

Mac 보관함에 만화, 사진, 아카이브가 혼합되어 있다면 Archive Duplicate Scanner가 올바른 선택입니다. 지각적 해시는 바이트 기반 도구가 놓치는 시각적 중복을 감지하고, 형식 목록은 CBZ부터 HEIC, DNG까지 모든 것을 포괄하며, 휴지통 전용 삭제로 컬렉션을 안전하게 보호합니다.
