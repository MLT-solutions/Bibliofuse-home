# iTunes나 iCloud 없이 iPhone에 만화 전송하는 방법

BiblioFuse는 iOS 17 이상이 설치된 iPhone 및 iPad에서 CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT 파일을 읽을 수 있으며, Mac 또는 PC의 어떤 브라우저에서든 만화를 직접 복사할 수 있는 내장 Wi-Fi 전송 기능을 갖추고 있습니다. iTunes, 케이블, iCloud 계정이 필요 없습니다. BiblioFuse 앱에서 Wi-Fi 전송 페이지를 열고 브라우저에 파일을 드래그 앤 드롭하면 몇 초 안에 라이브러리에 나타납니다.

## iTunes 없이 iPhone에 만화 전송하는 것이 표준이 된 이유

Apple은 macOS Catalina부터 독립 iTunes 데스크톱 앱을 제거했습니다. iOS 앱의 파일 공유는 Finder의 기기 패널로 이동했습니다: 케이블 연결, 기기 선택, 파일 공유 클릭, 앱 선택, 작은 창에 파일 드래그. 케이블 하나, 여러 단계.

Windows에서 iTunes는 여전히 존재하지만 무겁고 케이블이 필요합니다. 어댑터를 가지고 다니지 않는 노트북 사용자에게는 케이블 전송이 불필요한 장애물입니다.

## Wi-Fi 전송이 iPhone에 만화를 전송하는 최선의 방법인 이유

BiblioFuse의 Wi-Fi 전송은 케이블, iTunes, iCloud 계정, Mac이나 PC에 소프트웨어 설치가 필요 없습니다. 일반 가정용 Wi-Fi 연결에서 500MB CBZ 권이 15초 이내에 전송됩니다.

## BiblioFuse로 iTunes 없이 iPhone에 만화 전송하는 방법

1. iPhone 또는 iPad에서 **BiblioFuse**를 엽니다.
2. 화면 하단의 **전송** 탭을 탭합니다.
3. BiblioFuse가 로컬 IP 주소와 포트를 표시합니다 (예: `http://192.168.1.42:8080`).
4. Mac 또는 PC에서 브라우저를 열고 주소 표시줄에 해당 주소를 입력합니다.
5. BiblioFuse Wi-Fi 전송 페이지가 나타납니다. CBZ, EPUB, PDF 또는 기타 만화 파일을 페이지에 직접 드래그 앤 드롭합니다.
6. 파일이 즉시 업로드되어 iPhone의 BiblioFuse 라이브러리에 나타납니다.

iPhone과 컴퓨터는 같은 Wi-Fi 네트워크에 있어야 합니다. 배치당 파일 수 제한이 없습니다.

## 케이블 없이 iPhone에 만화를 전송하는 다른 방법

### 방법 2: 파일 앱과 iCloud Drive 또는 서드파티 클라우드 스토리지

만화가 iCloud Drive, Dropbox, Google Drive, OneDrive에 저장된 경우:

1. iPhone에서 **파일** 앱을 엽니다.
2. 클라우드 스토리지의 만화 파일로 이동합니다.
3. 파일을 길게 누르고 **"다음에서 열기" → BiblioFuse**를 선택합니다.

### 방법 3: AirDrop (Mac에서 iPhone만 가능)

1. Mac의 Finder에서 CBZ 또는 EPUB 파일을 하나 이상 선택합니다.
2. 우클릭하고 **공유 → AirDrop**을 선택합니다.
3. AirDrop 목록에서 iPhone을 선택합니다.
4. iPhone에서 전송을 수락합니다. **BiblioFuse**를 선택해 파일을 엽니다.

### 방법 4: Safari 다운로드 후 열기

iPhone의 Safari에서 만화 파일을 직접 다운로드하는 경우:

1. Safari에서 다운로드 링크를 탭합니다.
2. 다운로드 완료 후 다운로드 표시기에서 파일을 탭합니다.
3. 파일을 길게 누르고 **공유 → BiblioFuse에서 열기**를 선택합니다.

## 원활한 전송을 위한 팁

**Wi-Fi 전송 중 BiblioFuse를 전경에서 열어 두세요.** 로컬 웹 서버는 앱이 활성 상태일 때만 작동합니다. 다른 앱으로 전환하면 전송이 일시 중지됩니다.

**같은 Wi-Fi 네트워크를 사용하세요.** 두 기기 모두 같은 네트워크에 있어야 합니다. 게스트 네트워크와 클라이언트 격리 설정은 로컬 연결을 차단합니다.

**전송 전에 시리즈를 폴더로 정리하세요.** 한 번에 하나의 시리즈를 전송하면 처음부터 iPhone 라이브러리가 정리됩니다.

## 자주 묻는 질문

**컴퓨터 없이 iPhone에 만화를 전송할 수 있나요?**
네. iPhone에서 파일 앱을 열고 iCloud Drive, Dropbox 또는 호환 제공업체에 저장된 만화로 이동해 BiblioFuse에서 열기를 탭합니다.

**BiblioFuse의 Wi-Fi 전송은 모든 만화 형식과 호환되나요?**
네. Wi-Fi 전송은 CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT — BiblioFuse가 지원하는 모든 형식을 허용합니다.

**BiblioFuse Wi-Fi 전송을 사용하려면 계정이 필요한가요?**
아니요. Wi-Fi 전송은 앱에 내장된 로컬 네트워크 기능입니다. 계정, 구독, 인터넷 연결이 필요 없습니다.

**BiblioFuse Wi-Fi 전송 주소가 브라우저에서 로드되지 않는 이유는?**
두 기기 모두 같은 Wi-Fi 네트워크에 있어야 합니다. VPN이 활성화된 경우 일시적으로 비활성화하세요. 일부 라우터는 기본적으로 클라이언트 격리가 활성화되어 있습니다.

**BiblioFuse Wi-Fi 전송에 파일 크기 제한이 있나요?**
없습니다. BiblioFuse Wi-Fi 전송은 파일당 또는 세션 전체에 대한 크기 제한이 없습니다.

**iPhone 앱으로 파일 전송을 위한 iTunes 파일 공유의 대체재는 무엇인가요?**
Apple은 2019년 iTunes 앱을 삭제할 때 iOS 앱 파일 공유를 Mac의 Finder로 이전했습니다. BiblioFuse의 Wi-Fi 전송은 케이블이나 iTunes 없이 두 방법 모두를 대체합니다.
