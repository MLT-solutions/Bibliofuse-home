# Cara BiblioFuse Menyinkronkan Progres Baca di iPhone, iPad, dan Mac melalui iCloud

Kamu sedang membaca volume manga di iPhone saat perjalanan — sampai halaman 80, lalu beralih ke iPad di rumah dan harus mencari-cari melalui thumbnail halaman untuk menemukan di mana kamu berhenti. Atau kamu membaca satu bab di Mac dan keesokan paginya mengambil iPhone — yang masih menampilkan halaman satu.

Sinkronisasi progres baca terdengar sederhana, namun gagal dengan cara yang halus di sebagian besar pembaca. [BiblioFuse](/id/) menggunakan iCloud untuk menyinkronkan posisi baca yang tepat — nomor halaman, arah baca, dan pengaturan tampilan — di iPhone, iPad, dan Mac, tanpa langkah manual.

## Mengapa Sinkronisasi Progres Sulit Dilakukan dengan Benar

Pendekatan sederhana — menyimpan nomor halaman di dokumen cloud — gagal dalam beberapa hal: tidak mempertimbangkan volume dengan jumlah halaman berbeda dalam encoding yang berbeda, tidak menyinkronkan arah baca, dan tidak memiliki mekanisme untuk menandai volume sebagai "selesai" secara persisten di berbagai perangkat.

[BiblioFuse](/id/) menyimpan data progres sebagai metadata yang terikat ke hash konten file, bukan nama file. Bahkan jika kamu mengganti nama CBZ atau memindahkannya antar folder, data progres masih cocok dengan benar di perangkat mana pun.

## Menyiapkan Sinkronisasi iCloud

Di iPhone atau iPad:
1. Buka **Pengaturan** (pengaturan sistem)
2. Ketuk Apple ID → **iCloud** → **iCloud Drive**
3. Temukan **BiblioFuse** dan pastikan tombol aktif

Di Mac:
1. Buka **Pengaturan Sistem → Apple ID → iCloud**
2. Aktifkan iCloud Drive dan pastikan BiblioFuse ada dalam daftar

Setelah diaktifkan, BiblioFuse mengelola segalanya secara otomatis. Tidak ada tombol "Sinkronkan Sekarang" — perubahan disebarkan di latar belakang saat perangkat online.

## Yang Disinkronkan

**Posisi baca.** Nomor halaman yang tepat. Jika kamu menutup volume di halaman 87, setiap perangkat lain membukanya di halaman 87.

**Arah baca.** Jika kamu menyetel volume manga ke kanan-ke-kiri, pengaturan itu mengikuti volume ke setiap perangkat.

**Status selesai.** Menandai volume sebagai telah dibaca di satu perangkat menandainya di semua.

**Tag dan penilaian.** Tag dan penilaian bintang tersinkronisasi di semua perangkat.

## Cara Sinkronisasi File iCloud Drive Bekerja

[BiblioFuse](/id/) juga menggunakan iCloud Drive untuk file perpustakaan itu sendiri.

**Mac ke iPhone**: Buka Finder dan navigasi ke **iCloud Drive → BiblioFuse**. Letakkan CBZ, CBR, EPUB, atau PDF apa pun di folder itu. File muncul di BiblioFuse di iPhone dalam hitungan menit.

File yang ditambahkan melalui Wi-Fi Transfer juga disimpan di folder iCloud BiblioFuse dan tersedia di perangkat lain.

## Wi-Fi Transfer untuk Pemuatan Massal

Untuk pengaturan awal atau transfer batch besar, Wi-Fi Transfer lebih cepat dari sinkronisasi iCloud.

1. Buka **Pengaturan → Wi-Fi Import** di aplikasi
2. Aktifkan
3. Kunjungi URL yang ditampilkan di browser pada jaringan Wi-Fi yang sama
4. Seret koleksimu ke jendela browser

[BiblioFuse](/id/) menerima dengan kecepatan jaringan lokal: 50–100 MB/s di jaringan rumah biasa.

## Mac Home Library: Membaca Tanpa Menyalin File

Jika koleksi utamamu ada di Mac dan kamu tidak ingin memenuhi penyimpanan iPhone, Mac Home Library menawarkan pendekatan berbeda: streaming volume langsung dari Mac ke iPhone melalui Wi-Fi lokal. Tidak ada file yang disalin ke perangkat atau ke iCloud.

Cara mengaktifkan:
1. Pastikan BiblioFuse terinstal di Mac
2. Di aplikasi Mac, buka **Preferensi → Home Library** dan aktifkan berbagi
3. Di iPhone, buka BiblioFuse dan beralih ke tab Perpustakaan Mac

Progres baca dari sesi Mac Home Library tetap disinkronkan melalui iCloud.

## Membaca Offline

Jika membaca offline, BiblioFuse menyimpan progres secara lokal dan menyinkronkan saat perangkat terhubung berikutnya. Jika dua perangkat konflik, perangkat dengan nomor halaman lebih tinggi yang menang.

## Mulai Sekarang

[BiblioFuse](/id/) gratis di App Store untuk iPhone dan iPad. Sinkronisasi iCloud sudah termasuk tanpa tingkat premium. Aktifkan iCloud Drive untuk BiblioFuse di pengaturan perangkat dan progresmu mulai tersinkronisasi segera.
