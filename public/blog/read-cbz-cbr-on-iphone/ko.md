# iPhone에서 CBZ, CBR 만화 파일 읽는 방법

BiblioFuse는 iOS 17 이상의 iPhone 및 iPad에서 CBZ 및 CBR 만화 파일을 변환 없이 읽을 수 있게 해준다. 파일은 파일 앱에서 직접 열거나, 내장 Wi-Fi Transfer 기능을 통해 케이블이나 iTunes 없이 Mac에서 iPhone으로 몇 분 만에 전체 만화 시리즈를 무선으로 전송할 수 있다.

만화 챕터, 그래픽 노블, 스캔된 만화가 담긴 CBZ 또는 CBR 파일을 찾아 iPhone에서 눌렀는데 아무것도 열리지 않았나요? 어떤 앱도 나타나지 않았나요? 혼자가 아닙니다.

iOS는 CBZ 및 CBR 파일을 기본 지원하지 않습니다. Apple Books 앱은 EPUB과 PDF를 처리하지만 아카이브 기반 만화 형식은 완전히 무시합니다. iPhone에서 이 파일들을 제대로 읽으려면 전용 앱이 필요합니다. [BiblioFuse](/ko/)는 바로 이를 위해 만들어졌습니다 — CBZ, CBR, ZIP, RAR, EPUB, PDF, TXT를 지원하며, 긴 독서 세션과 대형 컬렉션을 위한 리더를 갖추고 있습니다.

이 가이드는 형식 설명부터 iPhone 파일 전송, 독서 환경 최적화까지 모든 것을 다룹니다.

## CBZ와 CBR이란 무엇인가

CBZ와 CBR은 만화 아카이브 형식입니다. 특수 인코딩이 아니라 확장자만 바꾼 일반 ZIP 및 RAR 파일입니다. `.cbz` 파일의 압축을 풀면 읽기 순서대로 나열된 이미지 폴더가 나옵니다: `001.jpg`, `002.jpg` 식으로요.

| 확장자 | 컨테이너 | 주요 용도 |
|--------|---------|---------|
| CBZ | ZIP | 서양 만화, 만가, 팬 스캔 |
| CBR | RAR | 구형 만화 릴리즈, 일본 스캔 |
| ZIP | ZIP | CBZ와 동일 (확장자 변경 없음) |
| RAR | RAR | CBR과 동일 (확장자 변경 없음) |

이 형식은 의도적으로 단순하게 설계되었습니다 — DRM 없음, 암호화 없음, 독점 구조 없음. 그래서 만화와 만가 커뮤니티에서 선호되는 형식입니다.

iPhone에서의 문제는 iOS가 ZIP 파일을 일반 아카이브로만 열기 때문에 내부 이미지를 만화로 표시하지 못한다는 것입니다. 순차 페이지 구조를 이해하는 리더가 필요합니다.

## 1단계 — BiblioFuse 설치

[BiblioFuse](/ko/)는 App Store에서 무료로 다운로드할 수 있습니다. 계정 불필요.

설치 후 파일 앱에 두 개의 폴더가 생성됩니다:
- **iCloud Drive → BiblioFuse** — 모든 기기 간 동기화 저장소
- **내 iPhone에 → BiblioFuse** — 로컬 전용 저장소

파일을 가져오기 전에 별도 설정이 필요 없습니다.

## 2단계 — 만화를 iPhone으로 가져오기

CBZ 또는 CBR 파일을 iPhone으로 옮기는 세 가지 주요 방법입니다.

### 파일 앱에서 공유하기

만화가 이미 iCloud Drive나 접근 가능한 Mac/PC에 있다면:

1. **파일** 앱 열기
2. 파일로 이동
3. 파일을 길게 누르고 **공유** 탭
4. 공유 메뉴에서 **BiblioFuse** 선택

파일이 즉시 가져와집니다.

### Wi-Fi 전송 — 대형 컬렉션에 최적

Wi-Fi 전송을 사용하면 케이블이나 iTunes 없이 Mac이나 PC에서 만화 폴더 전체를 BiblioFuse로 드래그할 수 있습니다.

1. BiblioFuse에서 **설정 → Wi-Fi 가져오기** 켜기
2. 로컬 URL이 표시됩니다 (예: `http://192.168.1.x:8080`)
3. 컴퓨터 브라우저에서 해당 URL 열기
4. CBZ 또는 CBR 파일 (또는 시리즈 폴더 전체)을 브라우저 창에 드래그
5. 파일이 로컬 네트워크를 통해 iPhone 라이브러리로 직접 전송됨

전송 후 Wi-Fi 가져오기를 끄세요. 파일이 라이브러리에 자동으로 나타납니다.

### iCloud Drive — 기기 간 접근

iPhone, iPad, Mac 모두에서 읽는다면 iCloud Drive에 저장하는 것이 최선입니다.

1. Mac에서 만화 폴더를 **iCloud Drive → BiblioFuse**에 복사
2. iPhone에서 파일 앱을 열고 다운로드 완료 대기
3. BiblioFuse 열기 — 폴더가 라이브러리에 자동으로 표시됨

여기 저장된 책은 iCloud를 통해 읽기 진행상황이 동기화됩니다.

## 3단계 — 권호 열고 읽기 시작

만화가 라이브러리에 있으면 표지를 탭해 엽니다. BiblioFuse가 아카이브를 로드하고 페이지를 추출해 첫 페이지를 즉시 표시합니다.

### 페이지 탐색

- **좌우 스와이프** — 한 페이지 앞으로/뒤로
- **오른쪽 가장자리 탭** — 다음 페이지
- **왼쪽 가장자리 탭** — 이전 페이지
- **가운데 탭** — 탐색 바 표시/숨기기

### 읽기 방향

일본 만화(오른쪽에서 왼쪽으로 읽음)는 읽기 방향을 바꿀 수 있습니다. 화면 가운데를 탭해 컨트롤을 열고 **오른쪽에서 왼쪽**으로 전환하세요.

### 양면 펼침 보기

많은 서양 만화는 두 페이지에 걸친 아트워크가 있습니다. iPhone을 가로 방향으로 돌리면 [BiblioFuse](/ko/)가 자동으로 양면 펼침 보기로 전환합니다.

### 확대

어떤 페이지든 핀치 줌으로 확대할 수 있습니다. BiblioFuse는 고해상도 렌더링을 유지합니다.

## 시리즈 정리

같은 시리즈의 여러 권을 가져오면 BiblioFuse가 폴더별로 그룹화합니다. 표지를 길게 누르면 별점(1~5점) 또는 커스텀 태그를 추가할 수 있습니다.

## 문제 해결

**파일을 전송했는데 라이브러리에 표시되지 않음.** 아래로 당겨 새로고침하세요. 파일 앱에서 **내 iPhone에 → BiblioFuse**에 파일이 있는지 확인하세요.

**페이지 순서가 잘못됨.** 일부 파일은 앞자리 0 없이 페이지 번호를 매깁니다. BiblioFuse는 알파벳 숫자 순으로 읽습니다. 순서가 잘못됐다면 아카이브 내 파일 이름 문제입니다.

**CBR 파일이 열리지 않음.** 매우 오래된 CBR 파일은 RAR5 형식을 사용합니다. CBZ로 재패키징을 시도해보세요.

## 다음 단계

첫 번째 시리즈를 가져와 정상적으로 읽을 수 있게 됐다면, [BiblioFuse](/ko/)의 다른 기능도 탐색해보세요. 도구 탭에서 기기 내에서 직접 압축 및 변환을 할 수 있습니다.


## 자주 묻는 질문

**iPhone에서 CBZ 파일을 어떻게 여나요?**
App Store에서 BiblioFuse를 설치합니다. 파일 앱에서 CBZ 파일을 탭하면 iOS가 BiblioFuse로 열기를 표시합니다. 설정의 Wi-Fi 가져오기에서 Wi-Fi Transfer도 사용할 수 있습니다.

**iPhone이 CBR 파일을 네이티브로 읽을 수 있나요?**
네이티브로는 읽을 수 없습니다. iOS에는 CBZ나 CBR에 대한 기본 지원이 없습니다. BiblioFuse는 iOS 17 이상에서 CBZ, CBR, ZIP, RAR을 변환 없이 네이티브로 읽습니다.

**CBZ와 CBR의 차이는 무엇인가요?**
CBZ는 만화 이미지를 담은 이름을 바꾼 ZIP 아카이브입니다. CBR은 이름을 바꾼 RAR 아카이브입니다. 내용은 동일하고 압축 방식만 다릅니다. BiblioFuse는 둘 다 직접 읽습니다.

**BiblioFuse가 큰 CBZ 파일을 메모리 부족 없이 읽을 수 있나요?**
네. BiblioFuse는 표시하는 데 필요한 페이지만 로드합니다. 500MB 이상의 CBZ 파일도 잘 작동합니다.

**iTunes 없이 iPhone에 전체 만화 시리즈를 전송하는 방법은?**
Wi-Fi Transfer를 사용합니다. BiblioFuse에서 설정의 Wi-Fi 가져오기로 이동해 서버를 켭니다. PC 브라우저에서 표시된 URL을 방문하고 시리즈 폴더 전체를 브라우저 창으로 드래그합니다.
