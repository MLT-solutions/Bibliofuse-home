# 如何在 iPhone 上阅读 CBZ 和 CBR 漫画

你找到了一个 CBZ 或 CBR 文件——一章漫画、一部图像小说或扫描版漫画——然后在 iPhone 上点开它。什么都没有发生，没有任何应用主动弹出来处理它。你并不孤单。

iOS 没有内置对 CBZ 或 CBR 格式的支持。苹果的"图书"应用支持 EPUB 和 PDF，但完全忽略了基于压缩包的漫画格式。要在 iPhone 上正确阅读这些文件，你需要一款专用应用。[BiblioFuse](/zh/) 正是为此而生——它支持 CBZ、CBR、ZIP、RAR、EPUB、PDF 和 TXT，阅读器专为长时阅读和大型书库设计。

本指南涵盖所有内容：这些格式是什么、如何将文件导入 iPhone，以及如何获得最佳阅读体验。

## CBZ 和 CBR 究竟是什么

CBZ 和 CBR 是漫画存档格式。它们并非特殊编码——只是将 ZIP 和 RAR 文件改了后缀名。解压任何 `.cbz` 文件，你会找到一组按阅读顺序排列的图片：`001.jpg`、`002.jpg`，依此类推。

| 后缀 | 容器 | 常见用途 |
|------|------|---------|
| CBZ | ZIP | 西方漫画、日漫、粉丝扫描版 |
| CBR | RAR | 早期漫画发布、日本扫描版 |
| ZIP | ZIP | 与 CBZ 相同，未改后缀 |
| RAR | RAR | 与 CBR 相同，未改后缀 |

该格式刻意设计得简单——没有 DRM、没有加密、没有专有结构。正因如此，它成为漫画和日漫社区的首选格式。

iPhone 上的问题在于：iOS 只将 ZIP 文件当作普通压缩包打开——不会将其中的图片以漫画形式显示。你需要一款能理解顺序页面结构的阅读器。

## 第一步 — 安装 BiblioFuse

[BiblioFuse](/zh/) 可在 App Store 下载，免费，无需账号。

安装后，它会在"文件"应用中创建两个文件夹：
- **iCloud 云盘 → BiblioFuse** — 用于在所有设备间同步的存储
- **我的 iPhone → BiblioFuse** — 用于仅在本机存储

导入文件前无需任何配置。

## 第二步 — 将漫画文件传到 iPhone

有三种主要方式将 CBZ 或 CBR 文件移至 iPhone。

### 从"文件"应用分享

如果漫画已在 iCloud 云盘或可访问的 Mac/PC 上：

1. 打开**文件**应用
2. 导航到该文件
3. 长按文件，点击**共享**
4. 在共享菜单中选择 **BiblioFuse**

文件会立即导入。

### Wi-Fi 传输 — 大型书库的最佳方式

Wi-Fi 传输让你无需数据线或 iTunes，直接将整个漫画文件夹从 Mac 或 PC 拖入 BiblioFuse。

1. 在 BiblioFuse 中，前往**设置 → Wi-Fi 导入**，开启开关
2. 屏幕上会显示一个本地 URL，例如 `http://192.168.1.x:8080`
3. 在电脑浏览器中打开该 URL
4. 将 CBZ 或 CBR 文件（或整个系列文件夹）拖入浏览器窗口
5. 文件通过局域网直接传输到 iPhone 书库

传输完成后，关闭 Wi-Fi 导入。文件会自动出现在书库中。

### iCloud 云盘 — 跨设备访问

如果你在 iPhone、iPad 和 Mac 上都阅读，将漫画存储在 iCloud 云盘中是最佳选择。

1. 在 Mac 上，将漫画文件夹复制到 **iCloud 云盘 → BiblioFuse**
2. 在 iPhone 上，打开"文件"等待下载完成
3. 打开 BiblioFuse——文件夹会自动出现在书库中

存储在此处的书籍会通过 iCloud 同步阅读进度。

## 第三步 — 打开册子，开始阅读

书库中有漫画后，点击任意封面即可打开。BiblioFuse 加载压缩包，提取页面，立即显示第一页。

### 页面导航

- **左右滑动** — 向前或向后翻一页
- **点击右边缘** — 下一页
- **点击左边缘** — 上一页
- **点击中央** — 显示或隐藏导航栏

### 阅读方向

对于日漫——从右至左阅读——可以切换阅读方向。点击屏幕中央打开控制项，切换为**从右至左**。

### 跨页显示

许多西方漫画有跨越两页的大幅插图。将 iPhone 横放，[BiblioFuse](/zh/) 会自动切换到双页视图，如原作般并排显示两页。

### 缩放

捏合手势即可放大任意页面。BiblioFuse 保持高分辨率渲染，放大细线条或小字时不会模糊。

## 整理系列漫画

导入同一系列的多册时，BiblioFuse 按文件夹分组显示。长按任意封面可添加评分（1–5 星）或自定义标签。

## 常见问题

**文件已传输但未出现在书库。** 下拉刷新书库视图。如果仍不显示，检查文件是否在"文件"应用的 **我的 iPhone → BiblioFuse** 中。

**页面顺序错乱。** 部分文件的页面命名没有前置零（如 `1.jpg` 排在 `10.jpg` 前）。BiblioFuse 按字母数字顺序读取文件。顺序错误是压缩包内文件命名的问题。

**CBR 文件无法打开。** 极旧的 CBR 文件使用 RAR5 格式。尝试将其重新打包为 CBZ。
