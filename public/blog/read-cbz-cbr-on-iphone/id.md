# Cara Membaca Komik CBZ dan CBR di iPhone

Kamu menemukan file CBZ atau CBR — bab manga, novel grafis, atau komik yang dipindai — lalu mengetuknya di iPhone. Tidak ada yang terbuka. Tidak ada aplikasi yang menawarkan diri. Kamu tidak sendirian.

iOS tidak memiliki dukungan bawaan untuk file CBZ atau CBR. Aplikasi Books Apple menangani EPUB dan PDF, tetapi mengabaikan format komik berbasis arsip sepenuhnya. Untuk membaca file-file ini dengan benar di iPhone, kamu butuh aplikasi khusus. [BiblioFuse](/id/) dibangun tepat untuk ini — membaca CBZ, CBR, ZIP, RAR, EPUB, PDF, dan TXT, dengan pembaca yang dirancang untuk sesi membaca panjang dan koleksi besar.

Panduan ini mencakup segalanya: apa itu format tersebut, cara memindahkan file ke iPhone, dan cara mendapatkan pengalaman membaca yang tepat.

## Apa Itu CBZ dan CBR

CBZ dan CBR adalah format arsip komik. Ini bukan encoding khusus — hanya file ZIP dan RAR biasa dengan ekstensi yang diubah namanya. Ekstrak file `.cbz` dan kamu akan menemukan folder gambar berurutan: `001.jpg`, `002.jpg`, dalam urutan baca.

| Ekstensi | Kontainer | Penggunaan umum |
|----------|-----------|----------------|
| CBZ | ZIP | Komik Barat, manga, scan penggemar |
| CBR | RAR | Rilisan komik lama, scan Jepang |
| ZIP | ZIP | Sama seperti CBZ tanpa ekstensi diganti |
| RAR | RAR | Sama seperti CBR tanpa ekstensi diganti |

Formatnya sengaja dibuat sederhana — tidak ada DRM, tidak ada enkripsi, tidak ada struktur kepemilikan. Itulah mengapa ini menjadi format pilihan di komunitas manga dan komik.

Masalah di iPhone: iOS hanya membuka file zip sebagai arsip generik — gambar di dalamnya tidak ditampilkan sebagai komik. Kamu butuh pembaca yang memahami struktur halaman berurutan.

## Langkah 1 — Pasang BiblioFuse

[BiblioFuse](/id/) tersedia di App Store. Gratis diunduh, tanpa akun.

Setelah dipasang, ia membuat dua folder di aplikasi File:
- **iCloud Drive → BiblioFuse** — untuk penyimpanan yang disinkronkan di semua perangkat
- **Di iPhone Saya → BiblioFuse** — untuk penyimpanan lokal saja

Tidak perlu konfigurasi apa pun sebelum mengimpor file.

## Langkah 2 — Pindahkan Komikmu ke iPhone

Ada tiga cara utama memindahkan file CBZ atau CBR ke iPhonemu.

### Bagikan dari Aplikasi File

Jika komikmu sudah ada di iCloud Drive atau Mac/PC yang bisa diakses via File:

1. Buka aplikasi **File**
2. Navigasi ke file tersebut
3. Tekan lama file lalu ketuk **Bagikan**
4. Pilih **BiblioFuse** dari menu berbagi

File langsung diimpor.

### Transfer Wi-Fi — Terbaik untuk Koleksi Besar

Transfer Wi-Fi memungkinkan kamu menyeret seluruh folder komik dari Mac atau PC ke BiblioFuse tanpa kabel atau iTunes.

1. Di BiblioFuse, buka **Pengaturan → Impor Wi-Fi** dan aktifkan
2. URL lokal akan muncul — misalnya `http://192.168.1.x:8080`
3. Buka URL itu di browser komputer
4. Seret file CBZ atau CBR (atau seluruh folder seri) ke jendela browser
5. File dipindahkan langsung ke perpustakaan iPhone

Setelah transfer selesai, matikan Impor Wi-Fi. Filenya otomatis muncul di perpustakaan.

### iCloud Drive — Untuk Akses Multi-Perangkat

Jika kamu membaca di iPhone, iPad, dan Mac, simpan komik di iCloud Drive.

1. Di Mac, salin folder komik ke **iCloud Drive → BiblioFuse**
2. Di iPhone, buka File dan tunggu unduhan selesai
3. Buka BiblioFuse — folder muncul otomatis di perpustakaan

Buku yang disimpan di sini menyinkronkan kemajuan bacaan melalui iCloud.

## Langkah 3 — Buka Volume dan Mulai Membaca

Setelah komikmu ada di perpustakaan, ketuk sampul apa saja untuk membukanya. BiblioFuse memuat arsip, mengekstrak halaman, dan menampilkan halaman pertama langsung.

### Navigasi Halaman

- **Geser kiri/kanan** — maju atau mundur satu halaman
- **Ketuk tepi kanan** — halaman berikutnya
- **Ketuk tepi kiri** — halaman sebelumnya
- **Ketuk tengah** — tampilkan atau sembunyikan bilah navigasi

### Arah Baca

Untuk manga — yang dibaca dari kanan ke kiri — kamu bisa membalik arah baca. Ketuk tengah layar untuk membuka kontrol, lalu ganti ke **Kanan ke Kiri**.

### Halaman Ganda

Banyak komik Barat memiliki ilustrasi yang membentang dua halaman. Putar iPhone ke mode lanskap dan [BiblioFuse](/id/) otomatis beralih ke tampilan dua halaman.

### Zoom

Cubit untuk memperbesar halaman mana saja. BiblioFuse mempertahankan rendering resolusi tinggi.

## Mengatur Seri

Saat mengimpor beberapa volume seri yang sama, BiblioFuse mengelompokkannya per folder. Tekan lama sampul untuk menambah rating atau tag khusus.

## Pemecahan Masalah

**File dipindahkan tapi tidak muncul di perpustakaan.** Tarik ke bawah untuk memperbarui. Periksa apakah file ada di **Di iPhone Saya → BiblioFuse** di aplikasi File.

**Halaman urutannya salah.** Beberapa file memberi nomor halaman tanpa nol di depan. BiblioFuse membaca secara alfanumerik. Jika urutannya salah, itu masalah penamaan dalam arsip itu sendiri.

**File CBR tidak bisa dibuka.** Beberapa CBR sangat lama menggunakan RAR5. Coba kemas ulang sebagai CBZ.
