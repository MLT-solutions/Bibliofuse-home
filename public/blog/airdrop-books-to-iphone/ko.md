# AirDrop으로 iPhone에 만화와 전자책을 전송하는 방법

BiblioFuse는 iOS 17 이상을 실행하는 iPhone 및 iPad에서 AirDrop으로 공유된 파일을 받을 수 있습니다. 누군가 CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT 파일을 AirDrop으로 보내면 공유 시트에서 바로 BiblioFuse로 열 수 있습니다. 파일은 BiblioFuse 라이브러리에 저장되고 iCloud를 통해 다른 Apple 기기로 자동 동기화됩니다.

## AirDrop이 iPhone으로 만화를 가져오는 가장 빠른 방법인 이유

AirDrop은 인터넷 연결 없이, 파일 크기 제한도 없이 로컬 Wi-Fi 또는 Bluetooth 속도로 파일을 전송합니다. 발신 Mac이나 iPad 근처에 있다면 400 MB짜리 CBZ 만화 파일이 10초 안에 iPhone에 도착합니다. 로그인할 계정도 없고 경유하는 클라우드 서비스도 없으며, 전송은 종단 간 암호화됩니다.

만화와 만화책 독자에게 AirDrop은 같은 사용자의 기기 간에 파일을 공유할 때 특히 유용합니다. Mac에서 새로운 만화 권을 다운받고 바로 iPhone으로 읽고 싶다면 AirDrop으로 바로 보내면 됩니다. USB 케이블도, 25 MB 제한이 있는 이메일 첨부도, 큰 파일이 iCloud Drive에 동기화되기를 기다릴 필요도 없습니다. [BiblioFuse](/ko/)가 파일을 받아 즉시 라이브러리에 추가합니다.

## iPhone에 만화나 전자책을 AirDrop으로 보내는 방법

Mac, iPad, 다른 iPhone 등 어떤 Apple 기기에서도 BiblioFuse가 설치된 iPhone으로 전송할 수 있습니다.

### Mac에서 보내기

1. Mac에서 **Finder**를 열고 만화 파일(CBZ, EPUB, PDF 등)을 찾습니다.
2. 파일을 우클릭하고 **「공유」→「AirDrop」**을 선택합니다.
3. 근처 AirDrop 수신자 목록에서 iPhone을 선택합니다.
4. iPhone에 파일 수락 여부를 묻는 알림이 나타납니다. **「수락」**을 탭합니다.
5. iOS가 어떤 앱으로 열지 물어봅니다. **BiblioFuse**를 탭합니다.
6. 파일이 BiblioFuse로 가져와지고 라이브러리에 나타납니다.

### iPad 또는 다른 iPhone에서 보내기

1. **파일** 앱을 열고 만화 파일을 찾습니다.
2. 파일을 길게 누른 후 **「공유」**를 탭합니다.
3. **AirDrop** 아이콘을 탭하고 iPhone을 선택합니다.
4. iPhone에서 파일을 수락하고 BiblioFuse로 엽니다.

### 만화 리더 앱에서 직접 공유하기

Mac에서 다른 리더 앱을 사용 중이고 iPhone으로 파일을 보내고 싶다면 해당 앱에서 파일을 데스크탑으로 드래그한 뒤 Finder에서 AirDrop으로 보내는 방법이 일반적입니다. 또는 macOS 공유 메뉴를 지원하는 앱에서 **「공유」→「AirDrop」**을 사용할 수 있습니다.

## 시작하기 전에 AirDrop이 켜져 있는지 확인하세요

AirDrop을 사용하려면 두 기기 모두 Wi-Fi와 Bluetooth가 활성화되어 있어야 합니다. 같은 네트워크에 연결할 필요는 없으며, AirDrop은 기기 간 직접 피어투피어 연결을 만듭니다. 안정적인 전송을 위해 두 기기를 약 9미터(30피트) 이내로 유지하세요.

**iPhone에서:** 제어 센터를 열고 네트워크 타일(왼쪽 상단 그룹)을 길게 누릅니다. **「AirDrop」**을 탭하고 원하는 대로 **「10분 동안 모두에게 공개」** 또는 **「연락처만」**을 선택합니다. 자신의 Mac에서 수신하는 경우 **「연락처만」**이 가장 안전합니다.

**Mac에서:** Finder를 열고 사이드바에서 **「AirDrop」**을 클릭한 후 창 하단에서 **「검색 허용 대상」**을 **「모두」** 또는 **「연락처만」**으로 설정합니다.

## 파일이 BiblioFuse에 도착한 후 일어나는 일

AirDrop으로 받은 파일을 수락하고 BiblioFuse를 대상 앱으로 선택하면, BiblioFuse는 해당 파일을 내부 라이브러리 폴더에 복사합니다. 그 이후:

- 파일이 관련 섹션(만화, 전자책 또는 전체 파일)에 라이브러리로 나타납니다.
- iCloud가 활성화되어 있다면 BiblioFuse가 설치된 다른 Apple 기기로 자동 동기화됩니다.
- BiblioFuse가 파일 형식(CBZ, EPUB, PDF 등)을 인식하고 올바른 리더 엔진으로 엽니다.
- 추가한 읽기 진행 상황, 북마크, 태그가 iCloud를 통해 기기 간에 동기화됩니다.

파일을 폴더에 수동으로 넣거나 이름을 바꿀 필요가 없습니다. BiblioFuse가 자동으로 가져오기를 처리합니다.

## AirDrop vs Wi-Fi Transfer: 어떤 것을 써야 할까?

BiblioFuse에는 AirDrop 없이도 Mac이나 PC(Windows 포함) 어디서나 브라우저에서 파일을 드래그할 수 있는 **Wi-Fi Transfer** 기능도 내장되어 있습니다. 상황별 추천 방법은 다음과 같습니다:

| 상황 | 최적 방법 |
|------|-----------|
| Mac, iPad, 또는 iPhone에서 보낼 때 | AirDrop |
| Windows PC에서 보낼 때 | Wi-Fi Transfer |
| Linux 머신에서 보낼 때 | Wi-Fi Transfer |
| 한 번에 많은 파일을 보낼 때 | Wi-Fi Transfer (여러 파일 드래그) |
| 자신의 Apple 기기 간에 파일 하나를 빠르게 보낼 때 | AirDrop |
| 친구 기기에서 공유할 때 | AirDrop |

Windows 사용자는 iPhone에서 BiblioFuse를 열고 **Wi-Fi Transfer** 버튼을 탭한 다음, 표시된 주소를 PC 브라우저에서 열면 됩니다. PC에 앱을 설치하지 않고도 여러 파일을 한 번에 드래그 앤 드롭으로 전송할 수 있습니다.

## 자주 묻는 질문

### AirDrop으로 CBZ 파일을 iPhone의 BiblioFuse에 직접 보낼 수 있나요?

네. iPhone에 CBZ 파일을 AirDrop으로 보내면 iOS가 어떤 앱으로 열지 묻는 공유 시트를 표시합니다. BiblioFuse를 선택하면 파일이 즉시 라이브러리로 가져와집니다. BiblioFuse는 iOS 17 이상에서 CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT 형식을 지원합니다.

### AirDrop은 용량이 큰 만화 파일도 전송할 수 있나요?

네. AirDrop은 파일 크기 제한이 없고 로컬 Wi-Fi 속도로 전송하므로 대용량 CBZ 또는 EPUB 파일에도 적합합니다. 기기가 가까이 있다면 300–500 MB 만화 파일도 보통 30초 이내에 전송됩니다.

### iPhone이 AirDrop 수신자 목록에 보이지 않는 이유는 무엇인가요?

가장 흔한 원인은 iPhone의 AirDrop이 **「수신 안 함」**으로 설정되어 있거나, 한 기기에서 Bluetooth가 꺼져 있거나, 기기가 너무 멀리 떨어져 있거나, iPhone 화면이 잠겨 있기 때문입니다. iPhone 제어 센터에서 AirDrop을 켜고 두 기기를 가까이 두고 화면이 잠기지 않도록 하세요.

### AirDrop으로 받은 EPUB 파일을 Apple Books 대신 BiblioFuse로 열 수 있나요?

네. AirDrop 파일을 수락한 후 iOS가 앱 선택 화면을 표시하면 **BiblioFuse**를 탭하세요. 공유 시트가 기본적으로 Apple Books를 보여준다면 **「더 보기」** 또는 **「다른 앱으로 열기」**를 탭해 BiblioFuse를 찾으세요. 파일 앱에서 EPUB 파일을 열고 BiblioFuse를 선택하면 EPUB의 기본 앱으로 설정할 수도 있습니다.

### Windows PC에서 iPhone으로 만화를 보내려면 어떻게 하나요?

AirDrop은 Apple 기기 간에만 작동하므로 Windows에서는 사용할 수 없습니다. 대신 BiblioFuse의 내장 **Wi-Fi Transfer** 기능을 사용하세요. iPhone의 BiblioFuse에서 전송 아이콘을 탭하고 표시된 URL을 Windows 브라우저에서 열면 파일을 페이지로 바로 드래그할 수 있습니다.

### Mac에서 AirDrop으로 보낸 파일을 Mac에서 삭제해도 iPhone에 남아 있나요?

네. BiblioFuse가 파일을 받아 가져오면 앱의 라이브러리 폴더에 자체 사본을 저장합니다. Mac에서 원본 파일을 삭제해도 iPhone의 BiblioFuse 내 사본에는 영향을 미치지 않습니다.

## 케이블 없이 만화 읽기를 시작하세요

AirDrop은 Apple 기기 간에 만화와 전자책을 이동하는 가장 빠른 방법입니다. BiblioFuse에 파일이 도착하는 순간 바로 읽기를 시작할 수 있습니다. 동기화 지연도, 계정 로그인도, 중간 단계도 없습니다.

App Store에서 [BiblioFuse](/ko/)를 다운로드하여 무선 독서 라이브러리를 구축하세요. Windows에서 전송하거나 파일을 대량으로 보낼 때는 내장 Wi-Fi Transfer 기능이 언제든 대안이 됩니다.
