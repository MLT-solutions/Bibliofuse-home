# 만화 양면 펼침 페이지 읽기: BiblioFuse가 와이드 이미지를 처리하는 방법

BiblioFuse는 iOS 17 이상을 실행하는 iPhone 및 iPad에서 CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT 파일을 읽습니다. 양면 펼침 페이지에 도달하면 BiblioFuse가 짝을 이루는 페이지를 자동으로 감지하고, 작가가 의도한 그대로 아트워크를 보존하며 나란히 표시할 수 있습니다. 단일 페이지, 나란히 보기, 전체 확대 모드 간에는 언제든지 전환할 수 있습니다.

## 만화 양면 펼침 페이지가 스마트폰에서 읽기 어려운 이유

슈퍼히어로 만화를 읽고 있습니다. 드디어 빌런이 정체를 드러내는 장면 — 숨이 멎을 듯한 두 페이지 가득한 거대한 스플래시 패널입니다. 그런데 iPhone에서는 22페이지가 보이고, 스와이프하면 23페이지가 나타납니다. 두 개의 단절된 반쪽. 극적인 효과가 사라져 버립니다.

이것이 바로 양면 펼침 페이지 문제입니다. 대부분의 만화 앱은 각 페이지를 독립적인 단위로 처리합니다. 22페이지와 23페이지가 사실 두 페이지로 나뉜 하나의 이미지라는 것을 앱이 알지 못하거나 — 신경 쓰지 않습니다. 그 결과, 만화에서 가장 강렬한 순간들이 작은 화면에서 가장 큰 손실을 겪게 됩니다.

[BiblioFuse](/ko/)는 이 문제를 해결하기 위해 만들어졌습니다. 자동 감지, 가로 방향 전환, 수동 컨트롤을 조합하여 양면 펼침 페이지를 처리함으로써, 아트워크가 작가가 그린 그대로 보이도록 합니다.

## 펼침 페이지가 잘못 처리될 때 무엇이 사라지는가

리더가 양면 펼침 페이지를 두 개의 별도 세로 페이지로 표시하면 여러 가지 문제가 동시에 발생합니다. 시각적 구도가 무너집니다 — 작가는 이미지를 하나의 전체로 읽히도록 그렸으며, 한 페이지에서 다른 페이지로 액션이 흐르도록 구성했습니다. 이를 분리하면 뇌가 두 개의 좁은 이미지를 정신적으로 연결해야 하므로 임팩트가 완전히 사라집니다.

텍스트 요소도 피해를 봅니다. 펼침 페이지 중앙(「거터」) 근처의 말풍선과 캡션은 종종 두 페이지에 걸쳐 있습니다. 한 번에 한 페이지만 보면 대사가 중간에 잘리거나 문맥상 이해하기 불가능해집니다.

EPUB이나 PDF 형식의 아트북과 대형 만화의 경우 문제가 더욱 심각합니다. 아트북의 양면 일러스트는 전체로 감상하도록 제작되었습니다. 얇은 두 조각으로 보는 것은 작품을 완전히 왜곡합니다.

## BiblioFuse가 만화 양면 펼침 페이지를 처리하는 방법

[BiblioFuse](/ko/)는 양면 펼침 페이지를 올바르게 읽기 위한 여러 도구를 제공합니다. 표준 작업 흐름은 다음과 같습니다:

1. **만화 파일을 엽니다.** BiblioFuse는 CBZ, CBR, ZIP, RAR, EPUB, PDF, TXT를 읽습니다. 변환이 필요 없습니다 — 파일을 바로 열면 됩니다.
2. **일반적으로 세로 모드로 읽습니다.** 단일 페이지는 화면을 채우며 한 장씩 표시됩니다. 대부분의 만화의 기본 설정입니다.
3. **펼침 페이지에서 가로로 회전합니다.** iPhone을 옆으로 돌리세요. BiblioFuse가 방향 변화를 감지하고 자동으로 두 페이지 보기로 전환합니다. 두 페이지가 나란히 나타나 전체 화면 너비를 채웁니다.
4. **핀치하여 세부 사항을 확대합니다.** 펼침 보기에서 이미지의 어느 부분이든 핀치하여 확대할 수 있습니다 — 거터 근처의 대사, 세밀한 배경 디테일, 구석의 액션 장면까지. 두 번 탭하면 전체 펼침 보기로 돌아갑니다.
5. **세로 방향으로 되돌립니다.** BiblioFuse가 즉시 단일 페이지 모드로 돌아옵니다. 읽기 위치가 유지됩니다.

세로와 가로 간의 전환은 원활합니다. 버튼을 탭할 필요가 없습니다 — 기기를 회전하는 것만으로 충분합니다.

## BiblioFuse는 만화의 오른쪽에서 왼쪽으로의 펼침을 지원합니까?

네. 만화 권을 오른쪽에서 왼쪽 읽기 방향으로 설정하면 펼침 레이아웃도 반전됩니다. 펼침의 오른쪽 페이지가 읽기 순서에서 「첫 번째」 페이지로 처리되는데, 이는 일본 만화에 올바른 방식입니다. 왼쪽 페이지가 그 뒤를 따릅니다. 이는 작가가 의도한 대로 액션이 펼침 페이지를 통해 올바르게 흐른다는 것을 의미합니다 — 패널 순서가 오른쪽에서 왼쪽으로 읽힙니다.

오른쪽에서 왼쪽 모드를 설정하려면, 권을 열고, 화면 중앙을 탭하여 컨트롤을 표시하고, 읽기 방향 아이콘을 탭한 다음 오른쪽에서 왼쪽을 선택하세요. 이 설정은 권별로 저장되므로 각 만화 시리즈에 한 번만 설정하면 됩니다.

## 실용적인 예시: 세 가지 유형의 양면 펼침 페이지

### 슈퍼히어로와 서양 만화 (CBZ/CBR)

슈퍼히어로 만화는 클라이맥스 순간에 스플래시 페이지와 양면 펼침 페이지를 많이 사용합니다. 대부분의 CBZ 파일은 두 페이지를 별도의 이미지 파일로 저장합니다 — 22페이지와 23페이지는 두 개의 별도 JPEG입니다. BiblioFuse는 가로로 회전할 때 이들을 결합하여 하나의 연속된 이미지로 표시합니다.

일부 오래된 CBZ 파일에는 펼침이 이미 단일 와이드 JPEG으로 결합되어 있습니다. BiblioFuse는 이것도 처리합니다 — 단일 와이드 이미지가 가로 화면을 이중화나 타일링 없이 채웁니다.

### 만화 (CBZ/CBR, 오른쪽에서 왼쪽)

만화 펼침에는 추가적인 복잡성이 있습니다: 읽기 방향이 반대입니다. 오른쪽에서 왼쪽으로 읽는 만화 펼침에서, 아카이브에서 물리적으로 오른쪽에 있는 페이지가 실제로는 펼침의 시작입니다. BiblioFuse는 오른쪽에서 왼쪽 모드가 활성화되어 있을 때 이를 존중하여 화면에 올바른 순서로 페이지를 배치합니다.

소년 만화와 청년 만화의 드라마틱하고 액션 넘치는 순간들은 양면 펼침으로 그려지는 경우가 많습니다 — 오다 에이이치로, 미우라 켄타로 같은 작가들의 특징적인 기법입니다. 이들을 의도된 형태로 보려면 올바른 페이지 페어링과 올바른 좌우 순서 모두가 필요합니다.

### 아트북과 대형 삽화 EPUB 또는 PDF

대형 아트북 — 컬렉터 에디션, 럭셔리 하드커버, 삽화 가이드 — 에는 전체 양면 일러스트가 포함되는 경우가 많습니다. 이들은 CBZ 대신 EPUB 또는 PDF로 패키징되는 경우가 많습니다. BiblioFuse는 두 형식 모두를 읽으며, 동일한 가로 회전 로직이 적용됩니다.

PDF 아트북의 경우 BiblioFuse가 전체 해상도로 렌더링합니다. 세밀한 일러스트를 확대해도 픽셀화가 발생하지 않습니다. 이는 회화적 아트워크나 시각적 밀도가 높은 책에서 특히 중요합니다.

## 펼침 감지를 위해 무언가를 구성해야 합니까?

기본 펼침 감지에는 구성이 필요하지 않습니다. 가로로 회전하면 페이지 기반 형식(CBZ, CBR, ZIP, RAR, PDF)에 대해 자동으로 두 페이지 보기가 활성화됩니다. EPUB 파일의 경우 펼침 동작은 EPUB가 어떻게 패키징되었는지에 따라 다르지만, 대부분의 만화 EPUB는 동일하게 작동합니다.

펼침이 잘못 정렬된 것처럼 보인다면 — 한 페이지가 너무 왼쪽이나 오른쪽에 치우쳐 있다면 — 가장 일반적인 원인은 원본 파일에 펼침 페이지가 비정상적인 순서로 저장되어 있기 때문입니다. 이는 BiblioFuse의 문제가 아니라 소스 파일의 문제입니다.

## 자주 묻는 질문

**iPhone에서 양면 펼침 페이지를 보려면 어떻게 합니까?**
BiblioFuse에서 만화를 읽는 동안 iPhone을 가로(수평) 모드로 회전하세요. BiblioFuse가 자동으로 두 페이지 보기로 전환하여 두 페이지를 나란히 표시해 화면을 채웁니다. 세로로 되돌리면 단일 페이지 모드로 돌아갑니다. 앱 설정을 변경할 필요가 없습니다.

**BiblioFuse는 CBZ 파일의 양면 펼침 페이지를 지원합니까?**
네. BiblioFuse는 CBZ, CBR, ZIP, RAR 파일을 읽고 iPhone 또는 iPad(iOS 17+)에서 가로로 회전할 때 짝을 이루는 페이지를 자동으로 나란히 표시합니다. 별도 페이지 펼침과 미리 결합된 와이드 이미지 모두 올바르게 처리됩니다.

**오른쪽에서 왼쪽으로 읽는 만화에서 양면 펼침은 어떻게 작동합니까?**
BiblioFuse에서 오른쪽에서 왼쪽 읽기 모드가 활성화되면 펼침 레이아웃이 반전되어 오른쪽 페이지가 먼저 나타납니다 — 일본 만화의 읽기 순서와 일치합니다. 권을 열고, 중앙을 탭하여 컨트롤을 표시하고, 읽기 방향 아이콘을 탭한 다음 오른쪽에서 왼쪽을 선택하세요.

**iPhone에서 양면 펼침 페이지를 확대할 수 있습니까?**
네. BiblioFuse의 가로 모드에서 펼침을 보는 동안 핀치하여 이미지의 어느 부분이든 확대할 수 있습니다. 두 번 탭하면 전체 펼침 보기로 돌아갑니다. BiblioFuse가 전체 해상도로 렌더링하므로 확대해도 세밀한 선과 작은 텍스트가 선명하게 유지됩니다.

**BiblioFuse는 EPUB 및 PDF 만화의 펼침을 표시합니까?**
네. BiblioFuse는 CBZ, CBR, ZIP, RAR, TXT 외에도 EPUB와 PDF를 읽습니다. 가로로 회전하면 이 형식에 대해서도 두 페이지 보기가 활성화됩니다. PDF 아트북의 경우 전체 해상도 렌더링으로 일러스트를 확대해도 선명하게 유지됩니다.

**펼침이 잘못 보인다면 — 페이지가 분리되거나 잘못된 순서로 있다면 어떻게 합니까?**
페이지가 잘못 정렬되거나 순서가 틀린 경우 문제는 대개 원본 CBZ 또는 PDF 파일이 어떻게 조립되었는지에 있습니다. 일부 스캔 릴리스는 펼침 페이지를 순서에 맞지 않게 저장합니다. BiblioFuse는 파일을 있는 그대로 읽습니다. 아카이브 내의 페이지를 재정렬할 수 없습니다. 펼침이 지속적으로 잘못 표시된다면 소스 파일을 확인하세요.
