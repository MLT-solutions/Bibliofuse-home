# BiblioFuse と Tailscale で自宅のマンガライブラリにリモートアクセスする

BiblioFuse は iOS 17 以降を搭載した iPhone・iPad で CBZ、CBR、EPUB、PDF、ZIP、RAR、TXT ファイルを読むことができます。Mac Home Library 機能を使えば、ファイルをコピーすることなく、ローカル Wi-Fi ネットワーク経由で Mac のコミックライブラリ全体を iPhone にストリーミングできます。これに Tailscale を組み合わせることで、同じストリーミング接続をインターネット越しに延長し、世界中どこからでも自宅ライブラリにアクセスできるようになります。

自宅の Mac に 2 TB のマンガを置いたまま外出しています。iPhone のストレージは 128 GB しかなく、全部入れるには到底足りません。クラウドにファイルを移したり、別の同期サービスに課金したりせず、旅先でもそのライブラリを読みたい。本記事では、[BiblioFuse](/ja/) と Tailscale の無料プランを使ってそれを実現する方法を解説します。

## BiblioFuse と Tailscale によるリモートコミックライブラリアクセスとは？

BiblioFuse の Mac Home Library 機能は、あなたの Mac をコミックコレクションのストリーミングサーバーに変えます。iPhone と Mac が同じ Wi-Fi ネットワーク上にあれば、iPhone の BiblioFuse から Mac のファイルをブラウズして開けます——コピーも同期も不要で、スマートフォンのストレージも消費しません。

制約は「同じネットワーク」という点です。自宅を離れた瞬間に接続が切れます。Mac はホームルーターの内側にあり、iPhone から直接アクセスできなくなるためです。

Tailscale はデバイス間に暗号化されたピアツーピアのメッシュネットワークを構築することでこの問題を解決します。物理的な場所に関係なく、iPhone と Mac は同じ仮想ネットワーク上に表示されます。BiblioFuse にとっては、Mac Home Library 接続がインターネット越しでも自宅 Wi-Fi と全く同じように機能するということを意味します。

## なぜ BiblioFuse と Tailscale はクラウドストレージより優れているのか？

クラウドストレージは手軽な代替手段に見えます——iCloud Drive や Google Drive にすべてを同期してどこからでもアクセスする、という方法です。しかし、大規模なコミックライブラリにとっては、クラウド同期には現実的な限界があります。

- **ストレージコスト。** iCloud 2 TB は月額 1,300 円。Tailscale の無料プランはストレージ料金なしで 100 台のデバイスをカバーします。
- **アップロード時間。** 家庭環境の回線で 2 TB をクラウドにアップロードするには数日かかります。Tailscale + BiblioFuse なら Mac から直接ストリーミングするので、アップロードは一切不要です。
- **プライバシー。** ファイルはネットワーク外に出ません。Tailscale はトラフィックをピアツーピアでルーティングし、中継サーバーは接続のネゴシエーションのみを処理してファイル内容には関与しません。
- **読書体験。** [BiblioFuse](/ja/) の Mac Home Library 経由のストリーミングは高速で、タップするとページがロードされます。クラウド同期ファイルは開く前に完全ダウンロードが必要です。

## ステップ 1——BiblioFuse で Mac Home Library を設定する

BiblioFuse の Mac Home Library は、ローカルネットワーク経由で Mac のコミックフォルダを iPhone アプリに提供する機能です。

1. iPhone と Mac の両方に **BiblioFuse** をインストールする（各プラットフォームの App Store から）
2. Mac で BiblioFuse を開き、**設定 → Mac Home Library** に移動する
3. **有効化** をクリック——Mac がローカルポートでライブラリの提供を開始します
4. iPhone で BiblioFuse を開き、**ライブラリ → ソースを追加 → Mac Home Library** に移動する
5. Mac がローカルネットワーク上に自動的に表示されるはずです。タップして接続します。

動作確認：iPhone から Mac 上のフォルダを参照して CBZ ファイルを開きます。ページが 1〜2 秒以内に読み込まれれば成功です。ローカル Wi-Fi で動作確認できたら、Tailscale を追加する準備が整いました。

## ステップ 2——Tailscale をインストールして設定する

Tailscale は WireGuard 上に構築された VPN です。無料プランは最大 100 台のデバイスをサポートし、ここで必要なすべての機能を網羅しています。

1. **tailscale.com** にアクセスして無料アカウントを作成する
2. Mac に **Tailscale アプリ** をインストールする（App Store または tailscale.com/download から入手可能）
3. iPhone に **Tailscale アプリ** をインストールする（App Store から）
4. 同じアカウントで両方のデバイスから Tailscale にサインインする
5. Mac の Tailscale アプリで Tailscale IP アドレス（通常は `100.x.x.x`）が表示されていることを確認する
6. iPhone で Tailscale アプリを開き、Mac がデバイスリストに緑色のステータスで表示されていることを確認する

これで両デバイスがプライベートな Tailscale ネットワーク上に存在し、物理的な場所に関わらず通信できるようになりました。

## ステップ 3——Tailscale 経由で BiblioFuse を Mac に接続する

Tailscale が起動していれば、BiblioFuse は Tailscale IP アドレスを使って Mac の Home Library にアクセスできます。

1. Mac で Tailscale IP アドレスをメモする（Tailscale のメニューバーアプリに表示——例：`100.64.0.2`）
2. iPhone で BiblioFuse を開き、**ライブラリ → ソースを追加 → Mac Home Library** に移動する
3. Mac が自動的に表示されない場合（別のネットワークにいるときは表示されません）、**アドレスを手動で入力** をタップする
4. Tailscale IP アドレスと BiblioFuse が使用するポート（デフォルト：`8080`）を入力する
5. **接続** をタップする

iPhone の BiblioFuse に Mac のライブラリが表示されます。フォルダを開いてボリュームをタップすれば、ローカル Wi-Fi と全く同じようにTailscale 経由でストリーミングされます。

## ステップ 4——外出中も接続を維持する

リモートアクセスを機能させるには、Mac が次の状態である必要があります。

- **スリープしない**——ネットワークアクセスのためにスリープを無効化：**システム設定 → バッテリー → オプション → ディスプレイがオフのときにコンピュータが自動的にスリープするのを防ぐ**（Mac デスクトップの場合）またはスケジュールを設定する
- **インターネットに接続**——旅行中も自宅のルーターは接続を維持します
- **Tailscale を起動**——Tailscale の環境設定でログイン時に起動するよう設定する

MacBook では「スリープを防ぐ」オプションには電源アダプタが必要です。自宅の Mac が電源に接続されていない MacBook の場合は、具体的なスケジュールを設定してください（例：06:00 に起動、23:00 にスリープ）。

## 実際の活用シーン

**旅行中に長編シリーズを読む。** 完全版のワンピース（107 巻、圧縮 CBZ で約 15 GB）が自宅の Mac に保存されています。Wi-Fi のない機内ではアクセスできませんが、インターネット接続があれば（ホテルの Wi-Fi、モバイルデータ）、[BiblioFuse](/ja/) がページをオンデマンドでストリーミングします。15 GB をダウンロードするのではなく、読み進めながら個別のページをダウンロードします。

**低速な接続での読書。** Tailscale はモバイルデータで効率的な WireGuard を使用します。BiblioFuse のツールタブで WebP 圧縮した CBZ ページは通常 200〜500 KB です。10〜20 Mbps の 4G 速度なら各ページは 0.1 秒未満で読み込まれます。2 Mbps の混雑したホテル Wi-Fi でも 1 秒以内に読み込めます。

**家族ライブラリの管理。** 配偶者の iPhone やタブレットを同じ Tailscale アカウントに追加する（無料プランで最大 100 台）と、家族全員が別々のクラウドサブスクリプションなしに同じ Mac ライブラリからストリーミングできます。

## トラブルシューティング

**Tailscale 有効化後、BiblioFuse に Mac が表示されない。** Mac の Tailscale IP アドレスを使って手動 IP 入力方法を使用してください。自動検出は mDNS ブロードキャストに依存しており、ネットワーク境界を越えられません。

**モバイルデータでページの読み込みが遅い。** BiblioFuse のツールタブまたは [BiblioFuse Web ツール](/ja/webapp/) を使って CBZ アーカイブを圧縮してページサイズを小さくしてください。品質 82 の WebP エンコード CBZ ファイルは PNG ベースのアーカイブより通常 75〜88% 小さく、リモートストリーミングが大幅に速くなります。

**Mac がスリープした。** Mac のシステム設定で「ディスプレイがオフのときにコンピュータが自動的にスリープするのを防ぐ」を有効にするか、Terminal で `caffeinate -i &` を実行して起動状態を維持してください。

**Tailscale 接続が切断される。** Tailscale はネットワーク状況の変化に応じて自動的に再接続します。接続に失敗した場合は、iPhone で BiblioFuse を閉じて再度開き、接続を再試行してください。

## よくある質問

**Tailscale なしで自宅のコミックライブラリにリモートアクセスできますか？**
はい、ただし代替手段はより複雑かプライバシーが低下します。ルーターのポートフォワーディングは Mac をインターネットに直接さらすことになりセキュリティリスクがあります。クラウド同期（iCloud、Google Drive）はまずライブラリ全体をアップロードする必要があります。Tailscale はポートを開かずに暗号化されたゼロ設定のリモートアクセスを提供します。

**このユースケースで Tailscale に費用はかかりますか？**
Tailscale の無料プランは 1 つの個人アカウントで最大 100 台のデバイスをストレージ料金なし・帯域制限なしでサポートします。自分のデバイス間で共有する個人用コミックライブラリには、無料プランで無期限に対応できます。

**モバイルデータ接続でのコミックストリーミング速度は？**
圧縮 CBZ ファイル（WebP ページ）では、個別ページは通常 200〜500 KB です。10 Mbps の 4G 接続なら各ページは 0.5 秒未満で読み込まれます。2 Mbps のホテル Wi-Fi でも読書体験はスムーズです。PNG ベースのアーカイブ（1 ページあたり 2〜5 MB）は明らかに遅く、リモートストリーミング前の圧縮で最も効果を発揮します。

**BiblioFuse は iPad でも Tailscale と動作しますか？**
はい。BiblioFuse は iPhone と iPad の両方をサポートしており、Tailscale も iPad で利用できます。設定は同一です——両方のアプリをインストールし、Tailscale にサインインして、Tailscale IP アドレスを使って Mac Home Library に接続します。

**Tailscale 使用中、コミックコレクションのプライバシーは守られますか？**
はい。Tailscale はデバイス間のすべてのトラフィックに WireGuard 暗号化を使用します。ファイルは暗号化されたトンネル内で iPhone と Mac 間を直接転送されます。Tailscale のサーバーは接続の調整を行いますが、ファイルの内容を見ることはありません。

**外出中に Mac が再起動した場合はどうなりますか？**
Tailscale がログイン時に起動するよう設定されており、BiblioFuse for Mac で Mac Home Library が起動時に実行されるよう設定されていれば、再起動後に自動的に接続が復元されます。継続性を確保するために、両方のアプリをそれぞれの設定でログイン時に起動するよう設定してください。
