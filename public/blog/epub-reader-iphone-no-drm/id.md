# Pembaca EPUB Bebas DRM Terbaik untuk iPhone — Tanpa Apple Books

Anda membeli kumpulan novel indie dari penerbit kecil. Anda mengunduh selusin ebook dari Humble Bundle. Anda mengambil file EPUB dari Project Gutenberg. Tidak satu pun dari mereka memiliki DRM — semuanya sepenuhnya milik Anda. Tetapi ketika mencoba menggunakan Apple Books, pengalamannya terasa kurang memuaskan: perpustakaan berantakan, metadata salah, dan aplikasinya tidak benar-benar ingin menjadi manajer EPUB untuk file yang Anda miliki sendiri. Aplikasi itu ingin menjadi toko.

[BiblioFuse](/id/) dibuat tepat untuk kebutuhan ini. Ini adalah pembaca iPhone dan iPad yang berjalan secara native untuk file EPUB bebas DRM — serta CBZ, CBR, PDF, dan lainnya — yang menjadikan perpustakaan lokal Anda sebagai warga kelas satu. Tidak ada toko, tidak perlu akun, tidak ada batasan file yang bisa dibuka.

Panduan ini menjelaskan mengapa Apple Books kurang ideal untuk membaca tanpa DRM, apa yang ditawarkan BiblioFuse sebagai gantinya, dan bagaimana cara memindahkan koleksi EPUB Anda ke iPhone tanpa kabel.

## Mengapa Apple Books Tidak Ideal untuk EPUB Bebas DRM

Apple Books bekerja dengan baik jika Anda membeli ebook langsung dari Apple. Untuk file yang diimpor dari tempat lain, ada banyak gesekan.

**Metadata tidak dapat diprediksi.** Apple Books membaca metadata EPUB saat impor, tetapi tidak selalu menampilkannya dengan benar. Informasi seri, nama penulis, dan gambar sampul sering kali tidak bertahan utuh setelah impor. Buku berakhir dengan judul yang salah atau sampul kosong.

**Struktur perpustakaan kaku.** Anda tidak dapat membuat hierarki folder sendiri atau menandai buku berdasarkan genre, status seri, atau prioritas baca. Semuanya berakhir dalam satu daftar besar.

**Aplikasi dirancang di sekitar pembelian.** Tampilan default menampilkan toko Apple Books. Bagian Daftar Keinginan dan Unggulan memakan ruang. Bagi seseorang dengan 200 file EPUB bebas DRM yang diimpor dari luar ekosistem Apple, ini adalah alat yang salah.

**Sideloading tidak praktis.** Memasukkan file ke Apple Books memerlukan aplikasi File atau sinkronisasi kabel, yang berfungsi tetapi tidak lancar. Tidak ada cara untuk mentransfer seluruh folder EPUB sekaligus melalui Wi-Fi.

Semua ini bukan berarti Apple Books adalah aplikasi yang buruk. Aplikasi ini sangat bagus untuk tujuan yang dirancangnya. Tetapi jika koleksi EPUB Anda berada di luar toko Apple, Anda memerlukan pembaca yang berbeda.

## Apa yang BiblioFuse Lakukan Secara Berbeda

[BiblioFuse](/id/) dibangun khusus untuk orang-orang yang mengelola perpustakaan ebook dan komik mereka sendiri.

**Membaca semua format utama secara native.** File EPUB, PDF, CBZ, CBR, ZIP, RAR, dan TXT semuanya terbuka tanpa konversi. Jika Anda memiliki perpustakaan campuran novel dan novel grafis, BiblioFuse menangani keduanya dalam satu aplikasi.

**Tag dan penilaian sepenuhnya di bawah kendali Anda.** Anda dapat menandai file apa pun dengan label khusus — "fiksi ilmiah", "untuk dibaca", "seri: Stormlight", apa pun yang masuk akal untuk koleksi Anda. Penilaian memungkinkan Anda mengurutkan dan memfilter berdasarkan kualitas. Ini adalah metadata yang Anda buat sendiri, bukan sesuatu yang diimpor dari toko.

**Transfer Wi-Fi bekerja tanpa kabel.** Untuk menambahkan file ke BiblioFuse, Anda membuka server web bawaan aplikasi dari browser di Mac atau PC dan menyeret file langsung ke dalamnya. Tidak perlu iTunes, tidak perlu kabel, tidak perlu akun cloud. Untuk folder berisi 50 EPUB, ini memakan waktu beberapa menit dan semuanya masuk ke perpustakaan dengan benar.

**Sinkronisasi iCloud membuat semuanya tersinkronisasi.** Jika Anda menggunakan BiblioFuse di iPhone, iPad, dan Mac, posisi baca, tag, dan penilaian Anda tetap tersinkronisasi secara otomatis melalui iCloud. Anda berhenti di bab 14 di iPhone; Anda melanjutkan di bab 14 di iPad.

**Sepenuhnya offline.** Tidak perlu akun untuk membaca. Tidak perlu koneksi jaringan setelah file diimpor. Aplikasi memproses semuanya di perangkat.

## Cara Mengimpor EPUB Bebas DRM ke BiblioFuse

Ada beberapa cara untuk memasukkan file EPUB ke BiblioFuse, tergantung dari mana file Anda berasal.

### Melalui Transfer Wi-Fi (Direkomendasikan untuk Banyak File)

Ini adalah cara tercepat untuk memindahkan koleksi dari Mac atau PC ke iPhone Anda.

1. Buka BiblioFuse di iPhone Anda.
2. Ketuk tombol **Wi-Fi Transfer** (ikon panah ke telepon di bilah alat atau tampilan perpustakaan).
3. BiblioFuse menampilkan URL lokal seperti `http://192.168.1.42:8080`.
4. Di Mac atau PC Anda, buka browser apa pun dan navigasikan ke URL tersebut.
5. Antarmuka web muncul. Seret file EPUB Anda — atau seluruh folder — ke area unggahan.
6. File ditransfer melalui jaringan lokal Anda dan muncul di perpustakaan BiblioFuse dalam hitungan detik.

Tidak perlu akun, tidak perlu kabel, tidak perlu iCloud. Hanya jaringan lokal dan browser.

### Melalui Aplikasi File

Jika EPUB Anda sudah ada di iPhone — di iCloud Drive, di folder aplikasi lain, atau di folder lokal — Anda dapat membukanya dari aplikasi File.

1. Buka aplikasi File dan navigasikan ke file EPUB.
2. Tekan lama file tersebut dan ketuk **Bagikan**.
3. Di lembar berbagi, ketuk **Buka di BiblioFuse**.
4. File diimpor ke perpustakaan Anda.

Untuk koleksi besar, Transfer Wi-Fi lebih cepat. Tetapi untuk file individual, cara lewat aplikasi File juga bekerja dengan baik.

### Melalui Mac Home Library (Untuk Pengguna Mac)

Jika perpustakaan EPUB Anda ada di Mac, fitur Mac Home Library BiblioFuse memungkinkan Anda melakukan streaming ke iPhone tanpa menyalin file sama sekali.

1. Instal BiblioFuse di Mac Anda.
2. Aktifkan Mac Home Library di aplikasi Mac.
3. Di iPhone, ketuk **Mac Library** di bilah sisi BiblioFuse.
4. Seluruh perpustakaan Mac Anda muncul dan dapat dibaca melalui jaringan Wi-Fi lokal Anda.

Ini bekerja dengan baik untuk koleksi besar di mana Anda tidak ingin menduplikasi penyimpanan. Mac Anda menyimpan file; iPhone Anda membacanya sesuai permintaan.

## Mengatur Perpustakaan EPUB di BiblioFuse

Setelah file Anda diimpor, BiblioFuse memberi Anda beberapa alat untuk menjaga semuanya tetap terkelola.

**Tag** bekerja seperti folder tetapi lebih fleksibel. Sebuah buku bisa memiliki beberapa tag. Anda bisa menandai buku dengan "fantasi" dan "seri" secara bersamaan. Ketuk ikon tag pada buku mana pun untuk menetapkan atau membuat tag. Filter perpustakaan Anda berdasarkan kombinasi tag apa pun dari bilah sisi.

**Penilaian** (1–5 bintang) memungkinkan Anda mengurutkan berdasarkan kualitas atau prioritas. Jika Anda telah membaca 200 buku dan ingin menemukan yang terbaik, filter 5 bintang langsung memunculkannya.

**Opsi pengurutan** mencakup judul, penulis, tanggal ditambahkan, ukuran file, dan penilaian. Dikombinasikan dengan filter tag, Anda bisa dengan cepat menemukan persis apa yang Anda cari dalam perpustakaan besar.

**Pengelompokan seri** (jika metadata ada dalam file EPUB) dapat menampilkan buku-buku yang dikelompokkan berdasarkan seri dengan urutan baca yang benar.
