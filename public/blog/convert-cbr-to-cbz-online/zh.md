# 如何在线将 CBR 转换为 CBZ：免费，无需软件

你有一文件夹的 CBR 文件——以 RAR 压缩包格式存储的漫画——而你的阅读器只支持 CBZ。或者你正在归档收藏，想要一种统一的开放格式。解决方案听起来很简单：把扩展名从 `.cbr` 改成 `.cbz`。但这样做行不通。

CBR 文件是 RAR 压缩包，CBZ 文件是 ZIP 压缩包，它们使用不同的压缩格式——不能仅靠重命名扩展名完成转换。你需要从 RAR 中提取图像，再重新打包成 ZIP。[BiblioFuse 在线转换工具](https://bibliofuse.com/zh/webapp/)直接在浏览器中完成这一切，文件不会上传到任何服务器。

## CBR 与 CBZ 的真正区别

CBR 和 CBZ 都是漫画压缩格式，都包含一系列图像文件（通常是 JPG 或 PNG），阅读器按顺序显示这些图像来模拟阅读漫画的体验。

区别在于底层压缩方式：

| 格式 | 压缩类型 | 开放标准？ |
|------|---------|----------|
| CBZ | ZIP | 是 |
| CBR | RAR | 否（专有格式） |

RAR 是 Eugene Roshal 开发的专有格式。打开 RAR 文件需要官方 WinRAR 软件或实现 RAR 规范的库，而该库必须明确集成到每款阅读器中。许多现代阅读器（包括基于浏览器的工具）只支持基于 ZIP 的 CBZ，因为 ZIP 规范是开放且广泛实现的。

CBZ 成为数字漫画的事实标准，部分原因是 ZIP 免版税且跨平台——每个操作系统都可以在不借助第三方软件的情况下打开 ZIP 文件。如果你正在长期归档收藏，CBZ 是更安全的选择。

## 转换原理

将 CBR 转换为 CBZ 包含三个步骤：

1. 从 CBR（RAR 压缩包）中提取图像
2. 将这些图像放入新的 ZIP 压缩包
3. 将 `.zip` 扩展名重命名为 `.cbz`

BiblioFuse 在线工具利用 WebAssembly 在浏览器中完成全部三个步骤。文件永远不会上传——提取、重新压缩和下载全部在你的设备本地进行。

如果你的漫画收藏包含版权内容、个人扫描或其他不希望上传到第三方服务器的内容，这一点尤为重要。

## 常见问题

**能一次转换多个 CBR 文件吗？**

可以。在文件选择器中选中所有 CBR 文件，工具会逐一处理。文件单独下载——每个 `.cbr` 对应一个 `.cbz`。如果之后想合并成一个卷，可以使用 CBZ 合并工具。

**转换过程中图像质量会改变吗？**

不会。压缩包内的图像不会重新编码。转换提取原始 JPG 或 PNG 文件，将它们放入新的 ZIP 中，不接触图像数据。转换后的 CBZ 与原始 CBR 在像素级别上完全一致。

**如果 CBR 文件有密码保护怎么办？**

受密码保护的 CBR 文件没有密码无法打开，网页工具无法绕过密码保护。如果你的 CBR 已加密，需要先解锁——SmartDecrypt 可以在 iPhone 和 Mac 上批量解锁受密码保护的 CBZ、PDF 和 ZIP 文件。

**我的 CBR 文件已转换，但 CBZ 打不开——是什么情况？**

部分标记为 `.cbr` 的文件实际上是 ZIP 压缩包（从 `.cbz` 重命名而来，或由默认使用 ZIP 的软件创建）。如果转换后的文件无法在阅读器中打开，可以尝试将原始 `.cbr` 重命名为 `.zip`，检查它是否已经是 ZIP 压缩包。也许只需直接将其重命名为 `.cbz` 而无需转换。

**有文件大小限制吗？**

没有服务器限制，因为文件从不上传。实际限制是设备的可用内存——非常大的 CBR 文件（1 GB 以上）可能在内存有限的设备上导致浏览器变慢。对于大多数 200 MB 以下的漫画文件，转换在几秒内完成。
