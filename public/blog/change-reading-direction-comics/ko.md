# BiblioFuse에서 만화 읽기 방향 바꾸는 방법: LTR·RTL 설정 가이드

BiblioFuse는 iOS 17 이상을 탑재한 iPhone과 iPad에서 왼쪽에서 오른쪽(LTR)과 오른쪽에서 왼쪽(RTL) 읽기 방향을 언제든지 전환할 수 있는 만화 리더 앱입니다. CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT 파일을 지원하며, 앱을 재시작하거나 파일을 다시 불러오지 않아도 읽는 도중 즉시 방향을 바꿀 수 있습니다. 일본 만화(manga)를 즐겨 읽는다면 RTL 모드를 켜서 일본식 페이지 넘김 방식을 그대로 경험할 수 있고, 서양 만화는 기본값인 LTR로 편안하게 읽을 수 있습니다.

## 만화와 망가에서 읽기 방향이 중요한 이유

BiblioFuse에서 일본 만화를 LTR로 읽으면 이야기 순서가 완전히 뒤집힙니다. 오른쪽에서 왼쪽으로 읽어야 할 컷이 반대 순서로 표시되고, 대사가 뒤섞이며, 액션 장면의 흐름이 끊겨버립니다. BiblioFuse에서 RTL로 전환하는 데 걸리는 시간은 단 5초, 그것만으로 모든 문제가 해결됩니다.

같은 문제가 웹툰, 한국 만화(manhwa), 그리고 독특한 컷 배치를 사용하는 일부 유럽 만화(bandes dessinées)에서도 발생합니다. 파일을 수정하거나 다시 불러올 필요 없이 읽기 세션 안에서 바로 해결할 수 있는 단 하나의 설정——이것이 여러 만화 문화권의 작품을 즐기는 독자들에게 [BiblioFuse](/{lang}/)가 최적의 도구인 이유입니다.

## BiblioFuse에서 읽기 방향 변경하는 방법

BiblioFuse에서 읽기 방향을 바꾸는 것은 읽기 화면 안에서 몇 번 탭하는 것으로 충분합니다.

1. iPhone 또는 iPad에서 BiblioFuse로 만화 또는 망가 파일을 엽니다.
2. 화면 중앙을 탭하여 리더 컨트롤을 불러옵니다.
3. 상단 또는 하단 툴바의 **설정 아이콘**（기어 또는 슬라이더 아이콘）을 탭합니다.
4. **읽기 방향** 옵션을 찾습니다——LTR（오른쪽 화살표）과 RTL（왼쪽 화살표）아이콘이 표시됩니다.
5. 일본 만화와 망가는 **RTL**, 서양 만화와 대부분의 소설은 **LTR**을 선택합니다.
6. 화면이 즉시 업데이트됩니다. 스와이프 또는 탭으로 새 페이지 순서를 확인하세요.

BiblioFuse는 파일별로 읽기 방향 설정을 기억합니다. CBZ 파일을 RTL로 변경하면, 다시 변경하기 전까지 그 파일은 항상 RTL로 열립니다. 라이브러리의 다른 파일에는 영향을 주지 않습니다.

## 전체 기본 읽기 방향 설정하기

주로 망가를 읽는다면 RTL을 전체 기본값으로 설정해 두면 매번 수동으로 바꾸는 번거로움을 없앨 수 있습니다.

1. **BiblioFuse**를 열고 **설정** 탭（또는 라이브러리 화면의 설정 기어 아이콘）으로 이동합니다.
2. **읽기 환경설정** 또는 **리더 설정**을 탭합니다.
3. **기본 읽기 방향**을 **RTL**로 설정합니다.
4. 이후 여는 모든 새 파일이 RTL로 기본 설정됩니다. 이미 개별 설정을 저장한 파일은 해당 설정이 유지됩니다.

라이브러리가 일본 망가 위주라면 특히 유용합니다——한 번만 설정하면 그 다음부터는 신경 쓸 필요가 없습니다.

## LTR vs RTL: 만화 종류별 어떤 방향을 써야 할까

만화 문화권마다 읽기 방향 관습이 다릅니다. 빠른 참고 가이드를 정리했습니다.

**RTL을 사용하는 경우:**
- 일본 망가（예: 원피스, 나루토, 베르세르크）
- 전통 형식의 한국 만화（웹툰 제외）
- 일본 방식을 따르는 일부 중국 만화（manhua）

**LTR을 사용하는 경우:**
- 서양 만화（Marvel, DC, Image, Dark Horse）
- 유럽 만화（bandes dessinées）
- 대부분의 EPUB 텍스트 전자책
- 세로 스크롤 모드의 웹툰과 장편 가로 만화

**세로 스크롤 모드**（웹툰용）는 페이지 방향과 별개의 설정입니다. 세로 스크롤 모드에서는 좌우 방향 설정이 시각적으로 아무 효과가 없는데, 수평 스와이프 대신 위아래 스크롤로 작동하기 때문입니다. 세로 스크롤 옵션은 읽기 방향과 같은 리더 설정 패널에 있습니다.

## 읽던 중에 방향을 바꿔도 읽던 위치를 잃지 않으려면

BiblioFuse는 방향을 변경해도 현재 읽기 위치를 그대로 보존합니다. 망가 47페이지를 읽는 도중 LTR에서 RTL로 전환해도 여전히 47페이지에 머물러 있습니다. 페이지 번호가 달라지거나 초기화되지 않습니다.

일부 망가 단행본에는 컬러 페이지, 작가 후기, 보너스 챕터 등 RTL 단행본 내에서도 LTR 형식으로 구성된 추가 콘텐츠가 포함되어 있을 수 있어 이 기능이 특히 유용합니다. 해당 섹션을 읽을 때만 방향을 바꾸고 다시 되돌리면 됩니다. 북마크를 찾아 이동할 필요가 없습니다.

## 자주 묻는 질문

### BiblioFuse에서 망가 읽기 방향은 어떻게 바꾸나요?

읽는 중에 화면 중앙을 탭해 리더 컨트롤을 불러온 뒤, 설정 아이콘을 탭하고 「읽기 방향」에서 **RTL**을 선택하세요. 변경 사항은 현재 페이지에 즉시 적용됩니다. BiblioFuse가 해당 파일에 설정을 저장하므로, 다음에 열 때도 자동으로 RTL로 표시됩니다.

### 읽기 방향을 바꾸면 페이지 번호도 바뀌나요?

아니요. LTR이든 RTL이든 총 페이지 수는 동일하게 유지됩니다. 방향을 전환하면 화면의 어느 쪽을 탭하거나 스와이프해야 다음 페이지로 넘어가는지만 달라질 뿐, 파일 내의 페이지 순서가 바뀌거나 재번호되지는 않습니다.

### 모든 망가에 RTL을 기본값으로 설정할 수 있나요?

네. **설정 → 읽기 환경설정 → 기본 읽기 방향**에서 RTL을 선택하세요. BiblioFuse가 이후 새로 여는 모든 파일에 RTL을 적용합니다. 이미 개별 방향 설정을 저장한 파일은 각자의 설정이 유지됩니다.

### BiblioFuse가 EPUB 전자책에서도 RTL을 지원하나요?

BiblioFuse는 EPUB 파일의 방향 전환을 지원하지만, 일본 망가 EPUB의 대부분은 파일 메타데이터에 읽기 방향 선언이 이미 포함되어 있습니다. BiblioFuse는 이 메타데이터를 존중하므로, 망가 EPUB는 대개 자동으로 올바른 방향으로 열립니다. 물론 리더 설정 패널에서 수동으로 재설정하는 것도 가능합니다.

### RTL 페이지 모드와 세로 스크롤 모드는 어떻게 다른가요?

RTL 페이지 모드는 수평 페이지 넘김 방향을 바꿉니다——왼쪽 스와이프가 뒤로 가기 대신 앞으로 가기가 됩니다. 세로 스크롤 모드는 수평 스와이프를 완전히 없애고 위에서 아래로의 연속 스크롤로 대체합니다. 이 모드는 웹툰과 세로 연재 만화용으로 설계된 것입니다. 두 설정 모두 같은 리더 설정 패널에 있으며 독립적으로 전환할 수 있습니다.

### BiblioFuse가 파일별로 읽기 방향을 기억하나요?

네. BiblioFuse는 각 파일에 대해 설정한 읽기 방향을 개별적으로 저장합니다. 망가는 RTL로, DC 만화는 LTR로, 웹툰은 세로 스크롤 모드로——모두 마지막으로 읽었을 때의 설정을 기반으로 자동으로 열립니다.

## 모든 만화를 작가가 의도한 방식으로 읽자

읽기 방향은 작은 설정이지만 독서 즐거움에 미치는 영향은 큽니다. BiblioFuse에서 망가를 RTL로 설정해 두면 컷이 작가가 의도한 흐름대로 이어지고, 말풍선 순서가 자연스럽게 읽히며, 두 페이지 펼침면도 올바르게 맞춰집니다.

지금 App Store에서 [BiblioFuse](/{lang}/)를 다운로드하고 망가 라이브러리를 열어보세요. 새 망가 파일을 iPhone으로 전송해야 한다면, 내장된 Wi-Fi 전송 기능을 사용하면 Mac이나 PC의 브라우저에서 파일을 직접 드래그할 수 있습니다——케이블 불필요, iTunes 불필요, iCloud 계정도 필요 없습니다.
