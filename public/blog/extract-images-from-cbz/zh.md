# 如何在线从 CBZ 文件提取图片（免费，无需安装软件）

CBZ 文件是一个将扩展名改为 `.cbz` 的 ZIP 压缩包，其中包含独立的图片文件——通常是 JPEG、PNG 或 WebP，每页漫画对应一张图片。您可以直接在浏览器中使用 [BiblioFuse 网页工具](/zh/webapp/) 提取这些图片，无需安装任何软件，也无需上传文件。一切操作均在客户端通过 WebAssembly 完成，您的文件永远不会离开您的设备。

## 您有一个 CBZ 文件——接下来怎么办？

您下载了一部数字漫画，收到了朋友发来的 CBZ 文件，或从阅读应用中导出了一个。现在您需要其中的单页图片——也许是为了打印喜欢的面板、编辑页面布局、分享特定场景，或者归档某系列的特定页面。访问原始图片本应简单，但大多数工具却让它比必要的更加复杂。

CBZ 文件并不是什么特殊格式。去掉 `.cbz` 扩展名，它就是一个普通的 ZIP 压缩包。图片就在里面，等待被访问。难点在于如何干净地完成这项操作，既不安装桌面应用程序，也不把文件发送到陌生的服务器。

## 为什么大多数方法都不够好用？

最显而易见的解决方案是将 `.cbz` 改名为 `.zip`，然后用操作系统内置工具解压。在桌面端可行——但在手机上操作很麻烦，需要使用文件管理器，还会留下一堆需要重命名和整理的文件。如果您有多个 CBZ 文件需要处理，这种方法也无法扩展。

基于云端的图片提取工具存在，但需要您将漫画文件上传到远程服务器。对于个人漫画、扫描原稿或任何您希望保持私密的内容，这种权衡是不可接受的。文件一旦上传，您就无法控制它被保留多长时间，也无法控制谁可以访问它。

CDisplay Ex、YACReader 或 calibre 等桌面应用程序可以打开 CBZ 文件，但需要安装，通常界面复杂，且在 iOS 或 Android 上不能以相同方式使用。如果您用的是平板电脑、Chromebook 或借用的电脑，就会陷入困境。

## 提取的 CBZ 图片能用来做什么？

### 提取喜爱的面板作为壁纸

数字漫画通常包含精彩的跨页画面或标志性面板，非常适合用作手机或桌面壁纸。提取图片，在任意图片编辑器中裁剪，就能获得来自您最喜欢系列的高分辨率壁纸，无需重新上传或截图导致的任何质量损失。

### 提取页面进行编辑或重新排序

如果您是创作者、译者或从事同人编辑工作并使用原始漫画页面，拥有单独的 JPEG 或 PNG 文件至关重要。您可能想重新排版一页、进行色彩校正，或在重新打包成新 CBZ 之前以不同顺序重新组合页面。先提取才能获得原始素材进行创作。

### 从长系列中归档特定页面

您有一套 200 册的漫画系列，只想保存每章的标题页用于同人归档。从每个 CBZ 提取图片可以让您精确挑选所需内容，而无需复制整个文件。提取的图片是标准的 JPEG 或 PNG 格式——与任何照片库、云盘或归档工具通用兼容。
