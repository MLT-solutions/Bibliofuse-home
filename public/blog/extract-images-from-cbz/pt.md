# Como Extrair Imagens de Arquivos CBZ Online (Grátis, Sem Instalar Nada)

Um arquivo CBZ é um arquivo ZIP renomeado com a extensão `.cbz`, contendo arquivos de imagem individuais — normalmente JPEG, PNG ou WebP — um por página de quadrinho. Você pode extrair essas imagens diretamente no navegador usando a [Ferramenta Web BiblioFuse](/pt/webapp/), sem instalar nada e sem enviar nenhum arquivo. Tudo roda no lado do cliente via WebAssembly, então seus arquivos nunca saem do seu dispositivo.

## Você Tem um Arquivo CBZ — E Agora?

Você baixou um quadrinho digital, recebeu um arquivo CBZ de um amigo, ou exportou um da sua app de leitura. Agora você precisa das imagens de páginas individuais — talvez para imprimir um painel favorito, editar um layout de página, compartilhar uma cena específica, ou arquivar páginas de uma série. Acessar as imagens brutas deveria ser simples, mas a maioria das ferramentas complica mais do que o necessário.

Arquivos CBZ não são um formato exótico. Remova a extensão `.cbz` e você tem um arquivo ZIP completamente comum. As imagens estão ali dentro, esperando para ser acessadas. O desafio é fazer isso de forma limpa, sem instalar um app de desktop ou enviar seus arquivos para um servidor desconhecido.

## Por Que a Maioria dos Métodos Fica Aquém?

A solução óbvia é renomear `.cbz` para `.zip` e extrair com a ferramenta integrada do sistema operacional. Funciona no desktop — mas é complicado no celular, exige acesso ao gerenciador de arquivos e deixa uma pasta bagunçada de arquivos para renomear e ordenar. Também não escala se você tiver vários arquivos CBZ para processar.

Ferramentas de extração baseadas em nuvem existem, mas exigem que você faça upload dos seus arquivos de quadrinhos para um servidor remoto. Para quadrinhos pessoais, arte digitalizada ou qualquer coisa que você queira manter privada, isso não é um compromisso aceitável. Uma vez enviado o arquivo, você não tem controle sobre por quanto tempo ele é retido ou quem pode acessá-lo.

Apps de desktop como CDisplay Ex, YACReader ou calibre podem abrir arquivos CBZ, mas exigem instalação, muitas vezes têm interfaces complexas e não estão disponíveis da mesma forma no iOS ou Android. Se você estiver em um tablet, Chromebook ou computador emprestado, você fica sem saída.

## Como Extrair Imagens de Arquivos CBZ Usando a Ferramenta Web BiblioFuse

A [Ferramenta Web BiblioFuse](/pt/webapp/) roda completamente no seu navegador via WebAssembly. Não há nada para instalar, nenhuma conta a criar, e nenhum arquivo jamais sai do seu dispositivo. Veja como usar:

**Passo 1 — Abrir a Ferramenta Web**

Acesse [bibliofuse.com/pt/webapp/](/pt/webapp/) em qualquer navegador moderno — Chrome, Safari, Firefox ou Edge. A ferramenta carrega completamente na sua aba do navegador. Sem login necessário.

**Passo 2 — Carregar Seu Arquivo CBZ**

Clique na área de upload ou arraste e solte seu arquivo CBZ na interface. A ferramenta lê o arquivo localmente usando a API de arquivos do navegador. Seu arquivo nunca é transmitido para nenhum servidor.

**Passo 3 — Processar o Arquivo**

A Ferramenta Web BiblioFuse abre o arquivo CBZ e lê os arquivos de imagem contidos nele. Você pode ver as páginas, inspecionar o conteúdo e escolher como prosseguir — baixar imagens individuais ou processar o arquivo ainda mais (redimensionar, recomprimir, mesclar com outros arquivos).

**Passo 4 — Baixar Suas Imagens**

Depois do processamento, baixe as imagens extraídas para o seu dispositivo. Os arquivos são salvos na sua pasta de downloads local, exatamente como apareciam dentro do arquivo CBZ.

O nível gratuito permite até 3 itens por lote, o que cobre a maioria dos casos de uso de edição única. O processamento é instantâneo em hardware moderno porque tudo roda localmente via WebAssembly — sem idas e vindas de rede, sem esperar na fila de um servidor.

## O Que Você Pode Fazer com as Imagens CBZ Extraídas?

### Extrair um Painel Favorito para Papel de Parede

Quadrinhos digitais frequentemente contêm páginas duplas impressionantes ou painéis icônicos que fazem ótimos papéis de parede para celular ou desktop. Extraia a imagem, recorte-a em qualquer editor de imagem e você tem um papel de parede em alta resolução da sua série favorita sem perda de qualidade por fazer novo upload ou captura de tela.

### Puxar Páginas para Editar ou Reordenar

Se você é criador, tradutor ou fan-editor trabalhando com páginas brutas de quadrinhos, ter os arquivos JPEG ou PNG individuais é essencial. Você pode querer re-letrar uma página, aplicar uma correção de cor, ou remontar as páginas em uma ordem diferente antes de reempacotar em um novo CBZ. Extrair primeiro fornece o material-fonte bruto para trabalhar.

### Arquivar Páginas Específicas de uma Série Longa

Você tem 200 edições em uma série de mangá e quer salvar apenas as páginas de título de capítulo para um arquivo de fãs. Extrair imagens de cada CBZ permite selecionar exatamente o que você precisa sem duplicar arquivos inteiros. As imagens são JPEG ou PNG padrão — universalmente compatíveis com qualquer biblioteca de fotos, drive na nuvem ou ferramenta de arquivo.

## Perguntas Frequentes Sobre Extração de Imagens de CBZ

**Como faço para extrair imagens de um arquivo CBZ?**

O método mais simples baseado em navegador é abrir a [Ferramenta Web BiblioFuse](/pt/webapp/), carregar seu arquivo CBZ e baixar as imagens extraídas. Alternativamente, você pode renomear o arquivo de `.cbz` para `.zip` e extraí-lo com a ferramenta de arquivo integrada do seu sistema operacional — mas isso só funciona convenientemente no desktop.

**Posso abrir um arquivo CBZ sem instalar nenhum software?**

Sim. A Ferramenta Web BiblioFuse roda completamente no seu navegador usando WebAssembly. Você não precisa instalar nada — nem mesmo uma extensão do navegador. Abra a URL, carregue seu arquivo e a ferramenta cuida do resto. Funciona no Windows, macOS, Linux, iOS e Android.

**A Ferramenta Web BiblioFuse é gratuita?**

O nível gratuito permite processar até 3 itens por lote sem precisar de conta. Você pode usá-la quantas vezes quiser sem custo para casos de uso de arquivo único ou pequeno lote.

**Extrair um CBZ faz upload dos meus arquivos em algum lugar?**

Não. A Ferramenta Web BiblioFuse é 100% do lado do cliente. Seus arquivos são lidos e processados localmente no seu navegador via WebAssembly. Nada é transmitido para nenhum servidor. Seus quadrinhos, suas páginas, sua privacidade.

**Quais formatos de imagem existem dentro de um arquivo CBZ?**

Arquivos CBZ normalmente contêm imagens JPEG para quadrinhos coloridos (tamanhos de arquivo menores, boa reprodução de cores) e PNG para conteúdo em preto e branco ou arte de linha onde qualidade sem perdas importa. Alguns arquivos CBZ modernos também incluem imagens WebP. A Ferramenta Web BiblioFuse lida com os três formatos.
