# How to Extract Images from CBZ Files Online (Free, No Software)

A CBZ file is a ZIP archive renamed with the `.cbz` extension, containing individual image files — typically JPEG, PNG, or WebP — one per comic page. You can extract those images directly in your browser using the [BiblioFuse Web Tool](/en/webapp/), with no software to install and no file upload required. Everything runs client-side via WebAssembly, so your files never leave your device.

## You Have a CBZ File — Now What?

You downloaded a digital comic, received a CBZ file from a friend, or exported one from your reader app. Now you need the individual page images inside — maybe to print a favourite panel, edit a page layout, share a specific scene, or archive pages from a series. Opening the raw images should be simple, but most tools make it harder than it needs to be.

CBZ files are not some exotic format. Strip away the `.cbz` extension and you have a perfectly ordinary ZIP archive. The images are sitting right there inside, waiting to be accessed. The challenge is doing it cleanly, without installing a desktop app or sending your files to a stranger's server.

## Why Most Methods Fall Short

The obvious solution is to rename `.cbz` to `.zip` and extract with your OS's built-in tool. That works on desktop — but it is clunky on mobile, requires file manager access, and leaves you with a messy folder of files you need to rename and sort. It also does not scale if you have multiple CBZ files to process.

Cloud-based image extraction tools exist, but they require you to upload your comic files to a remote server. For personal comics, scanned art, or anything you care about keeping private, that is not an acceptable trade-off. Once a file is uploaded, you have no control over how long it is retained or who can access it.

Desktop apps like CDisplay Ex, YACReader, or calibre can open CBZ files, but they require installation, often have complex interfaces, and are not available on iOS or Android in the same form. If you are on a tablet, Chromebook, or a borrowed computer, you are stuck.
