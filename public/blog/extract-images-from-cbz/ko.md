# CBZ 파일에서 이미지 온라인으로 추출하는 방법 (무료, 소프트웨어 불필요)

CBZ 파일은 `.cbz` 확장자로 이름이 변경된 ZIP 압축 파일로, 만화의 각 페이지에 해당하는 개별 이미지 파일(보통 JPEG, PNG 또는 WebP)이 담겨 있습니다. [BiblioFuse 웹 도구](/ko/webapp/)를 사용하면 소프트웨어 설치나 파일 업로드 없이 브라우저에서 직접 이미지를 추출할 수 있습니다. 모든 작업은 WebAssembly를 통해 클라이언트 측에서 처리되므로 파일이 기기를 떠나는 일이 없습니다.

## CBZ 파일이 있다면 — 다음은 무엇을 해야 할까요?

디지털 만화를 다운로드했거나, 친구에게 CBZ 파일을 받았거나, 리더 앱에서 내보낸 파일이 있을 것입니다. 이제 개별 페이지 이미지가 필요할 수 있습니다——좋아하는 컷을 인쇄하거나, 페이지 레이아웃을 편집하거나, 특정 장면을 공유하거나, 시리즈의 특정 페이지를 보관하기 위해서입니다. 원본 이미지에 접근하는 것은 간단해야 하지만 대부분의 도구는 필요 이상으로 복잡합니다.

CBZ 파일은 특수한 형식이 아닙니다. `.cbz` 확장자를 제거하면 완전히 일반적인 ZIP 압축 파일입니다. 이미지들은 그 안에 그대로 들어 있어 접근하기를 기다리고 있습니다. 과제는 데스크톱 앱을 설치하거나 낯선 서버에 파일을 보내지 않고 깔끔하게 작업하는 것입니다.

## 기존 방법들은 왜 불편할까요?

가장 명확한 해결책은 `.cbz`를 `.zip`으로 이름을 변경하고 운영체제의 내장 도구로 압축을 해제하는 것입니다. 데스크톱에서는 작동하지만 모바일에서는 번거롭고, 파일 관리자 접근이 필요하며, 이름을 변경하고 정렬해야 하는 파일이 뒤섞인 폴더가 남습니다. 여러 CBZ 파일을 처리해야 하는 경우에는 확장성도 없습니다.

클라우드 기반 이미지 추출 도구가 있지만, 만화 파일을 원격 서버에 업로드해야 합니다. 개인 만화, 스캔한 원고, 또는 프라이버시를 지키고 싶은 파일에 대해서는 이것이 받아들이기 어려운 타협입니다. 파일을 업로드하면 얼마나 오래 보관되는지, 누가 접근할 수 있는지를 제어할 수 없습니다.

CDisplay Ex, YACReader, calibre 같은 데스크톱 앱은 CBZ 파일을 열 수 있지만 설치가 필요하고, 복잡한 인터페이스를 가지는 경우가 많으며, iOS나 Android에서는 동일한 방식으로 이용할 수 없습니다. 태블릿, Chromebook, 또는 빌린 컴퓨터를 사용 중이라면 방법이 없습니다.

## BiblioFuse 웹 도구로 CBZ 파일에서 이미지 추출하는 방법

[BiblioFuse 웹 도구](/ko/webapp/)는 WebAssembly를 통해 브라우저에서 완전히 실행됩니다. 설치할 것도, 계정을 만들 필요도, 파일이 기기를 떠나는 일도 없습니다. 사용 방법은 다음과 같습니다:

**1단계 — 웹 도구 열기**

Chrome, Safari, Firefox, Edge 등 최신 브라우저에서 [bibliofuse.com/ko/webapp/](/ko/webapp/)으로 이동합니다. 도구는 브라우저 탭에서 완전히 로드됩니다. 로그인 불필요합니다.

**2단계 — CBZ 파일 불러오기**

업로드 영역을 클릭하거나 CBZ 파일을 인터페이스로 드래그 앤 드롭합니다. 도구는 브라우저의 파일 API를 통해 파일을 로컬에서 읽습니다. 파일은 어떤 서버에도 전송되지 않습니다.

**3단계 — 파일 처리**

BiblioFuse 웹 도구는 CBZ 압축 파일을 열고 내부의 이미지 파일을 읽습니다. 페이지를 확인하고, 내용을 검사하고, 진행 방식을 선택할 수 있습니다——개별 이미지 다운로드 또는 파일 추가 처리(크기 조정, 재압축, 다른 파일과 병합).

**4단계 — 이미지 다운로드**

처리가 완료되면 추출된 이미지를 기기에 다운로드합니다. 파일은 CBZ 압축 파일 내에 있던 그대로 로컬 다운로드 폴더에 저장됩니다.

무료 버전은 배치당 최대 3개 항목을 처리할 수 있으며, 대부분의 단일 호 사용 사례를 커버합니다. 모든 것이 WebAssembly를 통해 로컬로 실행되므로 최신 하드웨어에서 처리는 즉각적입니다——네트워크 왕복도, 서버 대기 줄도 없습니다.

## 추출한 CBZ 이미지로 무엇을 할 수 있을까요?

### 좋아하는 컷을 배경화면으로 추출

디지털 만화에는 멋진 전면 스프레드나 상징적인 컷이 자주 포함되어 있어 휴대폰이나 데스크톱 배경화면으로 훌륭합니다. 이미지를 추출하고 이미지 편집기에서 잘라내면, 재업로드나 스크린샷으로 인한 화질 저하 없이 좋아하는 시리즈의 고해상도 배경화면을 얻을 수 있습니다.

### 편집하거나 재배열할 페이지 꺼내기

원본 만화 페이지를 다루는 창작자, 번역가, 팬 편집자라면 개별 JPEG 또는 PNG 파일이 필수입니다. 페이지를 다시 레터링하거나, 색보정을 적용하거나, 새 CBZ로 재패키징하기 전에 다른 순서로 페이지를 재구성하고 싶을 수 있습니다. 먼저 추출하면 작업할 원시 소스 자료를 얻을 수 있습니다.

### 긴 시리즈에서 특정 페이지 보관

만화 시리즈에 200권이 있고 팬 아카이브를 위해 챕터 타이틀 페이지만 저장하고 싶다면, 각 CBZ에서 이미지를 추출하면 전체 파일을 복제하지 않고 원하는 것만 골라낼 수 있습니다. 이미지는 표준 JPEG 또는 PNG——어떤 사진 라이브러리, 클라우드 드라이브, 또는 보관 도구와도 범용적으로 호환됩니다.

## CBZ 이미지 추출에 관한 자주 묻는 질문

**CBZ 파일에서 이미지를 추출하려면 어떻게 하나요?**

가장 간단한 브라우저 기반 방법은 [BiblioFuse 웹 도구](/ko/webapp/)를 열고, CBZ 파일을 불러온 후, 추출된 이미지를 다운로드하는 것입니다. 또는 파일을 `.cbz`에서 `.zip`으로 이름을 변경하고 운영체제의 내장 압축 도구로 압축을 해제할 수도 있습니다——하지만 이 방법은 데스크톱에서만 편리합니다.

**소프트웨어 설치 없이 CBZ 파일을 열 수 있나요?**

네. BiblioFuse 웹 도구는 WebAssembly를 통해 브라우저에서 완전히 실행됩니다. 아무것도 설치할 필요가 없습니다——브라우저 확장 프로그램조차도요. URL을 열고 파일을 불러오면 도구가 나머지를 처리합니다. Windows, macOS, Linux, iOS, Android에서 작동합니다.

**BiblioFuse 웹 도구는 무료인가요?**

무료 버전은 계정 없이 배치당 최대 3개 항목을 처리할 수 있습니다. 단일 파일 또는 소량 배치 사용 사례에 대해 원하는 만큼 무료로 사용할 수 있습니다.

**CBZ를 추출하면 파일이 어딘가에 업로드되나요?**

아니요. BiblioFuse 웹 도구는 100% 클라이언트 측입니다. 파일은 WebAssembly를 통해 브라우저에서 로컬로 읽히고 처리됩니다. 어떤 서버에도 아무것도 전송되지 않습니다. 당신의 만화, 당신의 페이지, 당신의 프라이버시.

**CBZ 파일 내부에는 어떤 이미지 형식이 들어 있나요?**

CBZ 파일에는 일반적으로 컬러 만화용 JPEG 이미지(파일 크기가 작고 색재현이 좋음)와 무손실 품질이 중요한 흑백 또는 선화 콘텐츠용 PNG가 포함됩니다. 일부 최신 CBZ 파일에는 WebP 이미지도 포함됩니다. BiblioFuse 웹 도구는 세 가지 형식을 모두 처리합니다.
