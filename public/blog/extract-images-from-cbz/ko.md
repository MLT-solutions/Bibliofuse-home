# CBZ 파일에서 이미지를 온라인으로 추출하는 방법 (무료, 소프트웨어 불필요)

BiblioFuse 웹 도구를 사용하면 브라우저에서 직접 CBZ 파일의 개별 이미지를 추출할 수 있습니다. 소프트웨어 설치 불필요, 서버에 파일 업로드 없음, 계정 불필요. 처리는 탭 내의 WebAssembly에서 완전히 실행되므로 만화는 비공개로 유지됩니다.

## CBZ에서 이미지가 필요한 경우

CBZ 파일은 연속으로 번호가 매겨진 이미지(일반적으로 스캔된 만화나 망가의 JPEG 또는 PNG 페이지)를 포함하는 ZIP 아카이브입니다. 원본 이미지가 필요한 상황은 다양합니다:

- **특정 페이지 인쇄:** 프린터는 만화 아카이브가 아닌 이미지 파일이 필요
- **다른 형식으로 변환:** 일부 변환기는 이미지를 입력으로 받음
- **팬 아트나 참고 자료 만들기:** 원본 전체 해상도 페이지 파일이 필요
- **고품질 스캔 보관:** 이미지를 추출하여 스캔 품질과 메타데이터 확인
- **재패키징:** 큰 CBZ를 스토리 아크별로 작은 파일들로 분할

## BiblioFuse 웹 도구 작동 방식

[BiblioFuse Web](https://bibliofuse.com/ko/webapp/)은 WebAssembly(WASM)를 사용하여 브라우저 탭 내에서 직접 처리 로직을 실행합니다. 파일은 어떤 서버에도 전송되지 않습니다.

이는 PDF→CBZ 변환, CBZ 병합, EPUB 압축 등 모든 BiblioFuse 웹 도구에서 채택한 방식과 동일합니다.

## 단계별 가이드: CBZ에서 이미지 온라인으로 추출

### 1단계: BiblioFuse 웹 도구 열기

최신 브라우저에서 [BiblioFuse 웹 앱](https://bibliofuse.com/ko/webapp/)으로 이동합니다. 로그인, 확장 프로그램, 가입 불필요.

### 2단계: CBZ 추출기 선택

**CBZ에서 이미지 추출** 도구를 찾아 클릭하여 추출 인터페이스를 엽니다.

### 3단계: CBZ 파일 불러오기

**파일 선택**을 클릭하거나 CBZ 파일을 드롭 존으로 드래그합니다. 파일은 네트워크 요청 없이 로컬에서 직접 읽습니다.

### 4단계: 추출 옵션 선택

추출기에는 여러 옵션이 있습니다:
- **모든 이미지:** CBZ 아카이브의 모든 이미지 파일 추출
- **페이지 범위:** 지정한 페이지만 추출
- **형식 필터:** JPEG만 또는 PNG만 추출

### 5단계: 추출 및 다운로드

**추출**을 클릭합니다. WASM 엔진이 CBZ를 압축 해제하고 이미지를 ZIP 아카이브로 패키징합니다. 브라우저가 자동으로 다운로드를 시작합니다.

### 6단계: 압축 해제하여 이미지 사용

다운로드한 ZIP 파일을 아카이브 관리자로 엽니다. 읽기 순서대로 번호가 매겨진 추출된 이미지를 찾을 수 있습니다.

## CBZ vs. CBR

CBR 파일은 RAR 형식을 사용합니다. CBR이 있다면 먼저 BiblioFuse 웹 앱에서 CBZ로 변환한 후 이미지를 추출하세요.

## 개인 정보 및 보안

추출기가 브라우저에서 완전히 실행되므로 어떤 서버도 파일을 받거나 저장하지 않습니다:
- 개인 컬렉션은 기기를 떠나지 않음
- 계정 불필요
- 서버 비용으로 인한 파일 크기 제한 없음

## 자주 묻는 질문

**BiblioFuse CBZ 이미지 추출기가 정말 무료인가요?**
네. 모든 BiblioFuse 웹 도구는 제한 없이 무료입니다.

**어떤 이미지 형식을 지원하나요?**
JPEG, PNG, WebP, GIF, BMP, TIFF.

**비밀번호로 보호된 CBZ에서 이미지를 추출할 수 있나요?**
먼저 SmartDecrypt로 비밀번호 보호를 제거한 후 이미지를 추출하세요.

**처리 가능한 CBZ 파일 크기는?**
강제적인 제한이 없으며 브라우저의 사용 가능한 메모리가 유일한 제약입니다.

**추출된 이미지가 읽기 순서대로 나열되나요?**
네. 추출기는 읽기 순서를 결정하는 원본 파일 이름을 유지합니다.
