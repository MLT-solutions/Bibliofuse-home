# Cara Membaca Webtoon di iPhone: Gulir Vertikal Tanpa Batas di BiblioFuse

Kamu sudah mengunduh webtoon — manhwa Korea, webkomik pita panjang, atau seri vertikal dengan terjemahan penggemar — dan tersimpan di iPhone sebagai file CBZ atau EPUB. Setiap aplikasi baca komik yang kamu coba menampilkannya dengan orientasi yang salah, memotong panel yang seharusnya mengalir ke bawah. [BiblioFuse](https://bibliofuse.com/id/) menangani kasus ini secara langsung, dengan mode gulir vertikal yang dibangun khusus untuk format ini.

## Apa yang Membuat Webtoon Berbeda

Komik tradisional mengatur panel dalam kisi. Webtoon berbeda: dirancang untuk membaca di perangkat mobile, dengan panel tinggi dan sempit yang bertumpuk secara vertikal menjadi satu strip berkelanjutan. Format ini berasal dari platform seperti LINE Webtoon dan Naver, dan berkembang ke webkomik mandiri serta terjemahan scan manhwa.

Format vertikal ini menciptakan masalah bagi aplikasi yang dirancang untuk komik tradisional. Mode balik halaman mengganggu aliran. Tampilan dua halaman membuat panel sangat kecil. Bahkan aplikasi yang mendukung gulir vertikal sering melewatkan detail seperti sensitivitas geser atau ketahanan level zoom.

## Mode Gulir Vertikal BiblioFuse

[BiblioFuse untuk iPhone dan iPad](https://bibliofuse.com/id/) menyertakan mode gulir vertikal khusus. Alih-alih memperlakukan setiap gambar sebagai halaman, mode ini mengurutkannya secara berkelanjutan sehingga kamu bisa menggulir ke bawah sepanjang chapter tanpa ketukan atau sapuan. Bekerja untuk webtoon CBZ maupun EPUB.

Untuk beralih mode membaca di dalam komik:
1. Buka CBZ atau EPUB di BiblioFuse.
2. Ketuk tengah layar untuk menampilkan bilah alat pembaca.
3. Ketuk ikon **Tata Letak** (dua persegi panjang bertumpuk).
4. Pilih **Gulir Vertikal**.

Pembaca segera menyusun ulang konten. Gambar memenuhi lebar layar dan menggulir ke bawah bergerak melalui halaman tanpa jeda paginasi.

## Jarak Antara Gambar

Beberapa webtoon disusun sebagai satu gambar berkelanjutan per chapter. Yang lain dibagi menjadi gambar-gambar individual. BiblioFuse memberimu kontrol atas jarak di antara mereka:

- **Tanpa jarak**: gambar berjejer mulus, membuat chapter terasa seperti satu gambar berkelanjutan.
- **Jarak minimal**: pemisah tipis muncul antar halaman, berguna ketika halaman memiliki latar belakang putih.

Untuk sebagian besar webtoon, mode tanpa jarak memberikan pengalaman membaca yang paling autentik.

## Mengimpor File Webtoon

**File CBZ** adalah wadah paling umum untuk terjemahan. CBZ hanyalah arsip ZIP yang berisi gambar PNG atau JPG. BiblioFuse membaca CBZ secara native di iPhone dan iPad.

**File EPUB** umum untuk webtoon berlisensi resmi. BiblioFuse membaca EPUB tata letak tetap.

Metode transfer ke iPhone:
- **Transfer Wi-Fi**: buka BiblioFuse, pergi ke tab **Transfer**, dan server web lokal dimulai. Kunjungi alamat yang ditampilkan dari browser mana pun di jaringan yang sama untuk mengunggah file.
- **Aplikasi File**: bagikan CBZ atau EPUB dari aplikasi File iOS ke BiblioFuse menggunakan menu berbagi.

## Zoom dan Geser

Ketuk dua kali untuk memperbesar area; ketuk dua kali lagi untuk kembali ke lebar penuh. Zoom gulir vertikal bertahan di seluruh halaman. Menggeser horizontal saat diperbesar juga didukung.

## Membaca Seri Panjang

Seri manhwa tipikal memiliki 100 hingga 300+ chapter. BiblioFuse mengatur chapter sebagai file individual di dalam folder. Tombol **Berikutnya** muncul di akhir chapter untuk membuka chapter berikutnya tanpa kembali ke perpustakaan.

Progres membaca tersinkronisasi melalui iCloud di beberapa perangkat Apple.
