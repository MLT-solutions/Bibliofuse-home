# BiblioFuse на iPad: советы по чтению комиксов в полноэкранном режиме

iPhone удобен, но iPad — это то место, где чтение комиксов становится по-настоящему хорошим. iPad Pro 12,9 дюйма предлагает примерно в четыре раза большую площадь экрана, чем iPhone — достаточно, чтобы показать разворот почти в натуральную величину и снизить усталость глаз при долгих сессиях. [BiblioFuse](https://bibliofuse.com/ru/) создан, чтобы использовать все эти возможности. Вот настройки и привычки, которые максимально раскрывают потенциал чтения на iPad.

## Правильный режим отображения

Когда вы открываете комикс в BiblioFuse на iPad, первое решение — как страницы будут заполнять экран. Перейдите в **Настройки → Дисплей**:

- **По ширине** — расширяет каждую страницу горизонтально до края экрана. Хорошо для вертикально ориентированных страниц.
- **По высоте** — показывает страницу целиком с чёрными полосами по бокам. Ничего не обрезается.
- **Заполнить экран** — растягивает изображение для заполнения всего дисплея. Лучший вариант для высококачественных сканов.

Для стандартных американских комиксов на iPad Pro режим **По высоте** обеспечивает опыт, близкий к чтению бумажного издания.

## Разворот в горизонтальной ориентации

Поверните iPad горизонтально на развороте — [BiblioFuse](https://bibliofuse.com/ru/) автоматически определяет поворот и переключается на двухстраничный вид, точно как в печатном издании.

В **Настройки → Двухстраничная раскладка** вы управляете тем, показывается ли первая страница отдельно (обычно для обложек) или в паре, а также зазором между страницами.

Для манги справа налево порядок страниц автоматически меняется после установки **Направление чтения → RTL**.

## Бегунок с миниатюрами

На iPad нижняя панель бегунка достаточно широка, чтобы быть по-настоящему полезной. Нажмите в центр экрана, чтобы её открыть, и перетащите бегунок в любую точку. Миниатюры появляются при перемещении, позволяя навигацию по визуальному контенту, а не только по номеру страницы.

Удерживайте бегунок, чтобы установить **закладку** на текущей позиции.

## Синхронизация между устройствами через iCloud

Если вы также читаете на iPhone, BiblioFuse автоматически синхронизирует позицию чтения через iCloud. Откройте комикс на iPad, прочитайте 60 страниц, закройте приложение — iPhone продолжит с 60-й страницы.

Для основной библиотеки используйте [стриминг с Mac](https://bibliofuse.com/ru/blog/mac-home-library-streaming-iphone/). Запустите сервер BiblioFuse на Mac и подключитесь с iPad по Wi-Fi. Ваша библиотека на Mac — потенциально терабайты комиксов — транслируется прямо на iPad без копирования файлов.

## Яркость и ночное чтение

**Ночной режим BiblioFuse** — нажмите иконку солнца на панели читалки. Тёплый фильтр применяется к страницам, уменьшая синий свет без потери насыщенности цветов панелей.

## Организация библиотеки на iPad

BiblioFuse использует более крупную сетку на iPad, что делает визуальный просмотр удобным. Организуйте по папкам серий и называйте файлы последовательно — например, `Серия т1.cbz`, `Серия т2.cbz` — чтобы сетка читалась с первого взгляда.

Для массового импорта используйте передачу по Wi-Fi: значительно быстрее повторных переносов через AirDrop.

## Stage Manager и разделённый экран

На iPad Pro или iPad Air со Stage Manager BiblioFuse работает в изменяемом окне. Читайте комикс на двух третях экрана, пока Safari открыт на остальном.

Клавиши со стрелками на внешней клавиатуре работают для навигации по страницам — удобно с Magic Keyboard.

## Заключение

iPad превращает чтение комиксов в действительно отличный опыт — но только если приложение использует возможности экрана. Двухстраничный режим, большой бегунок с миниатюрами, синхронизация iCloud и стриминг с Mac делают [BiblioFuse на iPad](https://bibliofuse.com/ru/) гораздо большим, чем просто большой iPhone.
