# BiblioFuse di iPad: Tips Membaca Komik Layar Penuh

iPhone memang praktis, tapi iPad adalah tempat pengalaman membaca komik benar-benar memuaskan. iPad Pro 12,9 inci menawarkan sekitar empat kali luas layar iPhone — cukup untuk menampilkan halaman ganda hampir dalam ukuran sebenarnya dan mengurangi ketegangan mata dalam sesi membaca panjang. [BiblioFuse](https://bibliofuse.com/id/) dirancang untuk memaksimalkan semua ini. Berikut pengaturan dan kebiasaan yang paling mengoptimalkan pengalaman membaca di iPad.

## Pilih Mode Tampilan yang Tepat

Saat membuka komik di BiblioFuse di iPad, keputusan pertama adalah bagaimana halaman memenuhi layar. Buka **Pengaturan → Tampilan**:

- **Sesuaikan lebar** — memperluas setiap halaman secara horizontal ke tepi layar. Cocok untuk halaman portrait.
- **Sesuaikan tinggi** — menampilkan seluruh halaman dengan letterboxing di sisi. Tidak ada yang terpotong.
- **Isi layar** — meregangkan untuk mengisi seluruh tampilan. Terbaik untuk scan berkualitas tinggi.

Untuk komik Amerika standar di iPad Pro, **Sesuaikan tinggi** memberikan pengalaman paling mendekati membaca cetak.

## Halaman Ganda dalam Mode Landscape

Putar iPad ke orientasi landscape saat sampai di halaman ganda — [BiblioFuse](https://bibliofuse.com/id/) mendeteksi rotasi otomatis dan beralih ke tampilan dua halaman, persis seperti edisi cetak.

Di **Pengaturan → Tata Letak Dua Halaman**, Anda mengontrol apakah halaman pertama ditampilkan sendiri (umum untuk sampul) atau berpasangan, serta jarak antar halaman.

Untuk manga kanan ke kiri, urutan halaman terbalik otomatis setelah mengatur **Arah Baca → RTL**.

## Bilah Scrubber dengan Thumbnail

Di iPad, bilah scrubber bawah cukup lebar untuk benar-benar berguna. Ketuk bagian tengah layar untuk menampilkannya, lalu seret untuk melompat ke mana saja. Thumbnail muncul saat Anda menyeret sehingga bisa navigasi berdasarkan konten visual, bukan nomor halaman.

Tahan lama scrubber untuk menetapkan **bookmark** di posisi saat ini.

## Sinkronisasi Antar Perangkat dengan iCloud

Jika Anda juga membaca di iPhone, BiblioFuse menyinkronkan posisi baca secara otomatis melalui iCloud. Buka komik di iPad, baca 60 halaman, tutup aplikasi — iPhone Anda melanjutkan dari halaman 60.

Untuk perpustakaan utama, gunakan [Mac Home Library Streaming](https://bibliofuse.com/id/blog/mac-home-library-streaming-iphone/). Jalankan server streaming di Mac dan hubungkan dari BiblioFuse di iPad via Wi-Fi. Perpustakaan Mac Anda — bisa terabyte komik — streaming langsung ke iPad tanpa menyalin file.

## Kecerahan dan Membaca Malam

**Mode Malam BiblioFuse** — ketuk ikon matahari di toolbar pembaca. Filter hangat diterapkan ke halaman, mengurangi cahaya biru tanpa membuat panel terlihat pucat.

## Mengorganisir Perpustakaan iPad

BiblioFuse menggunakan grid lebih besar di iPad, membuat browsing visual praktis. Organisasikan berdasarkan folder seri dan beri nama file secara konsisten — mis. `Seri v01.cbz` — agar grid mudah dibaca sekilas.

Untuk impor massal, gunakan transfer Wi-Fi: jauh lebih cepat daripada transfer AirDrop berulang.

## Stage Manager dan Split View

Di iPad Pro atau iPad Air dengan Stage Manager, BiblioFuse berjalan di jendela yang dapat diubah ukurannya. Baca komik di dua pertiga layar sementara Safari terbuka di sisanya.

Tombol panah pada keyboard eksternal berfungsi untuk navigasi halaman, nyaman dengan Magic Keyboard.

## Kesimpulan

iPad mengubah membaca komik menjadi pengalaman yang benar-benar luar biasa — tapi hanya jika aplikasinya memanfaatkan layar. Mode halaman ganda, scrubber thumbnail besar, sinkronisasi iCloud, dan streaming Mac menjadikan [BiblioFuse versi iPad](https://bibliofuse.com/id/) jauh lebih dari sekadar iPhone yang lebih besar.
