# CBZ vs CBR vs EPUB — 漫画与电子书格式全解析

BiblioFuse 支持在运行 iOS 17 或更高版本的 iPhone、iPad 和 Mac 上原生读取 CBZ、CBR、EPUB、PDF、ZIP、RAR 和 TXT 格式，无需任何转换步骤。CBZ 和 CBR 是最适合漫画和漫画书的图像归档格式，而 EPUB 是专为小说和非虚构类书籍设计的可回流文本格式。

你正在看一个下载页面，上面列着四种文件类型：CBZ、CBR、EPUB、PDF，都声称是"同一本书"。该下载哪个？哪个能在手机、平板或 Mac 上使用？这些三字母后缀到底是什么意思？

本文一次讲清楚。读完之后，你会明白每种格式是什么、从哪里来、里面装着什么，以及在什么情况下该选哪个。

## 简短回答

| 格式 | 内容 | 适合用途 |
|------|------|---------|
| CBZ | ZIP 压缩包内的图片 | 漫画、日漫、图像小说 |
| CBR | RAR 压缩包内的图片 | 与 CBZ 相同，多见于老版本 |
| EPUB | ZIP 内的 HTML + 图片 | 文字电子书、小说 |
| PDF | 固定版式页面 | 文档、技术书籍 |

## CBZ — Comic Book ZIP

CBZ 全称 Comic Book ZIP，格式规则只有一条：把 ZIP 文件改个后缀名为 `.cbz`。没有特殊编码，没有神奇的文件头，也没有规范文档。用任何解压工具打开 `.cbz`，你会发现一个按阅读顺序排列的图片文件夹：`001.jpg`、`002.jpg`，直到最后一页。

CBZ 在 2000 年代初兴起，因为它解决了一个简单问题：漫画文件需要在网络上传播，而 ZIP 无处不在。`.cbz` 后缀告诉阅读软件："这是漫画，按顺序显示图片"，而不是"这是软件包，解压到文件夹"。

**内容：** 顺序图片，通常是 JPG 或 PNG，有时是 WebP。部分文件在根目录包含 `ComicInfo.xml`——一个存储书名、系列、期号、作者等信息的元数据文件。[BiblioFuse](/zh/) 会读取这些元数据，在书库中显示正确信息。

**兼容性：** 全平台通用。所有平台上的漫画阅读应用都支持 CBZ，是数字漫画收藏的默认格式。

## CBR — Comic Book RAR

CBR 全称 Comic Book RAR。概念与 CBZ 相同，只是容器是 RAR 压缩包而非 ZIP。

大多数漫画扫描发布的年代，RAR 在文件共享社区中非常流行，所以早期版本通常是 CBR 格式。内部结构完全相同，唯一区别是压缩算法和归档格式。[BiblioFuse](/zh/) 两种格式都支持。

**需要转换吗？** 如果你有大量 CBR 收藏想要"未来证明"，转换为 CBZ 是合理的——内部图片完全一样。转换无损，因为只是重新打包相同文件。

**关于 RAR5 的说明：** 一些非常旧的 CBR 文件使用较新的 RAR5 压缩标准，旧版软件无法打开。如果 CBR 文件无法打开，通常就是这个原因。重新打包为 CBZ 始终能解决问题。

## EPUB — Electronic Publication

EPUB（电子出版物）与 CBZ 和 CBR 完全不同。它是 W3C 的开放标准，专为文字内容丰富的电子书设计——小说、非虚构、参考书——内容需要根据不同屏幕尺寸和字体偏好自动调整。

EPUB 实质上是一个 ZIP 压缩包，包含：
- **HTML 或 XHTML 文件** — 章节正文，带语义标签
- **CSS 样式表** — 字体、行距、布局
- **图片** — 封面及插图
- **OPF 清单文件** — 列出所有文件，定义阅读顺序
- **NCX 或 NAV 文件** — 目录

与 CBZ 的关键区别：

- **自动适配：** 由阅读器决定字号、行距、页边距和背景色。同一本 EPUB 在不同设备上的显示因用户设置而异。
- **无障碍：** 屏幕阅读器可以朗读文本，无需 OCR。
- **可搜索：** 可以在 EPUB 中搜索任意词语。

**DRM：** 书店销售的商业 EPUB 通常使用 Adobe DRM 或 Apple FairPlay，将文件锁定到特定账号和应用。无 DRM 的 EPUB——来自古腾堡计划、多数小型出版商或作者直销——可在任何 EPUB 阅读器中使用，包括 [BiblioFuse](/zh/)。

## PDF — Portable Document Format

PDF 是固定版式格式。与 EPUB 不同，作者精确控制每页每个元素的位置。PDF 为打印而生。

对于漫画，PDF 可以用但不理想：文件通常较大，手机上的缩放体验不佳。对于技术手册、参考书和扫描文档，PDF 往往是最佳选择，因为固定版式能完整保留视觉结构。

## 该下载哪种格式？

**漫画或日漫：** CBZ 是最佳选择。如果只有 CBR 可用，下载 CBR——实际上完全相同。

**小说和文字电子书：** 始终选 EPUB。它自适应、易于访问，在任何屏幕尺寸上都无需费力缩放。

**扫描书籍或技术文档：** 如果版式很重要（图表、表格、多栏文本）选 PDF；如果来源提供 EPUB 且你更在意可读性，选 EPUB。

## BiblioFuse 的格式兼容性

[BiblioFuse](/zh/) 在 iPhone、iPad 和 Mac 上原生支持所有这些格式：CBZ、CBR、ZIP、RAR、EPUB、PDF 和 TXT。工具选项卡可在设备上直接处理格式转换：将 CBZ 内的图片压缩为 WebP 以缩减文件大小。

## 关于文件大小

| 格式 | 相对大小 | 原因 |
|------|---------|-----|
| CBZ（PNG） | 最大 | 无损图片 |
| CBZ（JPEG） | 中等 | 有损压缩 |
| CBZ（WebP） | 最小 | 现代压缩算法 |
| EPUB（文字） | 极小 | 文字压缩效率极高 |

一册日漫在 PNG CBZ 格式下通常约 300–400 MB。同一册压缩为 WebP 的 CBZ 仅 40–80 MB——减少 80%，在手机屏幕上肉眼看不出质量差异。一套 50 册的系列，省下的空间就非常可观了。

## 常见问题

**CBZ 和 CBR 漫画格式有什么区别？**
CBZ 和 CBR 都是顺序图像归档文件——CBZ 使用 ZIP 容器，CBR 使用 RAR。内部图像完全相同，只有外部归档格式不同。CBZ 是现代标准，因为 ZIP 是开放且无版权限制的；CBR 多见于早期版本。BiblioFuse 无需转换即可读取两者。

**能在漫画阅读器 App 中读取 EPUB 文件吗？**
可以，前提是 App 原生支持 EPUB。BiblioFuse 可以将无 DRM 的 EPUB 文件与 CBZ、CBR 一起管理，让你在同一书库中保存小说和图像小说。EPUB 是可回流格式，因此字体大小、行间距和页边距均可调整。

**漫画适合用 CBZ 还是 EPUB？**
CBZ 几乎始终是漫画的最佳选择。漫画页面是整页图像，而 CBZ 正是为此而设计的。下载漫画时，有 CBZ（或 CBR）就优先选择。

**如何将 CBR 转换为 CBZ？**
CBR 和 CBZ 包含相同的图像文件，只是容器不同。可以重新打包：解压 CBR，选中所有图像，压缩为 ZIP 后将扩展名改为 .cbz。BiblioFuse 的工具选项卡也可以直接在 iPhone 上完成 CBR 到 CBZ 的转换。

**为什么我的 CBZ 文件这么大？**
大型 CBZ 文件通常包含高分辨率 PNG 图像，这类图像无损且占用大量空间。以 80% 质量转换为 WebP 通常可将文件大小减少 75–88%，在手机或平板屏幕上看不出任何变化。BiblioFuse 内置压缩工具可在设备上完成处理。
