# Windows PCの漫画ライブラリをiPhoneにストリーミング — 自宅Wi-Fiと外出先5G対応

あなたの漫画コレクションはWindows PCに保存されています。CBZ、EPUB、PDFファイルが数百GB、自分流に整理されています。iPhoneの容量は64GBで、そのほとんどは既に使われています。明らかな解決策——ファイルをコピーする——は、すべてのファイルを2か所で管理し、常に容量不足に悩み、何か追加するたびに転送作業が必要になることを意味します。

[BiblioFuse](https://bibliofuse.com/ja/)は別のアプローチを取ります。Windows向けBiblioFuse PC Readerアプリには、組み込みのストリーミングサーバーが含まれています。iPhoneがPCに接続し、必要に応じてページをストリーミング——何もコピーされず、何もアップロードされず、iPhoneのストレージは空いたまま。Macユーザーが使っていたのと同じ機能が、Windowsでネイティブに利用可能になり、Microsoft Storeからダウンロードできます。

## PCライブラリストリーミングの仕組み

BiblioFuse PC ReaderはあなたのWindowsマシン上でローカルHTTPサーバーを実行します。iPhoneとPCが同じWi-Fiネットワーク上にある場合、iOSアプリはBonjour（AirPrintやAirDropを支えるゼロ設定ネットワークプロトコル）を使ってPCを自動的に検出します。IPアドレスを入力したりルーターを設定したりせずに、PCのライブラリがiPhoneに表示されます。

巻を開くと、BiblioFuseは最初の数ページを即座に取得し、読み進める間に次のページをプリフェッチします。CBZおよびCBRファイルはページごとにストリーミングされ、EPUBおよびPDFファイルはHTTP Rangeリクエストを使用して、全体をダウンロードせずに大きなファイルを素早く開きます。読書体験——右から左へのモード、縦読み漫画の縦スクロール、横向きの見開き表示——はローカルに保存されたファイルを読むのと同じです。唯一の違いはデータの出所だけです。

アプリウィンドウを閉じてもサーバーは動き続けます。終了する代わりにシステムトレイに移動します。つまり、PCで別のことをしている間も、iPhoneからライブラリにアクセスできます。ウィンドウを復元するには、トレイアイコンをダブルクリックします。サーバーを完全に停止するには、トレイアイコンを右クリックして「終了」を選択します。

## パート1：自宅Wi-Fiでのストリーミング

これが最も簡単な設定です。PCとiPhoneが同じネットワーク上にあり、BiblioFuseが自動的に検出を処理します。

### PCで

1. Microsoft Storeから**BiblioFuse PC Reader**をダウンロードしてインストールします。
2. アプリを開き、**設定 → ホームライブラリ**に移動します。
3. **フォルダーを追加**をクリックし、漫画や電子書籍が保存されているフォルダー（複数可）を選択します。複数のフォルダーを追加できます——漫画用、グラフィックノベル用、PDF用など。
4. **ライブラリ共有**を有効にします。サーバーはポート7341で即座に起動し、バックグラウンドでライブラリのスキャンを開始します。
5. オプションで、設定 → セキュリティで**パスワード**を設定してアクセスを保護できます。省略した場合、ローカルネットワーク上の誰でも接続できます——自宅では問題ありませんが、共有または公共のWi-Fiでは保護を推奨します。
6. ウィンドウを閉じても構いません。サーバーはシステムトレイで実行し続けます。

### iPhoneで

1. iPhoneで**[BiblioFuse](https://bibliofuse.com/ja/)**を開きます。
2. **ライブラリ**タブをタップします。
3. 上部のソース選択をタップします——PCが実行中で共有が有効なら、数秒以内に**PCホームライブラリ**が表示されます。
4. タップします。PCのフォルダー構造がすぐに表示されます。

設定は以上です。ライブラリ全体がiPhoneで閲覧・読書できるようになり、デバイスへのファイル転送はゼロです。PCに新しい巻を追加すると、次のライブラリ更新時にiPhoneビューに表示されます——再同期の手順は不要です。

### 対応フォーマット

BiblioFuse PC Readerはローカルで読めるすべてのフォーマットをストリーミングします：**CBZ、CBR、EPUB、PDF、TXT**。CBZ形式の漫画、CBR形式のグラフィックノベル、EPUB形式の電子書籍、PDF形式のスキャン本——すべて同じストリーミング接続でサポートされています。

### パフォーマンスのヒント

**5GHz Wi-Fiを使用してください。** ルーターが2.4GHzと5GHzの両方のバンドをサポートしている場合、PCとiPhoneの両方を5GHzバンドに接続してください。高解像度の大きなCBZファイルでは、高い帯域幅が明らかな違いをもたらします。

**SSD上のライブラリは高速です。** 漫画コレクションがHDDに保存されている場合、各新しい巻の最初のページ読み込みに少し遅延が生じます。ライブラリをSSD（内蔵または外付け）に移動するとこれが解消されます。

**iPhoneがPCを見つけられない場合：** 共有が実行中の場合、PCアプリは設定 → ホームライブラリにネットワークアドレスを表示します。Windowsファイアウォールがポート7341をブロックしていないか確認してください。iOSアプリのライブラリ → サーバーを追加 → 手動でPCのローカルIPアドレスを手動入力することもできます。

## パート2：外出先での5Gストリーミング

自宅Wi-Fiストリーミングは、両方のデバイスが同じネットワーク上にある場合のみ機能します。カフェ、ホテル、または5G環境からライブラリにアクセスするには、2つのネットワークを橋渡しする必要があります。解決策は[Tailscale](https://tailscale.com)——[BiblioFuse Tailscaleガイド](/ja/blog/tailscale-remote-library-access/)で説明されているのと同じ設定です。

Tailscaleはデバイス間にプライベートな暗号化メッシュネットワークを作成します。一度設定すれば、iPhoneはPCのストリーミングサーバーに、まるで同じローカルネットワーク上にいるかのようにアクセスできます——ポート転送もスタティックIPも不要で、どんなインターネット接続でも動作します。

### リモートストリーミングのためのTailscale設定

**Windows PCで：**

1. [tailscale.com](https://tailscale.com)またはMicrosoft StoreからTailscaleをダウンロードします。
2. Google、GitHub、Microsoftのアカウント、または無料のTailscaleアカウントでサインインします。
3. TailscaleはBiblioFuseと一緒にシステムトレイで動作します。PCのTailscale IPアドレス（`100.`で始まる）をメモしておきます。
4. BiblioFuse PC Readerが実行中でライブラリ共有が有効なことを確認します。

**iPhoneで：**

1. App StoreからTailscaleをインストールします。
2. 同じTailscaleアカウントでサインインします。
3. Tailscale VPNトグルを有効にします。
4. iPhoneでBiblioFuseを開きます。ライブラリ → サーバーを追加で、PCのTailscale IPアドレスとポート`7341`を入力します。
5. 接続をタップします。

TailscaleのIPが利用可能な場合、BiblioFuseは自動的にそれを優先するため、同じ保存済み接続が自宅（LAN経由）でも外出先（Tailscale経由）でも機能します。2つの別々の接続エントリは不要です。

### モバイルデータ通信時の期待値

Tailscaleは可能な限りピア・ツー・ピア接続を確立し、より良いパフォーマンスのためにリレーサーバーを回避します。一般的な5G接続では、漫画ページのストリーミングは快適に動作します。BiblioFuseは現在のページのみを送信し、次の数ページをプリフェッチします——CBZファイル全体を一度にストリーミングするわけではないので、200MBのアーカイブが200MBのデータ使用量を意味するわけではありません。画像解像度とサーバー圧縮設定に応じて、**1ページあたり約1〜3MB**を想定してください。

データプランが限られている場合は、外出前にWi-Fi経由で[BiblioFuse](https://bibliofuse.com/ja/)を使って個別の巻をダウンロードしましょう。iPhoneのPCライブラリビューで、任意の巻の隣にあるダウンロードアイコンをタップしてローカルにキャッシュします。その巻はその後、Tailscaleやデータ接続なしでオフラインで利用できます。

## PCとMacの違い

Macコンパニオンアプリを使ったことがある場合、iOS側からのPC体験はほぼ同じです。いくつかのWindows固有の違いがあります：

- **ポート：** PCサーバーはポート**7341**を使用します（Macアプリのデフォルトは8686）。
- **検出：** 両方ともサービスタイプ`_bibliofuse._tcp`でBonjour/mDNSを使用するため、iOSによる検出方法は同じです。
- **システムトレイ：** ウィンドウを閉じてもPCアプリはシステムトレイに残り、「終了」を選択するまでサーバーは停止しません。
- **設定ファイル：** PC設定は`%APPDATA%\BiblioFuse\settings.json`に保存されています。手動で確認またはリセットが必要な場合に参照できます。
- **同時稼働は1台のみ：** 現在のバージョンでは、BiblioFuse iOSはMacサーバーまたはPCサーバーのいずれかに接続します——両方同時には接続できません。iOSが最初に検出するのは、実行中かつアクティブなマシンです。

Macストリーミングの詳細については、[MacのライブラリをWi-Fi経由でiPhoneにストリーミング](/ja/blog/mac-home-library-streaming-iphone/)をご覧ください。

## よくある質問

**BiblioFuseにWindowsアプリはありますか？**
あります。BiblioFuse PC ReaderはMicrosoft Storeから入手できるネイティブWindowsアプリケーションです。CBZ、CBR、EPUB、PDF、TXTファイルをローカルで読み、ネットワーク経由でiOSアプリにライブラリ全体をストリーミングします。

**Windows PCからiPhoneに漫画をストリーミングするには？**
WindowsにBiblioFuse PC Readerをインストールし、設定でライブラリ共有を有効にします。iPhoneにBiblioFuseをインストールします。同じWi-Fiネットワーク上でiPhoneのBiblioFuseを開いてライブラリをタップすると——PCライブラリがBonjour検出によって自動的に表示されます。IPアドレス入力やルーター設定は不要です。

**外出先で5G経由でPCライブラリにアクセスできますか？**
はい、Tailscaleを使えば可能です。Windows PCとiPhoneにTailscaleをインストールし、同じアカウントでサインインして両デバイスで有効にします。BiblioFuseはTailscale接続を自動的に検出し、どこからでもPCにアクセスするために使用します——ホテルのWi-Fi、5G、どんなインターネット接続でも。

**PCからのストリーミングはiPhoneのストレージを使いますか？**
いいえ。ページはPCからオンデマンドで取得され、アプリ本体と読書進捗データを除いてiPhoneには何も保存されません。500GBのライブラリを閲覧・読書しても、iPhoneのファイルストレージは消費されません。

**ストリーミング中はPCを起動したままにする必要がありますか？**
はい。ストリーミングサーバーはPCで実行されるため、PCがスリープしたりサーバーが停止したりすると、iOS接続が切断されます。ストリーミング中のスリープを防ぐには、Windowsの設定 → 電源とスリープでスリープを「なし」に設定するか、PowerToysなどのツールを使用してください。

**PCストリーミングサーバーはどのポートを使用しますか？**
ポート7341です。このポートのトラフィックをブロックするファイアウォールがある場合、BiblioFuse PC Readerの例外を追加する必要があります。

## 始めましょう

[BiblioFuse PC Reader](https://bibliofuse.com/ja/)は今すぐMicrosoft Storeで入手できます。iPhone版BiblioFuseはApp Storeにあります。両方インストールし、PCアプリをライブラリフォルダーに指定すれば、次回アプリを開くときにはコレクション全体がiPhoneに——ケーブルなし、クラウドなし、コピーなしで。
