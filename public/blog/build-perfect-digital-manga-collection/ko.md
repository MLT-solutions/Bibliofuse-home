# 완벽한 디지털 만화 컬렉션 구축하는 방법: 압축, 중복 제거, 어디서나 읽기

BiblioFuse는 만화 수집가들이 실제로 사용하는 모든 형식——CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT——을 지원하는 iOS 리더입니다. iOS 17 이상이 설치된 iPhone 및 iPad에서 작동하며, 내장된 도구 탭을 통해 기기에서 직접 파일을 압축하고 변환할 수 있어 컴퓨터가 필요 없습니다. Mac용 Comic Duplicate Scanner와 [BiblioFuse 웹 도구](/ko/webapp/)를 함께 사용하면 완벽한 디지털 만화 컬렉션을 구축하고 관리하는 완전한 워크플로를 갖출 수 있습니다.

## 만화 수집가의 고민

몇 년 전부터 디지털 만화를 모으기 시작했습니다. 여기저기 다운로드하고, 기억도 가물가물한 킥스타터 번들도 있습니다. 어느 순간 베르세르크 1권이 세 개나 생겼고, 파일명도 조금씩 다르며, iPad는 항상 꽉 차있습니다. 익숙한 상황인가요?

대부분의 수집가는 결국 이 상황에 이릅니다. 문제는 관리 능력의 부재가 아닙니다. 파일이 너무 많은 곳에서 쏟아져 들어오기 때문입니다——상점 다운로드, 스캔 아카이브, 개인 립, 클라우드 서비스, 리더 앱 내보내기. 조용히 쌓이다 보면 어느새 중복 파일과 비대한 CBZ로 가득 찬 200 GB 미로가 됩니다.

## 파일 크기와 중복 파일이 만화 컬렉션을 망치는 이유

저장 공간은 한정적이고 만화는 이미지가 많습니다. 풀 해상도로 스캔된 단행본 하나가 400 MB에 달하기도 합니다. 이것을 500권 분량으로 곱하면, 중복을 제외하고도 200 GB에 이릅니다.

중복 파일은 생각보다 더 교묘합니다. 같은 권을 두 다른 곳에서 받으면 파일명이 다르고, 압축 수준이 다르며, 이미지 크기도 미세하게 다를 수 있어 단순 파일명 검사로는 발견되지 않습니다. 보유하고 있는 중복 파일마다 저장 공간이 낭비되고, 동기화 슬롯이 소모되며, 검색 결과가 오염됩니다.

전송 속도도 중요합니다. USB로 50 GB를 복사하거나 iCloud가 푸시하기를 기다리는 것은 고통스럽습니다. 올바른 워크플로는 불필요한 데이터가 기기에 도달하기 전에 제거합니다.

## 단계별 가이드: 완벽한 컬렉션 구축하기

### 1. 중복 파일 감사 및 제거 (Comic Duplicate Scanner)

압축하거나 정리하기 전에 먼저 중복된 파일을 찾아내세요. Mac 및 Windows용 **Comic Duplicate Scanner**는 지각적 해싱(perceptual hashing)을 사용하여 CBZ, CBR, 사진 아카이브 내부의 실제 이미지 내용을 비교합니다. 파일명이 바뀌었거나, 권이 재압축되었거나, 표지가 교체되었더라도 중복을 감지합니다.

라이브러리 폴더 전체를 스캔하세요. 자동 선택 전략으로 각 중복 쌍에서 품질이 낮은 파일을 표시하고, 목록을 검토한 뒤 휴지통으로 보내세요. Comic Duplicate Scanner는 확인 없이 영구 삭제하지 않습니다.

### 2. 큰 파일 압축하기 (BiblioFuse 웹 도구 또는 iOS 도구 탭)

중복 파일을 제거한 뒤에는 파일 크기를 줄이세요. [BiblioFuse 웹 도구](/ko/webapp/)는 WebAssembly를 사용하여 브라우저에서 완전히 실행됩니다——파일이 서버에 업로드되지 않습니다. CBZ, PDF, EPUB 파일을 한 번에 드래그하여 압축 수준을 선택하고 결과를 다운로드하세요. 무료 플랜은 배치당 3개를 처리합니다.

iPhone 또는 iPad에서 직접 작업하려면 [BiblioFuse](/ko/)를 열고 **도구** 탭으로 이동하세요. 컴퓨터 없이도 기기에서 파일을 압축, 변환, 병합할 수 있습니다.

### 3. 암호로 보호된 파일 해제 (SmartDecrypt)

팬 번역 아카이브나 일부 상점 다운로드 파일은 암호로 보호되어 있습니다. **SmartDecrypt PDF ZIP CBZ**(iOS 및 macOS)는 암호 프로필 저장소와 함께 일괄 해제 기능을 제공합니다——자주 사용하는 암호를 한 번 저장하고 폴더 전체에 탭 하나로 적용하세요. Rust 코어 덕분에 구형 기기에서도 빠르게 처리됩니다.

### 4. 태그, 평점, 컬렉션으로 정리하기 (BiblioFuse iOS)

깨끗하고 압축되고 해제된 라이브러리를 갖추었다면 이제 정리할 차례입니다. iPhone 및 iPad의 [BiblioFuse](/ko/)는 라이브러리의 모든 파일에 태그, 별점, 읽기 진행 상황 추적을 제공합니다. 시리즈별, 작가별, 읽기 상태별로 태그를 붙이세요——본인에게 맞는 방식이면 무엇이든 좋습니다. 평점으로 재독할 만한 권을 표시하고 건너뛸 권을 처리하세요.

검색창이 전체 라이브러리에서 즉시 원하는 권을 찾아주므로, 태그가 잘 정리된 컬렉션은 수천 권이 되어도 관리가 흐트러지지 않습니다.

### 5. 어디서나 읽기 (iCloud Sync, Wi-Fi 전송, Mac 홈 라이브러리)

완벽한 컬렉션이란 어디서든 접근 가능한 것입니다. BiblioFuse는 라이브러리를 기기에 가져오는 세 가지 방법을 제공합니다.

- **iCloud Sync** — 라이브러리가 iCloud Drive에 저장되어 동일한 Apple ID로 로그인된 모든 iPhone과 iPad에 자동 동기화됩니다.
- **Wi-Fi 전송** — Mac이나 PC의 브라우저에서 전송 페이지를 열고 파일을 드래그하면 즉시 BiblioFuse에 나타납니다. 케이블, iTunes, AirDrop 불필요.
- **Mac 홈 라이브러리** — BiblioFuse를 Mac의 폴더로 지정하면 iPhone이 Wi-Fi를 통해 Mac에서 직접 스트리밍합니다. 파일 복사 없이 64 GB iPhone에서 200 GB 전체 라이브러리에 접근 가능.

## 어떤 도구가 무엇을 하나요?

| 작업 | 도구 |
|------|------|
| 중복 권 찾기 | Comic Duplicate Scanner (Mac/Windows) |
| 파일 압축 / 변환 / 병합 | BiblioFuse 웹 도구 (브라우저) 또는 iOS 도구 탭 |
| 보호된 아카이브 해제 | SmartDecrypt PDF ZIP CBZ (iOS/macOS) |
| 읽기, 태그, 평점, 동기화 | BiblioFuse iOS (iPhone/iPad, iOS 17+) |
| Mac 라이브러리를 iPhone으로 스트리밍 | BiblioFuse Mac 홈 라이브러리 |

## 내 컬렉션을 iPhone으로 어떻게 전송하나요?

케이블 없는 가장 빠른 방법은 **Wi-Fi 전송**입니다. iPhone에서 BiblioFuse를 열고 Wi-Fi 전송 버튼을 탭하면 로컬 URL이 표시됩니다. Mac이나 PC의 브라우저에서 그 URL을 열고 CBZ 또는 EPUB 파일을 페이지에 드래그하세요. 현대적인 Wi-Fi 환경에서 전송 속도는 50~80 MB/s에 달하며, 1 GB 파일이 20초 이내에 도착합니다.

매우 큰 라이브러리에는 **Mac 홈 라이브러리**가 더욱 뛰어납니다. 아무것도 복사하지 않기 때문입니다. Mac이 개인 만화 서버가 되고 iPhone의 BiblioFuse가 파일을 직접 탐색하고 엽니다. Mac 라이브러리에 새 권을 추가하면 동기화 단계 없이 즉시 iPhone에서 볼 수 있습니다.

## 자주 묻는 질문

**BiblioFuse는 만화의 어떤 형식을 지원하나요?**
BiblioFuse는 CBZ, CBR, ZIP, RAR, EPUB, PDF, TXT 파일을 읽습니다. CBZ는 가장 일반적인 만화 아카이브 형식으로 즉시 열립니다. EPUB는 리플로어블 레이아웃의 만화와 라이트 노벨에 적합합니다. 모든 형식은 변환 없이 지원됩니다.

**Mac에서 중복 만화 권을 어떻게 찾나요?**
Comic Duplicate Scanner를 사용하세요. 아카이브 내부의 이미지 내용을 지각적 해싱으로 비교하므로, 파일명이 다르거나 재압축된 경우에도 중복을 찾습니다. 500권 라이브러리 전체 스캔은 보통 수 분이면 완료됩니다.

**화질 손실 없이 만화 파일을 압축할 수 있나요?**
네. BiblioFuse 웹 도구와 iOS 도구 탭은 파일 크기를 크게 줄이면서도 편안한 독서에 충분한 선명도를 유지하는 압축 수준을 제공합니다. 일상적인 독서용 복사본에는 적극적인 수준을 사용하면 크기를 50~70% 줄일 수 있습니다.

**만화 라이브러리를 iPhone에 동기화하는 방법은 무엇인가요?**
세 가지 옵션이 있습니다: 자동 배경 동기화에는 iCloud Drive, 브라우저에서 빠른 드래그 앤 드롭에는 Wi-Fi 전송, 파일 복사 없이 Mac 전체 라이브러리를 직접 스트리밍하려면 Mac 홈 라이브러리를 사용하세요.

**만화 파일 형식은 CBZ와 EPUB 중 어떤 것이 더 좋나요?**
스캔한 만화——전통적인 칸 배열 페이지——에는 CBZ가 더 좋습니다. 스캔 당시의 이미지를 그대로 보존하며 모든 만화 리더에서 열 수 있습니다. EPUB는 라이트 노벨처럼 텍스트 비중이 높은 콘텐츠에 더 적합합니다. BiblioFuse는 두 형식을 동등하게 잘 지원합니다.

**iPhone에서 암호로 보호된 CBZ 파일을 읽을 수 있나요?**
네, SmartDecrypt PDF ZIP CBZ를 사용하면 됩니다. 암호로 보호된 ZIP 및 CBZ 파일을 일괄 해제하고, 암호를 안전한 프로필에 저장하며, 공유 확장을 통해 해제된 파일을 BiblioFuse로 전달합니다. 각 암호는 한 번만 입력하면 됩니다.

## 오늘 완벽한 컬렉션으로 읽기 시작하세요

훌륭한 만화 컬렉션은 단순히 권을 보유하는 것이 아닙니다——찾을 수 있고, 기기에 맞출 수 있으며, 어디서나 부드럽게 열 수 있어야 합니다. [BiblioFuse](/ko/)는 리더, 전송 도구, 기기 내 처리 기능을 하나의 앱에 모두 담았습니다. Comic Duplicate Scanner와 SmartDecrypt와 함께 사용하면 처음부터 끝까지 완벽한 워크플로를 구성할 수 있으며, 컬렉션이 역대 최상의 상태가 될 것입니다.
