# BiblioFuse 북마크 활용하기: 만화에서 읽던 위치를 잃지 않는 방법

BiblioFuse의 북마크 기능을 사용하면 만화, 망가, EPUB의 어떤 페이지에든 사용자 이름을 붙여 저장하고 즉시 되돌아갈 수 있습니다. 이 기능은 iOS 17 이상의 iPhone과 iPad에서 작동하며, 모든 북마크는 iCloud를 통해 기기 전체에 자동으로 동기화됩니다. 챕터 중간이든 시리즈 중간이든 관계없이, 읽던 위치는 항상 저장됩니다.

## 만화 독자라면 누구나 겪는 바로 그 순간

긴 망가 시리즈의 3권을 밤늦게 절반까지 읽다가 읽기를 멈췄습니다. iPhone을 내려놓고 바쁜 일상이 이어졌고, 일주일 뒤 다시 집어 듭니다. 어디서 멈췄는지 기억하려고 페이지를 넘겨보지만 12화였는지 13화였는지 알 수가 없습니다. 앱을 확인해도 실수로 다른 권을 열었던 탓에 마지막 읽기 위치 표시가 엉뚱한 곳을 가리키고 있습니다.

BiblioFuse의 북마크는 바로 이런 순간을 위해 만들어졌습니다.

## iPhone에서 만화 북마크가 페이지 번호를 외우는 것보다 나은 이유는?

"247페이지만 기억하면 돼"라는 생각은 두세 개 이상의 시리즈를 동시에 읽는 순간 무용지물이 됩니다. 앱마다 페이지를 세는 방식이 다릅니다. 일부 시리즈는 여러 권으로 나뉘어 각 권이 1페이지부터 시작합니다. iPhone과 iPad를 번갈아 사용하면 단일 읽기 위치만 저장하는 앱에서 동기화 오류가 생길 수 있습니다.

BiblioFuse의 북마크는 이 모든 문제를 해결합니다. 이름이 있고, 영구적으로 저장되며, 대략적인 읽기 위치가 아닌 특정 페이지에 연결됩니다.

## iPhone에서 BiblioFuse 만화 북마크 사용하는 방법

BiblioFuse는 북마크 추가를 두 번의 탭으로 완료합니다. 처음부터 끝까지 작동 방식을 설명합니다.

### 북마크 추가 단계별 안내

1. [BiblioFuse](/ko/)에서 만화, 망가, EPUB 또는 PDF를 엽니다.
2. 저장하려는 페이지로 이동합니다.
3. 읽기 도구 모음의 북마크 아이콘(리본 또는 깃발 모양)을 탭합니다. 보통 화면 중앙을 탭하면 컨트롤이 나타납니다.
4. 북마크 이름을 입력하는 패널이 아래에서 올라옵니다. "2부 시작" 또는 "최애 컷 — 7화"처럼 설명적인 이름을 입력합니다.
5. 저장을 탭합니다. 북마크가 즉시 생성됩니다.
6. 나중에 돌아오려면 책의 상세 화면이나 리더 내에서 북마크 아이콘을 탭합니다. 전체 목록이 나타납니다.
7. 목록에서 원하는 북마크를 탭하면 해당 페이지로 바로 이동합니다.

책 한 권에 북마크를 무제한으로 추가할 수 있으므로 선택적으로 사용할 필요가 없습니다. 챕터마다 시작 페이지, 중요한 컷, 다시 보고 싶은 모든 페이지에 북마크를 추가하세요.

### 좋아하는 컷 북마크하기

북마크는 읽기 진행 상황 추적에만 사용하는 것이 아닙니다. 강렬한 더블 페이지나 캐릭터 등장 장면이 인상적이었다면 북마크를 추가하세요. "4권 최고의 전투 장면"이라고 이름을 붙이면 수백 페이지를 더 읽은 뒤에도 언제든지 돌아올 수 있습니다.

### 챕터 시작 부분 정리하기

긴 시리즈의 경우 각 챕터 첫 페이지에 챕터 번호가 포함된 이름으로 북마크를 추가하세요. 그러면 북마크 목록이 BiblioFuse가 지원하는 모든 형식——CBZ, CBR, ZIP, RAR, EPUB, PDF, TXT——에서 작동하는 개인 목차가 됩니다.

## iCloud 동기화가 북마크를 더욱 유용하게 만드는 이유는?

iPhone에서 만든 북마크는 추가 작업 없이 iPad와 Mac에도 자동으로 나타납니다. 이는 [BiblioFuse](/ko/)에 내장된 iCloud 동기화 덕분이며, 기존 Apple ID 외에 별도 계정이 필요하지 않습니다.

소파에서 iPhone으로 읽기 시작하고 북마크를 추가한 다음, 침실에서 iPad를 들고 같은 페이지로 바로 이동할 수 있습니다. 동기화는 백그라운드에서 이루어지므로 필요할 때 북마크가 준비되어 있습니다.

## 실용적인 시나리오: iPhone에서 만화 북마크를 최대한 활용하는 방법

### 여러 시리즈 동시에 추적하기

다섯 개의 서로 다른 망가 시리즈를 따라가고 있다면, 북마크는 각 시리즈에 이름이 붙은 저장 지점을 제공합니다. 책마다 "마지막으로 읽은 위치" 표시 하나에 의존하는 대신, 한 권 안에 여러 북마크를 둘 수 있습니다. 현재 읽기 위치용 하나, 공유하거나 다시 읽을 컷 용 여러 개로 구성할 수 있습니다.

### 중요한 컷을 참고 자료로 북마크하기

작가와 아티스트는 연속성 확인이나 컷 구성 연구를 위해 이전 권을 자주 다시 봅니다. BiblioFuse를 사용하면 각 책 안에 개인 참고 자료 라이브러리를 구축할 수 있습니다. 캐릭터 디자인이 바뀌는 컷이나 특정 스토리 라인이 처음 등장하는 페이지에 북마크를 추가하면 수백 페이지를 스크롤하지 않아도 몇 초 만에 찾을 수 있습니다.

### 친구와 페이지 참조 공유하기

친구에게 "이 컷 꼭 봐야 해"라고 말하고 싶을 때, 정확한 페이지 번호가 담긴 북마크가 있으면 정확하게 위치를 전달할 수 있습니다.

## 자주 묻는 질문

**BiblioFuse에서 페이지를 북마크하는 방법은?**
읽는 중에 화면 중앙을 탭하여 읽기 도구 모음을 표시한 다음 북마크 아이콘을 탭합니다. 이름을 입력할 수 있는 대화상자가 나타납니다. 저장을 탭하면 해당 책의 목록에 북마크가 추가됩니다.

**BiblioFuse 북마크가 iPad에도 동기화되나요?**
네. 북마크는 iCloud를 통해 동일한 Apple ID로 로그인된 모든 기기——iPhone, iPad, Mac——에 자동으로 동기화됩니다. 수동 동기화는 필요하지 않습니다.

**BiblioFuse에서 북마크에 이름을 붙일 수 있나요?**
네. 북마크를 만들 때마다 BiblioFuse가 사용자 이름 입력을 요청합니다. 챕터 참조, 장면에 대한 짧은 설명, 개인 메모 등 원하는 모든 내용을 입력할 수 있습니다.

**책 한 권에 북마크를 몇 개나 추가할 수 있나요?**
제한이 없습니다. 한 권에 원하는 만큼 북마크를 추가할 수 있어, 챕터마다 시작 페이지, 좋아하는 컷, 다시 보고 싶은 페이지를 모두 기록하는 것이 실용적입니다.

**EPUB 파일에서도 북마크가 작동하나요?**
네. BiblioFuse의 북마크는 앱이 지원하는 모든 형식——EPUB, PDF, CBZ, CBR, ZIP, RAR, TXT——에서 작동합니다. 파일 형식에 관계없이 북마크 경험은 동일합니다.

BiblioFuse의 북마크는 iPhone을 중요한 모든 시리즈, 챕터, 컷을 기억하는 독서 파트너로 만들어 줍니다. iCloud 동기화와 결합하면 나의 진행 상황이 모든 Apple 기기에서 이어집니다. 지금 [BiblioFuse](/ko/)를 다운로드하고 나만의 독서 생활에 맞춘 북마크 라이브러리를 만들어 보세요.
