# **Kebijakan Privasi - BiblioFuse Reader**

**Tanggal Efektif:** 2026-01-29

## **1. Pendahuluan**

BiblioFuse Reader ("Aplikasi") disediakan oleh Modern Logic Tech Solutions ("kami" atau "kami"). Kami mengutamakan privasi dan transparansi Anda.

## **2. Pengumpulan Data & Sinkronisasi Cloud**

Kami **tidak** menyimpan file pribadi, dokumen, atau buku Anda di server kami sendiri.

**Sinkronisasi Cloud:**

Aplikasi memfasilitasi sinkronisasi konten Anda (seperti EPUB, PDF, dan Buku Komik) di perangkat Anda menggunakan akun penyimpanan cloud pribadi Anda:

* **iCloud (iOS/macOS):** File disimpan dalam wadah iCloud pribadi Anda. Kami tidak memiliki akses ke file-file ini di luar wadah khusus Aplikasi.
* **Google Drive (Android):** File disimpan di Google Drive pribadi Anda. Aplikasi hanya mengakses file yang Anda pilih secara eksplisit atau folder yang Anda tentukan untuk sinkronisasi.

## **3. Pembayaran & Langganan**

**Pembelian Dalam Aplikasi:**

Aplikasi menawarkan fitur berbayar melalui pembelian satu kali ("Seumur Hidup") dan langganan ("Bulanan", "Tahunan").

* **Pemrosesan Pembayaran:** Semua transaksi keuangan diproses langsung oleh **Apple App Store** atau **Google Play Store**. Kami tidak mengumpulkan, menyimpan, atau memiliki akses ke informasi kartu kredit atau detail penagihan Anda.
* **Data Transaksi:** Kami mungkin menerima tanda terima transaksi anonim untuk memvalidasi status "Premium" Anda. Data ini tidak mencakup identitas pribadi Anda.

## **4. Layanan Pihak Ketiga**

Aplikasi menggunakan layanan pihak ketiga yang mungkin mengumpulkan informasi yang digunakan untuk mengidentifikasi perangkat Anda untuk fungsionalitas dan analitik:

* **Google Play Services**

## **5. Izin**

Aplikasi memerlukan izin berikut untuk berfungsi:

* **Akses Penyimpanan:** Diperlukan untuk membuka, membaca, dan mengelola file buku Anda secara lokal.
* **Akses Jaringan:** Diperlukan untuk mensinkronkan file Anda dengan penyedia cloud pilihan Anda (iCloud atau Google Drive) dan untuk memvalidasi lisensi premium.
