Como juntar vários ficheiros CBZ num só — ferramenta gratuita online

Descarregaste uma série de manga e cada capítulo veio como um ficheiro CBZ separado. Agora tens 40 ficheiros numa pasta — Capítulo 001.cbz até Capítulo 040.cbz — e sempre que acabas um capítulo no teu leitor tens de abrir o seguinte manualmente. Preferias ler o volume inteiro como um único CBZ. O problema é que juntá-los parece exigir instalar software ou pagar por um serviço online.

A BiblioFuse Web Tool junta ficheiros CBZ diretamente no browser, de forma gratuita. Nada é carregado para servidores. Não precisas de conta. O ficheiro resultante é descarregado diretamente para o teu computador.

Por que razão quereres juntar ficheiros CBZ

O padrão de um ficheiro por capítulo é comum no manga e em séries longas. Os grupos de scan empacotam cada capítulo separadamente. As lojas digitais vendem por vezes números avulsos mesmo que tenhas comprado uma série completa. Quando lês no telemóvel ou tablet, ter de navegar para o ficheiro seguinte a cada 30 páginas quebra o ritmo de leitura.

Um único CBZ para um volume ou arco completo oferece-te:

  • Leitura ininterrupta — o leitor trata-o como um ficheiro contínuo, gerindo corretamente as páginas duplas entre capítulos
  • Biblioteca mais organizada — um ficheiro por volume em vez de dezenas de capítulos soltos
  • Acompanhamento de progresso consistente — a percentagem lida e os marcadores funcionam ao longo de todo o volume, sem reiniciar a cada capítulo
  • Transferência mais simples — um único ficheiro para mover para o telemóvel em vez de muitos

Como funciona a junção de CBZ

O CBZ é um arquivo ZIP com imagens, uma por página, nomeadas sequencialmente para que o leitor saiba a ordem. Juntar ficheiros CBZ significa combinar essas sequências de imagens num único ZIP, renumeradas para que se leiam como uma história contínua.

A BiblioFuse Web Tool trata da renumeração automaticamente. Tu forneces os ficheiros e defines a ordem; a ferramenta extrai as imagens de cada CBZ, sequencia-as corretamente e empacota tudo num único CBZ de saída.

Como juntar ficheiros CBZ online

Passo 1: Abre a BiblioFuse Web Tool

Abre a BiblioFuse Web Tool num browser de secretária. Não é necessária conta nem início de sessão. A funcionalidade de junção está disponível na versão gratuita para lotes de até três ficheiros; faz upgrade para lotes maiores.

Passo 2: Adiciona os ficheiros CBZ por ordem

Clica na zona de largada ou arrasta os teus ficheiros CBZ para cima dela. A ordem importa — a ferramenta junta os ficheiros na sequência em que os adicionas. A maioria dos browsers permite selecionar vários ficheiros de uma vez a partir de uma pasta; se os teus ficheiros estiverem nomeados sequencialmente (Capítulo 001.cbz, Capítulo 002.cbz, etc.), selecioná-los todos de uma vez costuma adicioná-los na ordem certa.

Se precisares de reordenar ficheiros depois de os adicionar, podes arrastá-los para a posição correta na lista de ficheiros da ferramenta antes de iniciar a junção.

Passo 3: Executa a junção

Clica em Juntar. A ferramenta combina as imagens de todos os teus ficheiros CBZ num único arquivo, renumerando as páginas para formar uma sequência contínua. Um indicador de progresso mostra-te o estado. Quando terminar, o CBZ resultante está pronto para descarregar.

O nome do ficheiro de saída fica por defeito com o nome do primeiro ficheiro do lote, que podes alterar ao guardar.

Trabalhar com diferentes tipos de ficheiro

A BiblioFuse Web Tool consegue juntar mais do que apenas ficheiros CBZ. Se a tua série usa uma mistura de formatos — alguns capítulos em CBZ, outros em CBR, outros em ZIP — a ferramenta trata-os em conjunto. Não precisas de converter tudo para o mesmo formato antes de juntar. A ferramenta extrai as páginas de cada formato de entrada e empacota o resultado como CBZ.

Isto é útil para coleções de scans mais antigas, onde a consistência de formatos raramente era uma prioridade.

Qualidade de imagem após a junção

A junção não afeta a qualidade das imagens. A ferramenta combina os ficheiros de imagem dos teus arquivos sem os recodificar nem comprimir. Cada página no CBZ resultante é idêntica à página correspondente nos ficheiros originais.

Se também quiseres reduzir o tamanho do ficheiro ao mesmo tempo que juntas, a BiblioFuse Web Tool tem também uma etapa de compressão que podes executar antes ou depois da junção — recodificando imagens a um nível de qualidade escolhido para reduzir o ficheiro final. Mas é opcional; uma junção simples preserva tudo exatamente.

Criar volumes a partir de muitos capítulos

Um caso de uso comum é empacotar 10–12 capítulos num único CBZ do tamanho de um volume. Este fluxo de trabalho funciona bem:

  1. Reúne os capítulos que pertencem ao Volume 1 — para a maioria das séries são os capítulos 1–10 ou 1–12.
  2. Abre a BiblioFuse Web Tool e adiciona esses ficheiros de capítulos por ordem.
  3. Executa Juntar. Descarrega o resultado e renomeia-o para algo como Nome da Série Vol 01.cbz.
  4. Repete para o volume seguinte.

Cada ficheiro resultante é um volume limpo e autónomo que podes transferir para o teu leitor, catalogar na tua aplicação de biblioteca ou arquivar.

Após a junção: levar o ficheiro para o telemóvel

Depois de descarregares o CBZ resultante, a forma mais prática de o colocar num iPhone é através da funcionalidade de Transferência Wi-Fi do BiblioFuse.

Abre o BiblioFuse no teu iPhone, toca no botão de Transferência Wi-Fi e aparece um endereço de browser no ecrã. No computador, abre esse endereço num browser. Arrasta o teu ficheiro CBZ para a página. Transfere imediatamente através da tua rede local — sem USB, sem iTunes, sem conta na cloud.

Outras opções:

  • iCloud Drive: Coloca o ficheiro numa pasta do iCloud Drive ligada ao BiblioFuse
  • AirDrop: A partir de um Mac, envia o CBZ por AirDrop para o iPhone e abre-o no BiblioFuse
  • App Ficheiros: Liga via USB, copia para a pasta BiblioFuse na app Ficheiros

Perguntas frequentes

Qual é o número máximo de ficheiros que posso juntar?

A versão gratuita suporta até três ficheiros por lote. Para junções maiores, considera dividir o trabalho em várias passagens: junta os capítulos 1–3 num ficheiro parcial, depois junta esse ficheiro parcial com o lote seguinte, e assim por diante.

E se os meus capítulos estiverem em formato CBR?

O CBR é um arquivo RAR, não ZIP. A BiblioFuse Web Tool suporta CBR juntamente com CBZ e outros formatos — podes adicionar uma mistura de ficheiros CBZ e CBR e eles serão fundidos corretamente num único CBZ de saída.

A ferramenta preserva a ordem de leitura?

Sim. A ferramenta processa os ficheiros na ordem em que aparecem na lista, e as páginas dentro de cada ficheiro são adicionadas na sua sequência existente. Desde que adiciones os capítulos na ordem certa, o resultado lê-se corretamente.

Posso juntar ficheiros EPUB da mesma forma?

A BiblioFuse Web Tool também suporta a junção de EPUB, tratada como uma operação separada. Para conteúdo em estilo de banda desenhada — páginas digitalizadas em vez de texto refluível — o CBZ é geralmente o formato de destino mais adequado.

O ficheiro de saída preserva os metadados dos originais?

O CBZ resultante é um arquivo de imagens simples. Os metadados ComicInfo.xml dos originais não são combinados automaticamente. Se precisares de incluir metadados no ficheiro resultante, podes adicionar um ComicInfo.xml manualmente após descarregar o resultado.

Juntar ficheiros CBZ sem complicações

A BiblioFuse Web Tool trata da junção de CBZ como deve ser — no browser, de forma privada, gratuitamente, sem instalares nada. Tu forneces os ficheiros por ordem e ela trata do resto.

Para quem usa o BiblioFuse no iPhone ou iPad, a ligação entre a web tool e a app móvel torna o fluxo de trabalho perfeito: juntas no computador, transferes por Wi-Fi para o telemóvel, lês. Dezenas de ficheiros de capítulos transformam-se numa biblioteca de volumes organizada em poucos minutos.
