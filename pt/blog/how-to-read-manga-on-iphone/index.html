Ler Manga no iPhone: Modo RTL e Webtoon

22 de maio de 2026

Como ler manga no iPhone: da direita para a esquerda, rolagem vertical e páginas duplas

O BiblioFuse permite ler manga no iPhone e iPad nos formatos CBZ, CBR, ZIP ou RAR com o fluxo de paginas da direita para a esquerda, modo de rolagem vertical para webtoons e detecao automatica de paginas duplas na orientacao horizontal. Funciona no iOS 17 ou posterior e nao requer conta nem subscricao — os ficheiros sao carregados via Wi-Fi Transfer ou iCloud Drive sem cabo.

Ler manga num iPhone parece simples — são apenas imagens em ordem. Mas quem já tentou conhece os problemas: um leitor criado para banda desenhada ocidental impõe o fluxo de páginas da esquerda para a direita, quebrando a experiência de leitura em cada painel. Os webtoons (banda desenhada de rolagem vertical) aparecem como páginas individuais em vez de uma tira contínua. As páginas duplas surgem em modo retrato, esmagando artwork desenhado para ocupar toda a largura.

O manga tem requisitos específicos que a maioria dos leitores genéricos ignora. O BiblioFuse foi concebido para lidar com todos eles — modo da direita para a esquerda, rolagem vertical, deteção de páginas duplas e configurações de direção de leitura por série que persistem entre sessões.

Este guia explica exatamente como configurar cada uma dessas opções.

Instalar o BiblioFuse

O BiblioFuse está disponível gratuitamente na App Store. Não é necessária conta.

Uma vez instalado, lê CBZ, CBR, ZIP, RAR, EPUB, PDF e TXT — a gama completa de formatos que encontrará nas comunidades de manga. Os lançamentos de scans japoneses chegam habitualmente como CBZ ou ZIP com imagens JPG no interior; o BiblioFuse trata-os nativamente, sem conversão.

Transferir manga para o seu iPhone

O método mais rápido para uma coleção grande é a Transferência Wi-Fi. No BiblioFuse, aceda a Definições → Importar por Wi-Fi, ative a opção e visite o URL indicado em qualquer browser no seu Mac ou PC. Arraste as suas pastas de manga para a janela do browser — séries completas, não apenas volumes individuais.

Para volumes avulsos, utilize o menu Partilhar do iOS: abra a app Ficheiros, navegue até ao ficheiro CBZ, pressione longamente e partilhe para o BiblioFuse. Também pode copiar ficheiros para iCloud Drive → BiblioFuse no seu Mac e eles sincronizam automaticamente para o iPhone.

Depois de os ficheiros estarem na biblioteca, toque em qualquer capa para abrir o volume.

Modo de leitura da direita para a esquerda

A direção de leitura predefinida na maioria dos leitores é da esquerda para a direita: deslize para a esquerda para avançar e para a direita para recuar. Para o manga — que se lê da direita para a esquerda — isto parece ao contrário de imediato. Desliza para a esquerda pensando que avança e, em vez disso, recua.

No BiblioFuse:

  1. Abra um volume de manga
  2. Toque no centro do ecrã para mostrar os controlos
  3. Toque no ícone de direção de leitura (duas setas)
  4. Selecione Da direita para a esquerda

Agora deslize para a direita para avançar e para a esquerda para recuar — acompanhando o movimento natural dos seus olhos pelos painéis.

O BiblioFuse guarda esta configuração por volume, pelo que não precisa de a redefinir cada vez que abre o mesmo manga. As séries da esquerda para a direita (a maioria da banda desenhada ocidental, manhwa traduzido da esquerda para a direita) recordam a sua configuração de forma independente.

Zonas de toque para leitura da direita para a esquerda

Quando o modo da direita para a esquerda está ativo, as zonas de toque invertem-se:

  • Tocar na margem esquerda → página seguinte
  • Tocar na margem direita → página anterior
  • Tocar no centro → mostrar/ocultar controlos

Isto reproduz o funcionamento do manga físico — alcança o lado esquerdo do livro para virar para a página seguinte.

Modo de rolagem vertical para webtoons

Os webtoons (banda desenhada web coreana e muitos manga modernos online) são publicados como longas tiras verticais concebidas para rolagem contínua, não para virar página a página. Quando carrega um CBZ de webtoon num leitor de páginas padrão, vê fatias horizontais individuais em vez da tira completa — o que torna quase impossível acompanhar o fluxo dos painéis.

O BiblioFuse resolve isto com um modo de rolagem vertical:

  1. Abra o volume
  2. Toque no centro para mostrar os controlos
  3. Toque no ícone de disposição
  4. Selecione Rolagem vertical

As páginas empilham-se verticalmente numa tira contínua. Role para baixo para ler, tal como numa página web. Não há limites de página — o artwork flui como o criador pretendia.

Este modo funciona bem para:

  • Webtoons coreanos (LINE Webtoon, Naver Comics)
  • Manga web japonês moderno lançado em formato de tira longa
  • Manhwa
  • Banda desenhada com tradução de fãs formatada como tiras verticais

Para o manga tradicional publicado em formato de página (lançamentos Shonen Jump, volumes tankōbon), mantenha o modo de virar páginas.

Páginas duplas

Muitos volumes de manga incluem painéis a toda a largura — uma cena dramática que abrange duas páginas opostas, desenhada como uma única imagem larga. Em modo retrato num iPhone, estas páginas duplas aparecem como duas páginas separadas, perdendo-se o impacto visual do artwork.

Deteção automática em paisagem: Rode o iPhone para o modo paisagem. O BiblioFuse deteta a mudança de orientação e muda automaticamente para a vista de dupla página. Ambas as páginas aparecem lado a lado, ocupando o ecrã, exatamente como surgiriam no volume físico.

Páginas duplas da direita para a esquerda: Quando o modo da direita para a esquerda está ativo, a disposição das páginas duplas também se inverte — a ordem das páginas corresponde à direção de leitura, pelo que a página direita (a "primeira" página na ordem de leitura do manga) aparece no lado direito do ecrã.

Rode de volta para retrato e o BiblioFuse regressa imediatamente ao modo de página única.

Dicas para deteção de páginas duplas

Alguns ficheiros CBZ de grupos de scan mais antigos incluem páginas duplas como uma única imagem larga (já unida) em vez de duas páginas separadas. O BiblioFuse trata ambos os casos — imagens largas únicas em modo paisagem ocupam o ecrã sem duplicação. Não é necessário configurar nada.

Zoom e navegação

Juntar dois dedos para dar zoom funciona em qualquer página. O BiblioFuse renderiza as páginas em resolução total — dar zoom em traços finos, texto pequeno ou um painel de fundo detalhado não produz pixelização.

Duplo toque para ajustar devolve a página ao ecrã completo após o zoom.

Barra de progresso: Deslize a barra de progresso na parte inferior do ecrã para saltar para qualquer página do volume. Útil quando está a reler e quer ir diretamente a um capítulo ou cena específica.

Navegação por capítulos: Para manga em EPUB (utilizado por alguns editores), o BiblioFuse apresenta uma lista de capítulos. Para volumes CBZ/CBR, a barra de progresso é a sua ferramenta de navegação.

Organizar uma série de manga

Quando importa vários volumes da mesma série, o BiblioFuse agrupa-os por pasta. Se os seus ficheiros CBZ estiverem numa pasta chamada Demon Slayer, aparecem como uma única entrada de série na biblioteca. Toque na série para ver todos os volumes listados por ordem.

Pressione longamente qualquer volume para:

  • Adicionar uma classificação (1 a 5 estrelas)
  • Adicionar uma etiqueta (p. ex., "a ler", "concluído", "para ler")

A barra de filtros no topo da biblioteca permite filtrar por etiqueta, formato ou classificação. Se estiver a acompanhar quinze séries ativas, etiquete-as com "a ler atualmente" e filtre para ver apenas essas.

Sincronização do progresso de leitura

Se lê no iPhone e no iPad (ou Mac), o BiblioFuse sincroniza o progresso de leitura via iCloud. Abra o volume 3 de uma série no iPad durante um percurso, leia até à página 80 e quando mais tarde pegar no iPhone o progresso já lá estará.

Isto aplica-se apenas a ficheiros armazenados em iCloud Drive → BiblioFuse. Os ficheiros no armazenamento local do iPhone não sincronizam. Se transferiu por Wi-Fi Transfer, mova os ficheiros para o iCloud Drive na app Ficheiros se quiser sincronização entre dispositivos.

Brilho e conforto visual

Controlo de brilho: Deslize para cima ou para baixo no lado esquerdo do ecrã durante a leitura para ajustar o brilho sem sair do leitor. Útil ao passar de um ambiente luminoso para um quarto escuro.

Night Shift e True Tone do iOS: O BiblioFuse é uma app UIKit padrão, pelo que os ajustes de ecrã ao nível do sistema iOS (Night Shift, True Tone, Zoom de Ecrã) se aplicam normalmente.

Para sessões de leitura prolongadas, reduzir o brilho e ativar o Night Shift (Definições → Ecrã e Brilho → Night Shift) reduz significativamente o cansaço ocular.

Resolução de problemas

As páginas estão na ordem errada. A maioria dos ficheiros CBZ de manga usa nomes de ficheiro com zeros à esquerda (001.jpg, 002.jpg) para que ordenem corretamente. Se o seu CBZ foi criado sem esse preenchimento (1.jpg, 10.jpg, 2.jpg), as páginas ordenam alfanumericamente na ordem errada. A solução é renomear os ficheiros dentro do arquivo — extraia o CBZ, renomeie as imagens com números preenchidos e recompacte. O BiblioFuse lê a ordem que resultar da ordenação dos nomes de ficheiro.

O modo da direita para a esquerda é redefinido em novos volumes. O BiblioFuse guarda a direção de leitura por volume. Se importar um novo volume de uma série que já está a ler, terá de definir a direção uma vez ao abri-lo pela primeira vez. Depois disso, fica guardada.

As páginas do webtoon aparecem muito estreitas no modo de rolagem vertical. Alguns webtoons estão formatados como tiras muito altas e estreitas. Junte dois dedos para reduzir ligeiramente o zoom de forma a que a tira ocupe a largura do ecrã e depois role. O BiblioFuse lembra-se do nível de zoom entre páginas no modo de rolagem vertical.

As páginas duplas não estão a alinhar corretamente em paisagem. Se um CBZ tem páginas duplas montadas como um único JPEG largo, rodar para paisagem irá apresentá-lo centrado. Se a página dupla parecer desalinhada, o CBZ original provavelmente tem as duas páginas guardadas como ficheiros separados — o BiblioFuse une-as ao rodar. Se continuar incorreto, o próprio scan pode ter as páginas numa ordem invulgar.

O que ler a seguir

Agora que a sua configuração de leitura está pronta, o BiblioFuse tem mais algumas funcionalidades que vale a pena explorar. A Biblioteca Mac Home permite-lhe transmitir toda a sua coleção de manga de desktop para o iPhone por Wi-Fi sem copiar nada para o dispositivo — ideal se tiver terabytes de manga num Mac que prefere não duplicar. O separador Ferramentas trata da compressão — útil para reduzir volumes CBZ de 200 MB para 40–50 MB sem perda de qualidade visível, libertando espaço de armazenamento para mais séries.

As configurações certas fazem a diferença entre um leitor que dificulta a leitura e um que desaparece na experiência de leitura. Manga lido da direita para a esquerda, com páginas duplas em paisagem e webtoons em rolagem vertical, funciona exatamente como deve ser.

Perguntas frequentes

Qual e o melhor aplicativo de leitura de manga para iPhone? O BiblioFuse e criado especificamente para manga no iPhone e iPad. Suporta modo de leitura da direita para a esquerda, modo de rolagem vertical para webtoons, paginas duplas automaticas no modo horizontal e memoria de direcao de leitura por serie — basta configurar cada serie uma vez e permanece correto.

Como ativo o modo da direita para a esquerda para manga no iPhone? Abra um volume de manga no BiblioFuse, toque no centro da tela para mostrar os controles, toque no icone de direcao de leitura e selecione Direita para Esquerda. A direcao de deslizamento e as zonas de toque se invertem para seguir a ordem de leitura japonesa. O BiblioFuse salva essa configuracao por volume.

Posso ler webtoons (comics de rolagem vertical) no iPhone? Sim. No BiblioFuse, abra o volume, toque no icone de layout e selecione Rolagem Vertical. As paginas se empilham em uma faixa vertical continua — sem saltos de pagina, sem cortes horizontais. Este modo funciona para webtoons coreanos, manhwa e qualquer formato de faixa longa.

O BiblioFuse exibe paginas duplas corretamente? Sim. Gire o iPhone para o modo horizontal enquanto le e o BiblioFuse muda automaticamente para a visualizacao de duas paginas. No modo da direita para a esquerda, a ordem das paginas tambem se inverte para que o spread seja lido corretamente.

Quais formatos de manga o BiblioFuse suporta no iPhone? O BiblioFuse le CBZ, CBR, ZIP e RAR — os formatos usados por praticamente todos os lancamentos de scans de manga — alem de EPUB e PDF. Nenhuma conversao e necessaria; os arquivos abrem diretamente independentemente do formato.
