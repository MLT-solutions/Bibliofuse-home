Problemas conhecidos

iOS & macOS

Streaming requer Tailscale — modo somente LAN não disponível

A Segurança de Transporte de Aplicativos do iOS bloqueia HTTP simples para IPs locais. O Tailscale é necessário para Wi-Fi doméstico e streaming remoto. Um modo somente LAN sem Tailscale não está atualmente no roadmap.

Android

Sem streaming Mac/PC → Android

O Android usa uma base de código Flutter separada sem integração com iCloud. O suporte a streaming exigiria uma reescrita quase total e não está planejado.

Android

Acesso à rede SMB não verificado

O acesso a unidades SMB no Android não foi totalmente testado nas versões recentes.
