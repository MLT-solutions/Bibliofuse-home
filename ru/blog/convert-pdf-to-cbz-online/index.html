Как конвертировать PDF в CBZ онлайн (без программ и загрузки файлов)

Вы скачали отсканированную графическую новеллу в формате PDF. В Preview или Acrobat она открывается нормально, но в читалке для комиксов что-то не так: страницы грузятся медленно, масштабирование даёт размытую картинку, а перелистывание не такое резкое, каким должно быть. Вы знаете, что CBZ лучше подходит для комиксов. Проблема — как конвертировать, не устанавливая очередную настольную программу и не отправляя файлы на незнакомый сайт.

BiblioFuse Web Tool решает это прямо в браузере. Без регистрации. Без загрузки файлов. Без установки. Ваши файлы ни на секунду не покидают ваше устройство.

Почему PDF плохо подходит для комиксов

PDF создавался для документов — текстовых макетов, предназначенных для печати. Он отлично справляется со шрифтами, векторной графикой и переносом текста. Для этого он и создан.

С комиксами всё иначе. Отсканированный комикс — это последовательность полностраничных изображений, по одному на страницу. PDF оборачивает эти изображения в служебную структуру, которая увеличивает размер файла без какой-либо пользы, использует механизм отрисовки страниц, уступающий по скорости специализированному просмотрщику изображений, и не позволяет читалкам для комиксов заранее кешировать страницы впереди текущей позиции.

CBZ — это ZIP-архив с пронумерованной последовательностью файлов изображений. Почти никаких накладных расходов. Читалки для комиксов заранее распаковывают изображения, кешируют их в памяти и мгновенно перелистывают страницы. Качество масштабирования определяется исключительно файлом изображения, а не механизмом, пытающимся интерпретировать документный формат.

Когда вы конвертируете комикс из PDF в CBZ, вы извлекаете из PDF исходные изображения страниц и переупаковываете их в ZIP. Результат — файл, для работы с которым ваша читалка и создавалась.

Что делает BiblioFuse Web Tool

BiblioFuse Web Tool полностью работает в WebAssembly внутри браузера. WebAssembly выполняет скомпилированный код в изолированной среде, что позволяет приложению делать серьёзную обработку файлов — такую, которая обычно доступна только нативному настольному ПО — не отправляя ничего на сервер.

Файлы читаются с локального диска напрямую в память браузера. Конвертация выполняется во вкладке браузера. Результирующий файл записывается обратно на диск. Ничто ни в какой момент не проходит через серверы.

Это важно, если ваши комиксы содержат лицензионный контент, личные архивы или просто файлы, которые вы не хотите хранить в чужом облаке.

Как конвертировать PDF в CBZ

Шаг 1: Откройте инструмент

Зайдите на BiblioFuse Web Tool в любом настольном браузере — Chrome, Firefox, Safari, Edge. Логин и аккаунт не нужны. Бесплатный тариф поддерживает пакеты из трёх файлов одновременно.

Шаг 2: Добавьте PDF-файлы

Нажмите на зону загрузки или перетащите PDF-файлы прямо на неё. Можно добавить сразу несколько файлов. Инструмент покажет каждый файл с именем и определённым размером. Если файл слишком велик для доступной памяти браузера, вы увидите предупреждение до начала обработки.

Шаг 3: Выберите CBZ и конвертируйте

Выберите CBZ в качестве выходного формата в параметрах конвертации. Нажмите «Конвертировать». Инструмент последовательно обрабатывает каждый файл: извлекает изображения страниц из PDF и упаковывает их в CBZ-архив. Индикатор прогресса показывает, на каком этапе вы находитесь. После завершения конвертации выходные файлы появятся готовыми к загрузке.

Скачанные файлы сохраняют исходные имена, только расширение .pdf заменяется на .cbz.

Качество изображения при конвертации

Это самый частый вопрос, и ответ зависит от того, как создавался исходный PDF.

Отсканированные PDF: Самый распространённый случай для старых комиксов и томов манги. PDF содержит отсканированные изображения, хранящиеся непосредственно внутри него. Конвертация извлекает их без повторного кодирования. Изображения в CBZ идентичны изображениям в PDF — никакой потери качества, никакого дополнительного сжатия.

Векторные PDF: Некоторые PDF создаются из векторной графики или текстовых макетов. При конвертации таких файлов в CBZ каждая страница растеризируется с заданным разрешением. BiblioFuse Web Tool использует высокое разрешение по умолчанию, поэтому результат выглядит чётким при обычных уровнях масштабирования при чтении.

PDF со встроенным сжатием: Если при создании исходного PDF применялось сильное JPEG-сжатие, извлечённые изображения отражают это существующее сжатие. Инструмент не добавляет дополнительного сжатия, но и не может отменить то, которое уже встроено.

Для отсканированных комиксов — самого распространённого сценария конвертации — вы получаете извлечение без потерь. CBZ обычно такого же размера, что и PDF, или чуть меньше, качество не меняется.

Пакетная конвертация серии

Если у вас есть серия, охватывающая несколько PDF-томов, BiblioFuse Web Tool позволяет конвертировать весь пакет за одну сессию. Добавьте все PDF сразу, задайте CBZ как целевой формат и начните. Каждый файл обрабатывается и становится доступным для отдельной загрузки по мере завершения.

Бесплатный тариф обрабатывает до трёх файлов за один раз. Для более длинной серии запустите несколько пакетов в одной сессии браузера — перезагружать страницу между пакетами не нужно.

Перенос CBZ-файлов на телефон после конвертации

После завершения конвертации и загрузки CBZ-файлов есть несколько удобных способов перебросить их на телефон:

Передача по Wi-Fi (самый быстрый вариант): Если вы используете BiblioFuse на iPhone, встроенная функция передачи по Wi-Fi позволяет перетаскивать файлы из окна браузера на любом компьютере в той же сети. Откройте BiblioFuse, включите Wi-Fi-передачу и перетащите CBZ-файлы из браузера Mac или PC прямо в библиотеку. Без USB-кабеля, без iTunes, без облачного хранилища.

iCloud Drive: Положите файлы в папку iCloud Drive, связанную с BiblioFuse, и они синхронизируются автоматически, когда телефон подключён к Wi-Fi.

AirDrop: На Mac выберите CBZ-файлы в Finder, нажмите правой кнопкой и отправьте через AirDrop на iPhone. После завершения передачи нажмите «Открыть в BiblioFuse».

Частые вопросы

Справляется ли инструмент с PDF, содержащими одновременно изображения и текстовые слои?

Да. PDF со слоем отсканированных изображений и наложенным слоем OCR-текста обрабатываются корректно — инструмент извлекает визуальные изображения страниц. Текстовый слой не переносится в CBZ (CBZ — формат архива изображений по своей сути), но исходный PDF остаётся нетронутым.

Каков лимит размера файла?

Жёсткого лимита у инструмента нет. Практические ограничения определяются доступной памятью браузера — большинство настольных браузеров без проблем справляются с файлами в несколько гигабайт. Очень большие файлы на старом железе могут обрабатываться медленно, но как правило завершают обработку успешно.

Работает ли на мобильных браузерах?

Да, но производительность заметно ниже, чем на настольном компьютере. Для больших PDF рекомендуется использовать десктопный браузер.

Есть ли ограничение по количеству страниц?

Нет. Инструмент обрабатывает любое количество страниц в PDF.

Какой формат изображений используется в CBZ?

CBZ содержит тот формат изображений, который был сохранён в исходном PDF. Для отсканированных комиксов это обычно JPEG. Инструмент не перекодирует изображения в процессе конвертации, если только вы не включили сжатие как отдельный шаг.

Полный рабочий процесс: от PDF до телефона

BiblioFuse Web Tool выполняет конвертацию PDF в CBZ без установки программ, регистрации в облаке и загрузки файлов. Ваши файлы остаются на вашем устройстве на протяжении всего процесса.

В связке с BiblioFuse на iPhone или iPad вы получаете сквозной рабочий процесс без лишних шагов: конвертация в браузере, передача на телефон по Wi-Fi, чтение в приложении. Никакого специального менеджера комиксов для десктопа, никакого сервиса синхронизации, никаких промежуточных этапов.

Если у вас есть папка с PDF-комиксами, которые вы давно собирались конвертировать, откройте BiblioFuse Web Tool прямо сейчас и закиньте первый пакет.

Часто задаваемые вопросы

Как конвертировать PDF в CBZ без установки программ? Откройте BiblioFuse Web Tool в браузере на любом устройстве. Перетащите PDF-файл в зону загрузки — конвертация выполняется прямо в браузере на вашем устройстве. Файл не передаётся на серверы. Сохраните полученный CBZ и откройте его в BiblioFuse на iPhone или iPad.

Почему CBZ лучше PDF для чтения комиксов? CBZ оптимизирован для постраничной навигации в читалках: страницы загружаются мгновенно, масштабирование чёткое, а листание без задержек. PDF — универсальный формат для документов, который может содержать встроенный текст, шрифты и метаданные, замедляющие рендеринг в читалках для комиксов.

Загружаются ли мои файлы на серверы при конвертации PDF в CBZ? Нет. BiblioFuse Web Tool работает полностью в браузере с использованием JavaScript и WebAssembly. Файлы обрабатываются локально на вашем устройстве и никуда не передаются.

Какой размер PDF-файлов поддерживает инструмент? BiblioFuse Web Tool обрабатывает PDF-комиксы размером несколько сотен МБ в большинстве браузеров. Для очень больших файлов ограничивающим фактором является оперативная память браузера, а не ограничения загрузки.

Могу ли я конвертировать несколько PDF в CBZ за один раз? Да. Перетащите несколько PDF-файлов в окно инструмента — BiblioFuse Web Tool обработает их последовательно. Каждый PDF конвертируется в отдельный CBZ-файл с тем же именем.
