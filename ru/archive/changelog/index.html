Release history, known issues, and what's coming next.

2026-06-261.2.0
All platformsRelease

v1.2.0 — Локализация онбординга и поддержка языков Windows

Экраны онбординга теперь полностью локализованы на все 11 поддерживаемых языков. Метаданные App Store Connect обновлены с локализованными описаниями для всех 11 регионов. Приложение Windows теперь включает настройку языка, и все строки интерфейса полностью локализованы — соответствует поддержке 11 языков в приложении Mac.

2026-05-061.1.7
macOSRelease

v1.1.7 — macOS

Исправлено отображение цен в StoreKit, показывавшее неверную сумму в валюте на macOS после локализации App Store.

2026-05-021.1.4
macOSRelease

v1.1.4 — macOS

Реализован freemium-доступ на macOS: 7-дневный пробный период, разблокировка через StoreKit и ограничения функций для платных пользователей теперь активны в сборке для Mac App Store.

2026-04-221.1.0
macOSFix

v1.1.0–1.1.3 — Надёжность удаления файлов

Прямое удаление заменено на send2trash — файлы теперь отправляются в Корзину macOS вместо окончательного удаления. Исправлен баг откатного механизма NAS и проблема с молчаливым удалением строк. Полные исправления в рамках кросс-платформенного аудита кнопок.

2026-04-221.0.6
macOSFeature

Поддержка NAS и сетевых дисков

Сканирование папок на NAS и сетевых дисках через NSOpenPanel. Запросы Keychain сокращены с 4 до 1 при первом запуске. Улучшения UX покупок и повышение надёжности логики пробного периода.

2026-04-211.0.1
macOSFix

v1.0.1 — macOS

Исправлено восстановление покупки StoreKit при первом запуске. В меню «Справка» добавлена ссылка для связи с поддержкой.

2026-04-211.0.0
macOSRelease

v1.0.0 — Запуск на macOS

Первый выпуск в Mac App Store. Сканирование по перцептивному хешу для архивов комиксов (CBZ/CBR/ZIP/RAR) и фотографий. Нечёткое совпадение имён файлов с поддержкой Unicode. Умный автовыбор, постоянный список игнорируемых пар, кэш хешей для мгновенного повторного сканирования, сравнение нескольких папок, экспорт HTML/CSV.

2026-04-22
PC (Windows)Release

Windows — Отправка в Microsoft Store

Доступно в Microsoft Store. Сканирование по перцептивному хешу, дедупликация фотографий, нечёткое совпадение имён файлов, стратегии автовыбора, кэш хешей, сравнение нескольких папок, экспорт HTML/CSV. Бесплатно с 7-дневным пробным периодом; единовременная разблокировка безлимитного доступа.
