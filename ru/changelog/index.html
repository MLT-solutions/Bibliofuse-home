Известные проблемы

iOS & macOS

Стриминг требует Tailscale — режим только LAN недоступен

App Transport Security iOS блокирует обычный HTTP к локальным IP. Tailscale необходим как для домашнего Wi-Fi, так и для удалённого стриминга. Режим только LAN без Tailscale в настоящее время не запланирован.

Android

Нет стриминга Mac/PC → Android

Android использует отдельную кодовую базу Flutter без интеграции с iCloud. Поддержка стриминга потребует почти полной переработки и не планируется.

Android

Доступ к сети SMB не проверен

Доступ к дискам SMB на Android не был полностью протестирован в последних релизах.
