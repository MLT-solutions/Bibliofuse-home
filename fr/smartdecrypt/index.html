Déverrouillage automatique basé sur modèle

Enregistrez une fois, déverrouillez toujours

Les relevés bancaires, PDF d'assurance et bulletins de paie arrivent toujours avec le même modèle de mot de passe — votre numéro d'identité, date de naissance ou numéro de compte. Enregistrez un profil une fois et SmartDecrypt choisit automatiquement le bon mot de passe chaque fois que ce type de fichier arrive.

  • Correspondance par modèle de nom de fichier — par exemple « DBS_Statement_* » → votre mot de passe NRIC
  • Profils séparés pour relevés bancaires, PDF d'assurance, factures, bulletins de paie
  • Cycle de mot de passe automatique — essaie tous les profils correspondants en séquence

FAQ

Questions fréquemment posées

Quels formats de fichiers SmartDecrypt prend-il en charge ?

PDF, ZIP et 7z. Les archives ZIP et 7z non chiffrées sont extraites normalement sans demander de mot de passe. SmartDecrypt ne prend actuellement pas en charge RAR, EPUB ou autres formats.

SmartDecrypt contourne-t-il les mots de passe inconnus ?

Non. SmartDecrypt est conçu pour les fichiers auxquels vous avez déjà accès. Il utilise les mots de passe que vous avez enregistrés dans un profil — il ne casse ni ne force les mots de passe inconnus.

Mes mots de passe sont-ils sûrs ?

Oui. Les mots de passe sont stockés exclusivement dans le Trousseau système sur votre appareil — le même stockage sécurisé utilisé par Safari et vos autres applications. SmartDecrypt ne stocke jamais les mots de passe dans sa propre base de données et rien n'est téléchargé sur un serveur.

Qu'est-ce qu'un modèle de nom de fichier ?

Un modèle de nom de fichier est un nom partiel ou un mot-clé qui identifie les fichiers récurrents. Par exemple, si votre banque envoie des relevés mensuels nommés « BankStatement_2024-01.pdf », créez un profil avec le modèle « BankStatement » et le mot de passe. SmartDecrypt associe automatiquement les fichiers futurs à ce profil.

L'achat Pro est-il partagé entre iPhone et Mac ?

Oui. SmartDecrypt Pro est un achat universel unique. Achetez une fois sur iOS ou macOS et déverrouillez Pro sur les deux plates-formes en utilisant le même Apple ID.
