Problèmes connus

iOS & macOS

Le streaming nécessite Tailscale — mode LAN uniquement non disponible

La sécurité de transport des applications iOS bloque le HTTP simple vers les IP locales. Tailscale est requis pour le Wi-Fi domestique et le streaming distant. Un mode LAN uniquement sans Tailscale n'est pas actuellement prévu.

Android

Pas de streaming Mac/PC → Android

Android utilise une base de code Flutter distincte sans intégration iCloud. La prise en charge du streaming nécessiterait une réécriture quasi totale et n'est pas prévue.

Android

Accès réseau SMB non vérifié

L'accès aux lecteurs SMB sur Android n'a pas été entièrement testé dans les versions récentes.
