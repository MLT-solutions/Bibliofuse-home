Comment lire des webtoons sur iPhone : défilement vertical infini dans BiblioFuse

Vous avez téléchargé un webtoon — un manhwa coréen, un webcomic en longue bande ou une série verticale traduite par des fans — et il est stocké sur votre iPhone sous forme de fichier CBZ ou EPUB. Chaque lecteur de BD que vous avez essayé l'affiche dans le mauvais sens, découpant des cases conçues pour s'enchaîner vers le bas. BiblioFuse gère précisément ce cas, avec un mode défilement vertical conçu pour ce format.

Ce qui distingue les webtoons des BD traditionnelles

Les BD traditionnelles organisent les cases en grille. Les webtoons sont différents : conçus pour la lecture mobile, leurs cases hautes et étroites s'empilent verticalement en une bande continue. Le format est né sur des plateformes comme LINE Webtoon et Naver, et s'est répandu aux webcomics auto-publiés et aux scans de manhwa.

Ce format pose problème aux applications conçues pour les BD classiques. Le mode page par page brise le rythme. La vue double page rend les cases minuscules. Même les applications supportant le défilement vertical négligent souvent des détails comme la sensibilité du glissement ou la persistance du niveau de zoom.

Le mode défilement vertical de BiblioFuse

BiblioFuse pour iPhone et iPad inclut un mode défilement vertical dédié. Au lieu de traiter chaque image comme une page, il les juxtapose de façon continue pour que vous fassiez défiler tout le chapitre sans taper ni swiper. Fonctionne avec les webtoons CBZ et EPUB.

Pour passer en mode de lecture verticale :

  1. Ouvrez n'importe quel CBZ ou EPUB dans BiblioFuse.
  2. Appuyez au centre de l'écran pour afficher la barre d'outils du lecteur.
  3. Appuyez sur l'icône Disposition (deux rectangles superposés).
  4. Choisissez Défilement vertical.

Le lecteur réorganise le contenu immédiatement. Les images occupent toute la largeur de l'écran, et le défilement vers le bas avance dans les pages sans coupure de pagination.

Espace entre les images

Certains webtoons sont composés d'une seule image continue par chapitre. D'autres sont découpés en images individuelles. BiblioFuse vous laisse contrôler l'espace entre elles :

  • Sans espace : les images s'enchaînent parfaitement, donnant l'impression d'une seule image continue.
  • Espace minimal : un mince séparateur apparaît entre les pages, utile quand les pages ont un fond blanc.

Pour la plupart des webtoons, le mode sans espace offre l'expérience la plus authentique.

Importer des fichiers webtoon

CBZ est le conteneur le plus courant pour les scanlations. C'est simplement un ZIP contenant des images PNG ou JPG. BiblioFuse lit les CBZ nativement sur iPhone et iPad.

EPUB est courant pour les webtoons sous licence officielle. BiblioFuse lit les EPUB à mise en page fixe.

Méthodes de transfert sur iPhone :

  • Transfert Wi-Fi : ouvrez BiblioFuse, allez dans l'onglet Transfert, un serveur web local démarre. Visitez l'adresse affichée depuis n'importe quel navigateur sur le même réseau.
  • Application Fichiers : partagez un CBZ ou EPUB depuis Fichiers iOS vers BiblioFuse via le menu Partager.

Zoom et panoramique

Un double appui effectue un zoom sur une zone ; un second double appui revient à la vue complète. Le zoom persiste entre les pages. Le panoramique horizontal en mode zoom est également pris en charge.

Lecture de longues séries

Une série manhwa typique compte entre 100 et 300+ chapitres. BiblioFuse organise les chapitres comme des fichiers individuels dans un dossier. À la fin d'un chapitre, un bouton Suivant apparaît pour ouvrir le suivant sans revenir à la bibliothèque.

La progression de lecture se synchronise via iCloud sur plusieurs appareils Apple.

À retenir

Les webtoons nécessitent un lecteur qui comprend le format de bande verticale. Le mode défilement vertical de BiblioFuse gère les webtoons CBZ et EPUB nativement sur iPhone, avec des espaces configurables, un zoom persistant et une navigation fluide entre chapitres. Téléchargez BiblioFuse sur l'App Store pour commencer.
