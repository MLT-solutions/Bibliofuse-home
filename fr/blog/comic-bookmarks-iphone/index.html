Utiliser les signets dans BiblioFuse : ne perdez plus votre place dans un comics

BiblioFuse vous permet de placer des signets n'importe où dans un comics ou un ebook sur iPhone — pas seulement aux débuts de chapitre, mais sur n'importe quelle page. Les signets se synchronisent entre iPhone, iPad et Mac via iCloud, vous retrouvez donc vos positions enregistrées quel que soit l'appareil que vous utilisez.

Le problème : reprendre là où vous vous êtes arrêté

Vous êtes à la page 80 d'un volume de Saga. Votre téléphone s'éteint. Plus tard, vous ouvrez l'app sur votre iPad et le volume s'ouvre depuis le début. Un signet posé au bon moment vous aurait renvoyé exactement là où vous étiez.

Ce scénario se répète constamment chez les lecteurs qui alternent les appareils ou qui relisent de longues séries des années plus tard. Sans système de signets fiable, vous devez retenir votre position mentalement.

Comment fonctionnent les signets dans BiblioFuse

Le système de signets est conçu pour la rapidité : l'action la plus courante, poser un signet, ne demande que deux appuis. Les signets sont stockés par volume avec le numéro de page et un horodatage, et synchronisés via iCloud.

Quand vous ouvrez un volume avec un signet, BiblioFuse peut reprendre automatiquement à votre position de lecture. Les signets explicites vous permettent de marquer plusieurs points : une scène préférée, un cliffhanger à revoir, le début d'un nouvel arc narratif.

Téléchargez BiblioFuse pour iPhone pour profiter de l'expérience complète de signets et de synchronisation de lecture.

Comment ajouter un signet dans BiblioFuse

Étape 1 : Ouvrez un comics dans le lecteur

Lancez BiblioFuse et ouvrez n'importe quel comics, manga ou ebook. Naviguez jusqu'à la page que vous souhaitez marquer.

Étape 2 : Appuyez sur l'écran pour afficher les contrôles

Appuyez une fois au centre de la zone de lecture. Les contrôles apparaissent : une barre de navigation en haut et un curseur de page en bas. Repérez l'icône Signet dans la barre de navigation supérieure.

Étape 3 : Appuyez sur l'icône Signet

Appuyez sur l'icône de signet. BiblioFuse enregistre un signet pour la page actuelle. L'icône se remplit pour indiquer qu'un signet est actif sur cette page. Appuyez à nouveau pour le supprimer si vous l'avez placé par erreur.

Étape 4 : Accédez à vos signets

Appuyez sur le bouton Liste des signets pour ouvrir votre liste de signets pour ce volume. Chaque entrée affiche le numéro de page et la date d'enregistrement. Appuyez sur un signet pour y accéder directement.

Étape 5 : Naviguez entre les signets

Dans la liste des signets, appuyez sur n'importe quelle entrée pour revenir à cette page exacte. Vous pouvez supprimer un signet en faisant glisser l'entrée vers la gauche.

Signets selon votre style de lecture

Séance de lecture unique : Posez un signet quand vous arrêtez de lire. La prochaine fois que vous ouvrirez le volume, votre position enregistrée sera la première entrée.

Suivi des arcs narratifs : Les longs mangas comportent souvent plusieurs arcs dans un seul volume. Marquez la première page de chaque arc pour y revenir facilement.

Marquer des planches favorites : Les signets ne servent pas qu'à suivre votre progression. Utilisez-les pour sauvegarder des doubles pages spectaculaires ou des moments forts.

Signets vs. progression de lecture

BiblioFuse suit deux types de données de position. La progression de lecture est automatique : l'app enregistre la dernière page consultée à chaque fermeture du volume. Les signets sont intentionnels : vous les placez manuellement pour marquer des pages auxquelles vous souhaitez revenir.

Les deux se synchronisent via iCloud.

Synchronisation iCloud en pratique

Les signets se synchronisent via iCloud Drive dans le conteneur privé de BiblioFuse, sans compte BiblioFuse ni connexion externe. Cela fonctionne même hors ligne : BiblioFuse enregistre les signets localement et les synchronise avec iCloud dès la reconnexion.

Questions fréquentes

Les signets se synchronisent-ils sans connexion ? Oui. BiblioFuse enregistre les signets localement et les synchronise avec iCloud lors de la reconnexion.

Combien de signets puis-je ajouter à un volume ? Il n'y a pas de limite pratique.

Les signets survivront-ils à une désinstallation de BiblioFuse ? Oui, à condition qu'ils aient été synchronisés avec iCloud avant la désinstallation.

Les signets fonctionnent-ils avec les fichiers EPUB ? Oui. Les signets fonctionnent avec tous les formats pris en charge : CBZ, CBR, EPUB, PDF, ZIP, RAR et TXT.
