# Как извлечь изображения из CBZ-файлов онлайн (бесплатно, без программ)

Веб-инструмент BiblioFuse позволяет извлекать отдельные изображения из CBZ-файлов прямо в браузере — без установки программ, без загрузки файлов на сервер, без необходимости создавать аккаунт. Обработка выполняется полностью в WebAssembly в вашей вкладке, поэтому ваши комиксы остаются приватными.

## Когда нужны изображения из CBZ

CBZ-файл — это ZIP-архив с последовательно пронумерованными изображениями, обычно JPEG или PNG-страницами отсканированного комикса или манги. Вам могут понадобиться исходные изображения в таких ситуациях:

- **Печать конкретной страницы:** принтеру нужен файл изображения, а не архив комикса
- **Конвертация в другой формат:** некоторые конвертеры принимают изображения как входные данные
- **Создание фан-арта или справочных листов:** нужны оригинальные файлы в полном разрешении
- **Архивирование сканов:** извлечение позволяет проверить качество и метаданные
- **Переупаковка:** разделить большой CBZ на файлы поменьше по сюжетным аркам

## Как работает веб-инструмент BiblioFuse

[BiblioFuse Web](https://bibliofuse.com/ru/webapp/) использует WebAssembly (WASM) для выполнения логики обработки прямо в вашей вкладке браузера. Ваши файлы никогда не отправляются на сервер.

Тот же подход используется для всех веб-инструментов BiblioFuse: конвертация PDF в CBZ, объединение CBZ, сжатие EPUB и теперь извлечение изображений.

## Пошаговое руководство: извлечение изображений из CBZ онлайн

### Шаг 1: Откройте веб-инструмент BiblioFuse

Перейдите в [веб-приложение BiblioFuse](https://bibliofuse.com/ru/webapp/) в любом современном браузере. Без входа в систему, без расширений, без регистрации.

### Шаг 2: Выберите экстрактор CBZ

Найдите инструмент **Извлечь изображения из CBZ** и нажмите, чтобы открыть интерфейс.

### Шаг 3: Загрузите CBZ-файл

Нажмите **Выбрать файл** или перетащите CBZ-файл в область загрузки. Файл читается с вашего диска без сетевых запросов.

### Шаг 4: Выберите параметры извлечения

Экстрактор предлагает несколько вариантов:
- **Все изображения:** извлекает все файлы изображений из архива
- **Диапазон страниц:** извлечь только указанные страницы
- **Фильтр формата:** только JPEG или только PNG

### Шаг 5: Извлеките и скачайте

Нажмите **Извлечь**. Движок WASM распаковывает CBZ и упаковывает изображения в ZIP-архив. Браузер автоматически запускает загрузку.

### Шаг 6: Распакуйте и используйте изображения

Откройте загруженный ZIP-файл с помощью любого архиватора. Вы найдёте изображения с оригинальными именами в порядке чтения.

## CBZ vs. CBR

CBR-файлы используют формат RAR. Если у вас CBR, сначала конвертируйте его в CBZ (также доступно в веб-приложении BiblioFuse), затем извлеките изображения.

## Конфиденциальность и безопасность

Поскольку экстрактор работает полностью в браузере, никакой сервер не получает и не хранит ваши файлы:
- Личные коллекции никогда не покидают устройство
- Аккаунт не требуется
- Нет ограничений на размер файла

## Часто задаваемые вопросы

**Инструмент действительно бесплатный?**
Да. Все веб-инструменты BiblioFuse бесплатны без ограничений.

**Какие форматы изображений поддерживаются?**
JPEG, PNG, WebP, GIF, BMP и TIFF.

**Можно ли извлечь изображения из защищённых паролем CBZ?**
Сначала снимите защиту паролем с помощью SmartDecrypt, затем извлеките изображения.

**Насколько большой CBZ-файл можно обработать?**
Нет принудительного ограничения — ограничивает только доступная память браузера.

**Будут ли извлечённые изображения в порядке чтения?**
Да. Экстрактор сохраняет оригинальные имена файлов, определяющие порядок чтения.
