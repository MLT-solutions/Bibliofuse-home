# Как Открыть PDF-Выписки Банка с Паролем на iPhone

Каждый месяц банк присылает PDF. Он защищён паролем — ваш ИНН, дата рождения или последние шесть цифр номера счёта. Вы вводили это раньше, но не так недавно, чтобы помнить наизусть.

[SmartDecrypt PDF ZIP](/ru/smartdecrypt/) решает эту проблему: сохраните шаблон пароля один раз для каждого банка или учреждения. Каждая следующая выписка из этого источника разблокируется автоматически, когда вы её добавляете.

## Настройка SmartDecrypt для Банковских Выписок

**Шаг 1 — Создайте профиль для каждого учреждения**

Откройте SmartDecrypt и нажмите +, чтобы добавить новый профиль. Укажите название, шаблон имени файла (например, `Sberbank_*.pdf`) и пароль для этого учреждения.

**Шаг 2 — Добавьте файл**

Когда приходит ежемесячная выписка, нажмите «Поделиться» и выберите SmartDecrypt. Приложение сравнивает имя файла с сохранёнными профилями, применяет сохранённый пароль и разблокирует PDF.

**Шаг 3 — Несколько банков, несколько профилей**

- `Sberbank_*.pdf` → ваш СНИЛС
- `VTB_*.pdf` → ваша дата рождения
- `Rosgosstrah_*.pdf` → номер полиса

## Безопасное Хранение

Пароли хранятся в Apple Keychain. Они никогда не отправляются на сервер.

SmartDecrypt бесплатен для разблокировки одного файла за раз. SmartDecrypt Pro добавляет пакетную обработку с единовременной покупкой. Без подписки.
