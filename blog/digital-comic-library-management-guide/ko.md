# iPhone과 Mac에서 디지털 만화 라이브러리를 관리하는 완전 가이드

BiblioFuse는 iOS 17 이상을 지원하는 iPhone과 iPad용 네이티브 만화 및 전자책 리더로, CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT 파일을 읽을 수 있습니다. 태그, 평점, 컬렉션 기능으로 디지털 만화 라이브러리를 관리하며, Wi-Fi 전송으로 케이블이나 iTunes 없이 파일을 무선 전송합니다. 50권이든 5,000권이든 BiblioFuse 하나로 iPhone과 Mac을 연결하는 정돈된 라이브러리를 구축할 수 있습니다.

## 디지털 만화 라이브러리 관리가 생각보다 어려운 이유

Mac의 폴더 하나로 시작했습니다. 그러다 몇 가지 다운로드가 iPad로 넘어갔습니다. 어떤 파일은 CBZ 형식이고, 또 다른 것들은 ZIP 안에 흩어진 이미지 파일이며, EPUB 그래픽 노블 백로그는 또 전혀 다른 곳에 있습니다. 어느새 네 개의 불완전한 라이브러리가 생겨나고, 어떤 권을 읽었는지도 모르게 됩니다.

이것이 진지한 만화 독자가 매일 마주하는 현실입니다. 디지털 컬렉션은 빠르게 늘어나고, 적절한 도구가 없으면 디지털화의 모든 장점을 무색하게 만드는 혼란에 빠지게 됩니다.

## 파일 앱과 클라우드 드라이브가 부족한 이유

iPhone의 파일 앱은 파일 브라우저이지 독서 플랫폼이 아닙니다. 읽기 진행 상황의 개념이 없고, 스토리 아크나 시리즈별로 이슈에 태그를 달 방법도 없으며, 파일 이름 변경이나 압축을 위한 일괄 작업도 없습니다. iCloud Drive와 Google Drive는 파일을 동기화하지만 두 서비스 모두 업로드 시 이미지를 재압축해 아트워크 품질을 저하시킵니다. 어느 쪽도 만화 메타데이터를 이해하거나 기기 간에 '읽음' 상태를 유지하지 못합니다.

USB 케이블로 iPhone을 컴퓨터에 연결해야 하는 만화 리더는 구시대 유물입니다. 컬렉션을 서드파티 클라우드 서버에 업로드하도록 강요하는 앱도 마찬가지입니다. 현대적인 워크플로우는 파일을 로컬에 저장하고 개인 정보를 보호하며 빠르게 접근할 수 있어야 합니다.

## 만화 라이브러리를 단계적으로 정리하는 방법

iPhone과 Mac에서 깔끔하고 이동 가능하며 검색 가능한 만화 라이브러리를 구축하려면 다음 단계를 따르세요.

**1. 각 파일 유형에 맞는 형식을 선택하세요.**
스캔한 만화, 망가, 주로 이미지로 구성된 콘텐츠에는 CBZ를 사용하세요. CBZ는 이미지 파일의 이름만 바꾼 ZIP 아카이브로 만화 세계에서 가장 호환성 높은 형식이며, [BiblioFuse](/ko/)는 이를 기본으로 지원합니다. 리플로우 가능한 텍스트가 있는 그래픽 노블에는 EPUB을 사용하세요. BiblioFuse는 CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT를 읽습니다.

**2. 전송 전에 너무 큰 파일을 압축하고 크기를 줄이세요.**
고해상도 스캔은 이슈당 200–400MB를 쉽게 넘깁니다. [BiblioFuse 웹 도구](/ko/webapp/)는 CBZ, EPUB, PDF 파일을 위한 무료 브라우저 기반 일괄 압축 도구입니다. WebAssembly를 통해 브라우저에서 완전히 실행되며 서버에 아무것도 업로드되지 않습니다. 너무 큰 이슈들을 한꺼번에 드래그하고 목표 해상도를 선택한 뒤 전송 준비가 된 압축 버전을 다운로드하세요.

**3. 파일 이름을 일관되게 지정하세요.**
`시리즈 이름 v01 #001 (년도).cbz`와 같은 명명 규칙을 사용하면 정렬과 검색이 훨씬 쉬워집니다. 일관된 이름은 BiblioFuse 검색이 찾는 것을 즉시 발견하게 해주고 Mac 폴더도 사람이 읽기 쉬운 상태로 유지됩니다.

**4. Wi-Fi 전송으로 iPhone에 만화를 전송하세요.**
iPhone에서 BiblioFuse를 열고 Wi-Fi 전송 버튼을 탭합니다. 앱에 로컬 URL(예: `http://192.168.1.5:8080`)이 표시됩니다. Mac이나 PC의 브라우저에서 해당 URL을 열고 CBZ 또는 EPUB 파일을 브라우저 창에 직접 드래그 앤 드롭합니다. 케이블, iTunes, 클라우드 중개자 없이 수 초 안에 iPhone에 파일이 도착합니다.

**5. BiblioFuse에서 파일에 태그와 평점을 추가하세요.**
전송 후 태그 기능을 사용해 시리즈, 아크 또는 읽기 상태(예: "사가", "읽을 것", "완료")별로 이슈를 그룹화하세요. 다시 읽고 싶은 이슈에는 별점 평가를 부여하세요. 태그와 평점은 iCloud에 저장되므로 로그인한 모든 기기에 나타납니다.

**6. Mac Home Library로 Mac 라이브러리를 스트리밍하세요.**
전체 컬렉션이 Mac에 연결된 외장 드라이브에 있다면 모두 iPhone에 복사할 필요가 없습니다. Mac Home Library를 사용하면 iPhone의 BiblioFuse가 로컬 Wi-Fi를 통해 Mac 파일 시스템에서 직접 탐색하고 스트리밍할 수 있습니다. 600MB 이슈도 폰에 다운로드하지 않고 열 수 있습니다—필요 시 스트리밍됩니다.

**7. iCloud로 기기 간 읽기 진행 상황을 동기화하세요.**
BiblioFuse는 iCloud를 사용해 마지막으로 읽은 페이지, 태그, 평점을 iPhone, iPad, 향후 세션 간에 일관되게 유지합니다. 출퇴근 중 iPhone으로 읽기 시작하고 집에서 iPad로 정확히 멈춘 곳부터 이어 읽으세요.

## 케이블 없이 iPhone에 만화를 전송하는 방법은?

Wi-Fi 전송은 만화 리더 앱에서 가장 많이 요청되는 기능 중 하나이며, BiblioFuse에 기본 내장되어 있습니다. 정확한 사용 방법은 다음과 같습니다:

1. iPhone과 Mac(또는 PC)을 동일한 Wi-Fi 네트워크에 연결합니다.
2. iPhone에서 BiblioFuse를 열고 라이브러리 탭으로 이동합니다.
3. Wi-Fi 전송 아이콘을 탭합니다. 앱에 로컬 IP 주소와 포트가 표시됩니다.
4. Mac에서 Safari, Chrome 또는 아무 브라우저나 열고 해당 주소를 입력합니다.
5. 브라우저에 BiblioFuse 웹 인터페이스가 열립니다. Finder에서 CBZ, CBR, EPUB, PDF 또는 ZIP 파일을 브라우저 창에 직접 드래그합니다.
6. 파일이 Wi-Fi를 통해 전송되어 BiblioFuse 라이브러리에 즉시 나타납니다.

전송은 직접적이며 로컬 네트워크에서 기기 간으로 이루어져 클라우드 서비스가 관여하지 않습니다.

## iPhone에서 수백 권의 만화를 관리하는 방법은?

태그는 대형 라이브러리를 압도당하지 않고 관리하는 핵심입니다. 팔로우하는 각 시리즈별로 태그를 만들고 "현재 읽는 중"과 "완료" 태그도 추가하세요. 수 초 안에 태그 조합으로 라이브러리를 필터링할 수 있습니다. BiblioFuse에는 파일 이름을 검색하는 내장 검색 기능도 있어 특정 이슈 번호나 제목으로 바로 이동하기 쉽습니다.

수천 권의 매우 큰 컬렉션의 경우, Mac Home Library 기능 덕분에 모든 것을 iPhone에 한꺼번에 저장할 필요가 없습니다. 전체 아카이브를 Mac이나 NAS 드라이브에 보관하고 읽고 싶을 때 개별 이슈를 스트리밍하세요. 로컬에 다운로드가 필요한 것은 현재 읽기 목록뿐입니다.

## 자주 묻는 질문

**iPhone에서 만화를 관리하기에 가장 좋은 앱은 무엇인가요?**
BiblioFuse는 모든 주요 형식(CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT)을 지원하고 케이블 없이 무선 전송을 지원하며 iCloud를 통해 읽기 진행 상황을 동기화하기 때문에 iPhone 만화 라이브러리 관리에 강력한 선택입니다. iOS 17 이상에서 기본 기능 이용 시 구독 없이 작동합니다.

**Mac의 만화 파일을 iPhone과 동기화하려면 어떻게 정리해야 하나요?**
Mac 폴더 구조를 깔끔하게 유지하고 일관된 파일 이름을 사용하세요. Mac에서 파일 이름이 정리되면 BiblioFuse의 Wi-Fi 전송으로 iPhone에 일괄 이동하거나 Mac Home Library를 사용해 파일을 복사하지 않고 Mac 폴더를 직접 스트리밍하세요.

**CBZ 파일을 iPhone에서 변환 없이 읽을 수 있나요?**
네. BiblioFuse는 iPhone에서 CBZ 파일을 기본으로 읽습니다—변환이 필요 없습니다. Wi-Fi 전송으로 파일을 전송하면 리더에서 바로 열립니다. CBR, ZIP, RAR 아카이브도 마찬가지입니다.

**PC에서 iPhone으로 만화 파일을 전송하는 방법은?**
Mac과 동일한 과정입니다. PC와 iPhone을 동일한 Wi-Fi 네트워크에 연결하고 BiblioFuse에서 Wi-Fi 전송을 시작한 뒤 Windows 브라우저(Edge, Chrome, Firefox)에서 표시된 URL을 열고 파일을 브라우저 창에 드래그하면 됩니다. iTunes나 Windows 소프트웨어는 필요 없습니다.

**BiblioFuse가 iPhone과 iPad 간에 읽기 진행 상황을 동기화하나요?**
네. BiblioFuse는 iCloud를 사용해 모든 기기에서 자동으로 마지막으로 읽은 페이지, 태그, 평점을 동기화합니다. iCloud가 활성화된 한 진행 상황은 어디서나 일관되게 유지됩니다.

**BiblioFuse가 iPhone에서 관리할 수 있는 형식은 무엇인가요?**
BiblioFuse는 iPhone에서 CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT 파일을 관리합니다. 이 모든 형식을 하나의 라이브러리 안에서 열고, 읽고, 태그 달고, 평가하고, 정리할 수 있습니다.
