# PDFをCBZにオンラインで変換する方法（ソフト不要・アップロード不要）

スキャンしたグラフィックノベルをPDFでダウンロードしたが、コミックリーダーで読むとページ読み込みが遅く、ズームがぼやけ、ページめくりがもたつく。CBZのほうがコミックに向いているとわかっていても、新たなデスクトップアプリをインストールしたくないし、ファイルを知らないサイトに送るのも嫌だ。

[BiblioFuse Web Tool](/ja/webapp/)は、これをブラウザだけで解決する。アカウント不要。アップロード不要。インストール不要。ファイルはデバイスの外に出ない。

## なぜPDFはコミックに向かないのか

PDFは印刷用の文書レイアウト向けに設計されており、フォント・ベクターグラフィック・テキストのリフローを得意とする。コミックには向いていない。スキャンコミックは1ページごとの画像の連続だが、PDFはそれらを余分な構造で包んでファイルサイズを増やし、専用ビューアより遅いレンダラーを使い、先読みキャッシュも妨げる。

CBZは番号付き画像ファイルを収めたZIPアーカイブだ。オーバーヘッドがほとんどなく、リーダーは事前に画像を展開してメモリに保持し、ページを即座にめくれる。

## BiblioFuse Web Toolの仕組み

[BiblioFuse Web Tool](/ja/webapp/)はWebAssemblyを使ってブラウザ内で完全に動作する。ファイルはローカルディスクからブラウザのメモリへ直接読み込まれ、変換はブラウザのタブ内で実行され、出力ファイルはディスクに書き戻される。いかなる時点でもサーバーを経由しない。

## 変換手順

**ステップ1：ツールを開く**

[BiblioFuse Web Tool](/ja/webapp/)をデスクトップブラウザ（Chrome・Firefox・Safari・Edge）で開く。ログイン不要。無料プランは一度に3ファイルまで処理できる。

**ステップ2：PDFファイルを追加する**

ドロップゾーンをクリックするか、PDFを直接ドラッグする。複数ファイルを一度に追加可能。

**ステップ3：CBZを選んで変換する**

出力フォーマットにCBZを選び「変換」をクリックする。各ファイルが順次処理され、完了したものからダウンロードできる。ファイル名は元のままで、`.pdf` が `.cbz` に変わる。

## 画質について

**スキャンPDF：** 画像を再エンコードなしで取り出すため画質は完全に保たれ、ファイルサイズもほぼ変わらない。

**ベクター/レイアウトPDF：** 各ページをラスタライズして出力。デフォルトで高解像度を使用するため通常の閲覧ズームでは十分に鮮明。

**圧縮済みPDF：** 元のPDF作成時のJPEG圧縮がそのまま残る。追加圧縮はしないが、既存の圧縮は取り除けない。

## シリーズの一括変換と端末への転送

複数巻を変換するには全PDFを一度に追加すればよい。無料プランは1バッチ最大3ファイルで、同一セッション内でバッチを繰り返せる。

変換後のCBZをiPhoneに転送するには、BiblioFuseの[Wi-Fi転送](/ja/)が便利だ。同じネットワーク上のPCからブラウザでドラッグするだけでUSBもiTunesも不要。iCloud DriveやAirDropも使える。

## よくある質問

**OCRテキスト層が含まれるPDFは？** 視覚的なページ画像を抽出し、元のPDFは変更されない。

**ファイルサイズの上限は？** 明示的な上限はなく、ブラウザの利用可能メモリに依存する。

**モバイルブラウザでも使える？** 使えるが速度は遅い。大きなPDFにはデスクトップを推奨。

**CBZ内の画像フォーマットは？** 元のPDFに保存されているフォーマットのまま。変換中のトランスコードはない。

変換したいPDFコミックがあれば、今すぐ[BiblioFuse Web Tool](/ja/webapp/)を開いて最初のバッチをドロップしてみよう。
