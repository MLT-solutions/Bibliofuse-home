# Como ler webtoons no iPhone: rolagem vertical infinita no BiblioFuse

Você baixou um webtoon — um manhwa coreano, um webcomic de tira longa ou uma série vertical com tradução de fãs — e ele está no seu iPhone como arquivo CBZ ou EPUB. Todo leitor de quadrinhos que você tentou o exibe na orientação errada, dividindo painéis que foram feitos para fluir para baixo. O [BiblioFuse](https://bibliofuse.com/pt/) trata esse caso diretamente, com um modo de rolagem vertical criado especificamente para esse formato.

## O que torna os webtoons diferentes

Quadrinhos tradicionais organizam painéis em grade. Webtoons são diferentes: foram criados para leitura mobile, com painéis altos e estreitos que se empilham verticalmente em uma tira contínua. O formato surgiu em plataformas como LINE Webtoon e Naver, e se expandiu para webcomics autopublicados e scans de manhwa.

Esse formato vertical cria um problema para apps criados para quadrinhos tradicionais. O modo de virar páginas quebra o fluxo. A visualização de página dupla torna os painéis minúsculos. Até apps que suportam rolagem vertical frequentemente perdem detalhes como sensibilidade ao deslizamento e persistência do nível de zoom.

## O modo de rolagem vertical do BiblioFuse

O [BiblioFuse para iPhone e iPad](https://bibliofuse.com/pt/) inclui um modo de rolagem vertical dedicado. Em vez de tratar cada imagem como uma página, ele as organiza de forma contínua para que você role para baixo por todo o capítulo sem toques ou deslizamentos. Funciona tanto para webtoons CBZ quanto EPUB.

Para mudar o modo de leitura:
1. Abra qualquer CBZ ou EPUB no BiblioFuse.
2. Toque no centro da tela para exibir a barra de ferramentas do leitor.
3. Toque no ícone de **Layout** (dois retângulos empilhados).
4. Escolha **Rolagem vertical**.

O leitor reformata o conteúdo imediatamente. As imagens preenchem a largura da tela e rolar para baixo avança pelas páginas sem interrupções de paginação.

## Espaço entre imagens

Alguns webtoons são compostos como uma única imagem contínua por capítulo. Outros são divididos em imagens individuais. Quando as imagens são arquivos separados, o BiblioFuse permite controlar o espaço entre elas:

- **Sem espaço:** as imagens se encaixam perfeitamente, fazendo o capítulo parecer uma única imagem contínua.
- **Espaço mínimo:** um separador fino aparece entre páginas, útil quando têm fundos brancos.

Para a maioria dos webtoons, o modo sem espaço oferece a experiência de leitura mais autêntica.

## Importando arquivos de webtoons

**Arquivos CBZ** são o container mais comum para scanlations. Um CBZ é apenas um ZIP contendo imagens PNG ou JPG. O BiblioFuse lê CBZ nativamente no iPhone e iPad.

**Arquivos EPUB** são comuns para webtoons com licença oficial. O BiblioFuse lê EPUBs de layout fixo.

Métodos de transferência para o iPhone:
- **Transferência Wi-Fi:** abra o BiblioFuse, vá para a aba **Transferência** e um servidor web local inicia. Visite o endereço exibido de qualquer navegador na mesma rede.
- **App Arquivos:** compartilhe qualquer CBZ ou EPUB do app Arquivos do iOS para o BiblioFuse usando o menu compartilhar.

## Zoom e panorâmica

Dois toques fazem zoom em uma região; dois toques novamente voltam para a largura completa. O zoom de rolagem vertical persiste entre páginas. O movimento horizontal com zoom também é suportado.

## Lendo séries longas

Uma série manhwa típica tem de 100 a 300+ capítulos. O BiblioFuse organiza os capítulos como arquivos individuais dentro de uma pasta. Ao terminar um capítulo, um botão **Próximo** aparece para abrir o seguinte sem voltar à biblioteca.

O progresso de leitura sincroniza via iCloud em vários dispositivos Apple.

## Conclusão

Webtoons precisam de um leitor que entenda o formato de tira vertical. O modo de rolagem vertical do BiblioFuse trata webtoons CBZ e EPUB nativamente no iPhone, com espaços configuráveis, zoom persistente e navegação fluida entre capítulos.
