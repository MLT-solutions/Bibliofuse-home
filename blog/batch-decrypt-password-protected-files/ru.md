## Проблема расшифровки файлов по одному

У вас есть 50 банковских выписок — все PDF с паролем одного банка. Или папка из 30 CBZ-комиксов с фанатским переводом от одного издателя, все с одним паролем архива. Открытие каждого файла по отдельности — импорт, ввод пароля, расшифровка, сохранение — занимает несколько минут на файл. Для больших коллекций это часы однообразной работы.

Большинство инструментов расшифровки на iPhone обрабатывают файлы по одному. В них нет понятия пакетных операций или многократно используемых профилей паролей.

SmartDecrypt PDF ZIP CBZ решает эту проблему с помощью двух функций: **пакетной расшифровки** и **профилей паролей**. Выберите все файлы сразу, примените профиль и нажмите один раз.

[SmartDecrypt](/ru/smartdecrypt/) обрабатывает всё на устройстве — без облачного сервиса, без ожидания для каждого файла, без загрузок.

## Как работает пакетная расшифровка в SmartDecrypt

### Шаг 1: Создайте профиль пароля (рекомендуется)

Перед пакетной расшифровкой сохраните профиль пароля для коллекции. В SmartDecrypt перейдите в раздел «Профили», нажмите «Добавить профиль», введите имя (например, «Банковские выписки 2025») и укажите пароль. Профиль надёжно хранится на устройстве и никогда не синхронизируется с облаком.

### Шаг 2: Импортируйте несколько файлов

Откройте SmartDecrypt и нажмите «Импорт». Выберите несколько файлов в браузере «Файлов». Также можно выбрать несколько файлов в приложении «Файлы» iOS и поделиться ими всеми с SmartDecrypt сразу.

### Шаг 3: Примените пароль

Когда все файлы загружены, введите пароль один раз или выберите сохранённый профиль. Один и тот же пароль применяется ко всем файлам пакета.

### Шаг 4: Нажмите «Расшифровать»

SmartDecrypt обрабатывает каждый файл последовательно. Для типичных архивов каждый файл завершается менее чем за секунду. Пакет из 50 банковских выписок обрабатывается менее чем за минуту.

### Шаг 5: Получите доступ к расшифрованным файлам

Все расшифрованные файлы сохраняются в приложении «Файлы». Открывайте CBZ-файлы в [BiblioFuse](/ru/), импортируйте PDF в приложения для работы с документами или делитесь ими по мере необходимости.

## Когда использовать пакетную расшифровку

### Годовые финансовые документы

Банки и учреждения присылают выписки, налоговые формы и сводки по счетам в виде защищённых PDF. Накопление за год ежемесячных выписок означает 12 одинаковых операций расшифровки. Пакетная расшифровка обрабатывает все 12 одним нажатием.

### Импорт библиотеки комиксов

Если вы формируете цифровую библиотеку комиксов на iPhone с помощью BiblioFuse, а ваши комиксы пришли как защищённые CBZ-архивы, пакетная расшифровка — самый быстрый путь. Импортируйте все архивы сразу, расшифруйте за одну операцию.

### ZIP-резервные копии и коллекции документов

ZIP-архивы из экспортов приложений или личных резервных копий часто имеют защиту паролем. Пакетная расшифровка снимает пароли с целой папки файлов за один раз.

## Профили паролей: сохрани один раз, используй всегда

Хранилище профилей паролей делает пакетную расшифровку практичной. Большинство реальных коллекций предполагают один пароль для многих файлов. Профили хранятся надёжно на устройстве и никогда не синхронизируются с облачным сервисом.
