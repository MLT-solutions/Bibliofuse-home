# 비밀번호로 보호된 여러 파일을 한 번에 일괄 복호화하는 방법

SmartDecrypt PDF ZIP CBZ는 iPhone, iPad, Mac에서 비밀번호로 보호된 여러 PDF, CBZ, ZIP 파일을 한 번에 복호화합니다. 보호된 아카이브를 원하는 수만큼 선택하고 하나의 비밀번호나 저장된 비밀번호 프로필을 적용하면 앱이 업로드 없이, 파일별 대기 없이 모든 파일을 기기에서 처리합니다. 복호화 후 [BiblioFuse](/ko/)에서 CBZ 만화를 열어 iPhone에서 최상의 독서 경험을 즐기세요. SmartDecrypt의 일괄 복호화는 보호된 파일 컬렉션 전체를 처리하는 가장 빠른 방법입니다.

## 파일별 복호화의 문제점

같은 기관에서 온 비밀번호로 보호된 PDF 50개의 은행 명세서가 있습니다. 또는 같은 출판사의 팬 번역 CBZ 만화 30개 폴더에 모두 같은 아카이브 비밀번호가 있습니다. 각 파일을 개별적으로 열려면 — 가져오기, 비밀번호 입력, 복호화, 저장 — 파일당 몇 분이 걸립니다. 대용량 컬렉션은 수 시간의 반복 작업이 됩니다.

iPhone의 대부분의 복호화 도구는 파일을 하나씩 처리합니다. 일괄 작업이나 재사용 가능한 비밀번호 프로필 개념이 없습니다.

SmartDecrypt PDF ZIP CBZ는 두 가지 기능의 조합으로 이 문제를 해결합니다: **일괄 복호화**와 **비밀번호 프로필**. 모든 파일을 한 번에 선택하고 프로필을 적용한 다음 한 번 탭하면 됩니다.

[SmartDecrypt](/ko/)는 모든 것을 기기에서 처리합니다 — 클라우드 서비스 없음, 파일별 대기 없음, 업로드 없음.

## SmartDecrypt에서 일괄 복호화 작동 방식

### 1단계: 비밀번호 프로필 만들기 (권장)

일괄 복호화 전에 컬렉션의 비밀번호 프로필을 저장합니다. SmartDecrypt에서 프로필로 이동하고 프로필 추가를 탭한 다음 이름(예: "은행 명세서 2025")을 입력하고 비밀번호를 입력합니다. 프로필은 기기에 안전하게 저장되며 클라우드와 동기화되지 않습니다.

### 2단계: 여러 파일 가져오기

SmartDecrypt를 열고 가져오기를 탭합니다. 파일 브라우저에서 여러 파일을 선택합니다. iOS 파일 앱에서 여러 파일을 선택하고 공유 시트를 통해 모두 SmartDecrypt에 한 번에 공유할 수도 있습니다.

### 3단계: 비밀번호 적용

모든 파일이 로드된 상태에서 비밀번호를 한 번 입력하거나 저장된 프로필을 선택합니다. 같은 비밀번호가 배치의 모든 파일에 적용됩니다.

### 4단계: 복호화 탭

SmartDecrypt가 각 파일을 순서대로 처리합니다. 일반적인 아카이브의 경우 현대 iPhone에서 각 파일이 1초 미만으로 완료됩니다. 50개 은행 명세서 배치가 1분 이내에 완료됩니다.

### 5단계: 복호화된 파일 접근

모든 복호화된 파일이 파일 앱에 저장됩니다. [BiblioFuse](/ko/)에서 CBZ 파일을 열거나, PDF를 문서 앱에 가져오거나, 필요에 따라 공유합니다.

## 일괄 복호화를 사용해야 하는 경우

### 연간 재무 문서

은행과 기관은 명세서, 세금 양식, 계좌 요약을 비밀번호로 보호된 PDF로 보냅니다. 1년치 월별 명세서를 모으면 12번의 동일한 복호화 작업이 됩니다. 일괄 복호화로 한 번의 탭에 12개를 모두 처리합니다.

### 만화 라이브러리 가져오기

BiblioFuse를 사용해 iPhone에 디지털 만화 라이브러리를 구축 중이고 만화가 비밀번호로 보호된 CBZ 아카이브로 도착했다면, 일괄 복호화가 가장 빠른 경로입니다. 모든 아카이브를 한 번에 가져와 한 번의 작업으로 복호화하면 전체 컬렉션이 준비됩니다.

### ZIP 백업 및 문서 컬렉션

앱 내보내기나 개인 백업의 ZIP 아카이브에는 비밀번호 보호가 있는 경우가 많습니다. 일괄 복호화로 폴더 전체의 비밀번호를 한 번에 제거합니다.

## 비밀번호 프로필: 한 번 저장, 영원히 재사용

비밀번호 프로필 저장소는 일괄 복호화를 실용적으로 만드는 기능입니다. 대부분의 실제 컬렉션은 같은 비밀번호가 여러 파일에 적용됩니다. 프로필은 기기에 안전하게 저장되며 클라우드 서비스와 동기화되지 않습니다.

예. SmartDecrypt PDF ZIP CBZ는 유니버설 앱입니다: 같은 구매로 iPhone, iPad, Mac을 커버합니다. Mac 버전은 더 큰 파일을 더 빠르게 처리하고 드래그 앤 드롭 가져오기를 지원합니다. Finder에서 수백 개의 파일을 한 번에 선택하고 SmartDecrypt에 드롭할 수 있습니다.
