# Comment déchiffrer plusieurs fichiers protégés par mot de passe en une seule fois

SmartDecrypt PDF ZIP CBZ déchiffre plusieurs fichiers PDF, CBZ et ZIP protégés par mot de passe en une seule opération sur iPhone, iPad et Mac. Sélectionnez n'importe quel nombre d'archives protégées, appliquez un mot de passe ou un profil de mot de passe enregistré, et l'application traite chaque fichier sur l'appareil — sans envoi, sans attente par fichier. Après le déchiffrement, ouvrez vos bandes dessinées CBZ dans [BiblioFuse](/fr/) pour la meilleure expérience de lecture sur iPhone. Le déchiffrement par lot de SmartDecrypt est le moyen le plus rapide de traiter toute une collection de fichiers protégés.

## Le problème du déchiffrement fichier par fichier

Vous avez 50 relevés bancaires, tous des PDF protégés par mot de passe de la même institution. Ou un dossier de 30 bandes dessinées CBZ traduites par des fans du même éditeur, toutes avec le même mot de passe d'archive. Ouvrir chaque fichier individuellement — importer, saisir le mot de passe, déchiffrer, enregistrer — prend plusieurs minutes par fichier. Pour les grandes collections, cela représente des heures de travail répétitif.

La plupart des outils de déchiffrement sur iPhone traitent les fichiers un par un. Ils n'ont aucune notion d'opérations par lot ou de profils de mots de passe réutilisables.

SmartDecrypt PDF ZIP CBZ résout ce problème avec deux fonctionnalités combinées : le **déchiffrement par lot** et les **profils de mots de passe**. Sélectionnez tous les fichiers d'un coup, appliquez le profil et appuyez une seule fois.

[SmartDecrypt](/fr/smartdecrypt/) gère tout sur l'appareil — aucun service cloud, aucune attente par fichier, aucun envoi.

## Comment fonctionne le déchiffrement par lot dans SmartDecrypt

### Étape 1 : Créer un profil de mot de passe (recommandé)

Avant le déchiffrement par lot, enregistrez un profil de mot de passe pour la collection. Dans SmartDecrypt, allez dans Profils, appuyez sur Ajouter un profil, entrez un nom (par exemple "Relevés bancaires 2025") et saisissez le mot de passe. Le profil est enregistré en toute sécurité sur votre appareil et n'est jamais synchronisé avec un service cloud.

### Étape 2 : Importer plusieurs fichiers

Ouvrez SmartDecrypt et appuyez sur Importer. Sélectionnez plusieurs fichiers depuis le navigateur Fichiers. Vous pouvez également sélectionner plusieurs fichiers dans l'application Fichiers iOS et les partager tous avec SmartDecrypt en même temps.

### Étape 3 : Appliquer le mot de passe

Avec tous les fichiers chargés, tapez le mot de passe une fois ou sélectionnez le profil enregistré. Le même mot de passe s'applique à tous les fichiers du lot.

### Étape 4 : Appuyer sur Déchiffrer

SmartDecrypt traite chaque fichier en séquence. Pour des archives typiques, chaque fichier se complète en moins d'une seconde. Un lot de 50 relevés bancaires se termine en moins d'une minute.

### Étape 5 : Accéder aux fichiers déchiffrés

Tous les fichiers déchiffrés sont enregistrés dans l'application Fichiers. Ouvrez les fichiers CBZ dans [BiblioFuse](/fr/), importez les PDF dans une application de documents, ou partagez-les selon vos besoins.

## Quand utiliser le déchiffrement par lot ?

### Documents financiers annuels

Les banques et institutions envoient des relevés, formulaires fiscaux et résumés de compte sous forme de PDF protégés. Accumuler un an de relevés mensuels signifie 12 opérations de déchiffrement identiques. Le déchiffrement par lot les traite tous en un seul appui.

### Importations de bibliothèque de bandes dessinées

Si vous constituez une bibliothèque numérique de bandes dessinées sur iPhone avec BiblioFuse et que vos bandes dessinées sont des archives CBZ protégées, le déchiffrement par lot est le chemin le plus rapide. Importez toutes les archives d'un coup, déchiffrez en une opération, et toute votre collection est prête.

### Sauvegardes ZIP et collections de documents

Les archives ZIP provenant d'exports d'applications ou de sauvegardes personnelles portent souvent une protection par mot de passe. Le déchiffrement par lot supprime les mots de passe de tout un dossier en une fois.

## Profils de mots de passe : enregistrez une fois, réutilisez toujours

Le magasin de profils de mots de passe est la fonctionnalité qui rend le déchiffrement par lot pratique. La plupart des collections réelles impliquent le même mot de passe appliqué à de nombreux fichiers. Les profils sont stockés en toute sécurité sur l'appareil et ne sont jamais synchronisés avec un service cloud.

## Foire aux questions

**Puis-je déchiffrer plusieurs fichiers protégés par mot de passe en même temps sur iPhone ?**
Oui. SmartDecrypt PDF ZIP CBZ prend en charge le déchiffrement par lot pour PDF, CBZ et ZIP. Importez plusieurs fichiers, saisissez le mot de passe une fois ou sélectionnez un profil enregistré, et appuyez sur Déchiffrer. Tous les fichiers sont traités sur l'appareil sans aucun envoi.

**Qu'est-ce qu'un profil de mot de passe dans SmartDecrypt ?**
Un profil de mot de passe est une paire nom-mot de passe enregistrée en toute sécurité sur votre appareil. Au lieu de retaper le même mot de passe pour chaque fichier, vous créez un profil une fois et le sélectionnez pour les déchiffrements futurs. Les profils ne sont jamais synchronisés avec un service cloud.

**Combien de fichiers puis-je déchiffrer par lot en même temps ?**
Il n'y a pas de limite fixe. SmartDecrypt traite les fichiers en séquence. En pratique, 50 relevés PDF ou archives CBZ typiques se complètent en moins d'une minute sur un iPhone moderne.

**Le déchiffrement par lot fonctionne-t-il aussi pour les fichiers ZIP ?**
Oui. SmartDecrypt gère PDF, ZIP et CBZ en mode lot. Vous pouvez mélanger les types de fichiers dans un seul lot s'ils partagent le même mot de passe.

**Le déchiffrement par lot est-il sûr pour les documents sensibles ?**
Oui. SmartDecrypt fonctionne entièrement sur votre appareil. Aucun fichier n'est envoyé. Le moteur Rust traite tout localement et les fichiers déchiffrés sont uniquement enregistrés dans l'application Fichiers locale.

**SmartDecrypt fonctionne-t-il sur Mac pour le déchiffrement par lot ?**
Oui. La version Mac prend en charge le déchiffrement par lot avec importation par glisser-déposer depuis le Finder. Vous pouvez déposer des centaines de fichiers en une seule fois.
