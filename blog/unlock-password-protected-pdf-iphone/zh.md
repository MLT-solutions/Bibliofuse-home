# 如何在 iPhone 和 Mac 上解锁受密码保护的 PDF

[SmartDecrypt PDF ZIP CBZ](/zh/smartdecrypt/) 可以在 iPhone、iPad 和 Mac 上移除受保护 PDF 文件的密码，无需将您的文档发送至任何外部服务器。它还能解锁受密码保护的 ZIP 和 CBZ 压缩包，并可通过内置批量模式一次性解密多个文件。如果您曾需要在 iPhone 上解锁受密码保护的 PDF 却无从下手，SmartDecrypt 正是专为此而设计的工具。

## 为什么受密码保护的 PDF 会在 iPhone 上阻止您访问？

您收到来自银行、雇主或出版商的 PDF 文件，通过电子邮件、文件共享链接或即时通讯应用发来。您在 iPhone 上打开它，却不是直接显示内容，而是要求输入密码。您输入密码，文件打开一次——但您无法添加注释、复制文本，也无法分享已解锁的版本。有时您需要将解密后的文件传递给另一个根本不支持输入密码的应用程序。

iOS 没有内置方式可以永久移除 PDF 密码。Apple 的"文件"应用和"图书"应用虽然可以在每次输入密码后打开受保护的 PDF，但无法移除密码并保存解锁副本。每次重新打开文件，密码提示都会再次出现。

## 为什么在线 PDF 解锁网站是个糟糕的选择

搜索"在线解锁 PDF"，会出现几十个网络服务。方法看似简单：上传文件，获取解锁版本。问题在于您的文档在此过程中发生了什么。

银行对账单、工资单、医疗记录和法律合同恰恰是那些最需要用密码保护的文件。将它们上传到不知名的网络服务器，意味着您的敏感数据离开了您的设备，落入您无法掌控的基础设施中。许多免费服务会将上传的文件保留数天乃至数周。有些甚至是数据收集的幌子。即使是拥有正规隐私政策的服务，当本地应用完全可以在设备上完成同样工作时，也是一种不必要的风险。

SmartDecrypt PDF ZIP CBZ 完全在您的 iPhone 或 Mac 上运行。不上传任何内容。基于 Rust 的解密引擎在本地完成所有工作，解锁后的文件保存回您的"文件"应用。
