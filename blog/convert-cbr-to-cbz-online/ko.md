# CBR을 CBZ로 온라인 변환하는 방법: 무료, 소프트웨어 설치 불필요

BiblioFuse 웹 도구는 CBR 만화 아카이브를 브라우저에서 직접 CBZ 형식으로 변환합니다 — 소프트웨어 설치가 필요 없고, 파일을 서버에 업로드할 필요도 없습니다. CBR 파일은 RAR 아카이브이고, CBZ 파일은 동일한 만화 이미지를 담은 ZIP 아카이브입니다. BiblioFuse는 이미지를 표준 ZIP 기반 CBZ 형식으로 재패키징하여 모든 아트워크의 품질을 완전히 보존합니다. 어떤 브라우저에서든 도구를 열고, CBR 파일을 드롭하면 몇 초 안에 변환된 CBZ를 다운로드할 수 있습니다.

당신의 만화 컬렉션에는 수십 개—어쩌면 수백 개—의 .cbr 파일이 있을 것입니다. CBR은 RAR 압축 표준을 기반으로 한 최초의 만화 아카이브 형식이었습니다. CBZ는 이후에 더 개방적인 ZIP 형식을 사용하며 등장했습니다. 오늘날 대부분의 최신 만화 리더, 라이브러리, 아카이브 도구는 CBZ를 선호합니다. 소프트웨어를 설치하거나 제3자 서버에 파일을 업로드하지 않고 온라인으로 CBR을 CBZ로 변환하는 것이 컬렉션을 표준화하는 가장 깔끔한 방법입니다.

## CBR과 CBZ의 차이점은 무엇인가요?

CBR과 CBZ는 모두 만화 아카이브 형식입니다. 같은 원리로 작동합니다: 순차적인 만화 페이지 이미지(보통 JPEG 또는 PNG) 폴더를 하나의 아카이브 파일로 압축합니다. 유일한 실질적 차이는 압축 알고리즘입니다:

- **CBR** — RAR 압축을 사용합니다. RAR은 Eugene Roshal이 개발한 독점 형식입니다. 초기 만화 스캐너들은 좋은 압축률로 인해 이것을 선호했습니다. `.cbr` 확장자는 "Comic Book RAR"의 약자입니다.
- **CBZ** — ZIP 압축을 사용합니다. ZIP은 추가 소프트웨어 없이 모든 운영 체제에서 광범위하게 지원되는 개방형 표준입니다. `.cbz` 확장자는 "Comic Book ZIP"의 약자입니다.

두 형식 내부의 이미지는 동일합니다. CBR을 CBZ로 변환해도 아트워크, 페이지 순서, 품질이 변하지 않습니다 — 같은 파일을 다른 아카이브 컨테이너로 재패키징할 뿐입니다.

## 왜 온라인으로 CBR을 CBZ로 변환해야 하나요?

**호환성:** CBZ는 사실상 모든 만화 리더, 라이브러리 앱, 아카이브 도구에서 기본적으로 지원됩니다. 일부 앱 — 특히 iOS, 웹, 최신 플랫폼에서 — ZIP 기반 아카이브만 지원합니다. CBZ는 어디서나 작동하지만, CBR은 그렇지 않을 때가 있습니다.

**개방형 표준:** ZIP은 완전히 개방된 형식입니다. RAR은 독점 사양을 가지고 있으며 특정 디코딩 라이브러리가 필요합니다. 장기 아카이브에는 CBZ가 더 미래 지향적인 선택입니다.

**파일 관리:** macOS와 Windows는 추가 앱 없이 ZIP 파일을 기본적으로 열 수 있습니다. 확장자를 `.zip`으로 변경하면 Finder나 파일 탐색기에서 CBZ 내용을 탐색할 수 있습니다. CBR 파일은 WinRAR이나 전용 압축 해제 도구가 필요합니다.

**웹 호환성:** [BiblioFuse 웹 도구](https://bibliofuse.com/ko/webapp/)는 ZIP 기반 아카이브(CBZ)에서만 작동합니다. CBR 파일을 먼저 변환하면 온라인 만화 도구 전체와 호환됩니다.

## 온라인으로 CBR을 CBZ로 변환하는 방법 — 단계별 안내

BiblioFuse 웹 도구는 WebAssembly를 사용하여 파일을 브라우저에서 완전히 처리합니다. 서버에 아무것도 업로드되지 않으며, 개인 데이터도 전송되지 않습니다. 파일은 전체 과정 동안 사용자 기기에 남아 있습니다.

### 1단계: BiblioFuse 웹 도구 열기

Chrome, Firefox, Safari 또는 Edge에서 [bibliofuse.com/ko/webapp/](https://bibliofuse.com/ko/webapp/)에 접속합니다. 계정이나 로그인이 필요하지 않습니다.

### 2단계: CBR 파일 가져오기

**파일 추가**를 클릭하거나 CBR 파일을 도구의 드롭 영역으로 직접 드래그합니다. 도구는 CBR(RAR 기반) 아카이브를 수락하고 내부 이미지를 추출합니다.

무료 버전에서는 배치당 최대 3개의 파일을 처리할 수 있습니다. 세 개를 한 번에 드롭하면 단일 작업으로 변환할 수 있습니다.

### 3단계: 출력 형식 선택

도구 옵션에서 출력 형식으로 **CBZ**를 선택합니다. 이렇게 하면 변환기가 추출된 이미지를 `.cbz` 확장자를 가진 ZIP 아카이브로 재패키징합니다.

### 4단계: 변환 및 다운로드

**변환**을 클릭합니다(또는 해당 작업 버튼). 도구가 브라우저에서 파일을 처리합니다 — 업로드 없이, 서버 응답 대기 없이. 처리가 완료되면 다운로드 링크가 나타납니다. 클릭하여 변환된 CBZ 파일을 Mac 또는 PC에 저장합니다.

다운로드된 파일은 원본 CBR과 동일한 이미지를 ZIP 형식으로 재패키징한 표준 CBZ 아카이브입니다.

## 변환 후 품질과 파일 크기

CBR에서 CBZ로의 변환은 이미지에 대해 무손실입니다. 만화 페이지 이미지(아카이브 내 JPEG, PNG 또는 WebP 파일)는 RAR 컨테이너에서 추출되어 재인코딩이나 압축 변경 없이 ZIP 컨테이너에 직접 배치됩니다.

원본 CBR과 결과 CBZ 사이에 파일 크기가 약간 다를 수 있습니다. ZIP과 RAR은 서로 다른 압축 알고리즘을 사용하므로, ZIP으로 압축된 동일한 이미지가 RAR 원본보다 약간 더 크거나 작은 파일을 생성할 수 있습니다. 차이는 일반적으로 5% 미만이며 이미지 품질은 동일합니다.

변환된 CBZ의 파일 크기도 줄이고 싶다면, BiblioFuse 웹 도구는 선택적 압축 단계를 제공합니다 — 아카이브 내 이미지를 리사이즈하고 재압축하여 파일 용량을 줄일 수 있습니다.

## 여러 CBR 파일을 CBZ로 일괄 변환하기

무료 사용자의 경우, BiblioFuse 웹 도구는 배치당 최대 3개의 파일을 처리합니다. 여러 CBR 파일을 한 번에 드롭하여 단일 패스로 변환합니다:

1. **파일 추가**를 클릭하여 2~3개의 CBR 파일을 선택합니다(또는 모두 한 번에 도구로 드래그).
2. 모든 파일의 출력 형식을 **CBZ**로 설정합니다.
3. **변환**을 클릭합니다. 모든 파일이 브라우저에서 병렬로 처리됩니다.
4. 변환된 각 CBZ를 개별적으로 다운로드합니다.

대규모 컬렉션의 경우 3개씩 배치로 반복합니다. 한 번에 수십 또는 수백 개의 파일을 변환해야 하는 경우에도, 도구는 각 세트에 대해 동일한 프로세스를 처리합니다 — 계정, 대기열, 일일 제한 없음.

## 개인 정보 보호: 업로드하지 않는 것이 중요한 이유

많은 온라인 파일 변환 도구는 파일을 클라우드 서버에 업로드하고, 거기서 처리한 후 돌려보내는 방식으로 작동합니다. 파일이 제3자 서버를 통과하며 — 잠재적으로 저장되거나, 기록되거나, 제어할 수 없는 방식으로 사용될 수 있습니다.

BiblioFuse 웹 도구는 다릅니다. WebAssembly를 사용하여 변환 로직을 브라우저에서 직접 실행합니다. CBR 파일은 기기를 절대 떠나지 않습니다. 이는 특히 직접 구매하거나 스캔한 만화에 중요합니다 — 개인 정보나 저작권 노출이 없습니다.

## 변환 후 iPhone에서 CBZ 파일 읽기

CBR 파일을 CBZ로 변환한 후, iPhone의 [BiblioFuse](https://bibliofuse.com/ko/)에 가져오는 것은 간단합니다:

1. Wi-Fi 전송, AirDrop 또는 파일 앱을 통해 iPhone으로 CBZ 파일을 전송합니다.
2. BiblioFuse가 로컬 라이브러리에 가져옵니다.
3. 즉시 열고 읽습니다 — CBZ는 BiblioFuse의 기본 형식입니다.

참고: iPhone의 BiblioFuse는 CBR 파일도 기본적으로 읽을 수 있으므로, iOS에서 읽기 위해 변환이 엄격히 필요하지는 않습니다. 하지만 다른 사람들과 공유하거나, 다른 도구를 사용하거나, 장기 아카이브를 구축하려면 CBZ가 더 나은 형식입니다.

## 자주 묻는 질문

### CBR과 CBZ의 차이점은 무엇인가요?

CBR 파일은 RAR 압축을 사용하고, CBZ 파일은 ZIP 압축을 사용합니다. 둘 다 동일한 유형의 만화 페이지 이미지(JPEG, PNG 또는 WebP)를 포함합니다. 유일한 차이는 이미지를 감싸는 아카이브 형식입니다. CBR을 CBZ로 변환하면 이미지를 변경하지 않고 ZIP 아카이브에 재패키징합니다.

### 소프트웨어를 설치하지 않고 CBR을 CBZ로 변환할 수 있나요?

네. BiblioFuse 웹 도구는 WebAssembly를 사용하여 브라우저에서 완전히 실행됩니다. Chrome, Firefox, Safari 또는 Edge에서 열어보세요 — 다운로드, 설치, 계정이 필요하지 않습니다. CBR 파일을 드롭하고 변환된 CBZ를 다운로드하면 됩니다.

### CBR을 CBZ로 변환하면 이미지 품질이 떨어지나요?

아니요. CBR에서 CBZ로의 변환은 무손실 컨테이너 교환입니다. 아카이브 내의 이미지 파일은 재인코딩 없이 추출되고 재패키징됩니다. 품질은 원본과 동일합니다. ZIP과 RAR 압축의 차이로 인해 파일 크기가 몇 퍼센트 변경될 수 있지만, 아트워크는 변하지 않습니다.

### 온라인 도구로 CBR 파일을 변환하는 것이 안전한가요?

BiblioFuse 웹 도구로는 안전합니다. 이 도구는 WebAssembly를 사용하여 브라우저에서 로컬로 파일을 처리합니다 — CBR 파일은 서버에 절대 업로드되지 않습니다. 파일 업로드가 필요한 변환 도구는 개인 정보 보호가 취약합니다. 개인 정보가 중요하다면, 업로드 없는 브라우저 기반 도구를 사용하세요.

### CBR에서 CBZ로 변환하는 데 얼마나 걸리나요?

파일 크기와 기기 프로세서에 따라 다르지만, 일반적으로 파일당 몇 초입니다. 100MB CBR 파일은 대부분의 최신 노트북에서 5초 이내에 변환됩니다. 도구가 브라우저에서 파일을 병렬로 처리하므로 일괄 변환도 효율적입니다.

### iPhone의 BiblioFuse가 변환 없이 CBR을 읽는 이유는 무엇인가요?

iOS용 BiblioFuse에는 내장 RAR 압축 해제 기능이 있어, 변환 단계 없이 CBR 파일을 기본적으로 열 수 있습니다. 변환은 다른 도구, 온라인 리더와의 호환성이나 장기 아카이브를 위해 CBZ가 필요할 때 유용합니다 — BiblioFuse가 요구하기 때문이 아닙니다.

## 만화 아카이브 변환을 위한 원스톱 솔루션

온라인으로 CBR을 CBZ로 변환하는 것은 더 호환 가능하고, 개방적이며, 미래 지향적인 만화 라이브러리를 위한 빠른 단계입니다. [BiblioFuse 웹 도구](https://bibliofuse.com/ko/webapp/)는 무료, 개인 정보 보호, 즉각적으로 이를 가능하게 합니다 — 가입 없이 어떤 브라우저에서나 열어보세요. CBR 파일을 드롭하고 CBZ를 받아 ZIP 기반 표준을 지원하는 모든 리더에 가져오세요.
