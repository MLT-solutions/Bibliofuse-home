# CBR을 CBZ로 온라인 변환하는 방법: 무료, 소프트웨어 불필요

CBR 파일이 가득 찬 폴더가 있습니다—RAR 압축 형식으로 저장된 만화들입니다—그런데 리더는 CBZ만 엽니다. 또는 컬렉션을 아카이빙하면서 일관된 오픈 형식을 원합니다. 해결책은 간단해 보입니다: 확장자를 `.cbr`에서 `.cbz`로 바꾸면 될 것 같죠. 하지만 그것은 작동하지 않습니다.

CBR 파일은 RAR 아카이브입니다. CBZ 파일은 ZIP 아카이브입니다. 내부에서 다른 압축 형식을 사용하므로 확장자만 바꿀 수는 없습니다. RAR에서 이미지를 추출하여 ZIP으로 다시 압축해야 합니다. [BiblioFuse 온라인 변환기](https://bibliofuse.com/ko/webapp/)는 이것을 브라우저에서 처리하며, 어떤 서버에도 파일을 업로드하지 않습니다.

## CBR vs CBZ: 실제 차이점은?

CBR과 CBZ 모두 만화 아카이브 형식입니다. 둘 다 이미지 파일 시퀀스(주로 JPG 또는 PNG)를 포함하며, 리더가 이를 순서대로 표시하여 만화 읽기를 시뮬레이션합니다.

차이점은 내부 압축 방식에 있습니다:

| 형식 | 아카이브 유형 | 오픈 표준? |
|------|-----------|---------|
| CBZ | ZIP | 예 |
| CBR | RAR | 아니요 (독점 형식) |

RAR은 Eugene Roshal이 개발한 독점 형식입니다. RAR 파일을 열려면 공식 WinRAR 소프트웨어나 RAR 사양을 구현하는 라이브러리가 필요하며, 해당 라이브러리는 각 리더에 명시적으로 포함되어야 합니다. 브라우저 기반 도구를 포함한 많은 현대 리더들이 ZIP 기반 CBZ만 지원하는데, ZIP 사양이 개방적이고 널리 구현되어 있기 때문입니다.

CBZ가 디지털 만화의 사실상 표준이 된 것은 ZIP이 로열티가 없고 크로스플랫폼이기 때문입니다. 모든 운영 체제는 타사 소프트웨어 없이 ZIP 파일을 열 수 있습니다. 장기 아카이빙을 목적으로 한다면 CBZ가 더 안전한 선택입니다.

## 변환 작동 방식

CBR을 CBZ로 변환하는 데는 세 단계가 필요합니다:

1. CBR(RAR 아카이브)에서 이미지 추출
2. 동일한 이미지를 새 ZIP 아카이브에 넣기
3. `.zip` 확장자를 `.cbz`로 변경

BiblioFuse 온라인 도구는 WebAssembly를 사용하여 브라우저에서 이 세 단계를 모두 수행합니다. 파일은 절대 업로드되지 않으며—추출, 재압축, 다운로드 모두 기기에서 로컬로 진행됩니다.

## 단계별 가이드: CBR을 CBZ로 온라인 변환

1. [bibliofuse.com/ko/webapp/](https://bibliofuse.com/ko/webapp/)으로 이동합니다.
2. 도구 모음에서 **변환** 도구를 엽니다.
3. **파일 선택**을 클릭하거나 `.cbr` 파일을 드롭 영역으로 드래그합니다.
4. 도구가 이미지를 추출하고, 새 ZIP 아카이브를 생성하고, `.cbz` 파일을 자동으로 다운로드합니다.

일괄 변환—여러 CBR 파일을 한 번에—의 경우 파일 선택기에서 모든 파일을 선택하세요. 각 파일은 개별적으로 처리되고 다운로드됩니다. 도구에서 부과하는 대기열 제한은 없지만, 매우 큰 파일(400MB 이상)은 기기의 처리 능력에 따라 더 오래 걸릴 수 있습니다.

## 브라우저 기반 변환이 개인 정보를 보호하는 이유

변환은 WebAssembly(WASM)를 통해 브라우저에서 완전히 실행됩니다. 이것은 게임과 동영상 편집기가 플러그인 없이 브라우저에서 실행되도록 하는 것과 동일한 기술로, 계산 집약적인 작업이 브라우저 샌드박스에서 기본적으로 실행됩니다.

[BiblioFuse 웹 도구](https://bibliofuse.com/ko/webapp/)를 사용할 때:

- CBR 파일이 브라우저 메모리로 읽혀집니다
- 이미지가 메모리에서 추출됩니다
- 새 ZIP 파일이 메모리에서 구성됩니다
- ZIP이 기기로 다운로드됩니다

어떤 시점에서도 데이터가 컴퓨터를 떠나지 않습니다. 변환 중 서버 호출이 없습니다. 브라우저의 네트워크 인스펙터(F12 → 네트워크)를 열어 확인할 수 있습니다: 유일한 요청은 초기 페이지 로드뿐이고, 업로드는 표시되지 않습니다.

컬렉션에 라이선스 콘텐츠, 개인 스캔 또는 타사 서버에 업로드하고 싶지 않은 다른 것들이 포함된 경우 이것은 중요합니다.

## iPhone 또는 iPad에서 CBR 파일 변환

BiblioFuse 웹 도구는 모바일 브라우저에서도 작동합니다. iPhone 또는 iPad의 Safari 또는 Chrome에서 [bibliofuse.com/ko/webapp/](https://bibliofuse.com/ko/webapp/)으로 이동하세요.

iOS에서 **파일 선택**을 탭하고 파일 앱, iCloud Drive 또는 연결된 클라우드 스토리지 제공업체에서 CBR 파일을 찾아보세요. 다운로드된 CBZ 파일은 다운로드 폴더로 이동하며, 거기서 읽기를 위해 [BiblioFuse](https://bibliofuse.com/ko/)로 직접 공유할 수 있습니다.

## 변환 후: BiblioFuse에서 CBZ 파일 읽기

iPhone의 BiblioFuse는 CBZ 파일을 기본적으로 지원합니다—추가 변환이 필요 없습니다. 변환된 파일을 읽으려면:

1. Mac 또는 iPhone에서 웹 도구를 사용하여 CBR을 CBZ로 변환합니다.
2. Wi-Fi 전송 또는 AirDrop을 통해 CBZ 파일을 iPhone으로 전송합니다.
3. 파일 앱에서 파일을 선택하고 BiblioFuse로 공유하여 가져옵니다.

참고: iPhone의 BiblioFuse는 CBR 파일도 직접 읽을 수 있습니다—RAR 추출을 기본적으로 처리합니다. 변환하는 주된 이유는 파일을 아카이빙하거나, 다른 사람과 공유하거나, 워크플로우의 다른 도구가 특별히 CBZ를 요구하는 경우입니다.

## 자주 묻는 질문

**여러 CBR 파일을 한 번에 변환할 수 있나요?**

예. 파일 선택기에서 모든 CBR 파일을 선택하면 도구가 각각을 처리합니다. 파일은 개별적으로 다운로드되며—`.cbr`당 하나의 `.cbz`가 생성됩니다. 이후 하나의 볼륨으로 합치고 싶다면 CBZ 병합 도구를 사용하세요.

**변환 중 이미지 품질이 바뀌나요?**

아니요. 아카이브 내의 이미지는 재인코딩되지 않습니다. 변환은 원본 JPG 또는 PNG 파일을 추출하여 이미지 데이터에 손대지 않고 새 ZIP에 넣습니다. 변환된 CBZ는 원본 CBR과 픽셀 단위로 동일합니다.

**CBR 파일이 비밀번호로 보호되어 있다면?**

비밀번호로 보호된 CBR 파일은 비밀번호 없이 열 수 없습니다. 웹 도구는 비밀번호 보호를 우회할 수 없습니다. CBR이 암호화되어 있다면 먼저 잠금을 해제하세요—SmartDecrypt는 iPhone과 Mac에서 비밀번호로 보호된 CBZ, PDF, ZIP 파일을 일괄 잠금 해제할 수 있습니다.

**CBR 파일이 변환됐는데 CBZ가 열리지 않습니다—무슨 일인가요?**

`.cbr`로 표시된 일부 파일은 실제로는 ZIP 아카이브입니다(`.cbz`에서 이름이 변경되었거나 ZIP을 기본으로 사용하는 소프트웨어에서 생성된 것). 변환이 리더에서 열리지 않는 파일을 생성한 경우, 원본 `.cbr`을 `.zip`으로 이름을 바꿔서 이미 ZIP 아카이브인지 확인해 보세요. 변환 없이 직접 `.cbz`로 이름만 바꾸면 되는 경우도 있습니다.

**파일 크기 제한이 있나요?**

파일이 업로드되지 않으므로 서버 부과 제한은 없습니다. 실제 제한은 기기의 사용 가능한 RAM입니다—매우 큰 CBR 파일(1GB 이상)은 메모리가 제한된 기기에서 브라우저를 느리게 만들 수 있습니다. 대부분의 200MB 미만 만화 파일은 몇 초 안에 변환이 완료됩니다.

## 핵심 정리

CBR과 CBZ는 비슷해 보이지만 내부적으로 다른 아카이브 형식을 사용합니다. CBR을 CBZ로 변환한다는 것은 만화 이미지를 RAR에서 ZIP 형식으로 재아카이빙하는 것을 의미합니다—[BiblioFuse 온라인 도구](https://bibliofuse.com/ko/webapp/)는 파일 업로드 없이 브라우저에서 완전히 처리합니다. 이미지 품질은 보존되고, 프로세스는 개인 정보를 보호하며, 결과로 생성된 CBZ 파일은 오픈 ZIP 기반 형식을 지원하는 모든 리더에서 작동합니다.
