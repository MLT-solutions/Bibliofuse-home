# iTunes나 케이블 없이 iPhone으로 만화 전송하는 방법

BiblioFuse에는 내장 Wi-Fi 전송 기능이 있어 Mac이나 PC의 브라우저에서 CBZ, CBR, EPUB, PDF, ZIP 파일을 케이블, iTunes, 클라우드 업로드 없이 iPhone 라이브러리로 바로 드래그할 수 있습니다. iOS 17 이상에서 작동하며, 일반 가정용 Wi-Fi 네트워크에서 15–40 MB/s의 전송 속도로 150 MB 만화 한 권을 약 5초 만에 전송합니다.

만화 라이브러리가 Mac에 있습니다. CBZ 파일 200개, 시리즈별로 정리되어 있고 각 권은 80~200MB. 실제로 읽는 건 iPhone입니다. 침대에서, 지하철에서, 점심시간에. 그 파일들을 iPhone으로 옮기는 게 아무도 제대로 다루지 않는 문제입니다.

이메일은 25MB 제한이 있습니다. AirDrop은 되긴 하지만 파일을 하나씩 보내면서 매번 승인해야 합니다. iCloud Drive로 동기화할 수도 있지만, 만화 20GB를 먼저 업로드하고 다운로드가 완료되길 기다리는 건 현실적이지 않습니다. iTunes 파일 공유도 여전히 존재하지만 케이블이 필요하고, 대부분의 사람들이 이미 포기한 수준의 인내심을 요구합니다.

[BiblioFuse](/ko/)는 Wi-Fi 전송 기능으로 이 문제를 해결합니다. 내장 로컬 웹 서버를 통해 Mac이나 PC의 브라우저에서 만화 라이브러리 전체를 iPhone으로 바로 끌어다 놓을 수 있습니다. 케이블 불필요, 컴퓨터에 소프트웨어 설치 불필요, 클라우드 업로드 불필요. 이 가이드에서 전체 과정을 안내합니다.

## Wi-Fi 전송의 실제 작동 방식

Wi-Fi 전송은 iPhone에서 소형 HTTP 서버를 실행하며, 로컬 네트워크에서만 접근할 수 있습니다. 제공된 주소(예: `http://192.168.1.x:8080`)를 브라우저에서 열면 드래그 앤 드롭 업로드 인터페이스가 나타납니다. 데스크톱에서 파일이나 폴더를 브라우저 창에 끌어다 놓으면 [BiblioFuse](/ko/)가 Wi-Fi를 통해 직접 수신합니다.

핵심 단어는 *직접*입니다. 파일이 클라우드 서비스를 거치지 않고 컴퓨터에서 iPhone으로 이동합니다. 제3자 서버에 아무것도 업로드되지 않습니다. 어떤 계정에도 로그인할 필요가 없습니다. 유일한 요구 사항은 두 기기가 같은 Wi-Fi 네트워크에 있어야 한다는 것입니다.

전송 속도는 인터넷 연결이 아닌 로컬 네트워크에 따라 결정됩니다. 일반 가정용 라우터에서 15~40MB/s를 기대할 수 있습니다. 150MB 만화 한 권은 약 5초면 전송됩니다.

## 1단계 — BiblioFuse 설치

아직 설치하지 않았다면 App Store에서 [BiblioFuse](/ko/)를 설치하세요. 무료 다운로드이며 계정이나 로그인이 필요하지 않습니다.

설치 후 iOS 파일 앱에 두 개의 폴더가 생성됩니다:
- **iCloud Drive → BiblioFuse** — 동기화 저장소, 모든 Apple 기기에서 접근 가능
- **내 iPhone → BiblioFuse** — 로컬 전용 저장소, 더 빠르고 동기화 지연 없음

Wi-Fi 전송으로 전송된 파일은 기본적으로 로컬 저장소 폴더에 저장됩니다.

## 2단계 — iPhone에서 Wi-Fi 전송 활성화

1. BiblioFuse 열기
2. **설정** 탭 (라이브러리 화면 오른쪽 하단)
3. **Wi-Fi 가져오기** 항목에서 토글 켜기
4. 로컬 URL이 표시됨: `http://192.168.x.x:8080`

전송 중에는 BiblioFuse를 열어두고 화면을 켠 상태로 유지하세요. 앱이 백그라운드로 이동하면 iOS가 서버를 일시 정지시킵니다.

## 3단계 — 컴퓨터에서 URL 열기

Mac이나 PC에서 Safari, Chrome, Firefox, Edge 등 아무 브라우저나 열고 BiblioFuse에 표시된 URL을 입력하세요. 드롭 영역이 있는 간단한 페이지가 보입니다.

아무것도 설치할 필요가 없습니다. 확장 프로그램도, 앱도, 계정도 필요 없습니다. 페이지는 iPhone에서 직접 제공됩니다.

## 4단계 — 만화 파일 드래그

Finder(또는 Windows 탐색기)에서 CBZ, CBR, ZIP, RAR 또는 EPUB 파일을 브라우저의 드롭 영역으로 드래그하세요. 다음을 드래그할 수 있습니다:

- 파일 하나
- 여러 파일 동시에
- 폴더 전체

폴더를 드래그하면 BiblioFuse가 안의 모든 파일을 폴더 구조를 유지한 채 수신합니다. 40권이 들어있는 `Berserk` 폴더를 드래그하면 라이브러리에 해당 폴더 이름 아래 모두 정리된 상태로 도착합니다.

진행 표시줄에서 각 파일의 업로드 상황을 확인할 수 있습니다. 만화 10GB 같은 대용량 배치도 방치해두고 돌릴 수 있습니다. 개별 파일을 클릭하거나 승인할 필요가 없습니다.

## 5단계 — Wi-Fi 전송 끄기

전송이 완료되면 BiblioFuse로 돌아가 Wi-Fi 전송을 끄세요. 보안 조치입니다. 서버는 로컬 네트워크에서만 접근 가능하지만, 사용 후 끄면 같은 네트워크의 다른 사람이 접근하는 것을 막을 수 있습니다.

전송된 파일은 라이브러리에 자동으로 나타납니다. 표지를 탭해서 읽기 시작하세요.

## 전송 속도 참고

현대적인 802.11ac(Wi-Fi 5) 가정용 네트워크 기준 대략적인 수치입니다:

| 파일 크기 | 전송 시간 |
|-----------|-----------|
| 50 MB (소형 만화 한 권) | 약 2초 |
| 150 MB (일반적인 만화 한 권) | 약 5초 |
| 500 MB (대형 CBZ) | 약 15초 |
| 10 GB (시리즈 전체) | 약 5분 |

오래된 802.11n 네트워크나 혼잡한 메시 시스템은 더 느릴 수 있습니다. 대용량 배치를 전송할 때는 iPhone을 라우터 가까이 두세요.

## iPhone에 만화를 넣는 다른 방법들

Wi-Fi 전송이 대형 컬렉션에는 가장 빠른 방법이지만, 유일한 방법은 아닙니다.

**파일 앱에서 공유** — 만화가 이미 iCloud Drive나 Mac에 연결된 USB 드라이브에 있다면, 파일 앱에서 길게 눌러 공유 → BiblioFuse를 탭하세요. 파일 한두 개에 가장 편리합니다.

**iCloud Drive 동기화** — Mac에서 `iCloud Drive → BiblioFuse` 폴더에 파일을 복사하면 iPhone에 자동으로 동기화됩니다. 독서 진행 상황도 이 방식으로 동기화됩니다. 여러 기기에서 읽는 경우 유용합니다. 단점: iCloud에 먼저 업로드해야 하므로 대형 컬렉션은 시간이 걸릴 수 있습니다.

**AirDrop** — 개별 파일에 적합합니다. AirDrop 아이콘을 탭하고 iPhone을 선택하면 BiblioFuse가 파일을 열겠다고 제안합니다. 권수가 많은 시리즈에는 적합하지 않습니다.

**Tailscale / 원격 접속** — Mac과 같은 네트워크에 있지 않은 경우, Tailscale을 사용해 기기 간 사설 가상 네트워크를 만들고 원격으로 Wi-Fi 전송이나 iCloud 동기화를 사용할 수 있습니다. 고급 설정이지만 어디서든 집 라이브러리에 접근할 수 있게 됩니다.

## 문제 해결

**브라우저에서 URL에 접근할 수 없습니다.**
두 기기가 같은 Wi-Fi 네트워크에 있는지 확인하세요. 일부 라우터는 기기 간 격리 기능("클라이언트 격리" 또는 "AP 격리")을 사용합니다. 기기끼리 서로 보이지 않는다면 라우터 설정을 확인하세요.

**파일이 전송됐는데 라이브러리에 나타나지 않습니다.**
라이브러리에서 아래로 당겨 새로고침하세요. 파일 앱의 **내 iPhone → BiblioFuse** 폴더를 확인해 파일이 도착했는지 확인하세요. BiblioFuse 폴더 밖에 있는 파일은 색인되지 않습니다.

**전송이 중간에 멈췄습니다.**
화면이 잠기면서 iOS가 앱을 일시 정지시킨 것으로 보입니다. BiblioFuse로 돌아가 Wi-Fi 전송이 여전히 켜져 있는지 확인하고 브라우저에서 업로드를 다시 시작하세요. 이미 업로드된 파일은 안전합니다.

**CBR 파일에 "지원되지 않는 형식"이라고 표시됩니다.**
매우 오래된 CBR 아카이브 중 일부는 RAR5를 사용하며, 이는 범용으로 지원되지 않습니다. CBZ로 재패키징하세요: RAR을 추출하고, 모든 이미지를 선택한 다음 ZIP으로 압축하고 `.zip`을 `.cbz`로 이름 바꾸세요. BiblioFuse의 도구 탭에서도 기기에서 직접 형식 변환을 할 수 있습니다.

## 다음 단계

라이브러리가 iPhone에 들어오면 [BiblioFuse](/ko/)는 관리를 편리하게 해줄 몇 가지 도구를 제공합니다. 도구 탭에서는 기기에서 CBZ 파일을 압축할 수 있습니다. 200MB 파일을 화질 손상 없이 40MB로 줄일 수 있는 경우도 있습니다. Mac 홈 라이브러리 기능은 한 걸음 더 나아가 iPhone으로 파일을 전송하는 대신, Mac의 전체 만화 라이브러리를 Wi-Fi로 실시간 스트리밍할 수 있게 해줍니다. 복사가 아예 필요 없습니다.

이것이 전체 그림입니다. 예전 iTunes 동기화가 1시간 걸리던 것을 몇 분 만에 끝낼 수 있습니다.

## 자주 묻는 질문

**iTunes 없이 iPhone으로 만화를 무선으로 전송하는 방법은?**
BiblioFuse에서 설정 → Wi-Fi 가져오기로 이동하여 켭니다. 로컬 URL이 나타납니다. 같은 Wi-Fi 네트워크의 Mac이나 PC 브라우저에서 해당 URL을 열고 CBZ, CBR 또는 EPUB 파일을 브라우저 창에 드래그하세요. 파일이 라이브러리에 자동으로 들어옵니다.

**BiblioFuse Wi-Fi 전송 속도는 얼마나 빠른가요?**
현대 802.11ac(Wi-Fi 5) 가정용 네트워크에서 15–40 MB/s를 기대할 수 있습니다. 150 MB 만화 한 권은 약 5초, 10 GB 시리즈는 약 5분 정도 걸립니다. 속도는 라우터와 네트워크 혼잡도에 따라 다릅니다.

**시리즈 폴더 전체를 한 번에 전송할 수 있나요?**
그렇습니다. 브라우저의 드롭 영역에 폴더를 드래그하면 BiblioFuse가 폴더 내 모든 파일을 받아 폴더 이름을 라이브러리에 유지합니다. 40개 챕터가 들어있는 폴더는 그 폴더 이름 아래 정리되어 도착합니다.

**집 네트워크 밖에서도 Wi-Fi 전송이 작동하나요?**
표준 Wi-Fi 전송은 두 기기가 같은 로컬 네트워크에 있어야 합니다. 원격 접근을 위해 Tailscale을 사용하면 Mac과 iPhone 사이에 개인 암호화 네트워크가 생성되어 인터넷 연결이 있는 어디서든 Wi-Fi 전송을 사용할 수 있습니다.

**전송이 완료되기 전에 멈추면 어떻게 되나요?**
화면이 잠기고 iOS가 BiblioFuse를 일시 중단하면 전송도 일시 정지됩니다. 이미 업로드된 파일은 안전합니다. BiblioFuse를 다시 열고 Wi-Fi 전송이 여전히 켜져 있는지 확인한 다음 업로드를 재시작하세요. 대부분의 브라우저가 미완료 파일을 재시도합니다.
