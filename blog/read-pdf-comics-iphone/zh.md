# 用 BiblioFuse 在 iPhone 上阅读 PDF 漫画

你有一批 PDF 格式的漫画——旧期号的扫描件、数字下载或朋友分享的文件——想在 iPhone 上轻松阅读，不想被繁琐的应用或订阅费用困扰。[BiblioFuse](https://bibliofuse.com/zh/) 原生支持 PDF 漫画，逐页阅读，操控体验与 CBZ、EPUB 文件完全一致。

## 为什么 PDF 漫画在 iPhone 上很麻烦

PDF 是通用格式，听起来很方便，但当你尝试在小屏幕上阅读一个 400 页的扫描文件时，问题就来了。iOS 内置的「文件」应用可以打开 PDF，但并非为漫画阅读而设计——没有专为序列艺术优化的全屏模式，没有阅读进度追踪，也没有自然的翻页手势。

大多数专用漫画阅读器只支持 CBZ 和 CBR，PDF 用户只能将就。

[BiblioFuse](https://bibliofuse.com/zh/) 填补了这一空缺。它将 PDF 漫画与其他格式一视同仁：全屏阅读、滑动翻页、放大镜头格，以及通过 iCloud 同步阅读进度。

## 如何将 PDF 漫画添加到 BiblioFuse

**方法一：Wi-Fi 从电脑传输**

打开 BiblioFuse，点击**工具** → **Wi-Fi 传输**。iPhone 会显示一个本地 URL，在 Mac 或 PC 的浏览器中打开它，将 PDF 文件拖入上传区域即可。无需云服务，全程在局域网内完成。

**方法二：从「文件」或 iCloud Drive 导入**

如果你的 PDF 已在 iCloud Drive 或 Dropbox 中：
1. 打开「文件」应用找到 PDF。
2. 长按文件，选择**共享**。
3. 在分享菜单中选择 **BiblioFuse**。

**方法三：从 Mac AirDrop**

在 Finder 中右键点击 PDF → **共享 → AirDrop**，选择你的 iPhone。

## 在 BiblioFuse 中阅读 PDF

点击库中的 PDF 即可打开。BiblioFuse 将每页以全屏显示：

- **左右滑动**翻页
- **双指捏合缩放**镜头格；双击恢复
- **点击屏幕中央**呼出快速导航栏

对于日式从右往左阅读的漫画，进入**设置 → 阅读方向**，切换为从右到左（RTL）。

## 跨设备进度同步

BiblioFuse 通过 iCloud 同步阅读位置。在 iPhone 上读到哪，iPad 上就从哪接着读，无需账号，无需服务器。

## PDF 还是 CBZ？

如果可以选择，CBZ 通常更好：页面以单独图片存储，加载更快，文件也可能更小。[BiblioFuse 网页工具](https://bibliofuse.com/zh/webapp/) 可在浏览器中免费将 PDF 转换为 CBZ，无需上传任何文件。

## 小结

PDF 漫画在 BiblioFuse 中与 CBZ、EPUB 体验完全一致：全屏手势阅读 + iCloud 进度同步。为获得最佳长期体验，建议用[免费工具](https://bibliofuse.com/zh/webapp/)将 PDF 转换为 CBZ。
