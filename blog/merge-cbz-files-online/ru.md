# Как объединить несколько файлов CBZ в один — бесплатный онлайн-инструмент

Вы скачали серию манги, и каждая глава пришла отдельным файлом CBZ. Теперь в папке 40 файлов — от Глава 001.cbz до Глава 040.cbz — и каждый раз, дочитав главу, приходится вручную открывать следующий файл. Было бы удобнее читать весь том как единый CBZ. Проблема в том, что для объединения, казалось бы, нужно устанавливать программы или платить за онлайн-сервис.

[BiblioFuse Web Tool](/ru/webapp/) объединяет файлы CBZ прямо в браузере, бесплатно. Ничего не загружается на сервер. Учётная запись не нужна. Готовый файл скачивается прямо на компьютер.

## Зачем объединять файлы CBZ

Схема «одна глава — один файл» широко распространена в манге и длинных сериях. Группы сканлейтеров упаковывают каждую главу отдельно. Цифровые магазины иногда продают выпуски поштучно, даже если вы купили всю серию. Читая на телефоне или планшете, приходится переключаться на следующий файл каждые 30 страниц — это сбивает ритм чтения.

Один объединённый CBZ для всего тома или арка даёт:

- **Непрерывное чтение** — ридер воспринимает его как единый файл и правильно обрабатывает разворотные страницы на стыке глав
- **Упорядоченную библиотеку** — один файл на том вместо десятков разрозненных глав
- **Единое отслеживание прогресса** — процент прочитанного и закладки работают для всего тома, а не сбрасываются с каждой главой
- **Удобный перенос** — один файл вместо множества

## Как работает объединение CBZ

CBZ — это ZIP-архив с изображениями, по одному на страницу, пронумерованными по порядку. Объединить файлы CBZ — значит соединить эти последовательности изображений в один ZIP, перенумеровав их так, чтобы история читалась непрерывно.

[BiblioFuse Web Tool](/ru/webapp/) выполняет перенумерацию автоматически. Вы указываете файлы и порядок их следования; инструмент извлекает изображения из каждого CBZ, выстраивает их в нужном порядке и упаковывает всё в один выходной CBZ.

## Как объединить файлы CBZ онлайн

**Шаг 1: Откройте BiblioFuse Web Tool**

Откройте [BiblioFuse Web Tool](/ru/webapp/) в браузере на компьютере. Регистрация и вход не требуются. Функция объединения доступна бесплатно для пакетов до трёх файлов; для больших пакетов нужно перейти на платный план.

**Шаг 2: Добавьте файлы CBZ в нужном порядке**

Нажмите на зону загрузки или перетащите на неё файлы CBZ. Порядок важен — инструмент объединяет файлы в той последовательности, в которой вы их добавляете. В большинстве браузеров можно выбрать несколько файлов сразу из папки; если ваши файлы названы по порядку (Глава 001.cbz, Глава 002.cbz и т. д.), выбор всех сразу обычно добавит их в правильном порядке.

Если нужно изменить порядок после добавления, можно перетащить файлы в нужную позицию в списке перед началом объединения.

**Шаг 3: Запустите объединение**

Нажмите «Объединить». Инструмент соединяет изображения из всех файлов CBZ в один архив, перенумеровывая страницы для непрерывной последовательности. Индикатор прогресса показывает ход выполнения. По завершении объединённый CBZ готов к скачиванию.

Имя выходного файла по умолчанию совпадает с именем первого файла в пакете — его можно изменить при сохранении.

## Разные форматы файлов

[BiblioFuse Web Tool](/ru/webapp/) умеет объединять не только CBZ. Если в вашей серии встречаются разные форматы — одни главы в CBZ, другие в CBR, третьи в ZIP — инструмент справится с ними вместе. Конвертировать всё в один формат перед объединением не нужно. Инструмент извлекает страницы из каждого входного формата и упаковывает результат как CBZ.

Это удобно для старых коллекций сканов, где единообразие форматов редко было приоритетом.

## Качество изображений после объединения

Объединение не затрагивает качество изображений. Инструмент соединяет файлы изображений из исходных архивов без перекодирования и сжатия. Каждая страница в объединённом CBZ идентична соответствующей странице в оригинальных файлах.

Если вы хотите одновременно с объединением уменьшить размер файла, в [BiblioFuse Web Tool](/ru/webapp/) есть шаг сжатия — его можно запустить до или после объединения, перекодируя изображения с выбранным уровнем качества. Но это необязательно; простое объединение сохраняет всё без изменений.

## Сборка томов из множества глав

Типичный сценарий — упаковать 10–12 глав в один CBZ размером с том. Удобный рабочий процесс:

1. Соберите главы, входящие в Том 1 — как правило, это главы 1–10 или 1–12.
2. Откройте [BiblioFuse Web Tool](/ru/webapp/) и добавьте эти файлы по порядку.
3. Запустите «Объединить». Скачайте результат и переименуйте, например, в `Название серии Том 01.cbz`.
4. Повторите для следующего тома.

Каждый итоговый файл — это аккуратный самостоятельный том, который можно перенести на ридер, добавить в библиотечное приложение или архивировать.

## После объединения: перенос файла на телефон

После скачивания объединённого CBZ самый удобный способ перенести его на iPhone — функция Wi-Fi Transfer в BiblioFuse.

Откройте [BiblioFuse](/ru/) на iPhone, нажмите кнопку Wi-Fi Transfer, и на экране появится адрес браузера. Откройте его в браузере на компьютере. Перетащите объединённый CBZ на страницу. Передача мгновенная через локальную сеть — без USB, без iTunes, без облачных аккаунтов.

Другие варианты:
- **iCloud Drive:** Поместите файл в папку iCloud Drive, связанную с BiblioFuse
- **AirDrop:** С Mac отправьте CBZ через AirDrop на iPhone и откройте в BiblioFuse
- **Приложение «Файлы»:** Подключитесь через USB и скопируйте в папку BiblioFuse в приложении «Файлы»

## Часто задаваемые вопросы

**Сколько файлов можно объединить?**

В бесплатном тарифе — до трёх файлов за раз. Для больших объединений можно разбить задачу на несколько проходов: сначала объедините главы 1–3 в промежуточный файл, затем объедините его со следующим пакетом и так далее.

**Что если главы в формате CBR?**

CBR — это RAR-архив, а не ZIP. [BiblioFuse Web Tool](/ru/webapp/) поддерживает CBR наряду с CBZ и другими форматами — можно добавлять смесь файлов CBZ и CBR, и они корректно объединятся в один CBZ.

**Сохраняется ли порядок чтения?**

Да. Инструмент обрабатывает файлы в том порядке, в котором они стоят в списке, а страницы внутри каждого файла добавляются в существующей последовательности. Если вы добавите главы в правильном порядке, результат будет читаться корректно.

**Можно ли точно так же объединять файлы EPUB?**

BiblioFuse Web Tool поддерживает объединение EPUB — оно реализовано как отдельная операция. Для контента в стиле комиксов — отсканированные страницы, а не текст с перетеканием — CBZ обычно предпочтительнее.

**Сохраняются ли метаданные из исходных файлов?**

Объединённый CBZ — это обычный архив изображений. Метаданные ComicInfo.xml из исходных файлов не объединяются автоматически. Если нужно сохранить метаданные, можно добавить ComicInfo.xml вручную после скачивания результата.
