# iPhone에서 비밀번호로 보호된 CBZ 및 ZIP 파일 여는 방법

SmartDecrypt PDF ZIP CBZ는 파일을 서버에 업로드하지 않고 iPhone, iPad, Mac에서 비밀번호로 보호된 CBZ 및 ZIP 아카이브를 직접 열 수 있습니다. 앱은 Rust 기반 엔진을 사용하여 기기에서 CBZ 만화 아카이브와 ZIP 파일을 복호화하고 잠금 해제된 복사본을 파일 앱에 저장합니다. 복호화 후 [BiblioFuse](/ko/)에서 CBZ 만화를 열어 iPhone에서 최상의 독서 경험을 즐기세요. App Store에서 SmartDecrypt PDF ZIP CBZ를 다운로드하면 몇 초 만에 아카이브 비밀번호를 제거할 수 있습니다.

## 비밀번호로 보호된 CBZ 파일이 대부분의 만화 리더에서 열리지 않는 이유

출판사, 팬 번역 그룹 또는 자신의 백업 시스템에서 CBZ 만화 아카이브를 다운로드합니다. 만화 리더에서 열려고 하면 첫 페이지 대신 비밀번호 입력 창이 나타나거나 아무것도 열리지 않습니다. BiblioFuse를 포함한 대부분의 iPhone 만화 리더는 일반 CBZ 및 CBR 파일을 문제없이 열 수 있지만, 암호화된 아카이브를 복호화하도록 설계되지는 않았습니다. 복호화는 파일이 리더에 도달하기 전에 수행해야 하는 별도 작업입니다.

바로 여기서 SmartDecrypt가 활약합니다. 만화 리더 앞 단계에서 작동합니다: 먼저 아카이브를 복호화한 다음 잠금 해제된 파일을 [BiblioFuse](/ko/) 또는 다른 앱에서 엽니다.

## 온라인 ZIP 복호화 서비스를 사용하지 말아야 하는 이유

빠른 검색을 하면 ZIP 및 CBZ 아카이브를 온라인으로 잠금 해제한다고 주장하는 수십 개의 서비스가 나옵니다. 워크플로는 편리해 보이지만 문제는 첫 번째 단계인 업로드에 있습니다.

만화 아카이브에는 종종 라이선스 콘텐츠가 포함되어 있습니다. ZIP 파일에는 개인 문서, 사진 또는 백업이 포함될 수 있습니다. 제3자 서버에 업로드하면 콘텐츠가 기기를 떠나 제어할 수 없는 인프라에 저장됩니다. 무료 서비스는 종종 파일을 보관합니다.

SmartDecrypt는 완전히 기기에서 실행됩니다. 업로드 없음, 네트워크 요청 없음, 파일에 대한 제3자 접근 없음.

## iPhone에서 비밀번호로 보호된 CBZ 또는 ZIP 파일 여는 방법

1. **SmartDecrypt PDF ZIP CBZ 설치** — iPhone 또는 iPad의 App Store에서.
2. **파일 찾기.** 파일 앱, 이메일 첨부 파일 또는 Safari 다운로드에서 보호된 CBZ 또는 ZIP을 찾습니다.
3. **SmartDecrypt에 공유.** 파일의 공유를 탭하고 공유 시트에서 SmartDecrypt를 선택합니다. 파일이 앱에 즉시 로드됩니다. SmartDecrypt를 직접 열어 파일을 가져올 수도 있습니다.
4. **비밀번호 입력.** 아카이브의 비밀번호를 입력합니다. 저장된 비밀번호 프로필이 있으면 목록에서 선택합니다.
5. **복호화 탭.** SmartDecrypt가 기기에서 아카이브를 처리하고 잠금 해제된 복사본을 파일 앱에 저장합니다. 원본 보호 파일은 변경되지 않습니다.
6. **리더에서 열기.** 복호화된 파일을 탭하고 [BiblioFuse](/ko/)를 선택하여 읽습니다. 비밀번호 없이 아카이브가 정상적으로 열립니다.
