# SmartDecrypt PDF ZIP Kini Tersedia dalam Bahasa Portugis, Spanyol, Indonesia, dan Mandarin

SmartDecrypt PDF ZIP CBZ kini mendukung bahasa Portugis, Spanyol, Bahasa Indonesia, dan Mandarin Sederhana — menghadirkan antarmuka aplikasi lengkap dalam empat bahasa baru. Aplikasi ini membuka kunci PDF, arsip ZIP, dan 7z yang dilindungi kata sandi di iPhone, iPad, Mac, dan Windows seluruhnya di perangkat, tanpa unggahan. [Unduh SmartDecrypt PDF ZIP](https://apps.apple.com/ca/app/smartdecrypt-pdf-zip/id6763979229) di App Store, atau dapatkan di Microsoft Store untuk Windows.

## Mengapa Bahasa-Bahasa Ini Penting

Hingga pembaruan ini, SmartDecrypt PDF ZIP hanya tersedia dalam bahasa Inggris. Artinya, para kolektor dan profesional di Brasil, Amerika Latin, Indonesia, dan Tiongkok menggunakan alat dekripsi yang sepenuhnya fungsional dalam bahasa yang bukan bahasa mereka. Pola kata sandi, label status antrean, panel preferensi — semuanya dalam bahasa Inggris.

Bagi seseorang yang mengelola ratusan laporan PDF yang dilindungi, arsip ZIP terenkripsi, atau file CBZ komik yang terkunci dengan nama file dalam bahasa Portugis, Spanyol, Indonesia, atau karakter Mandarin, hambatan ini nyata.

Pembaruan ini menghilangkan hambatan tersebut sepenuhnya.

## Yang Baru — 4 Paket Bahasa

Versi terbaru menambahkan lokalisasi UI lengkap untuk:

- **Portugis** (Português) — Portugis Brasil dan Eropa
- **Spanyol** (Español) — mencakup Amerika Latin dan Spanyol
- **Bahasa Indonesia** — untuk komunitas pengumpul arsip Indonesia yang besar
- **Mandarin Sederhana** (简体中文) — untuk pembaca manhua dan pengguna Tiongkok daratan

Setiap string yang terlihat di aplikasi diterjemahkan: menu, tombol toolbar, label antrean file, lencana status, dialog pencocokan pola, perintah peningkatan Pro, dan panel preferensi. Tidak ada yang tersisa dalam bahasa Inggris ketika Anda mengatur bahasa sistem ke salah satu dari empat bahasa ini.

Aplikasi secara otomatis mengikuti bahasa sistem operasi Anda. Di iOS, buka **Pengaturan → Umum → Bahasa & Wilayah** dan tambahkan bahasa pilihan Anda ke bagian atas. Di macOS, atur di **Preferensi Sistem → Umum → Bahasa & Wilayah**. Di Windows, ubah bahasa tampilan di **Pengaturan → Waktu & bahasa → Bahasa & wilayah**. Mulai ulang SmartDecrypt dan antarmuka langsung berubah — tanpa instal ulang, tanpa unduhan paket bahasa.

## Apa yang Dilakukan SmartDecrypt PDF ZIP

Jika Anda belum pernah menggunakannya, inilah yang sebenarnya dilakukan [SmartDecrypt PDF ZIP](https://apps.apple.com/ca/app/smartdecrypt-pdf-zip/id6763979229).

### Buka Kunci File yang Dilindungi Tanpa Mengetik Kata Sandi Setiap Kali

SmartDecrypt menyimpan profil kata sandi bernama menggunakan Apple Keychain. Anda membuat profil sekali, menautkannya ke pola nama file (misalnya, `rekening-*.pdf` atau `slip-gaji-*.zip`), dan aplikasi secara otomatis mencocokkan kata sandi yang tepat saat Anda menjatuhkan file. Tidak perlu salin-tempel dari aplikasi catatan. Tidak perlu mencari di email lama.

### Format File yang Didukung

- **PDF** — PDF yang dilindungi kata sandi dari bank, penyedia penggajian, lembaga pemerintah
- **ZIP** — arsip ZIP terenkripsi yang digunakan untuk unduhan komik, paket dokumen, dan transfer file
- **7z** — format 7-Zip sumber terbuka, umum untuk arsip besar

### Dekripsi Massal dengan Satu Ketukan

Dengan SmartDecrypt Pro (pembelian universal sekali untuk iOS dan macOS), Anda dapat memuat beberapa file sekaligus dan membiarkan aplikasi menelusuri profil yang disimpan secara otomatis. Jatuhkan 20 slip gaji, terapkan profil Anda, dan aplikasi memproses setiap file secara berurutan — sepenuhnya di perangkat.

### Penyimpanan Kata Sandi yang Aman di Keychain

Kata sandi disimpan di Apple Keychain di iOS dan macOS. Di Windows, disimpan di manajer kredensial sistem. SmartDecrypt tidak pernah menulis kata sandi ke file database atau mengirimkannya ke server.

### Pemrosesan 100% di Perangkat

Semua dekripsi berjalan secara lokal menggunakan inti Rust FFI. File Anda tidak pernah meninggalkan perangkat Anda.
