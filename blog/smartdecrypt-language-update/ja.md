# SmartDecrypt PDF ZIP がポルトガル語・スペイン語・インドネシア語・中国語に対応

SmartDecrypt PDF ZIP CBZ がポルトガル語、スペイン語、バハサ・インドネシア語、簡体字中国語に対応しました。4 つの新言語でアプリの完全な UI を提供します。iPhone、iPad、Mac、Windows でパスワード保護された PDF、ZIP、7z をすべてデバイス上で解除でき、ファイルのアップロードは不要です。App Store から [SmartDecrypt PDF ZIP をダウンロード](https://apps.apple.com/ca/app/smartdecrypt-pdf-zip/id6763979229)するか、Windows 向けに Microsoft Store から入手できます。

## なぜこれらの言語が重要だったのか

このアップデートまで、SmartDecrypt PDF ZIP は英語のみの対応でした。ブラジル、ラテンアメリカ、インドネシア、中国のコレクターや専門家は、自国語以外の言語で完全に機能する復号ツールを使用していました。パスワードパターン、キューのステータスラベル、設定パネル——すべて英語でした。

ポルトガル語、スペイン語、インドネシア語、または中国語の文字を使ったファイル名で、保護された PDF の明細書、暗号化された ZIP アーカイブ、ロックされた CBZ コミックファイルを何百も管理する人にとって、この障壁は現実的な問題でした。

このアップデートでそれを完全に解消します。

## 新機能——4 言語パック

最新バージョンでは以下の言語の完全な UI ローカライズが追加されました：

- **ポルトガル語**（Português）——ブラジル葡萄牙語とヨーロッパ葡萄牙語
- **スペイン語**（Español）——ラテンアメリカとスペインをカバー
- **バハサ・インドネシア語**——インドネシアの大規模なアーカイブ収集コミュニティ向け
- **簡体字中国語**（简体中文）——マンファ読者と中国大陸ユーザー向け

アプリ内のすべての表示文字列が翻訳されています：メニュー、ツールバーボタン、ファイルキューラベル、ステータスバッジ、パターンマッチダイアログ、Pro アップグレードのプロンプト、設定パネル。これら 4 つの言語のいずれかにシステム言語を設定すると、英語は残りません。

アプリは OS の言語設定に自動的に従います。iOS では **設定 → 一般 → 言語と地域** で優先言語をリストの先頭に移動します。macOS では **システム設定 → 一般 → 言語と地域** で設定します。Windows では **設定 → 時刻と言語 → 言語と地域** で表示言語を変更します。SmartDecrypt を再起動すると UI が即座に切り替わります。

## SmartDecrypt PDF ZIP でできること

まだ使ったことがない方のために、[SmartDecrypt PDF ZIP](https://apps.apple.com/ca/app/smartdecrypt-pdf-zip/id6763979229) の主な機能を紹介します。

### 毎回パスワードを入力せずに保護ファイルを解除

SmartDecrypt は Apple キーチェーンを使って名前付きパスワードプロファイルを保存します。プロファイルを一度作成してファイル名パターン（例：`明細-*.pdf` や `給与明細-*.zip`）に紐付けると、ファイルをドロップしたときに自動的に正しいパスワードが適用されます。

### 対応ファイル形式

- **PDF** — 銀行・給与プロバイダー・政府機関のパスワード保護 PDF
- **ZIP** — コミックのダウンロードやファイル転送に使われる暗号化 ZIP
- **7z** — 大規模アーカイブに一般的なオープンソースの 7-Zip 形式

### 1 タップでバッチ復号

SmartDecrypt Pro（iOS と macOS の買い切りユニバーサル購入）を使えば、複数のファイルを一度に読み込み、保存済みプロファイルを自動的に順番に試せます。20 枚の給与明細をドロップしてプロファイルを適用するだけで、すべてのファイルがデバイス上で処理されます。

### キーチェーンへの安全なパスワード保存

パスワードは iOS と macOS では Apple キーチェーン、Windows ではシステム資格情報マネージャーに保存されます。データベースファイルへの書き込みやサーバーへの送信は一切行いません。

### 100% デバイス上での処理

すべての復号は Rust FFI コアを使ってローカルで実行されます。ファイルはデバイスの外に出ません。
