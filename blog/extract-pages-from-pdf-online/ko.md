# PDF에서 페이지를 온라인으로 무료로 추출하는 방법

BiblioFuse 웹 도구를 사용하면 PDF에서 페이지를 온라인으로 추출할 수 있습니다. 브라우저에서 바로 작동하므로 소프트웨어 설치, 계정 생성, 서버에 파일 업로드가 전혀 필요하지 않습니다. BiblioFuse는 PDF, EPUB, CBZ 등 만화 및 전자책 형식을 지원합니다. 모든 처리는 WebAssembly(WASM)를 통해 사용자의 기기에서 완전히 로컬로 이루어지므로 문서의 프라이버시가 완벽하게 보호됩니다. 최신 브라우저에서 [BiblioFuse Web](https://bibliofuse.com/ko/webapp/)을 열고 몇 초 만에 PDF 페이지를 추출해 보세요.

200페이지짜리 스캔 만화 PDF가 있는데 참고용으로 15~30페이지만 필요하신가요? 아니면 PDF 형식으로 다운로드한 만화를 스마트폰에서 읽기 전에 챕터별 파일로 나누고 싶으신가요? 기존 선택지들은 불편하기만 합니다. 데스크톱 PDF 편집기는 유료이거나 구독이 필요하고, 클라우드 서비스는 파일을 제3자 서버에 업로드하며, 브라우저 확장 프로그램은 원하지 않는 광범위한 권한을 요구합니다. 개인 컬렉션에는 파일을 전혀 보지 않고 처리하는 도구가 명백한 선택입니다. [BiblioFuse Web](https://bibliofuse.com/ko/webapp/)이 바로 그렇게 만들어졌습니다.

## PDF에서 페이지를 온라인으로 추출할 때 기존 도구의 문제점은 무엇인가요?

대부분의 인기 있는 온라인 PDF 페이지 추출 도구는 문서를 외부 서버에 업로드해야 합니다. 강력한 개인정보 보호 정책이 있더라도 파일은 네트워크를 통해 전송되고, 처리 중에는 제3자 서버에 임시로 저장되며, 접근 로그에 나타날 수 있습니다. 개인 컬렉션, 실제로 소유한 책의 스캔본, 기밀 문서에는 이것이 용납되지 않습니다.

BiblioFuse는 전혀 다른 방식을 택합니다. 처리 엔진이 WASM 모듈로 브라우저에 한 번 다운로드되면, 이후 모든 처리는 로컬에서 이루어집니다. PDF 파일은 절대로 기기를 떠나지 않습니다. 브라우저의 네트워크 탭을 열어서 직접 확인해 보세요. 페이지가 로드된 후 추출 중에 파일 데이터가 브라우저를 떠나지 않는 것을 확인할 수 있습니다.

## BiblioFuse는 어떻게 PDF에서 페이지를 온라인으로 추출하나요?

BiblioFuse 웹 도구는 WebAssembly를 사용하여 브라우저 탭 내에서 PDF 처리 로직을 직접 실행합니다. 추출 도구를 열면 작은 WASM 번들이 서버에서 한 번 다운로드됩니다. 그 이후로는 모든 처리가 기기에서 로컬로 이루어집니다. PDF는 네트워크를 통해 전송되지 않습니다.

이것은 BiblioFuse의 모든 웹 도구에 적용되는 동일한 프라이버시 우선 아키텍처입니다. PDF 변환, CBZ 병합, EPUB 압축, ZIP에서 CBZ로 이름 변경이 모두 마찬가지입니다. 처리 엔진은 브라우저에서 실행되고, 서버는 도구 자체만 제공합니다.

## 단계별 가이드: PDF에서 페이지를 온라인으로 무료로 추출하기

### 1단계: BiblioFuse 웹 도구 열기

Chrome, Firefox, Safari 또는 Edge에서 [BiblioFuse Web](https://bibliofuse.com/ko/webapp/)에 접속합니다. 로그인, 가입, 확장 프로그램이 필요하지 않습니다.

### 2단계: PDF 페이지 추출 도구 선택

웹 앱 페이지에서 **PDF 페이지 추출** 도구를 찾아 클릭하여 인터페이스를 엽니다.

### 3단계: PDF 불러오기

**파일 선택**을 클릭하거나 PDF를 드롭 영역으로 드래그합니다. 파일은 브라우저의 File API를 통해 로드됩니다. 네트워크 요청 없이 로컬 저장소에서 직접 읽습니다.

### 4단계: 페이지 범위 지정

다음 형식 중 하나로 추출할 페이지를 입력합니다.

- 단일 페이지: `5`
- 연속 범위: `5–20`
- 혼합 목록: `1, 3, 7–15, 22`

범위와 개별 페이지를 필요에 따라 자유롭게 조합할 수 있습니다. 출력 PDF에는 지정한 순서대로 해당 페이지들이 포함됩니다.

### 5단계: 추출 및 다운로드

**추출**을 클릭합니다. WASM 엔진이 PDF 구조를 읽고, 요청된 페이지 객체를 새 문서에 복사하고, 브라우저 다운로드를 시작합니다. 일반적인 만화 PDF(50~300 MB)는 몇 초 안에 완료됩니다.

### 6단계: 추출된 PDF 사용하기

다운로드한 파일을 PDF 뷰어에서 엽니다. iPhone으로 옮기고 싶다면, BiblioFuse의 Wi-Fi 전송 기능을 사용해 Mac 브라우저에서 iPhone 라이브러리로 직접 드래그할 수 있습니다. 케이블이 필요 없습니다.

## 페이지를 추출할 때 무엇이 보존되나요?

BiblioFuse 추출기는 PDF 구조 수준에서 작동합니다. 스크린샷이 아닌 PDF 페이지 객체 자체를 복사합니다. 이것이 의미하는 바는 다음과 같습니다.

- **텍스트와 벡터 그래픽은 어떤 확대 배율에서도 선명하고 선택 가능한 상태로 유지**됩니다
- **내장된 폰트가 함께 이전**되어 추출된 파일에서도 텍스트가 동일하게 표시됩니다
- **스캔된 이미지의 해상도는 변경되지 않습니다** — 600 DPI 스캔은 추출 후에도 600 DPI입니다
- **추출된 페이지의 주석**은 출력 파일에 보존됩니다

이것은 'PDF로 인쇄'하는 추출 방법과 근본적으로 다릅니다. 후자는 각 페이지를 스크린샷하여 래스터 이미지로 내장하므로 OCR, 텍스트 검색 기능, 고배율에서의 품질이 모두 손실됩니다.

## PDF 페이지 추출의 주요 활용 사례

**스캔된 만화 권을 챕터별로 나누기**
스캔된 한 권에는 일반적으로 하나의 큰 PDF에 8~15개의 챕터가 포함되어 있습니다. PDF 뷰어의 썸네일 스트립을 사용하여 챕터 시작 페이지 번호를 확인하고, 각 범위에 대해 개별적으로 추출을 실행합니다. 챕터당 하나의 PDF 파일이 생성되어 BiblioFuse에서 시리즈별로 정리하기 훨씬 쉬워집니다.

**인쇄용 전페이지 일러스트 추출하기**
일부 만화 PDF에는 인쇄하거나 배경화면으로 사용하고 싶은 전페이지 컬러 일러스트가 포함되어 있습니다. 단일 페이지를 추출한 후 이미지 도구로 JPEG로 변환합니다. 추출 중에는 재인코딩이 발생하지 않습니다.

**스캔에서 빈 페이지나 손상된 페이지 제거하기**
스캔된 책에는 때로 처음과 끝에 빈 페이지가 포함되거나, 스캔 오류가 있는 몇 페이지가 있습니다. 유효한 범위만 추출하여 불필요한 페이지 없는 깔끔한 PDF를 만들 수 있습니다.

**챕터 발췌본 준비하기**
팬 번역 작업, 교정자와 챕터 공유, 독서 모임을 위한 미리보기 생성 등에 — 전체 권을 공유하는 대신 관련 페이지만 추출합니다.

**챕터를 권으로 재조합하기**
여러 파일에서 챕터 범위를 추출한 다음, BiblioFuse 웹 도구의 **PDF 병합** 기능을 사용하여 하나의 권으로 합칩니다. 품질 손실 없이 전체 컬렉션을 이런 방식으로 재구성할 수 있습니다.

## 프라이버시 상세 설명

서버 기반 PDF 도구는 제공업체가 관리하는 인프라에서 파일을 처리합니다. 강력한 개인정보 보호 정책이 있더라도 파일은 네트워크를 통해 전송되고, 처리 중에는 서버에 저장되며, 접근 로그에 나타날 수 있습니다. BiblioFuse Web은 이것을 완전히 방지합니다. WASM 번들이 한 번 다운로드되고, 이후 모든 작업은 로컬에서 이루어집니다. PDF를 받거나 저장하는 서버가 없습니다.

이것이 가장 중요한 경우는 다음과 같습니다.
- 실제로 소유한 책의 개인 스캔본
- 재구성 중인 세금 문서나 법적 계약서
- 개인 사용 라이선스 하의 독립 출판사 만화
- 제3자와 공유하고 싶지 않은 모든 문서

## 파일 크기와 브라우저 메모리는 어떻게 되나요?

부과된 파일 크기 제한은 없습니다. 제약은 브라우저의 사용 가능한 메모리입니다. 최신 데스크톱(Chrome, Firefox, Safari 또는 Edge)에서는 수백 MB의 PDF 파일이 일상적으로 처리됩니다. 500 MB 이상의 파일은 메모리를 확보하기 위해 다른 브라우저 탭을 닫은 상태에서 처리하는 것이 좋습니다.

모바일 브라우저(iOS의 Safari, Android의 Chrome)도 추출 도구를 실행할 수 있습니다. 대용량 PDF의 경우 데스크톱 브라우저가 메모리 처리를 더 안정적으로 합니다. 전형적인 단권 만화 PDF(50~200 MB)는 현재 iPhone 및 Android 기기에서 잘 처리됩니다.

## 자주 묻는 질문

**페이지를 추출하면 PDF 품질이 떨어지나요?**
아니요. BiblioFuse는 PDF 구조 수준에서 페이지를 추출합니다. 이미지 해상도, 벡터 그래픽, 내장된 폰트가 정확하게 보존됩니다. 재렌더링이나 래스터화는 발생하지 않습니다.

**연속되지 않은 페이지를 추출할 수 있나요?**
네. `1, 5, 10–15, 20`과 같이 쉼표로 구분된 목록을 입력하세요. 추출된 PDF에는 지정한 순서대로 해당 페이지들이 포함됩니다.

**PDF 페이지 추출 도구가 정말 무료인가요?**
네. BiblioFuse의 모든 웹 도구는 계정 없이 무료로 사용할 수 있습니다. 무료 티어는 배치당 최대 3개 항목을 처리합니다.

**PDF가 비밀번호로 보호되어 있으면 어떻게 하나요?**
추출 전에 비밀번호를 제거해야 합니다. SmartDecrypt(iPhone과 Mac에서 사용 가능)를 사용하여 파일을 복호화한 다음, 잠금이 해제된 PDF에서 페이지 추출기를 실행하세요.

**iPhone의 Safari에서 작동하나요?**
네. BiblioFuse 웹 도구는 iOS의 Safari에서 작동합니다. 대용량 PDF 파일의 경우 데스크톱 브라우저가 더 안정적이지만, 일반적인 만화 크기의 PDF는 iPhone에서도 잘 작동합니다.

**추출된 PDF에 내장된 텍스트와 OCR 레이어가 보존되나요?**
네. PDF에 내장된 텍스트 — 스캔 소프트웨어가 추가한 OCR 레이어를 포함하여 — 는 추출된 페이지에 보존됩니다.
