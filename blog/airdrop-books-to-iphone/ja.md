# AirDropを使ってiPhoneに漫画と電子書籍を転送する方法

BiblioFuseは、iOS 17以降を搭載したiPhoneとiPad上でAirDrop経由で共有されたファイルを受け取ることができます。CBZ、CBR、EPUB、PDF、ZIP、RAR、TXTファイルをAirDropで受け取った場合、共有シートから直接BiblioFuseで開くことができます。ファイルはBiblioFuseのライブラリに保存され、iCloud経由で他のAppleデバイスに自動的に同期されます。

## AirDropがiPhoneに漫画を取り込む最速手段のひとつである理由

AirDropはインターネット接続不要で、ファイルサイズの制限もなく、ローカルWi-FiまたはBluetoothの速度でファイルを転送します。送信元のMacやiPadの近くにいれば、400 MBのCBZ漫画ファイルが10秒以内にiPhoneに届きます。アカウントへのログインも、経由するクラウドサービスも不要で、転送はエンドツーエンドで暗号化されています。

漫画やマンガを読む方にとって、AirDropは自分のデバイス間でファイルを共有するときに特に便利です。Macで新しいマンガの巻をダウンロードして、すぐにiPhoneで読みたい場合、AirDropなら直接送れます。USBケーブルも不要、25 MBの制限があるメール添付も不要、大容量ファイルのiCloud Drive同期を待つ必要もありません。[BiblioFuse](/ja/)がファイルを受け取り、瞬時にライブラリに追加します。

## AirDropでiPhoneに漫画や電子書籍を送る手順

この操作は、Mac、iPad、または別のiPhoneなど、どのAppleデバイスからでも、BiblioFuseが入っているiPhoneに向けて行えます。

### Macからの場合

1. MacでFinderを開き、漫画ファイル（CBZ、EPUB、PDFなど）を探します。
2. ファイルを右クリックして**「共有」→「AirDrop」**を選択します。
3. 近くのAirDrop受信者リストからiPhoneを選びます。
4. iPhoneにファイルの受け取りを確認する通知が表示されます。**「承認」**をタップします。
5. iOSがどのアプリでファイルを開くか尋ねます。**BiblioFuse**をタップします。
6. ファイルがBiblioFuseにインポートされ、ライブラリに表示されます。

### iPadまたは別のiPhoneからの場合

1. **「ファイル」**アプリを開き、漫画ファイルを探します。
2. ファイルを長押しして**「共有」**をタップします。
3. **AirDrop**アイコンをタップし、iPhoneを選択します。
4. iPhoneでファイルを承認し、BiblioFuseで開きます。

### 漫画リーダーアプリから直接共有する場合

Mac上で別のリーダーアプリを使用していて、iPhoneにファイルを送りたい場合は、そのアプリからファイルをデスクトップにドラッグし、Finderを使ってAirDropで送るのが一般的です。あるいは、macOSの共有メニューに対応しているアプリであれば、**「共有」→「AirDrop」**を使うこともできます。

## 始める前にAirDropがオンになっていることを確認する

AirDropを使用するには、両方のデバイスでWi-FiとBluetoothが有効になっている必要があります。同じネットワークに接続している必要はなく、AirDropはデバイス間で直接ピアツーピア接続を確立します。安定した転送のために、両デバイスを約9メートル（30フィート）以内に保ってください。

**iPhoneの場合：**コントロールセンターを開き、ネットワークタイル（左上のグループ）を長押しします。**「AirDrop」**をタップして、好みに応じて**「10分間全員に公開」**または**「連絡先のみ」**を選択します。自分のMacから受け取る場合は、**「連絡先のみ」**が最も安全です。

**Macの場合：**Finderを開き、サイドバーで**「AirDrop」**をクリックし、ウィンドウ下部の**「自分を検出可能にする相手」**を**「全員」**または**「連絡先のみ」**に設定します。

## BiblioFuseにファイルが届いた後の動作

AirDropで送られたファイルを承認してBiblioFuseを送信先として選ぶと、BiblioFuseはファイルを内部のライブラリフォルダにコピーします。それ以降：

- ファイルはライブラリの該当セクション（漫画、電子書籍、またはすべてのファイル）に表示されます。
- iCloudが有効になっていれば、BiblioFuseが入っている他のAppleデバイスに自動的にファイルが同期されます。
- BiblioFuseはファイルの形式（CBZ、EPUB、PDFなど）を認識し、適切なリーダーエンジンで開きます。
- 追加した読書の進捗、ブックマーク、タグはiCloud経由でデバイス間で同期されます。

ファイルをフォルダに手動で移動したり、名前を変更したりする必要はありません。BiblioFuseが自動的にインポートを処理します。

## AirDropとWi-Fi Transfer：どちらを使うべきか？

BiblioFuseには、AirDropなしでもMacやPC（Windowsを含む）の任意のブラウザからファイルをドラッグできる**Wi-Fi Transfer**機能も内蔵されています。状況に応じた選び方は以下の通りです：

| 状況 | 最適な方法 |
|------|-----------|
| Mac、iPad、またはiPhoneから送信する | AirDrop |
| Windows PCから送信する | Wi-Fi Transfer |
| Linuxマシンから送信する | Wi-Fi Transfer |
| 一度に大量のファイルを送る | Wi-Fi Transfer（複数ファイルをドラッグ） |
| 自分のAppleデバイス間でファイルを素早く1件送る | AirDrop |
| 友人のデバイスから共有する | AirDrop |

Windowsユーザーの方は、iPhoneでBiblioFuseを開き、**「Wi-Fi Transfer」**ボタンをタップして、表示されたアドレスをPCのブラウザで開くだけです。PC側にアプリをインストールすることなく、複数のファイルを一度にドラッグ＆ドロップで送れます。

## よくある質問

### AirDropでCBZファイルをiPhoneのBiblioFuseに直接送れますか？

はい。iPhoneにCBZファイルをAirDropで送ると、iOSがどのアプリで開くかを尋ねる共有シートを表示します。BiblioFuseを選ぶと、ファイルが即座にライブラリにインポートされます。BiblioFuseはiOS 17以降でCBZ、CBR、EPUB、PDF、ZIP、RAR、TXTに対応しています。

### AirDropは大きな漫画ファイルにも対応していますか？

はい。AirDropにはファイルサイズの制限がなく、ローカルWi-Fiの速度で転送するため、大きなCBZやEPUBファイルにも適しています。デバイスが近くにあれば、300〜500 MBの漫画ファイルでも通常30秒以内に転送が完了します。

### iPhoneがAirDropの受信者として表示されないのはなぜですか？

よくある原因としては、iPhoneのAirDropが**「受信しない」**に設定されている、どちらかのデバイスでBluetoothが無効になっている、デバイスが離れすぎている、またはiPhoneの画面がロックされているなどが挙げられます。iPhoneのコントロールセンターでAirDropをオンにして、両デバイスを近くに置き、画面をロック解除した状態にしてください。

### AirDropで受け取ったEPUBをApple Booksではなくて BiblioFuseで開けますか？

はい。AirDropファイルを承認した後にiOSがアプリ選択画面を表示したら、**BiblioFuse**をタップしてください。共有シートのデフォルトがApple Booksになっている場合は、**「その他」**または**「別のアプリで開く」**をタップしてBiblioFuseを探してください。「ファイル」アプリからEPUBを開いてBiblioFuseを選ぶことで、EPUBのデフォルトアプリとして設定することもできます。

### Windows PCからiPhoneに漫画を送るにはどうすればいいですか？

AirDropはAppleデバイス間でのみ機能するため、Windowsからは使用できません。代わりにBiblioFuseの内蔵**Wi-Fi Transfer**機能を使ってください。iPhoneのBiblioFuseで転送アイコンをタップし、表示されたURLをWindowsのブラウザで開いて、ファイルをページにドラッグするだけです。

### MacからAirDropで送ったファイルをMacから削除しても、iPhoneに残りますか？

はい。BiblioFuseがファイルを受け取ってインポートした時点で、アプリのライブラリフォルダに独自のコピーを保存します。Macから元のファイルを削除しても、iPhone上のBiblioFuse内のコピーには影響しません。

## ケーブルなしで漫画読書をはじめよう

AirDropは、Appleデバイス間で漫画や電子書籍を移動する最速の方法です。BiblioFuseにファイルが届いたら、すぐに読み始められます。同期の遅延も、アカウントのログインも、余計な手順も一切ありません。

App Storeから[BiblioFuse](/ja/)をダウンロードして、ワイヤレス読書ライブラリを作り始めましょう。Windowsからの転送や大量ファイルの転送には、内蔵のWi-Fi Transfer機能がいつでも代替手段として使えます。
