# AirDrop으로 iPhone에 만화와 전자책 전송하는 방법

iPhone에 만화나 전자책 몇 권을 옮겨야 할 때, AirDrop이 가장 빠른 방법입니다. 케이블 없이, 추가 앱 없이, 서비스 로그인 없이. Mac에서 파일을 선택하고 AirDrop 목록에서 iPhone을 선택하면 [BiblioFuse](https://bibliofuse.com/ko/)에 몇 초 만에 직접 전달됩니다.

이 가이드는 AirDrop을 통한 CBZ, CBR, EPUB, PDF, ZIP 파일 전송, BiblioFuse에서 열리도록 파일을 올바르게 수락하는 방법, 그리고 AirDrop이 기기를 표시하지 않거나 전송이 멈출 때의 해결책을 다룹니다.

## AirDrop이 하는 일

AirDrop은 Wi-Fi Direct를 사용하여 암호화된 피어 투 피어 연결을 생성합니다. 기기가 같은 네트워크에 있을 필요가 없고, 아무것도 Apple 서버를 거치지 않습니다. 두 기기가 1미터 이내에 있으면 150MB CBZ 파일이 보통 3~6초 안에 전송됩니다.

iPhone이 만화 파일(CBZ, CBR)이나 전자책(EPUB, PDF)을 받으면 iOS가 어느 앱으로 열지 묻습니다. **BiblioFuse**를 탭하면 파일이 라이브러리에 직접 가져와집니다.

## 단계별: Mac에서 iPhone으로

### 1. 두 기기에서 AirDrop 활성화

**iPhone에서:** 제어 센터를 열고 네트워크 타일을 길게 누른 다음 **AirDrop**을 탭합니다. **연락처만** 또는 **10분 동안 모든 사람**을 선택합니다.

**Mac에서:** Finder를 열고 **이동 → AirDrop** (또는 Shift+Command+R)을 선택합니다. 창 하단에서 AirDrop 가시성을 설정합니다.

### 2. 파일 선택

Finder에서 만화나 전자책 파일로 이동합니다. Command+클릭 또는 Shift+클릭으로 여러 파일을 선택할 수 있습니다. 5~10개 파일씩 전송하는 것이 20개 이상을 한 번에 보내는 것보다 안정적입니다.

### 3. 전송 시작

선택한 파일을 우클릭하고 **공유 → AirDrop**을 선택합니다. 근처 기기가 표시되는 공유 시트가 열립니다. iPhone이 몇 초 내에 나타납니다. 클릭합니다.

iPhone에 파일을 수락할지 묻는 알림이 나타납니다. **수락**을 탭합니다.

### 4. BiblioFuse를 대상 앱으로 선택

수락 후 iOS가 각 파일을 어느 앱에서 열지 묻습니다. [BiblioFuse](https://bibliofuse.com/ko/)가 목록에 표시됩니다. 탭하면 파일이 직접 가져와집니다.

실수로 이 프롬프트를 닫으면 파일이 **파일 → 다운로드**에 있습니다. 거기서 탭하여 BiblioFuse로 엽니다.

## iPhone에서 iPhone으로 보내기

프로세스는 양방향으로 동일합니다. 보내는 iPhone에서:

1. **파일** 앱을 열고 만화 파일로 이동합니다.
2. 파일을 길게 누른 다음 **공유**를 탭합니다. 여러 파일: 선택 → 파일 선택 → 공유.
3. 공유 시트에서 **AirDrop**을 탭합니다.
4. 받는 iPhone을 선택합니다.

## 일반적인 문제 해결

**Mac의 AirDrop 목록에 iPhone이 나타나지 않음**

두 기기 모두 블루투스와 Wi-Fi가 켜져 있어야 합니다. 여전히 나타나지 않으면 iPhone에서 AirDrop을 껐다가 켜거나 Mac에서 블루투스를 껐다가 켭니다.

연락처만은 Mac과 iPhone이 연락처/Apple ID를 공유해야 합니다. 다른 Apple ID를 사용 중이라면 둘 다 **10분 동안 모든 사람**으로 설정합니다.

**전송이 "대기 중" 또는 "보내는 중"에서 멈춤**

기기를 더 가까이 이동시키고, 한 번에 더 적은 파일을 시도하고, 저전력 모드가 아닌지 확인합니다. 대용량 파일이 계속 실패하면 [BiblioFuse Wi-Fi 전송](https://bibliofuse.com/ko/)을 사용합니다.

|---------|------------|
| Mac 라이브러리에서 50개 이상 파일 | BiblioFuse Wi-Fi 전송 |
| 지속적인 동기화, 대규모 컬렉션 | [Mac 라이브러리 스트리밍](https://bibliofuse.com/ko/blog/mac-home-library-streaming-iphone/) |
| Windows 또는 Android에서 파일 | 파일 앱을 통한 USB 케이블 |

## 결론

AirDrop은 Apple 기기 간에 만화나 전자책을 전송하는 가장 빠른 방법입니다. 양쪽에서 AirDrop을 활성화하고, 파일을 선택하고, 공유 시트에서 iPhone을 클릭하고, 폰에서 수락하고, [BiblioFuse](https://bibliofuse.com/ko/)를 목적지로 선택합니다 — 대부분의 파일에서 전체 과정이 30초 미만입니다.
