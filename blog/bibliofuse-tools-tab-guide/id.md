# Menggunakan Tab Alat BiblioFuse: Kompres, Konversi, dan Gabungkan di iPhone

Tab Alat BiblioFuse memungkinkan kamu mengompres, mengonversi, dan menggabungkan file CBZ, EPUB, PDF, dan RAR langsung di iPhone atau iPad — tanpa komputer atau unggahan ke cloud. Alat Kompres mengodekan ulang gambar ke WebP, biasanya memotong ukuran file sebesar 60-75%. Konversi menangani perubahan PDF ke CBZ dan RAR ke CBZ, sementara Gabungkan mengombinasikan beberapa file bab menjadi satu arsip volume.

Kamu sedang istirahat makan siang, telepon di tangan, dan menyadari koleksi komik kamu menghabiskan penyimpanan iPhone. File CBZ dari seri manga itu masing-masing berukuran 80–120 MB. Kamu ingin mengompresnya, tapi laptop ada di rumah dan tidak mau menunggu sampai malam. Atau mungkin kamu punya beberapa file bab dari satu seri dan ingin menggabungkannya menjadi satu volume sebelum membaca panjang.

[BiblioFuse](/id/) menangani keduanya tanpa komputer. Tab Alat — yang terintegrasi dalam aplikasi iOS — memungkinkan kamu mengompres, mengonversi, dan menggabungkan komik dan ebook langsung di iPhone atau iPad. Semua berjalan di perangkat. Tidak ada file yang diunggah ke mana pun. Tidak perlu aplikasi desktop.

Panduan ini membahas setiap fungsi di tab Alat: apa yang dilakukan masing-masing, kapan menggunakannya, dan cara menjalankannya.

## Apa yang Dilakukan Tab Alat

Sebagian besar aplikasi pembaca komik bersifat hanya-baca. Kamu mengimpor file, membacanya, dan selesai. Kompresi dan konversi membutuhkan perangkat lunak desktop terpisah atau alat web.

BiblioFuse mengintegrasikan pemrosesan file langsung ke dalam pembaca. Artinya, aplikasi yang sama yang mengatur dan menampilkan perpustakaanmu juga bisa membentuk ulang file di dalamnya — memperkecilnya, mengubah formatnya, atau menggabungkannya — tanpa keluar dari iPhone.

Tab Alat menawarkan tiga fungsi utama:

- **Kompres** — mengurangi ukuran file dengan mengodekan ulang gambar di dalam CBZ atau EPUB
- **Konversi** — mengubah file dari satu format ke format lain (mis., CBZ ke EPUB)
- **Gabung** — menggabungkan beberapa file menjadi satu arsip

Setiap operasi bersifat non-destruktif secara default: [BiblioFuse](/id/) membuat file baru dan membiarkan aslinya tidak tersentuh.

## Kompres: Membuat File Lebih Kecil Tanpa Merusak Kualitas

Alat Kompres mengurangi ukuran file dengan mengodekan ulang gambar di dalam arsip kamu. Untuk file CBZ dan CBR — yang pada dasarnya merupakan arsip ZIP dari file gambar — ini bisa memotong ukuran file secara dramatis.

### Kapan Menggunakannya

- Perpustakaanmu mendekati batas penyimpanan dan kamu ingin membebaskan ruang tanpa menghapus konten
- Kamu mentransfer file melalui koneksi lambat dan ingin file lebih kecil
- Kamu mengunduh scan yang dikodekan dengan kualitas yang tidak perlu tinggi (300 DPI untuk pembaca layar, misalnya)

### Cara Mengompres File

1. Buka BiblioFuse dan pergi ke tab **Alat** (ikon kunci inggris di bagian bawah layar).
2. Ketuk **Kompres**.
3. Ketuk **Tambah File** dan pilih satu atau lebih file CBZ, CBR, atau EPUB dari perpustakaanmu.
4. Pilih pengaturan kompresi:
   - **Kualitas** — slider dari lossless hingga agresif. Untuk manga dan komik, kualitas 75–85% biasanya tidak terlihat di layar.
   - **Format** — pilih JPEG (lebih kecil) atau WebP (kompresi lebih baik pada kualitas yang sama).
   - **Dimensi maksimum** — secara opsional batasi resolusi gambar. Scan dengan lebar 3000px jarang terlihat lebih baik dari 1600px di layar ponsel.
5. Ketuk **Kompres** untuk memulai pemrosesan.

BiblioFuse menampilkan bilah kemajuan untuk setiap file. Selesai, versi yang dikompresi muncul di perpustakaanmu sebagai file baru. Aslinya tidak berubah.

### Seberapa Banyak yang Bisa Dihemat?

Hasilnya bervariasi tergantung bagaimana file sumber dikodekan. CBZ dengan gambar PNG resolusi tinggi sering kali bisa dikurangi 60–75% dengan mengonversi ke JPEG dengan kualitas 80%. File yang sudah dikompresi dengan pengaturan JPEG agresif tidak akan menyusut jauh lagi. Aplikasi menampilkan ukuran file sebelum dan sesudah sehingga kamu bisa memutuskan apakah akan menyimpan versi yang dikompresi.

## Konversi: Mengubah Format File di iPhone

Alat Konversi mengubah file dari satu format ke format lain. Tidak semua format cocok untuk setiap situasi, dan terkadang kamu akan menerima atau mengunduh file dalam format yang kurang nyaman dari yang kamu inginkan.

### Skenario Konversi Umum

**CBZ ke EPUB** — File CBZ adalah arsip gambar. Mengonversi ke EPUB membuat dokumen yang bisa mengalir jika file berisi teks, atau menyematkan gambar dalam paket EPUB yang ditangani secara berbeda oleh beberapa aplikasi membaca.

**PDF ke CBZ** — Jika kamu punya komik yang dipindai dan disimpan sebagai PDF (umum untuk scan lama), mengonversinya ke CBZ memisahkan setiap halaman menjadi file gambar individual. Ini sering memberikan kontrol yang lebih baik atas kualitas tampilan di pembaca.

**RAR ke CBZ** — RAR adalah format arsip terkompresi yang digunakan oleh beberapa scan lama. CBZ lebih didukung secara universal. Mengonversi dari RAR ke CBZ membuat file dapat dibaca di aplikasi yang tidak mendukung RAR.

### Cara Mengonversi File

1. Di tab Alat, ketuk **Konversi**.
2. Ketuk **Tambah File** dan pilih file yang ingin kamu konversi.
3. Pilih format target dari dropdown.
4. Ketuk **Konversi**.

File yang dikonversi disimpan ke perpustakaanmu. Aslinya tidak dihapus.

## Gabung: Menggabungkan Beberapa File Menjadi Satu

Alat Gabung mengambil beberapa file dan menggabungkannya menjadi satu arsip. Ini paling berguna untuk unduhan bab per bab yang seharusnya bersama sebagai satu volume.

### Kapan Menggunakannya

- Kamu mengunduh bab 1–12 dari seri manga secara terpisah dan ingin membacanya sebagai satu file
- Kamu punya arsip multi-bagian yang dibagi untuk diunduh dan perlu digabungkan kembali
- Kamu ingin mengonsolidasikan seri panjang menjadi volume untuk navigasi yang lebih mudah

### Cara Menggabungkan File

1. Di tab Alat, ketuk **Gabung**.
2. Ketuk **Tambah File** dan pilih file yang akan digabungkan. Kamu bisa menambahkan sebanyak yang dibutuhkan.
3. Atur urutannya — seret item dalam daftar untuk menempatkannya dalam urutan membaca. Ini penting: hasil penggabungan akan menggunakan urutan ini.
4. Beri nama file keluaran.
5. Ketuk **Gabung**.

BiblioFuse menggabungkan konten gambar dari semua file yang dipilih menjadi satu arsip CBZ dalam urutan yang kamu tentukan. File yang digabungkan muncul di perpustakaanmu. Aslinya tidak dihapus.

### Tips untuk Menggabungkan

**Periksa urutan sebelum menggabungkan.** Jika bab 7 muncul sebelum bab 4, file yang digabungkan akan salah dan kamu harus mengulangi operasi. Luangkan waktu sejenak untuk mengatur file dengan benar.

**Gunakan format sumber yang konsisten.** Menggabungkan file CBZ bersama bekerja dengan bersih. Mencampur format (beberapa CBZ, beberapa PDF) bisa menghasilkan hasil yang tidak terduga. Jika perlu menggabungkan format campuran, konversi ke CBZ terlebih dahulu.

**Gabung menghasilkan keluaran CBZ.** Terlepas dari format masukan, hasil yang digabungkan adalah arsip CBZ.

## Pemrosesan Di Perangkat: Apa Artinya Sebenarnya

Ketiga alat — Kompres, Konversi, dan Gabung — berjalan sepenuhnya di iPhone atau iPad kamu. Tidak ada server yang terlibat. File kamu tidak meninggalkan perangkat.

Ini penting karena dua alasan:

**Privasi.** Koleksi komik dan ebook bersifat pribadi. Beberapa orang memiliki koleksi besar konten yang mereka beli secara sah dari sumber yang tidak lagi ada, atau konten dari area abu-abu yang mereka lebih suka tidak kirim melalui server pihak ketiga.

**Keandalan.** Pemrosesan di perangkat bekerja tanpa koneksi jaringan. Di pesawat, di lokasi terpencil, atau di mana saja tanpa internet yang andal, tab Alat tetap berfungsi.

Kompromisnya adalah kecepatan. Prosesor iPhone yang kuat, namun mengompres 50 file CBZ beresolusi tinggi akan membutuhkan waktu lebih lama di ponsel daripada di desktop. BiblioFuse memproses file di latar belakang, sehingga kamu bisa terus membaca saat kompresi berjalan.

## Menyatukan Semuanya: Alur Kerja Nyata

Begini cara tab Alat masuk ke dalam rutinitas praktis:

Kamu mengunduh 15 bab seri manga baru, masing-masing sebagai file CBZ terpisah, masing-masing sekitar 60 MB. Kamu ingin:

1. **Menggabungkan** bab 1–5 menjadi Volume 1, bab 6–10 menjadi Volume 2, bab 11–15 menjadi Volume 3.
2. **Mengompres** setiap volume untuk mengurangi dampak penyimpanan.
3. Membaca Volume 1 sementara Volume 2 dan 3 masih dikompresi di latar belakang.

Semua ini terjadi di iPhone kamu, tanpa komputer, tanpa mengunggah apa pun. Dari awal hingga akhir: sekitar 10–15 menit tergantung ukuran file dan generasi perangkat keras.

Jika kamu telah menggunakan BiblioFuse terutama sebagai pembaca, tab Alat layak untuk dijelajahi. Ini mengubah iPhone kamu menjadi stasiun pemrosesan file yang mampu — tidak perlu desktop.

## Pertanyaan yang Sering Diajukan

**Apa yang dilakukan alat kompresi BiblioFuse?**
Alat kompresi mengodekan ulang gambar dalam file CBZ dan EPUB ke format WebP, biasanya mengurangi ukuran file 60-75% dengan sedikit atau tanpa kehilangan kualitas yang terlihat di layar iPhone atau iPad. Pemrosesan terjadi sepenuhnya di perangkat menggunakan perangkat keras kompresi gambar iPhone.

**Bisakah saya mengompres file komik di iPhone tanpa komputer?**
Ya. Tab Alat BiblioFuse berjalan sepenuhnya di perangkat pada iOS 17 atau lebih baru. Pilih file CBZ atau EPUB di perpustakaanmu, kirim ke antrian alat, pilih Kompres dan iPhone memproses file secara lokal. Tidak perlu desktop.

**Bagaimana penggabungan file CBZ bekerja di BiblioFuse?**
Masuk ke mode pilih dengan menekan lama file CBZ pertama, lalu ketuk bab lainnya untuk menambahkan. Ketuk tombol Alat di bilah bawah, pilih Gabungkan dan BiblioFuse menggabungkan file sesuai urutan yang kamu tentukan. Hasilnya masuk ke folder export di iPhone Saya -> BiblioFuse -> export.

**Format apa yang didukung alat kompresi BiblioFuse?**
Alat kompresi mendukung file CBZ dan EPUB. Alat konversi mengonversi PDF ke CBZ dan RAR ke CBZ. Alat gabung mengombinasikan beberapa CBZ menjadi satu file CBZ. PDF ke CBZ berguna untuk file PDF yang dipindai dengan gambar yang mendapat manfaat dari pengodean ulang WebP.

**Di mana file output setelah menggunakan tab Alat?**
Di iOS, file yang diproses muncul otomatis di iPhone Saya -> BiblioFuse -> export. Dari sana kamu bisa memindahkannya ke iCloud Drive -> BiblioFuse untuk sinkronisasi antar perangkat, atau berbagi ke app lain. Di Android, kamu memilih folder output saat menjalankan alat.
