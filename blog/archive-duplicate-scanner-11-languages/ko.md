# Comic Duplicate Scanner 이제 11개 언어 지원 — 중복인지 몰랐던 만화를 찾아보세요

대규모 만화 컬렉션을 관리하다 보면 중복 파일이 중복처럼 보이지 않는 경우가 있습니다. 하나의 파일은 `comic_E1-5.cbz`, 다른 하나는 `ComiX_E1-10.cbz`. 이름도 다르고 화수 범위도 달라 보이지만 — 안의 페이지는 겹칩니다. [Comic Duplicate Scanner](/ko/archive/)는 바로 이를 위해 만들어졌습니다. 파일을 열어 파일명이 아닌 페이지 이미지 자체를 비교합니다.

이 기능이 이제 11개 언어로 제공됩니다. Comic Duplicate Scanner는 영어, 스페인어, 프랑스어, 네덜란드어, 포르투갈어, 러시아어, 중국어 간체, 일본어, 한국어, 인도네시아어, 말레이어로 전체 UI 현지화를 완성했습니다.

## 파일명 기반 중복 감지의 한계

표준 중복 파일 찾기 도구는 이름, 크기 또는 해시를 비교합니다. 만화 압축 파일에서는 이 방식이 실패합니다:

- **같은 내용, 다른 이름**: `Spider-Man_001.cbz`와 `spiderman_issue01.cbz`는 같은 만화지만 파일명 비교로는 잡히지 않음
- **화수 범위 겹침**: `manga_vol1_ch1-5.cbz`와 `manga_vol1_ch3-8.cbz`는 서로 다른 압축 파일에 동일한 챕터를 포함
- **재인코딩 파일**: 다른 JPEG 품질로 재내보내기한 CBZ는 해시가 다르지만 페이지 내용은 동일

[Comic Duplicate Scanner](/ko/archive/)는 각 압축 파일에서 페이지 이미지 샘플을 추출해 지각 해싱으로 시각적으로 비교함으로써 이 문제를 우회합니다.

## 시각적 비교 작동 방식

Comic Duplicate Scanner는 각 CBZ, CBR, ZIP, RAR 압축 파일을 열어 첫 페이지들을 추출하고 각 페이지의 지각 해시를 생성합니다. 충분한 해시가 일치하면 파일명에 관계없이 해당 쌍이 잠재적 중복으로 표시됩니다.

일치 감도를 직접 조정할 수 있습니다. 엄격한 임계값은 더 가까운 시각적 일치를 요구하고, 느슨한 임계값은 경계선 케이스도 잡아냅니다.

Comic Duplicate Scanner는 스캔 및 일치 미리보기가 무료. Comic Duplicate Scanner Pro는 대량 이동/삭제 작업을 활성화하는 일회성 구매입니다. 구독 없음.
