v1.2.0 — 新手引导本地化 & Windows 语言支持

新手引导屏幕现已在所有 11 种支持语言中完全本地化。App Store Connect 元数据已更新，为所有 11 个地区提供本地化描述。Windows 应用现在包含语言设置，所有界面字符串已完全本地化——与 Mac 应用的 11 语言支持保持一致。

2026-05-061.1.7

v1.1.7 — macOS

修复了 macOS 上 App Store 本地化后 StoreKit 显示错误货币金额的问题。

2026-05-021.1.4
macOSRelease

v1.1.4 — macOS

在 macOS 上接入了免费增值访问流程：Mac App Store 版本现已启用 7 天试用限制、StoreKit 解锁和按功能的高级权限限制。

2026-04-221.1.0
macOSFix

v1.1.0–1.1.3 — 删除可靠性改进

将直接删除替换为 send2trash，使文件移入 macOS 废纸篓而非永久删除。修复了 NAS 回退 bug 和静默行删除问题。完成了跨平台按钮审计的全部正确性修复。

2026-04-221.0.6
macOSFeature

支持 NAS 和网络卷

通过 NSOpenPanel 扫描 NAS 和网络卷上的文件夹。首次启动时钥匙串提示从 4 次减少到 1 次。应用内购体验改进及试用逻辑强化。

2026-04-211.0.1
macOSFix

v1.0.1 — macOS

修复了首次启动时 StoreKit 购买未能恢复的问题。在"帮助"菜单中添加了支持邮件链接。

2026-04-211.0.0
macOSRelease

v1.0.0 — macOS 正式发布

Mac App Store 首发版本。针对漫画档案（CBZ/CBR/ZIP/RAR）和照片的感知哈希扫描。支持 Unicode 的文件名模糊匹配。智能自动选择、永久忽略对列表、哈希缓存（实现即时重新扫描）、多文件夹对比及 HTML/CSV 导出。

2026-04-22
PC (Windows)Release

Windows — 提交至 Microsoft Store

已上架 Microsoft Store。感知哈希扫描、照片去重、文件名模糊匹配、自动选择策略、哈希缓存、多文件夹对比和 HTML/CSV 导出。免费提供 7 天试用，一次性购买解锁无限制使用。
