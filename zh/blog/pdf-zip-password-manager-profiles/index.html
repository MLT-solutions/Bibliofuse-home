为 CBZ 和 PDF 收藏保存密码 — 再也不用重复输入

SmartDecrypt PDF ZIP CBZ 可在 iPhone、iPad 和 Mac 上存储命名密码配置文件，让你无需反复输入受保护的 PDF、CBZ 和 ZIP 文件的密码。只需创建一次配置文件，SmartDecrypt 在批量解密时便会自动应用，单次点击即可处理数百个文件，全程在设备本地完成，无需上传至云端。

每位漫画和 PDF 收藏爱好者都面临的烦恼

你花了多年时间建立起一个数字图书馆：来自 Kickstarter 众筹活动的漫画压缩包、PDF 画册、粉丝汉化版、订阅档案，以及各出版商发布的内容——其中许多都受到密码保护。每次收到新一批文件，你都要重复同样的流程：打开文件、输入密码、解密、再来一遍。

密码本身很少是问题所在，你知道它，甚至已经保存在备忘录应用里。真正的阻力来自机械重复：30 个文件意味着 30 次密码输入操作。对于那些定期到达的收藏——每月账单、每周新刊——这已经成了一项周而复始的琐事。

iPhone 上的大多数解密工具一次只能处理一个文件，而且在不同会话之间没有任何记忆功能。

SmartDecrypt PDF ZIP CBZ 通过密码配置文件解决了这一问题：为你常用的每个密码提供一个有名称、安全且完全本地化的存储空间。使用 SmartDecrypt，只需保存一次配置文件，以后再也不用输入那个密码。

密码配置文件是如何工作的？

SmartDecrypt 中的密码配置文件是一个命名条目，可在你的设备上安全存储密码。给它起一个描述性的名称——"X 出版社档案"、"2025 年银行账单"、"ABC 系列粉丝字幕组"——下次遇到来自同一来源的文件时，直接选择该配置文件即可，无需再次输入密码。

配置文件使用受设备安全飞地保护的本地安全存储。它们永远不会同步到任何云服务，永远不会离开你的设备，并且可以随时删除。你可以根据收藏数量创建任意数量的配置文件。

如何创建和使用密码配置文件

第一步：打开"配置文件"标签页

在 iPhone、iPad 或 Mac 上启动 SmartDecrypt，然后导航至底部工具栏中的"配置文件"标签页（Mac 上为侧边栏）。

第二步：添加新配置文件

点击 + 按钮（Mac 上点击"添加配置文件"），为该收藏输入一个描述性名称——确保它在列表中一眼可辨，然后输入密码。

第三步：保存并使用

点击"保存"，配置文件将立即出现在你的配置文件列表中。导入文件进行解密时，点击"选择配置文件"，无需手动输入密码。SmartDecrypt 会将已保存的密码应用到批次中的每一个文件。

第四步：随时管理配置文件

随时返回"配置文件"页面，对配置文件进行重命名、编辑或删除。当出版商更改档案密码时，只需几秒钟更新配置文件，此后所有新批次将自动使用新密码。

密码配置文件的实际应用场景

每月财务文件

银行和金融机构发送 PDF 时通常使用固定密码——往往是你的出生日期或账号。无需每月重新查找密码，只需将其保存为一次性配置文件。当每月的账单批次到达时，选择配置文件，一次点击解密所有文件。

订阅漫画档案

独立出版商和粉丝小组通常以订阅者密码分发 CBZ 档案，新刊每周或每月更新。保存的配置文件意味着下载后只需一次点击即可解密整批文件，让你的 BiblioFuse 图书馆始终保持最新状态，毫无摩擦。

多来源收藏

如果你有来自多个不同来源的文件，且每个来源使用不同的密码，则可以为每个来源创建单独的配置文件。选择来源 A 的文件，应用配置文件 A；选择来源 B 的文件，应用配置文件 B。你再也不用猜测哪个密码对应哪批文件。

在 Mac 上也能使用吗？

可以。SmartDecrypt PDF ZIP CBZ 在 iPhone、iPad 和 Mac 上均可使用，一次购买通用。在 Mac 上，你可以将整个文件夹的受密码保护文件拖入 SmartDecrypt，选择已保存的配置文件，一次性完成全部解密。

Mac 共享扩展与 Finder 无缝集成：在 Finder 中选中文件，右键点击，选择"共享"→ SmartDecrypt，选择配置文件，解密完成。整个操作流程只需三次点击，而且你在 iPhone 上创建的密码配置文件始终可用，无需任何额外设置。

常见问题解答

如何在 SmartDecrypt 中保存密码？ 打开 SmartDecrypt，进入"配置文件"标签页，点击 + 添加新配置文件，输入描述性名称和密码，然后点击"保存"。配置文件会安全存储在你的设备上，每次解密该收藏的文件时均可选用。

SmartDecrypt 中保存的密码安全吗？ 安全。SmartDecrypt 使用受设备安全飞地保护的本地安全存储来保存配置文件。密码永远不会被上传，不会与任何云服务同步，也不会离开你的设备。只有你才能访问它们。

我可以为不同的收藏创建多个密码配置文件吗？ 可以。SmartDecrypt 对已保存的配置文件数量没有限制，你可以为每个出版商、每部系列、每个文件来源各创建一个配置文件，数量随意。

SmartDecrypt 会自动将配置文件应用到匹配的文件吗？ SmartDecrypt 不会根据文件名自动匹配配置文件——你需要在解密步骤中主动选择使用哪个配置文件。这让你拥有完全的掌控权：选定文件批次，选定配置文件，然后一键全部解密。

我可以编辑或删除已保存的密码配置文件吗？ 可以。进入"配置文件"标签页，点击要更改的配置文件，然后进行编辑或删除。编辑配置文件会更新保存的密码以供日后使用，不会影响已解密的文件。

密码配置文件也适用于 ZIP 和 CBZ 文件吗？ 适用。SmartDecrypt PDF ZIP CBZ 支持三种格式的密码配置文件。只要文件共享相同的密码，你就可以使用单个配置文件批量解密由 PDF、CBZ 和 ZIP 混合组成的收藏。

再也不用两次输入同一个密码

密码配置文件将 SmartDecrypt PDF ZIP CBZ 变成了一个"一次设置、永久生效"的解密系统。新收藏到来时创建一次配置文件，此后来自同一来源的所有批次只需一次点击即可完成解密——无论是 iPhone、iPad 还是 Mac，始终在本地处理，无需上传。在 App Store 搜索 SmartDecrypt PDF ZIP CBZ 立即开始使用。
