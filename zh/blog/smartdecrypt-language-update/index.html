SmartDecrypt PDF ZIP 现已支持葡萄牙语、西班牙语、印尼语和中文

SmartDecrypt PDF ZIP CBZ 现已支持葡萄牙语、西班牙语、印尼语（Bahasa Indonesia）和简体中文——将完整的应用界面带到四种新语言。该应用可在 iPhone、iPad、Mac 和 Windows 上完全本地解锁受密码保护的 PDF、ZIP 和 7z 文件，无需上传。现在即可从 App Store 下载 SmartDecrypt PDF ZIP，或从 Microsoft Store 获取 Windows 版本。 访问 SmartDecrypt 页面，了解完整功能。

为什么这些语言如此重要

在此次更新之前，SmartDecrypt PDF ZIP 仅提供英文界面。这意味着巴西、拉丁美洲、印度尼西亚和中国的收藏家和专业人士一直在使用一款功能完整的解密工具，却不得不面对不熟悉的语言界面。密码模式、队列状态标签、偏好设置面板——全部都是英文。

对于管理数百个受保护的 PDF 账单、加密的 ZIP 存档或含有中文文件名的 CBZ 漫画文件的用户来说，这种障碍切实存在。

此次更新彻底消除了这一障碍。

新增内容——4 种语言包

最新版本新增了以下语言的完整界面本地化：

  • 葡萄牙语（Português）——巴西葡萄牙语和欧洲葡萄牙语
  • 西班牙语（Español）——覆盖拉丁美洲和西班牙
  • 印尼语（Bahasa Indonesia）——面向印度尼西亚庞大的存档收藏社区
  • 简体中文（简体中文）——面向漫画读者和中国大陆用户

应用中所有可见的字符串均已翻译：菜单、工具栏按钮、文件队列标签、状态徽章、模式匹配对话框、Pro 升级提示和偏好设置面板。当您将系统语言设置为这四种语言之一时，界面不会残留任何英文内容。

应用会自动跟随操作系统语言设置。在 iOS 上，前往设置 → 通用 → 语言与地区，将您的首选语言移至列表顶部。在 macOS 上，在系统设置 → 通用 → 语言与地区中进行设置。在 Windows 上，在设置 → 时间和语言 → 语言和地区中更改显示语言。重启 SmartDecrypt 后界面立即切换——无需重新安装，无需下载语言包。

SmartDecrypt PDF ZIP 能做什么

如果您还没有使用过，下面是 SmartDecrypt PDF ZIP 的核心功能介绍。

无需每次输入密码即可解锁受保护文件

SmartDecrypt 使用 Apple 钥匙串存储命名密码配置文件。您只需创建一次配置文件，将其与文件名模式关联（例如 账单-*.pdf 或 工资条-*.zip），当您拖入文件时，应用会自动匹配正确的密码。无需从备忘录复制粘贴，无需翻找旧邮件。

支持的文件格式

  • PDF — 来自银行、薪资服务商、政府机构的受密码保护 PDF
  • ZIP — 用于漫画下载、文档包和文件传输的加密 ZIP 存档
  • 7z — 开源 7-Zip 格式，常用于大型存档

一键批量解密

使用 SmartDecrypt Pro（iOS 和 macOS 的一次性通用购买），您可以一次加载多个文件，让应用自动遍历已保存的配置文件。拖入 20 份工资单，应用一个配置文件，应用将依次处理每个文件——完全在本地进行，不向任何服务器发送任何内容。

密码安全存储在钥匙串中

密码在 iOS 和 macOS 上存储于 Apple 钥匙串中。在 Windows 上，存储于系统凭据管理器中。SmartDecrypt 从不将密码写入数据库文件或发送至服务器。

100% 本地处理

所有解密使用 Rust FFI 内核在本地运行。您的文件永远不会离开您的设备。
