在 iPhone 上用 OCR 提取漫画和漫画文字

BiblioFuse 内置设备端 OCR，可从 iPhone 上的漫画和漫画页面提取文字，支持日语（含竖排文字）、中文、韩语和西方文字，适用于 CBZ、CBR、EPUB 和 PDF 文件。整个过程完全在神经网络引擎上本地运行，无需网络连接或上传服务器，在 iPhone 14 或更新型号上可在两秒内完成识别。

你正在读第三册漫画，主角突然提到一个用日文写的门派名称，你想查查汉字的含义。或者你遇到了一句精彩对白，想保存下来。问题是：漫画页面是图片，不是文字——你无法直接点按气泡框来复制内容。

BiblioFuse 的 OCR 功能解决了这个问题。它可以扫描任意页面或选定区域，提取其中的文字，以可复制的字符串形式呈现给你。支持 CBZ、CBR、EPUB 和 PDF 格式，兼容日文、中文、韩文及西文字符。所有处理均在设备本地完成，无需联网。

为何漫画 OCR 识别难度更高

普通文档（收据、带嵌入字体的 PDF）的 OCR 早已是成熟技术。漫画页面更难识别，原因有三：

文字置于有弧形边缘的气泡框中，叠加在带有复杂线条的插图上，背景干扰大，容易混淆 OCR 系统。

字体风格化或手绘，尤其是西方漫画，常用自定义字体模拟手写，或为强调效果而变形字母。

漫画有独特的排版惯例，日文漫画文字通常为竖排、从右到左，紧密填充在窄小的气泡框中。

BiblioFuse 针对漫画和漫画版式专门训练 OCR 模型，分别为横排西文和竖排 CJK（中文、日文、韩文）文字建立独立模型，识别效果明显优于通用文档扫描。

提取文字的用途

  • 查找外语词汇：读日文漫画时，提取对话文字，粘贴到词典或翻译应用中查询。
  • 保存引言：精彩对白、旁白、章节标题，一键复制。
  • 在系列中检索：提取人物姓名或专有名词，在其他卷中搜索出处。
  • 无障碍阅读：将扫描版漫画中的文字提取出来，输入屏幕阅读器或文字转语音应用。
  • 阅读未授权翻译：对于从未正式翻译的漫画，OCR + 机器翻译可提供基本的阅读体验。

如何在 BiblioFuse 中使用 OCR

OCR 内置于阅读器界面，无需切换模式。

扫描整页：

  1. 阅读时点按页面中央，调出工具栏。
  2. 点按工具栏中的 OCR 按钮（文本光标图标）。
  3. BiblioFuse 扫描当前页面，在底部面板显示提取的文字。
  4. 点按任意文字块即可复制，或使用全部复制。

扫描指定区域：

  1. 长按页面任意位置。
  2. 拖动绘制选择框，圈住目标区域（气泡框、旁白框或标题）。
  3. 松开手指，BiblioFuse 扫描该区域并显示结果。
  4. 点按复制或分享。

对于内容密集的页面或只需要单个气泡框内文字的情况，区域选择模式更为准确。

语言检测与 CJK 支持

BiblioFuse 在大多数情况下会自动检测字符类型。日文竖排文字可正确处理——BiblioFuse 会重建从上到下、从右到左的列式阅读顺序，以正确顺序呈现提取的文字。

对于繁体中文和简体中文，BiblioFuse 会在 OCR 输出中区分字符集。韩国漫画（manhwa） 通常使用横排文字，由与其他 CJK 文字相同的处理流程处理。

识别精度

  • 高分辨率扫描（600 DPI 以上）：精度极高。
  • 标准版本（300 DPI）：干净文字精度高；风格化字体偶有错误。
  • 低分辨率或高压缩扫描：JPEG 失真会影响精度。可先在 BiblioFuse 的阅读设置中开启超分辨率放大，再运行 OCR。
  • 手绘字体：变化较大。印刷体识别效果好于手写体。

提取后的操作

BiblioFuse 的文字提取面板支持 iOS 标准分享操作：

  • 复制到剪贴板，粘贴到任意应用
  • 直接分享至翻译应用、备忘录或通讯应用
  • 在 Safari 中搜索

日文文字可直接在 iOS 系统词典中查找，将提取的汉字复制到剪贴板后，打开"翻译"应用或词典应用即可查询。

使用技巧

用区域选择框选单个气泡：整页扫描提取所有内容，但气泡密集时，选框单个气泡可获得更干净的结果。

低画质扫描先开启超分辨率：在阅读器设置中开启图像放大，可提升 OCR 输入图像的清晰度。

读漫画时保持手机竖直：竖排文字 OCR 对旋转角度敏感，轻微倾斜可能影响列顺序重建。

OCR 在本地运行

BiblioFuse 的所有 OCR 处理均在你的 iPhone 上本地完成，页面不会上传至任何服务器，无任何外部 API 调用。这对于包含个人文件或私密内容的库尤为重要。

立即开始

OCR 功能已内置于 BiblioFuse iOS 版中，无需额外购买，可在 App Store 免费下载。导入一本漫画或漫画，在阅读时点按 OCR 按钮即可体验。对于漫画爱好者来说，这是一个用过就难以割舍的功能。

常见问题

BiblioFuse 的 OCR 支持哪些语言？ BiblioFuse 支持日语（含从上到下的竖排文字）、简体和繁体中文、韩语，以及西方文字（拉丁、西里尔、希腊字母）。语言会自动识别，但如果自动识别在混合页面上效果不佳，可以在 OCR 设置中手动指定语言。

BiblioFuse 中 OCR 如何使用？ 阅读漫画或漫画时，点按阅读器控件中的 OCR 按钮。出现选择框后，将其拖到任意分格或对话气泡上。BiblioFuse 在神经网络引擎上本地处理选定区域并返回提取的文字。点按文字即可复制或分享。

OCR 需要网络连接吗？ 不需要。BiblioFuse OCR 完全在设备上使用 Apple 神经网络引擎运行，不会向外部服务器上传任何内容。可在飞行模式、无流量、无 Wi-Fi 的情况下正常使用。

为什么 BiblioFuse OCR 识别日语比 iPhone 相机更好？ BiblioFuse 专为漫画页面文字优化：气泡框中的小字、漫画的竖排文字、艺术字体和倾斜文字。系统 Vision 应用未针对此场景调优。BiblioFuse 还按阅读顺序保留文字，因此复制粘贴到词典 App 只需一次点按。

BiblioFuse OCR 支持哪些文件格式？ OCR 适用于 CBZ、CBR、EPUB 和 PDF——BiblioFuse 支持的所有主要漫画和电子书格式。可以从任何以图片渲染的页面提取文字，包括 PDF 和 EPUB 中的扫描页面。
