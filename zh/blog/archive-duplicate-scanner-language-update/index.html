Comic Duplicate Scanner 现已支持葡萄牙语、西班牙语、印尼语和简体中文

Comic Duplicate Scanner 现已支持葡萄牙语、西班牙语、印度尼西亚语和简体中文，将完整的应用界面带入四种新语言。该应用使用感知哈希技术查找重复的漫画压缩包（CBZ、CBR、ZIP、RAR）和照片，Mac 和 Windows 均可免费使用。好消息是：应用现在支持简体中文。立即下载，用你的语言体验。

国际用户为何需要这次更新

管理大型漫画或漫画集合本就不易，更不用说还要面对语言障碍。此前，中国大陆、新加坡和马来西亚的用户不得不用英文界面操作 Comic Duplicate Scanner——即便他们通过中文社区发现了这款应用。菜单是英文的，扫描结果标签是英文的，导出报告也是英文的。

对于管理数万份中文漫画（漫画/manhwa）文件的用户来说，这种不便是实实在在的。这次更新彻底解决了这个问题。

新增内容——4种新语言包

最新版 Comic Duplicate Scanner 新增了以下语言的完整界面本地化：

  • 简体中文（你现在正在使用！）
  • 葡萄牙语（Português）
  • 西班牙语（Español）
  • 印度尼西亚语（Bahasa Indonesia）

"完整界面支持"意味着应用中所有可见字符串均已翻译——菜单、工具栏按钮、结果表格的列标题、相似度分数标签、确认对话框、偏好设置面板，以及带缩略图的 HTML 导出报告。

应用会自动跟随操作系统的语言设置。在 macOS 上，前往系统设置 → 通用 → 语言与地区，将简体中文移至列表顶部。在 Windows 上，在设置 → 时间和语言 → 语言和地区中更改。重启 Comic Duplicate Scanner 后，界面即刻切换。

Comic Duplicate Scanner 的功能

Comic Duplicate Scanner 使用感知哈希技术检测重复文件。它打开每个压缩包（CBZ、CBR、ZIP、RAR），计算其中实际页面图像的感知哈希值。两个文件可以有完全不同的名称、不同的文件大小，甚至使用不同的压缩方式——只要页面看起来相同，它们就会被判定为重复。

除压缩包外，应用还支持照片去重：JPEG、PNG、HEIC、WEBP、BMP 和 GIF。

其他核心功能：

  • 模糊文件名匹配——无需打开文件即可识别"Spider-Man #001"与"spiderman_001"的匹配。完全支持 Unicode，中文文件名（汉字）历来均可正确处理。
  • 智能自动选择——按规则决定保留哪个副本：最大文件、最新、最旧或页数最多。
  • 仅安全删除——将文件移至 macOS 废纸篓或 Windows 回收站，绝不永久删除。
  • 哈希缓存（SQLite）——结果存储在本地。重新扫描文件夹时，只处理新增文件，速度极快。
  • 丰富导出——带缩略图和相似度分数的 CSV 及 HTML 报告。
  • 完全本地——无云端、无分析、无需账户。

谁将从此次更新中受益

中文漫画（漫画、manhwa、manhua）库可能增长极为迅速。中文文件名（汉字）此前已被 Unicode 感知的模糊匹配器正确处理。现在，整个界面——扫描结果、偏好设置面板、导出报告——均已支持简体中文。

中国大陆、新加坡和马来西亚使用简体中文的用户现在可以在完全本地化的环境中管理他们的漫画收藏。

常见问题

Comic Duplicate Scanner 支持中文文件名吗？ 是的。模糊文件名匹配完全支持 Unicode，始终能正确处理中文（汉字）文件名。这次新增的简体中文界面是额外的功能——底层匹配引擎早已支持中文内容。

如何将应用切换到简体中文？ 应用会自动跟随系统语言设置。在 macOS 上，在系统设置 → 通用 → 语言与地区中将简体中文设为首选语言。在 Windows 上，在设置 → 时间和语言中更改。更改设置后重启应用。

应用是免费的吗？ Comic Duplicate Scanner 对最多 1,000 个文件的库免费使用。对于更大的收藏，一次性购买 19.99 美元即可解锁无限扫描。Mac 版包含 7 天免费试用。

支持哪些文件格式？ 漫画压缩包：CBZ、CBR、ZIP、RAR。照片：JPEG、PNG、HEIC、WEBP、BMP、GIF。

我的数据会被发送到任何地方吗？ 不会。Comic Duplicate Scanner 完全在本地运行——您的文件和哈希值永远不会离开您的设备。无云端，无数据追踪，无需账户。
