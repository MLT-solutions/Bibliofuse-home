CBZ、CBR、RAR、ZIP——漫画压缩包格式哪种最好？

你下载了一批漫画，一半是 .cbz，几个是 .cbr，还有一个干脆是 .rar。打开阅读器，一切都能读——但你开始疑惑：格式真的有区别吗？需要统一转换成一种格式吗？它们之间到底有什么不同？

以下是主流漫画压缩包格式的清晰对比，以及为什么 BiblioFuse 能无缝打开所有格式。

这些格式的本质

CBZ、CBR、ZIP、RAR 都不是漫画专用格式——它们都是通用压缩包容器，里面只是一堆图片文件（通常是 JPEG 或 PNG，越来越多使用 WebP）。阅读器把压缩包在内存中解压后按顺序显示图片。

因此，格式选择几乎完全取决于压缩算法和兼容性，与漫画本身的功能无关。

CBZ——大多数收藏的最佳选择

CBZ 就是把 ZIP 文件改后缀名为 .cbz，仅此而已。解压一个 CBZ 文件，你得到的就是一个图片文件夹。

CBZ 是最佳默认格式的原因：

  • ZIP 是开放免授权格式，所有操作系统和工具原生支持
  • macOS、Windows、Linux 无需额外软件即可打开
  • 几乎所有漫画阅读器都支持 CBZ 的读写
  • 无专利或授权问题（不像 RAR）
  • 无需重新压缩整个包即可添加、删除或重排页面

压缩效率：ZIP 使用 Deflate 压缩，速度快但效率不是最高。对于已经压缩过的 JPEG 图片来说，CBZ 和 CBR 的文件大小差异通常在 5% 以内。PNG 图片的差距略大（10–20%），但根本原因在于 PNG 本身体积较大。

结论：从零开始建立收藏，用 CBZ。

CBR——兼容性好但技术上处于劣势

CBR 是把 RAR 文件改后缀名为 .cbr。RAR 是 WinRAR 作者 Eugene Roshal 开发的专有格式，压缩率略优于 ZIP，尤其对 PNG 等未压缩图片格式效果更明显。

CBR 的问题：

  • RAR 是专有格式，创建 RAR 文件需要授权版 WinRAR 或 Unix 版 RAR
  • 开源的 UnRAR 库只能读取，不能创建 RAR 文件
  • RAR 5 引入了新的头部格式，部分旧阅读器无法处理
  • 重新打包或编辑 CBR 需要额外工具

何时会遇到 CBR：大多数 CBR 文件来自 2000 年代初中期 WinRAR 流行的年代。新发布的漫画很少用 CBR，因为它比 CBZ 没有明显优势。

需要把 CBR 转换为 CBZ 吗？ 如果存储紧张且有大量 PNG 为主的 CBR，转换能节省空间。对于大多数 JPEG 格式的收藏，差异可以忽略不计，转换并不值得。BiblioFuse 原生支持 CBR，无需转换即可阅读。

RAR 和 ZIP——基础格式

普通 .rar 和 .zip 文件在漫画阅读器中的效果与 CBR 和 CBZ 完全相同，区别仅在于文件扩展名。部分扫描工具保存文件时使用原始压缩包扩展名而不是 .cbz 或 .cbr，BiblioFuse iOS 版直接支持 .zip 和 .rar 文件，无需重命名。

CB7 和 CBT——小众格式

CB7 是把 7-Zip 文件（.7z）改后缀名为 .cb7。7-Zip 的 LZMA2 压缩对文本等可压缩数据效果显著，但漫画多为已压缩的 JPEG，实际节省空间有限，且兼容性较差。

CBT 是把 TAR 文件改后缀名为 .cbt。TAR 不做任何压缩，只是打包文件。偶尔出现在 Linux 来源的压缩包中。两种格式普及度都极低，建议坚持用 CBZ。

PDF——不是压缩包，但很常见

PDF 是文档格式而非图片容器。漫画 PDF 使用嵌入图片，格式本身增加了布局开销，逐页编辑也需要专用工具。出版商的数字版漫画和扫描版漫画单行本经常以 PDF 形式出现。BiblioFuse 可以读取 PDF，但图片型 PDF 的阅读体验通常不如等效的 CBZ——跳转更慢，无逐页元数据，每张图片的文件体积也更大。

如需将漫画 PDF 转换为 CBZ，可使用 BiblioFuse 网页工具，完全在浏览器中运行，无需上传文件。

BiblioFuse 如何处理所有这些格式

BiblioFuse（iPhone 和 iPad 版） 原生支持 CBZ、CBR、ZIP、RAR、CB7、CBT、EPUB 和 PDF，无需在导入前转换收藏。

所有处理在设备本地完成，不会上传任何文件到服务器。对于大型多格式漫画库，BiblioFuse 按系列组织文件，通过 iCloud 同步追踪每本书的阅读进度——无论底层格式是 CBZ 还是 CBR。

BiblioFuse 的工具栏支持直接在 iPhone 上进行格式转换：选择 CBR 压缩包，点击转换，选择 CBZ——应用在应用内完成重新打包，无需离开。

实用建议

从零开始建立新收藏： 用 CBZ。开放、通用兼容、易于操作。

已有 CBR 收藏： 无需转换，除非有特定原因（磁盘空间、某个工具不支持 CBR）。BiblioFuse 和所有现代阅读器都完美支持 CBR。

混合格式书库： 不必为了统一而强制转换。像 BiblioFuse 这样支持多种格式的阅读器，让你无需维护单一格式的收藏。

出版商 PDF 漫画： 保持 PDF 格式即可，除非需要编辑或合并卷册，此时通过 BiblioFuse 网页工具转换为 CBZ 才有意义。

格式之争远没有拥有一个能处理所有格式的阅读器重要。有了这样的阅读器，格式就变得透明无感。
