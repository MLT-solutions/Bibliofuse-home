SmartDecrypt 现已支持 11 种语言

2026年6月28日

SmartDecrypt 现已支持 11 种语言 — 用自己的语言解锁银行 PDF

你的银行账单以加密 PDF 的形式到达。密码是你的身份证号、出生日期或账户后六位 — 你输过几十次却还是记不住。SmartDecrypt PDF ZIP 就是为此而生：只需保存一次密码规律，之后每张账单拖进来就自动解锁。

这一体验现在支持 11 种语言。SmartDecrypt PDF ZIP CBZ 将完整的界面本地化扩展至英语、西班牙语、法语、荷兰语、葡萄牙语、俄语、简体中文、日语、韩语、印度尼西亚语和马来语 — 覆盖逾 30 亿母语使用者。

为什么银行账单是核心使用场景

大多数人不会把自己定义为"解密工具用户"——直到银行开始每个月发来带密码的 PDF。

问题在于密码。银行、保险公司和政府机构通常使用个人识别信息作为密码 — 身份证号、出生日期或保单号。每家机构用的格式不同，没有一个与你平时的密码相同。

SmartDecrypt 通过文件名模式匹配解决这个问题。你创建一个配置：「凡是文件名匹配 招商银行_*.pdf 的，密码是我的身份证号」。下次这类文件到达时，拖进 SmartDecrypt，几秒内即可解锁。

应用内所有可见文字均已翻译 — 菜单、工具栏按钮、文件队列、状态标签、模式匹配对话框、Pro 升级提示和偏好设置面板。

SmartDecrypt 自动跟随设备语言。iOS：设置 → 通用 → 语言与地区。macOS：系统设置 → 通用 → 语言与地区。Windows：设置 → 时间和语言 → 语言和区域。

支持的文件格式

PDF — 银行、保险公司和政府机构用于账单的格式。SmartDecrypt 解除 PDF 保护并保存无密码副本。

ZIP — 加密 ZIP 压缩包，常用于文档包和下载内容。SmartDecrypt 在应用内直接解压。

7z — 开源 7-Zip 格式，常用于大型压缩包。完整支持加密和非加密文件。

模式匹配的工作原理

收到文件时，SmartDecrypt 将文件名与已保存的配置对比。创建一个包含名称、模式（如 平安保险_*.pdf）和密码的配置文件。匹配到文件时，SmartDecrypt 自动应用已保存的密码。

密码存储在 Apple 钥匙串或 Windows 凭据管理器中，绝不发送至任何服务器。

SmartDecrypt 免费支持每次解锁一个文件。SmartDecrypt Pro 是一次性通用购买，解锁批量处理功能。无需订阅。

前往 App Store 或 Microsoft Store 下载 SmartDecrypt PDF ZIP。11 种语言更新现已在所有平台上线。
