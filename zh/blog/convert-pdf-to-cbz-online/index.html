如何在线将 PDF 转换为 CBZ（无需软件，无需上传）

你下载了一部扫描版漫画，格式是 PDF。在 Preview 或 Acrobat 里打开没问题，但放到漫画阅读器里体验就差远了——翻页迟缓，缩放模糊，完全不是阅读漫画该有的感觉。你知道 CBZ 格式更适合漫画，却不想再装一个桌面软件，也不想把文件上传到不明网站。

BiblioFuse Web Tool 直接在浏览器里解决这个问题。无需注册，无需上传，无需安装。文件始终留在你的设备上。

为什么 PDF 不适合漫画

PDF 为文档排版而生，擅长处理字体、矢量图形和文字回流。但漫画本质上是一张张全页图片的序列，PDF 的内部结构对这些图片来说只会增加文件体积、拖慢渲染速度，还会妨碍阅读器提前缓存页面。

CBZ 则是一个包含有序编号图片文件的 ZIP 压缩包，几乎没有多余开销。阅读器可以提前解压并缓存图片，翻页即时响应，缩放质量完全取决于图片本身。

将 PDF 漫画转为 CBZ，本质上就是把 PDF 里的页面图片提取出来，重新打包成 ZIP。

BiblioFuse Web Tool 是如何工作的

BiblioFuse Web Tool 基于 WebAssembly 在浏览器内运行，能够完成原本只有桌面软件才能处理的文件转换任务。你的文件从本地磁盘直接读入浏览器内存，转换在浏览器标签页内完成，输出文件写回本地磁盘——全程没有任何数据经过服务器。

如果你的漫画涉及版权内容或私人文档，这一点尤为重要。

转换步骤

第一步：打开工具

在桌面浏览器（Chrome、Firefox、Safari、Edge 均可）访问 BiblioFuse Web Tool，无需登录。免费版每次最多处理三个文件。

第二步：添加 PDF 文件

点击上传区域，或直接将 PDF 文件拖入。可以同时添加多个文件。如果某个文件超出浏览器可用内存，转换前会有提示。

第三步：选择 CBZ 并开始转换

在输出格式中选择 CBZ，点击"转换"。工具会依次处理每个文件，完成后下载文件名与原文件相同，扩展名从 .pdf 变为 .cbz。

图片质量说明

  • 扫描版 PDF：直接提取原始扫描图片，无重新编码，无质量损失，CBZ 与 PDF 文件大小基本相当。
  • 矢量/排版 PDF：转换时会将页面栅格化，工具默认使用较高分辨率，正常阅读缩放下效果清晰。
  • 已压缩 PDF：若 PDF 创建时已做过 JPEG 压缩，提取的图片会保留原有压缩状态，工具不会叠加额外压缩。

批量转换与传输到手机

如需转换整套系列，可一次性导入所有 PDF，免费版支持每批最多三个文件，同一会话内可连续运行多批，无需刷新页面。

转换完成后，如果你在 iPhone 上使用 BiblioFuse，可通过内置的 Wi-Fi 传输功能，直接从电脑浏览器拖拽 CBZ 文件到应用的书库，无需 USB 或云端同步。也可通过 iCloud Drive 或 AirDrop 传输。

常见问题

PDF 同时包含扫描图层和 OCR 文字层，能处理吗？ 可以，工具会提取视觉图层，原始 PDF 不受影响。

文件大小有限制吗？ 无硬性限制，实际取决于浏览器可用内存，桌面浏览器通常可处理数 GB 的文件。

支持移动端浏览器吗？ 支持，但性能比桌面端慢，大文件建议使用桌面浏览器。

CBZ 内图片是什么格式？ 保持 PDF 原有格式，扫描版通常为 JPEG，不做额外转码。

立即打开 BiblioFuse Web Tool，把第一批 PDF 漫画拖进去开始转换吧。
