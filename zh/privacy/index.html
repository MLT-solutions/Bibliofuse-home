隐私政策

BiblioFuse 以本地处理、无广告、无追踪为核心。本政策说明哪些内容会保留在你的设备上，以及使用平台服务时可能涉及的有限数据。

隐私政策 - BiblioFuse Reader

生效日期: 2026-01-29

1. 简介

BiblioFuse Reader("应用程序")由Modern Logic Tech Solutions("我们"或"我们的")提供。我们优先考虑您的隐私和透明度。

2. 数据收集和云同步

我们不会在我们自己的服务器上存储您的个人文件、文档或书籍。

云同步:

该应用程序使用您的个人云存储账户促进您的内容(如EPUB、PDF和漫画书)在您的设备上进行同步:

  • iCloud (iOS/macOS): 文件存储在您的私有iCloud容器中。我们无法访问应用程序特定容器之外的这些文件。
  • Google Drive (Android): 文件存储在您的个人Google云端硬盘中。该应用程序仅访问您明确选择的文件或您指定用于同步的文件夹。

3. 付款和订阅

应用内购买:

该应用程序通过一次性购买("终身")和订阅("月度"、"年度")提供付费功能。

  • 付款处理: 所有金融交易直接由Apple App Store或Google Play商店处理。我们不收集、存储或访问您的信用卡信息或账单详情。
  • 交易数据: 我们可能会收到匿名交易收据以验证您的"高级"状态。这些数据不包括您的个人身份。

4. 第三方服务

该应用程序使用可能收集用于识别您的设备以实现功能和分析的信息的第三方服务:

  • Google Play Services
  • Apple StoreKit(用于购买验证)

5. 权限

该应用程序需要以下权限才能运行:

  • 存储访问: 需要在本地打开、读取和管理您的图书文件。
  • 网络访问: 需要将您的文件与您选择的云提供商(iCloud或Google Drive)同步,并验证高级许可证。

6. Windows 应用程序 (BiblioFuse for Windows)

Microsoft Store 付款:

Windows 应用程序提供一次性应用内购买("BiblioFuse Pro — 永久解锁"),由 Microsoft Store 专门处理。我们不收集、存储或访问您的付款详情。购买验证通过 Microsoft Store API (Windows.Services.Store) 执行,该 API 在您的设备和 Microsoft 之间直接通信。我们不接收任何个人身份信息或账单详情。

权限:

  • 本地文件访问: 需要在您的设备上打开、读取和显示电子书文件。
  • 本地网络访问: 需要将您的库流式传输到同一 Wi-Fi 网络上的 iOS 设备。流式传输完全可选,由用户启动。不会将任何数据传输到我们的服务器 — 所有流式传输都是本地网络上的点对点传输。

遥测和分析:

Windows 应用程序不收集遥测、崩溃报告或使用分析。关于您使用的任何数据都不会发送给我们或第三方。

7. 联系我们

如果您对本隐私政策有任何疑问,请通过以下方式联系我们:
