중복을 찾으세요. 이름을 바꿔도 찾아냅니다.

Comic Duplicate Scanner는 만화 아카이브를 열고 내부의 실제 이미지를 비교합니다 — 파일 이름이나 크기만이 아닙니다. 한 파일이 comic_E1-5이고 다른 파일이 ComiX_E1-10이라도 처음 몇 페이지가 일치하면 중복으로 표시됩니다.

심층 아카이브 스캔

아카이브를 열고 내부 페이지를 비교

대부분의 중복 검색 도구는 파일명이나 크기만 비교합니다. Comic Duplicate Scanner는 각 CBZ, CBR, ZIP 또는 RAR 파일을 열고 실제 페이지에서 썸네일을 추출해 비교합니다. comic_E1-5와 ComiX_E1-10 — 이름도 다르고 에피소드 범위도 달라도 — 시작 페이지가 일치하면 감지됩니다.

  • 아카이브를 열고 페이지 이미지를 비교 — 파일명만이 아님
  • 구성 가능한 유사도 임계값 — 90% 권장
  • 해시 캐시로 반복 스캔을 거의 즉시 완료
  • 해상도와 색상 모드가 달라도 중복을 감지합니다 — 동일 페이지의 1200px 흑백 스캔과 3000px 컬러 스캔도 중복으로 표시됩니다

사진 및 파일명 매칭

사진과 유사한 파일 이름 — 모두 지원

아카이브 외에도 dHash 알고리즘을 사용하여 중복 JPG, PNG, HEIC, WEBP, BMP, GIF 사진을 감지합니다. 파일명 퍼지 매칭은 형식만 다른 복사본 — "Spider-Man #001" vs "Spider Man 001" — 을 구성 가능한 임계값으로 감지합니다.

  • 사진 형식: JPG, PNG, HEIC, WEBP, BMP, GIF
  • 구성 가능한 기호 제거 기능이 있는 RapidFuzz 알고리즘
  • 유니코드 지원 — 일본어, 한국어, 중국어 및 모든 문자 집합에서 작동
스마트 자동 선택

한 번의 클릭으로 보관할 올바른 파일 표시

규칙 설정 — 가장 크거나, 가장 작거나, 가장 새로운 것, 가장 오래된 것, 또는 페이지 수가 가장 많은 파일 유지 — 자동 선택이 한 번의 클릭으로 모든 중복을 표시합니다. 아무것도 수정되기 전에 결과를 검토하세요. 확인할 때까지 아무것도 이동하지 않습니다.

  • 다섯 가지 규칙: 가장 큰 것, 가장 작은 것, 가장 새로운 것, 가장 오래된 것, 가장 많은 페이지
  • 마지막 복사본 안전 경고로 실수로 인한 전체 삭제 방지
  • 의도적인 변형 쌍을 위한 영구 무시 목록

스캔 결과를 스프레드시트용 CSV 또는 썸네일, 유사도 점수, 정렬 가능한 열이 포함된 완전한 HTML 보고서로 내보냅니다. 다중 폴더 모드는 두 개의 별도 폴더를 비교합니다 — 정리된 라이브러리와 다운로드 폴더 사이의 중복을 찾는 데 유용합니다.

FAQ

자주 묻는 질문

감지적 해싱이란 무엇이며 왜 중요한가요?

감지적 해싱은 원시 파일 바이트가 아닌 이미지의 시각적 구조를 비교합니다. 두 파일은 이름, 크기, 메타데이터가 달라도 내용이 같아 보이면 감지적 해싱이 일치시킵니다. 표준 중복 검색기는 파일 이름이 변경되거나 다시 내보내지면 실패하지만 Comic Duplicate Scanner는 그렇지 않습니다.

이름이 바뀐 중복 파일을 찾을 수 있나요?

네. 파일 이름을 바꾸는 것은 내용을 변경하지 않습니다. Comic Duplicate Scanner는 각 아카이브 내의 이미지를 비교하므로 파일 이름과 상관없이 중복이 감지됩니다.

삭제가 영구적인가요?

아니요. 표시된 파일은 시스템 휴지통(macOS) 또는 재활용 통(Windows)으로 이동되며 영구 삭제되지 않습니다. 휴지통을 비우기 전에 언제든지 파일을 복구할 수 있습니다.

어떤 파일 형식이 지원되나요?

만화 아카이브: CBZ, CBR, ZIP, RAR. 사진: JPG, PNG, HEIC, WEBP, BMP, GIF. 파일명 퍼지 매칭은 모든 파일 유형에서 작동합니다.

만화뿐만 아니라 사진도 작동하나요?

네. 사진 모드를 선택하여 JPG, PNG, HEIC, WEBP, BMP, GIF 파일을 스캔하세요. iPhone 사진 라이브러리 내보내기, 스캔된 문서, 디지털 아트 컬렉션에 작동합니다.

구매 전에 체험판이 있나요?

최대 1,000개 파일을 영원히 무료로 스캔할 수 있습니다. Mac App Store에서 7일 무료 체험으로 무제한 액세스가 해제되며, Pro 일회성 구매는 구독 없이 $19.99입니다.

컬러 스캔과 동일한 콘텐츠의 흑백 버전을 매칭할 수 있나요?

예. dHash 알고리즘은 색상 값이 아닌 밝기 그래디언트로 작동합니다. 동일한 페이지의 풀컬러 스캔과 흑백 버전은 거의 동일한 해시 값을 생성하여 중복으로 표시됩니다.

이미지 해상도가 중복 감지에 영향을 미치나요?

아니요. dHash는 비교 전에 모든 이미지를 작은 고정 그리드로 리사이즈하므로, 동일 페이지의 1200px와 3000px 스캔은 같은 해시 값을 생성합니다. 소스 간 해상도 차이는 완전히 무시됩니다.
