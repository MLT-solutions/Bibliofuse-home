Comic Duplicate Scanner

Release history, known issues, and what's coming next.

2026-06-261.2.0
All platformsRelease

v1.2.0 — 온보딩 현지화 & Windows 언어 지원

온보딩 화면이 지원되는 11개 언어 모두에서 완전히 현지화되었습니다. App Store Connect 메타데이터가 11개 지역의 현지화된 설명으로 업데이트되었습니다. Windows 앱에 이제 언어 설정이 포함되고 모든 UI 문자열이 완전히 현지화되어 Mac 앱의 11개 언어 지원과 일치합니다.

2026-05-061.1.7
macOSRelease

v1.1.7 — macOS

App Store 현지화 후 macOS에서 StoreKit이 잘못된 통화 금액을 표시하던 오류를 수정했습니다.

2026-05-021.1.4
macOSRelease

v1.1.4 — macOS

macOS에 프리미엄 접근 플로우를 적용했습니다. Mac App Store 빌드에서 7일 체험 제한, StoreKit 잠금 해제, 기능별 프리미엄 제한이 활성화되었습니다.

2026-04-221.1.0
macOSFix

v1.1.0–1.1.3 — 삭제 안정성 개선

직접 삭제를 send2trash로 교체하여 파일이 영구 삭제되지 않고 macOS 휴지통으로 이동하도록 했습니다. NAS 폴백 버그와 자동 행 삭제 문제를 수정했습니다. 크로스 플랫폼 버튼 감사 정확성 수정 완료.

2026-04-221.0.6
macOSFeature

NAS 및 네트워크 볼륨 지원

NSOpenPanel을 통해 NAS 드라이브 및 네트워크 볼륨의 폴더를 스캔합니다. 첫 실행 시 키체인 프롬프트를 4회에서 1회로 줄였습니다. IAP UX 개선 및 체험 로직 강화.

2026-04-211.0.1
macOSFix

v1.0.1 — macOS

첫 실행 시 StoreKit 구매가 복원되지 않던 문제를 수정했습니다. 도움말 메뉴에 지원 이메일 링크를 추가했습니다.

2026-04-211.0.0
macOSRelease

v1.0.0 — macOS 출시

Mac App Store 최초 출시. 만화 아카이브(CBZ/CBR/ZIP/RAR)와 사진을 위한 감각적 해시 스캔. 유니코드 지원 파일명 퍼지 매칭. 스마트 자동 선택, 영구 무시 쌍 목록, 즉시 재검색을 위한 해시 캐시, 다중 폴더 비교, HTML/CSV 내보내기.

2026-04-22
PC (Windows)Release

Windows — Microsoft Store 제출

Microsoft Store에서 이용 가능합니다. 감각적 해시 스캔, 사진 중복 제거, 파일명 퍼지 매칭, 자동 선택 전략, 해시 캐시, 다중 폴더 비교, HTML/CSV 내보내기 포함. 7일 무료 체험; 일회성 무제한 잠금 해제.
