작동 원리

3단계. 설정 필요 없음.

1

평소처럼 탐색

ContentCue에서 아무 웹사이트나 열고 평소대로 읽으세요. 사이트별 설정도, 계정도 필요 없습니다.

2

광고가 방해함

팝업이나 리다이렉트가 나타나 다른 페이지로 이동합니다. ContentCue가 그 순간 정확한 스크롤 위치를 자동으로 저장합니다.

3

이어서 읽기 탭

이어서 읽기 버튼이 나타납니다. 한 번 탭하면 읽던 정확한 줄로 돌아갑니다.

위치 복원

한 번의 탭. 정확한 위치로 돌아가기.

ContentCue는 페이지 방해 요소(팝업, 리다이렉트 배너, 실수로 광고 탭하기 등)를 감지하고 스크롤 위치를 자동으로 저장합니다. 위치가 저장되면 이어서 읽기 버튼이 나타납니다. 탭하면 원래 위치로 돌아갑니다.

  • 탐색할 때마다 자동으로 위치 저장
  • 위치가 저장되면 언제든 이어서 읽기 버튼 표시
  • 모든 콘텐츠 사이트에서 작동 — 사이트별 설정 불필요
시각적 광고 구분

콘텐츠와 광고 구분하기

ContentCue는 페이지 내 광고 주변에 색상 테두리를 그려서 읽는 콘텐츠와 즉시 구분할 수 있게 해줍니다. 이는 광고 차단이 아닙니다 — 모든 광고가 정상적으로 로드되고 표시됩니다. 단지 시각적 명확성을 제공할 뿐입니다.

  • 색상 테두리로 페이지 내 광고 강조
  • 모든 광고가 정상적으로 로드되고 표시됨
  • Pro에서 테두리 색상 커스터마이징

FAQ

자주 묻는 질문

ContentCue는 광고 차단기인가요?

아니요. ContentCue는 광고를 차단, 숨김 또는 로드하지 않습니다. 모든 광고가 정상적으로 표시됩니다. ContentCue는 광고가 읽기를 방해할 때 스크롤 위치를 저장하여 탐색을 개선하고, 페이지 내 광고 주변에 색상 테두리를 그려서 명확하게 보일 수 있도록 합니다.

ContentCue는 어떤 사이트에서 작동하나요?

ContentCue는 완전한 웹 브라우저이므로 모든 웹사이트에서 작동합니다 — Webtoons, Tapas, LINE Webtoon, 독립 작가 페이지, 만화 검색 사이트, WordPress 블로그 등 수천 개 사이트에서요. 사이트별 설정이 필요 없습니다.

위치 복원은 어떻게 작동하나요?

ContentCue는 읽는 동안 스크롤 위치를 모니터링합니다. 탐색할 때(광고 링크 탭, 리다이렉트 또는 뒤로 버튼), 정확한 위치를 저장합니다. 브라우저 UI에 이어서 읽기 버튼이 나타납니다. 탭하면 정확히 읽던 위치로 돌아갑니다.

ContentCue는 제 열람 데이터를 수집하나요?

아니요. ContentCue는 Apple WebKit을 사용하여 기기 내에서만 모든 탐색을 처리합니다. 열람 기록, 방문한 URL, 개인 데이터가 수집되거나 전송되지 않습니다.

Pro 업그레이드에는 무엇이 포함되나요?

Pro는 일회성 $2.99 구매입니다. 무제한 북마크, 커스텀 광고 테두리 색상, 조정 가능한 자동 스크롤 속도를 사용할 수 있습니다. 구독이 아닙니다 — 한 번 구매하면 영구적으로 사용할 수 있습니다.
