왜 ContentCue인가

웹에서 읽기는 불편합니다. 그 짜증나는 부분을 고쳤습니다.

ContentCue는 휴대폰으로 읽고 듣는 일상의 불편함을 해결하기 위해 만들어졌습니다 — 광고를 차단하거나 좋아하는 사이트를 망가뜨리지 않습니다.

읽던 위치를 계속 잃어버립니다

팝업이 페이지를 가로채거나, 리디렉션이 실행되거나, 잠깐 앱을 전환했을 뿐인데 — 읽던 문단을 다시 찾아 헤매게 됩니다.

ContentCue가 정확한 위치를 자동 저장하고 한 번의 탭으로 되돌려줍니다.

광고가 콘텐츠와 뒤섞입니다

본문 속 광고는 기사처럼 보이도록 설계되어 실수로 탭하면 흐름이 끊깁니다.

컬러 테두리로 모든 광고를 즉시 알아볼 수 있습니다 — 아무것도 차단하지 않으면서 명확하게.

콘텐츠가 세 개의 앱에 흩어져 있습니다

기사용 브라우저, 별도의 RSS 앱, 팟캐스트용 앱 — 확인할 곳이 세 군데, 플레이어도 세 개입니다.

웹페이지, RSS 피드, 팟캐스트를 하나의 앱, 하나의 오디오 플레이어로.

정작 읽을 시간이 없습니다

출퇴근길, 헬스장, 운전 중에 읽을거리는 쌓여갑니다 — 하필 눈을 쓸 수 없을 때에요.

어떤 페이지나 읽지 않은 피드든 오디오로 바꿔 화면 잠금 상태, CarPlay, Vision Pro에서 들으세요.

작동 원리

3단계. 설정 필요 없음.

1

평소처럼 탐색

ContentCue에서 아무 웹사이트나 열고 평소대로 읽으세요. 사이트별 설정도, 계정도 필요 없습니다.

2

광고가 방해함

팝업이나 리다이렉트가 나타나 다른 페이지로 이동합니다. ContentCue가 그 순간 정확한 스크롤 위치를 자동으로 저장합니다.

3

이어서 읽기 탭

이어서 읽기 버튼이 나타납니다. 한 번 탭하면 읽던 정확한 줄로 돌아갑니다.

위치 복원

한 번의 탭. 정확한 위치로 돌아가기.

ContentCue는 페이지 방해 요소(팝업, 리다이렉트 배너, 실수로 광고 탭하기 등)를 감지하고 스크롤 위치를 자동으로 저장합니다. 위치가 저장되면 이어서 읽기 버튼이 나타납니다. 탭하면 원래 위치로 돌아갑니다.

  • 탐색할 때마다 자동으로 위치 저장
  • 위치가 저장되면 언제든 이어서 읽기 버튼 표시
  • 모든 콘텐츠 사이트에서 작동 — 사이트별 설정 불필요
시각적 광고 구분

콘텐츠와 광고 구분하기

ContentCue는 페이지 내 광고 주변에 색상 테두리를 그려서 읽는 콘텐츠와 즉시 구분할 수 있게 해줍니다. 이는 광고 차단이 아닙니다 — 모든 광고가 정상적으로 로드되고 표시됩니다. 단지 시각적 명확성을 제공할 뿐입니다.

  • 색상 테두리로 페이지 내 광고 강조
  • 모든 광고가 정상적으로 로드되고 표시됨
  • Pro에서 테두리 색상 커스터마이징
모든 광고가 정상 표시됨 — 단지 더 잘 보이게 도와줍니다
RSS 리더

피드를 구독하세요. 놓친 헤드라인을 들어보세요.

즐겨찾는 뉴스 피드를 구독하고 가장 자주 읽는 채널을 상단에 고정하세요. 전체 구독 목록을 OPML로 가져오거나 내보낼 수 있습니다. 깔끔한 집중 화면으로 기사를 읽거나, TTS를 켜서 읽지 않은 헤드라인을 개인 라디오처럼 연속으로 들으세요.

  • 즐겨 찾는 피드를 팔로우하고 자주 읽는 채널을 고정하세요
  • 구독 목록을 OPML로 가져오기/내보내기
  • TTS가 읽지 않은 헤드라인을 연속으로 읽어주며 오디오 브리핑처럼 재생
팟캐스트 플레이어

검색하고, 스트리밍하고, 좋아하는 프로그램을 다운로드하세요.

내장 팟캐스트 디렉터리에서 검색하고, 좋아하는 프로그램을 고정하며, 에피소드를 스트리밍하거나 다운로드해 오프라인으로 나중에 들을 수 있습니다. 잠금 화면에서 재생을 제어하고 수면 타이머도 내장되어 있습니다.

  • 즐겨찾는 프로그램을 고정할 수 있는 내장 팟캐스트 디렉터리
  • 에피소드를 스트리밍하거나 다운로드해 오프라인 청취
  • 잠금 화면 재생 제어와 내장 수면 타이머
웹사이트 TTS

어떤 페이지든 재생을 탭하세요. 화면이 잠겨도 계속 들을 수 있어요.

운전 중이나 요리, 산책 중에도 ContentCue가 기사를 읽어줍니다. 화면을 잠가도 백그라운드에서 재생이 계속되며, 마지막 기사와 위치에서 빠르게 다시 시작합니다.

  • 한 번의 탭으로 어떤 기사든 텍스트 음성 변환
  • 화면이 잠겨도 백그라운드에서 오디오 재생 지속
  • 마지막 기사와 위치에서 빠르게 이어서 재생
Apple Vision Pro 네이티브 앱

거실 크기의 나만의 독서실.

Vision Pro의 ContentCue는 iPad 포팅이 아닌 진정한 visionOS 네이티브 앱입니다. 시선 친화적 컨트롤의 플로팅 윈도우에서 탐색하고 읽고 들은 뒤, 재생을 작은 코너 플레이어로 줄여 다른 일을 하면서 계속 들으세요.

핀치로 페이지를 넘기는 읽기 모드 — 아무 곳에서나 빠르게 핀치하면 페이지가 넘어가며, 버튼을 조준할 필요가 없습니다

CarPlay

내 피드와 프로그램을 차량 디스플레이에서.

iPhone을 연결하면 저장된 팟캐스트와 RSS 피드와 함께 ContentCue가 CarPlay에 나타납니다. 차량 화면에서 에피소드나 기사 브리핑을 시작하고, 대기열의 다음 항목을 확인하고, 재생 속도를 변경하세요 — 모두 네이티브 CarPlay 인터페이스에서.

  • 저장된 팟캐스트와 RSS 피드를 차량 디스플레이에서 탐색
  • 대기열 보기에서 지금 재생 중인 항목과 다음 항목을 확인
  • 휴대폰을 들지 않고 재생 속도 조절
iCloud 동기화 및 Siri 단축어

iPhone, iPad, Vision Pro 어디서나 이어집니다.

  • 북마크, RSS 피드, 팟캐스트 구독 정보가 iCloud로 동기화
  • "Siri야, ContentCue 재생해"로 손 안 대고 브리핑이나 팟캐스트 시작
  • RSS와 팟캐스트에 별도의 소스·속도 제어가 가능한 5개의 내장 단축어

FAQ

자주 묻는 질문

ContentCue는 광고 차단기인가요?

아니요. ContentCue는 광고를 차단, 숨김 또는 로드하지 않습니다. 모든 광고가 정상적으로 표시됩니다. ContentCue는 광고가 읽기를 방해할 때 스크롤 위치를 저장하여 탐색을 개선하고, 페이지 내 광고 주변에 색상 테두리를 그려서 명확하게 보일 수 있도록 합니다.

ContentCue는 어떤 사이트에서 작동하나요?

ContentCue는 완전한 웹 브라우저이므로 모든 웹사이트에서 작동합니다 — Webtoons, Tapas, LINE Webtoon, 독립 작가 페이지, 만화 검색 사이트, WordPress 블로그 등 수천 개 사이트에서요. 사이트별 설정이 필요 없습니다.

위치 복원은 어떻게 작동하나요?

ContentCue는 읽는 동안 스크롤 위치를 모니터링합니다. 탐색할 때(광고 링크 탭, 리다이렉트 또는 뒤로 버튼), 정확한 위치를 저장합니다. 브라우저 UI에 이어서 읽기 버튼이 나타납니다. 탭하면 정확히 읽던 위치로 돌아갑니다.

ContentCue에 RSS 리더가 있나요?

네. 어떤 RSS 피드든 구독하고, 즐겨찾는 채널을 고정하고, 전체 구독 목록을 OPML로 가져오거나 내보낼 수 있습니다. TTS를 켜면 읽지 않은 헤드라인을 개인 라디오처럼 끊김 없는 오디오 브리핑으로 들을 수 있습니다.

ContentCue에서 팟캐스트를 들을 수 있나요?

네. 내장 팟캐스트 디렉터리에서 검색하고, 팔로우하는 프로그램을 고정하고, 에피소드를 스트리밍하거나 다운로드해 오프라인으로 들을 수 있습니다. 재생은 잠금 화면에서도 작동하며 내장 수면 타이머도 포함되어 있습니다.

ContentCue가 웹페이지를 소리 내어 읽어주나요?

네. 어떤 기사든 재생을 탭하면 ContentCue가 텍스트 음성 변환으로 읽어줍니다. 화면이 잠겨도 백그라운드에서 오디오가 계속 재생되며, 앱이 마지막 기사와 위치에서 빠르게 다시 시작합니다.

ContentCue가 iPhone, iPad, Vision Pro 간에 동기화되나요?

네, Premium으로 가능합니다. 북마크, RSS 피드, 팟캐스트 구독이 Apple Vision Pro를 포함한 모든 Apple 기기에서 iCloud를 통해 자동 동기화됩니다.

ContentCue를 Apple Vision Pro에서 사용할 수 있나요?

네. ContentCue는 네이티브 visionOS 앱을 제공합니다 — 플로팅 윈도우에서 탐색하고, 빠른 핀치로 기사를 넘기고, 재생을 작은 코너 플레이어로 줄일 수 있습니다. 유니버설 구매로 한 번 구매하면 iPhone, iPad, Vision Pro에서 사용할 수 있으며 Premium도 세 기기에서 공유됩니다.

ContentCue가 CarPlay를 지원하나요?

네. 저장된 팟캐스트와 RSS 피드와 함께 ContentCue가 차량의 CarPlay 디스플레이에 나타납니다. 차량 화면에서 바로 재생을 시작하고, 현재 대기열을 확인하고, 재생 속도를 변경할 수 있습니다.

ContentCue는 제 열람 데이터를 수집하나요?

아니요. ContentCue는 Apple WebKit을 사용하여 기기 내에서만 모든 탐색을 처리합니다. 열람 기록, 방문한 URL, 개인 데이터가 수집되거나 전송되지 않습니다.

ContentCue Premium에는 무엇이 포함되나요?

무료 버전에는 전체 읽기 브라우저와 함께 최대 3개의 북마크, 3개의 RSS 피드, 3개의 저장된 팟캐스트 프로그램이 포함됩니다. Premium은 이 제한을 없애고(북마크·피드·프로그램 무제한) 기기 간 iCloud 동기화, Siri 단축어, 광고 테두리 색상 사용자화, 자동 스크롤 속도 조절을 추가합니다. 7일 무료 체험이 포함된 연간 구독 또는 일회성 평생 구매로 이용할 수 있습니다.
