Stream Your Windows PC Comic Library to iPhone — Home Wi-Fi and 5G

2026년 6월 10일

Windows PC 만화 라이브러리를 iPhone으로 스트리밍 — 홈 Wi-Fi와 외출 시 5G

만화 컬렉션이 Windows PC에 있습니다. 수백 GB의 CBZ, EPUB, PDF 파일이 원하는 대로 정리되어 있습니다. iPhone은 64GB 저장 공간이 있는데 대부분이 이미 꽉 찼습니다. 가장 명확한 해결책 — 파일 복사 — 은 모든 것을 두 곳에서 관리하고, 끊임없이 용량 부족을 겪으며, 새 파일을 추가할 때마다 전송해야 한다는 것을 의미합니다.

BiblioFuse는 다른 방법을 제시합니다. Windows용 BiblioFuse PC Reader 앱에는 내장 스트리밍 서버가 포함되어 있습니다. iPhone이 PC에 연결해 필요에 따라 페이지를 스트리밍합니다 — 아무것도 복사되지 않고, 아무것도 업로드되지 않으며, iPhone 저장 공간은 그대로 남아 있습니다. Mac 사용자들이 사용하던 것과 동일한 기능이 이제 Windows에서 네이티브로 제공되며, Microsoft Store에서 다운로드할 수 있습니다.

PC 라이브러리 스트리밍 작동 방식

BiblioFuse PC Reader는 Windows 머신에서 로컬 HTTP 서버를 실행합니다. iPhone과 PC가 같은 Wi-Fi 네트워크에 있을 때, iOS 앱은 Bonjour(AirPrint와 AirDrop을 지원하는 것과 동일한 제로 구성 네트워킹 프로토콜)를 사용해 자동으로 PC를 발견합니다. IP 주소를 입력하거나 라우터를 구성하지 않아도 iPhone에 라이브러리가 나타납니다.

볼륨을 열면 BiblioFuse가 처음 몇 페이지를 즉시 가져오고 읽는 동안 다음 페이지를 미리 로드합니다. CBZ와 CBR 파일은 페이지별로 스트리밍되고, EPUB와 PDF 파일은 HTTP Range 요청을 사용해 전체를 다운로드하지 않고도 큰 파일을 빠르게 열 수 있습니다. 독서 경험 — 오른쪽에서 왼쪽 모드, 웹툰용 세로 스크롤, 가로 모드 펼침 보기 — 은 로컬에 저장된 파일을 읽는 것과 동일합니다. 유일한 차이는 데이터의 출처뿐입니다.

앱 창을 닫아도 서버는 계속 실행됩니다. 종료하는 대신 시스템 트레이로 이동합니다. 즉, PC를 다른 용도로 사용하는 동안에도 iPhone에서 라이브러리에 접근할 수 있습니다. 창을 복원하려면 트레이 아이콘을 더블클릭하세요. 서버를 완전히 중지하려면 트레이 아이콘을 우클릭하고 종료를 선택하세요.

1부: 홈 Wi-Fi에서 스트리밍

가장 간단한 설정입니다. PC와 iPhone이 같은 네트워크에 있고, BiblioFuse가 자동으로 검색을 처리합니다.

PC에서

  1. Microsoft Store에서 BiblioFuse PC Reader를 다운로드하여 설치합니다.
  2. 앱을 열고 설정 → 홈 라이브러리로 이동합니다.
  3. 폴더 추가를 클릭하고 만화와 전자책이 저장된 폴더(또는 여러 폴더)를 선택합니다. 여러 폴더를 추가할 수 있습니다 — 만화용, 그래픽 노블용, PDF용.
  4. 라이브러리 공유를 활성화합니다. 서버가 포트 7341에서 즉시 시작되고 백그라운드에서 라이브러리 스캔을 시작합니다.
  5. 선택적으로 설정 → 보안에서 비밀번호를 설정해 접근을 보호할 수 있습니다. 건너뛰면 로컬 네트워크의 누구든 연결할 수 있습니다 — 가정용으로는 괜찮지만, 공유 또는 공공 Wi-Fi에서는 잠그는 것을 권장합니다.
  6. 창을 닫아도 됩니다. 서버는 시스템 트레이에서 계속 실행됩니다.

iPhone에서

  1. iPhone에서 **BiblioFuse**를 엽니다.
  2. 라이브러리 탭을 탭합니다.
  3. 상단의 소스 선택을 탭합니다 — PC가 실행 중이고 공유가 활성화되어 있으면 몇 초 안에 PC 홈 라이브러리가 표시됩니다.
  4. 탭하세요. PC의 폴더 구조가 즉시 나타납니다.

설정이 완료되었습니다. 전체 라이브러리를 iPhone에서 검색하고 읽을 수 있으며, 기기에 파일이 전송되지 않습니다. PC에 새 볼륨을 추가하면 다음 라이브러리 새로고침 시 iPhone 뷰에 나타납니다 — 재동기화 단계가 필요 없습니다.

지원 형식

BiblioFuse PC Reader는 로컬에서 읽을 수 있는 모든 형식을 스트리밍합니다: CBZ, CBR, EPUB, PDF, TXT. CBZ의 만화, CBR의 그래픽 노블, EPUB의 전자책, PDF의 스캔 도서 — 모두 동일한 스트리밍 연결을 통해 지원됩니다.

성능 팁

5GHz Wi-Fi를 사용하세요. 라우터가 2.4GHz와 5GHz를 모두 지원하는 경우, PC와 iPhone 모두 5GHz 대역에 연결되어 있는지 확인하세요. 고해상도 대용량 CBZ 파일에서는 더 높은 대역폭이 눈에 띄는 차이를 만듭니다.

SSD의 라이브러리가 더 빠릅니다. 만화 컬렉션이 HDD에 있는 경우, 각 새 볼륨의 첫 페이지 로딩에 약간의 지연이 발생합니다. 라이브러리를 SSD(내장 또는 외장)로 이동하면 이 문제가 해결됩니다.

iPhone이 PC를 찾지 못할 경우: 공유가 실행 중일 때 PC 앱은 설정 → 홈 라이브러리에 네트워크 주소를 표시합니다. Windows 방화벽이 포트 7341을 차단하고 있는지 확인하세요. iOS 앱에서 라이브러리 → 서버 추가 → 수동으로 PC의 로컬 IP 주소를 직접 입력할 수도 있습니다.

2부: 외출 시 5G로 스트리밍

홈 Wi-Fi 스트리밍은 두 기기가 같은 네트워크에 있을 때만 작동합니다. 카페, 호텔, 또는 5G 환경에서 라이브러리에 접근하려면 두 네트워크를 연결해야 합니다. 해결책은 Tailscale — BiblioFuse Tailscale 가이드에서 설명된 것과 동일한 설정입니다.

Tailscale은 기기 간에 프라이빗하고 암호화된 메시 네트워크를 만듭니다. 한 번 설정하면 iPhone은 마치 같은 로컬 네트워크에 있는 것처럼 PC의 스트리밍 서버에 접근할 수 있습니다 — 어떤 인터넷 연결에서도, 포트 포워딩이나 고정 IP 없이.

원격 스트리밍을 위한 Tailscale 설정

Windows PC에서:

  1. tailscale.com 또는 Microsoft Store에서 Tailscale을 다운로드합니다.
  2. Google, GitHub, Microsoft 또는 무료 Tailscale 계정으로 로그인합니다.
  3. Tailscale은 BiblioFuse와 함께 시스템 트레이에서 실행됩니다. PC의 Tailscale IP 주소(100.으로 시작)를 메모해 두세요.
  4. BiblioFuse PC Reader가 라이브러리 공유가 활성화된 상태로 실행 중인지 확인합니다.

iPhone에서:

  1. App Store에서 Tailscale을 설치합니다.
  2. 같은 Tailscale 계정으로 로그인합니다.
  3. Tailscale VPN 토글을 활성화합니다.
  4. iPhone에서 BiblioFuse를 엽니다. 라이브러리 → 서버 추가에서 PC의 Tailscale IP 주소와 포트 7341을 입력합니다.
  5. 연결을 탭합니다.

BiblioFuse는 Tailscale IP를 사용할 수 있을 때 자동으로 우선 사용하므로, 동일하게 저장된 연결이 집에서(LAN 경유)도 외출 시(Tailscale 경유)도 작동합니다. 두 개의 별도 연결 항목이 필요하지 않습니다.

모바일 데이터 사용 시 기대치

Tailscale은 가능한 경우 피어-투-피어 연결을 설정해 더 나은 성능을 위해 릴레이 서버를 우회합니다. 일반적인 5G 연결에서 만화 페이지 스트리밍은 잘 작동합니다. BiblioFuse는 현재 페이지만 전송하고 다음 몇 페이지를 미리 로드합니다 — CBZ 파일 전체를 한 번에 스트리밍하지 않으므로, 200MB 아카이브가 200MB 데이터 사용을 의미하지 않습니다. 이미지 해상도와 서버 압축 설정에 따라 페이지당 약 1~3MB를 예상하세요.

제한된 데이터 플랜에서 읽으려면 집을 나서기 전에 Wi-Fi에서 BiblioFuse를 사용해 개별 볼륨을 다운로드하세요. iPhone의 PC 라이브러리 뷰에서 원하는 볼륨 옆의 다운로드 아이콘을 탭해 로컬에 캐시합니다. 그 볼륨은 이후 Tailscale이나 데이터 연결 없이 오프라인에서 사용 가능합니다.

PC vs Mac: 다른 점은?

Mac 동반 앱을 사용한 적이 있다면, iOS 측에서 PC 경험은 거의 동일합니다. Windows 특유의 몇 가지 차이점:

  • 포트: PC 서버는 포트 7341을 사용합니다(Mac 앱의 기본값은 8686).
  • 검색: 둘 다 서비스 유형 _bibliofuse._tcp로 Bonjour/mDNS를 사용하므로, iOS가 동일한 방식으로 검색합니다.
  • 시스템 트레이: 창을 닫아도 PC 앱은 시스템 트레이에 남아 있고, 종료를 선택할 때까지 서버는 멈추지 않습니다.
  • 설정 파일: PC 설정은 %APPDATA%\BiblioFuse\settings.json에 저장됩니다. 수동으로 검사하거나 재설정이 필요한 경우 참조할 수 있습니다.
  • 동시에 하나의 서버만 활성: 현재 버전에서 BiblioFuse iOS는 Mac 서버 또는 PC 서버 중 하나에만 연결됩니다 — 동시에 둘 다 연결할 수 없습니다. iOS가 처음 검색하는 것은 실행 중이고 활성 상태인 머신입니다.

Mac 스트리밍에 대한 자세한 내용은 Wi-Fi로 Mac 만화 라이브러리를 iPhone에 스트리밍을 참조하세요.

자주 묻는 질문

BiblioFuse에 Windows 앱이 있나요? 네. BiblioFuse PC Reader는 Microsoft Store에서 제공되는 네이티브 Windows 애플리케이션입니다. 로컬에서 CBZ, CBR, EPUB, PDF, TXT 파일을 읽고 네트워크를 통해 전체 라이브러리를 iOS 앱에 스트리밍합니다.

Windows PC에서 iPhone으로 만화를 스트리밍하는 방법은? Windows에 BiblioFuse PC Reader를 설치하고 설정에서 라이브러리 공유를 활성화합니다. iPhone에 BiblioFuse를 설치합니다. 같은 Wi-Fi 네트워크에서 iPhone의 BiblioFuse를 열고 라이브러리를 탭하면 — PC 라이브러리가 Bonjour 검색을 통해 자동으로 나타납니다. IP 주소나 라우터 구성이 필요 없습니다.

집에서 멀리 있을 때 5G로 PC 라이브러리에 접근할 수 있나요? 네, Tailscale을 사용하면 됩니다. Windows PC와 iPhone에 Tailscale을 설치하고, 같은 계정으로 로그인한 뒤 두 기기에서 활성화합니다. BiblioFuse는 Tailscale 연결을 자동으로 감지하고 어디서든 PC에 접근하는 데 사용합니다 — 호텔 Wi-Fi, 5G, 어떤 인터넷 연결도 가능합니다.

PC에서 스트리밍하면 iPhone 저장 공간을 사용하나요? 아니요. 페이지는 PC에서 온디맨드로 가져오며, 앱 자체와 읽기 진행 데이터를 제외하고 iPhone에는 아무것도 저장되지 않습니다. 500GB 라이브러리를 탐색하고 읽어도 파일에 iPhone 저장 공간이 사용되지 않습니다.

스트리밍이 작동하려면 PC가 켜져 있어야 하나요? 네. 스트리밍 서버는 PC에서 실행되며, PC가 절전 모드로 들어가거나 서버가 중지되면 iOS 연결이 끊어집니다. 스트리밍 중 절전을 방지하려면 Windows 설정 → 전원 및 절전에서 절전을 절대 안 함으로 설정하거나 PowerToys 같은 도구를 사용하세요.

PC 스트리밍 서버는 어떤 포트를 사용하나요? 포트 7341입니다. 방화벽이 이 포트의 트래픽을 차단하는 경우 BiblioFuse PC Reader에 대한 예외를 추가해야 합니다.

시작하기

BiblioFuse PC Reader는 지금 Microsoft Store에서 이용할 수 있습니다. iPhone용 BiblioFuse는 App Store에 있습니다. 둘 다 설치하고, PC 앱이 라이브러리 폴더를 가리키도록 하면, 다음에 앱을 열 때 전체 컬렉션이 iPhone에 — 케이블 없이, 클라우드 없이, 복사 없이.
