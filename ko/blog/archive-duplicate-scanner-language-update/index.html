Comic Duplicate Scanner가 이제 포르투갈어, 스페인어, 바하사 인도네시아어, 간체 중국어를 지원합니다 — 앱 전체 인터페이스를 4개의 새로운 언어로 제공합니다. 이 앱은 지각 해싱을 사용하여 중복 만화 아카이브(CBZ, CBR, ZIP, RAR)와 사진을 찾아내며, Mac과 Windows에서 무료로 사용할 수 있습니다. 지금 다운로드하세요.

국제 수집가들이 이 업데이트를 필요로 했던 이유

방대한 만화나 만화책 컬렉션을 관리하는 것은 언어 장벽 없이도 충분히 어렵습니다. 지금까지 영어권 이외의 수집가들은 Comic Duplicate Scanner를 영어로 사용해야 했습니다. 영어 메뉴, 영어 결과 레이블, 영어 내보내기 보고서.

번역된 만화나 해외판 코믹스 파일을 수만 개 관리하는 독자들에게 이런 불편함은 실질적인 장벽이었습니다. 이번 업데이트로 그 장벽을 완전히 제거합니다.

새로운 기능 — 4가지 새 언어 팩

Comic Duplicate Scanner 최신 버전에서 다음 언어의 전체 UI 로컬라이제이션이 추가되었습니다:

  • 포르투갈어 (Português)
  • 스페인어 (Español)
  • 바하사 인도네시아어 (Bahasa Indonesia)
  • 간체 중국어 (简体中文)

"전체 UI 지원"은 앱의 모든 표시 문자열이 번역되었음을 의미합니다 — 메뉴, 툴바 버튼, 결과 테이블의 열 헤더, 유사도 점수 레이블, 확인 대화상자, 기본 설정 패널, 그리고 썸네일이 있는 HTML 내보내기 보고서까지 포함됩니다.

앱은 운영 체제의 언어 설정을 자동으로 따릅니다. macOS에서는 시스템 설정 → 일반 → 언어 및 지역으로 이동하여 원하는 언어를 목록 맨 위로 이동하세요. Windows에서는 설정 → 시간 및 언어 → 언어 및 지역에서 변경할 수 있습니다. Comic Duplicate Scanner를 재시작하면 인터페이스가 즉시 전환됩니다.

Comic Duplicate Scanner 소개

Comic Duplicate Scanner는 지각 해싱을 사용하여 중복 파일을 감지합니다. 각 아카이브(CBZ, CBR, ZIP, RAR)를 열고 내부의 실제 페이지 이미지의 지각 해시를 계산합니다. 두 파일은 이름, 파일 크기, 심지어 압축 방식이 완전히 달라도 — 페이지가 같아 보이면 일치합니다. 파일 이름을 바꿔도 중복을 숨길 수 없습니다.

기타 주요 기능:

  • 퍼지 파일명 매칭 — 파일을 열지 않고도 "Spider-Man #001"과 "spiderman_001"의 일치를 감지합니다. Unicode 인식으로 한국어 파일명도 항상 올바르게 처리합니다.
  • 스마트 자동 선택 — 규칙에 따라 어떤 복사본을 유지할지 선택: 가장 큰 파일, 최신, 가장 오래된 것, 또는 페이지 수가 가장 많은 것.
  • 안전한 삭제만 가능 — 파일을 macOS 휴지통 또는 Windows 휴지통으로 이동합니다. 영구 삭제는 없습니다.
  • 해시 캐시(SQLite) — 결과는 로컬에 저장됩니다. 새 파일을 추가한 후 폴더를 다시 스캔하면 새 파일만 처리됩니다.
  • 풍부한 내보내기 — 썸네일과 유사도 점수가 포함된 CSV 및 HTML 보고서.
  • 100% 로컬 — 클라우드 없음, 분석 없음, 계정 불필요.

자주 묻는 질문

Comic Duplicate Scanner는 한국어 파일명을 지원하나요? 네. 퍼지 파일명 매칭은 완전히 Unicode를 인식하며 한국어(한글)를 포함한 파일명을 항상 올바르게 처리해 왔습니다.

앱 언어를 변경하려면 어떻게 해야 하나요? 앱은 시스템 언어 설정을 자동으로 따릅니다. macOS에서는 시스템 설정 → 일반 → 언어 및 지역에서 설정하세요. 설정 변경 후 앱을 재시작하세요.

앱은 무료인가요? Comic Duplicate Scanner는 최대 1,000개 파일의 라이브러리까지 무료입니다. 더 큰 컬렉션의 경우 USD 19.99의 일회성 구매로 무제한 스캔이 가능합니다. Mac 버전에는 7일 무료 체험이 포함됩니다.

지원 파일 형식은 무엇인가요? 만화 아카이브: CBZ, CBR, ZIP, RAR. 사진: JPEG, PNG, HEIC, WEBP, BMP, GIF.

내 데이터가 어딘가로 전송되나요? 아니요. Comic Duplicate Scanner는 완전히 로컬에서 작동합니다 — 파일과 해시가 장치를 벗어나지 않습니다.
