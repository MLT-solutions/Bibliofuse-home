파일명이 다른 중복 만화 찾는 방법

만화 라이브러리를 정리하면서 표준 중복 파일 찾기를 실행합니다. "중복 없음"이라고 나옵니다. 하지만 뭔가 이상한 걸 알고 있습니다 — 1-5화 파일과 1-10화 파일이 있는데, 앞의 5화는 명백히 겹칩니다.

Comic Duplicate Scanner가 해결하도록 만들어진 바로 이 문제입니다: 파일명이나 크기가 아닌 실제 페이지 내용을 비교해 중복 만화 압축 파일을 찾아냅니다.

표준 중복 찾기가 만화에 통하지 않는 이유

  • 화수 묶음 방식이 다른 경우: 한 출처에서 manga_ch1-5.cbz, 다른 출처에서 manga_ch1-10.cbz를 다운로드
  • 이름 규칙이 다른 경우: Spider-Man_001.cbz와 spiderman_issue01.cbz가 같은 권일 수 있음
  • 재인코딩 파일: 두 출처에서 같은 만화지만 JPEG 품질이 달라 해시가 완전히 다름
  • 형식이 다른 경우: CBZ와 CBR로 동일한 내용

Comic Duplicate Scanner의 감지 방식

Comic Duplicate Scanner는 각 압축 파일 안의 실제 페이지 이미지에 지각 해싱을 사용합니다.

  1. 두 압축 파일(CBZ, CBR, ZIP, RAR)을 열기
  2. 각각 첫 페이지 추출
  3. 각 페이지의 지각 해시 생성
  4. 두 압축 파일 간 해시 비교

설정 가능한 임계값 이상으로 해시가 일치하면 파일명이나 크기에 관계없이 해당 쌍이 잠재적 중복으로 표시됩니다.

엄격한 임계값은 더 가까운 시각 일치를 요구하고, 느슨한 임계값은 더 많은 경계선 케이스를 잡아냅니다. 기본값은 오탐 없이 실제 중복을 잡도록 설정되어 있습니다.

Comic Duplicate Scanner는 스캔과 미리보기가 무료. Pro는 일괄 작업을 활성화합니다. 구독 없음.
