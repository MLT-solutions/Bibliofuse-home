CBZ를 PDF로 온라인 변환하는 방법 — 무료, 개인정보 보호, 업로드 불필요

2026년 6월 7일

CBZ를 PDF로 온라인 변환하는 방법 — 무료, 개인정보 보호, 업로드 불필요

CBZ 파일은 전용 만화 리더 앱에서 잘 작동하지만, 세상은 여전히 PDF를 기본으로 사용합니다. 스캔한 호를 동료에게 보내거나, 컬렉션을 범용 포맷으로 보관하거나, PDF만 지원하는 앱에서 파일을 열어야 할 때 CBZ를 PDF로 변환하는 작업이 필요합니다. 그런데 대부분의 솔루션은 제대로 처리하지 못합니다.

온라인 변환기는 파일을 원격 서버에 업로드하고, 데스크톱 소프트웨어는 설치와 유료 라이선스를 요구합니다. 파일이 기기를 떠나지 않고 빠르게 포맷을 변환하고 싶을 때 두 방법 모두 이상적이지 않습니다.

더 좋은 방법이 있습니다. BiblioFuse 웹앱은 WebAssembly를 사용해 브라우저 내에서 완전히 CBZ를 PDF로 변환합니다 — 서버 없음, 계정 없음, 어떤 데이터도 전송되지 않습니다.

CBZ와 PDF 모두 중요한 이유

CBZ(Comic Book ZIP)는 디지털 만화의 표준 포맷입니다. 순서대로 이름 붙인 이미지(주로 JPEG 또는 WebP)를 담은 ZIP 아카이브로, 만화 리더가 페이지별로 표시합니다. 포맷이 컴팩트하고 전용 앱의 지원이 넓으며 이미지 품질을 잘 보존합니다.

반면 PDF는 범용 문서 포맷입니다. 모든 기기, 운영체제, 이메일 클라이언트에서 특별한 리더 없이 열 수 있습니다. Kindle에서 읽거나, 이메일로 보내거나, 인쇄하거나, 보관하거나, CBZ를 인식하지 못하는 수백 개의 앱에서 사용할 수 있습니다.

디지털 만화 라이브러리를 관리한다면 파일 공유나 PDF 중심 워크플로에 콘텐츠를 넣을 때 언젠가 이 경계를 넘어야 합니다.

BiblioFuse가 브라우저에서 CBZ를 PDF로 변환하는 방법

BiblioFuse 웹앱은 WebAssembly(WASM)를 사용해 완전히 브라우저 내에서 동작하므로, 변환 로직이 서버로 데이터를 전송하지 않고 브라우저 안에서 실행됩니다.

변환 과정:

  1. 브라우저가 WASM 모듈을 로드 — 거의 네이티브 속도로 실행되는 컴파일된 샌드박스 바이너리
  2. CBZ 파일을 선택하면 브라우저 메모리에 로드
  3. WASM 모듈이 ZIP 아카이브를 압축 해제하고 이미지 시퀀스를 읽음
  4. 이미지를 표준 페이지 크기의 PDF로 조합
  5. PDF가 브라우저 다운로드로 기기에 직접 저장 — 어디에도 업로드되지 않음

일반적인 단일 호 만화(520MB) 변환은 수 초면 완료됩니다. 대용량 볼륨이나 고해상도 스캔은 3060초 소요될 수 있습니다.

단계별 안내: CBZ를 PDF로 온라인 변환

1단계 데스크톱 또는 모바일의 최신 브라우저(Chrome, Firefox, Safari, Edge)에서 bibliofuse.com/ko/webapp/을 엽니다.

2단계 변환 도구를 클릭하거나 CBZ 파일을 드롭 영역으로 드래그합니다.

3단계 출력 포맷으로 CBZ → PDF를 선택합니다.

4단계 변환을 클릭합니다. 파일 압축 해제와 재조합이 브라우저 내에서 완전히 이루어집니다.

5단계 PDF를 다운로드합니다. 로컬에 저장되며 계정이 필요 없고 서버에 업로드되는 것도 없습니다.

회원가입 불필요, 클라우드 작업 대기 불필요, 서버 무료 플랜의 용량 제한 없음.

기술 노트: CBZ 내부 구조

CBZ 파일은 단순히 .cbz 확장자를 가진 ZIP 아카이브입니다. .zip으로 이름을 바꿔 열면 001.jpg, 002.jpg 등 순서대로 이름 붙인 이미지 파일이 담긴 폴더가 보입니다.

BiblioFuse 변환기는:

  • ZIP 인덱스를 읽어 페이지 순서를 결정
  • 각 이미지(JPEG, PNG, WebP, AVIF) 압축 해제
  • 원본 해상도로 각 이미지를 PDF 페이지에 삽입
  • 이미지 크기에 맞게 PDF 페이지 크기 설정
  • 표준 PDF 파일 출력

이미지가 원본 해상도로 삽입되므로 출력 PDF는 CBZ와 시각적으로 동일합니다 — 변환 과정에서 품질 손실 없음.

자주 묻는 질문

변환기가 내 파일을 저장하나요?
아닙니다. 변환은 WebAssembly를 통해 브라우저 안에서만 완전히 실행됩니다. 파일은 기기를 떠나지 않습니다.

최대 파일 크기는?
서버가 관여하지 않으므로 서버 측 제한이 없습니다. 실용적 한계는 브라우저 가용 메모리로, 현대 기기에서는 보통 수 GB입니다.

PDF가 CBZ와 같은 모습인가요?
네. 이미지는 원본 해상도로 삽입됩니다. 유일한 차이는 컨테이너 포맷이며 시각적 콘텐츠는 동일합니다.

만화를 더 스마트하게 관리하세요

CBZ를 PDF로 변환하는 것은 디지털 만화 라이브러리 관리의 한 단계일 뿐입니다. BiblioFuse iOS 앱은 CBZ, CBR, EPUB, PDF 지원, Mac에서 Wi-Fi 전송, iCloud 독서 진행 동기화, 변환 및 압축 도구를 포함한 완전한 독서 환경을 제공합니다.

설치 없이 어떤 기기에서든 빠른 포맷 변환이 필요하다면 bibliofuse.com/ko/webapp/이 가장 빠른 방법입니다.
