iPhone에서 만화 읽기: 오른쪽에서 왼쪽, 세로 스크롤, 펼침 페이지 설정

2026년 5월 22일

iPhone에서 만화 읽는 방법: 오른쪽에서 왼쪽, 세로 스크롤, 양면 페이지

BiblioFuse는 iPhone 및 iPad에서 CBZ, CBR, ZIP 또는 RAR 형식의 망가를 올바른 오른쪽에서 왼쪽 페이지 흐름, 웹툰용 세로 스크롤 모드, 가로 방향에서 자동 양면 페이지 감지로 읽을 수 있게 해준다. iOS 17 이상에서 실행되며 계정이나 구독이 필요 없다 — 파일은 케이블 없이 Wi-Fi Transfer 또는 iCloud Drive를 통해 불러온다.

iPhone에서 만화를 읽는 건 간단해 보입니다. 그냥 순서대로 이미지를 보여주면 되니까요. 하지만 막상 해본 사람이라면 그 문제를 잘 압니다. 서양 코믹스용으로 만들어진 리더는 왼쪽에서 오른쪽으로 페이지를 넘기게 강제해 모든 컷의 독서 흐름을 망가뜨립니다. Webtoon(세로 스크롤 만화)은 긴 연속 이미지 대신 개별 페이지로 쪼개어 표시됩니다. 양면 스프레드는 세로 모드에서 두 장으로 분리되어, 화면 전체를 가득 채우도록 그린 작화가 반쪽짜리로 보이게 됩니다.

manga에는 특수한 요구사항이 있는데, 대부분의 범용 리더는 이를 무시합니다. BiblioFuse는 이 모든 것을 처리하도록 설계되었습니다. 오른쪽에서 왼쪽 모드, 세로 스크롤, 양면 페이지 감지, 그리고 세션 사이에서도 유지되는 작품별 읽기 방향 설정이 모두 포함되어 있습니다.

이 가이드에서는 각 기능을 설정하는 방법을 구체적으로 설명합니다.

BiblioFuse 설치하기

BiblioFuse는 App Store에서 무료로 제공됩니다. 계정이 필요 없습니다.

설치하면 CBZ, CBR, ZIP, RAR, EPUB, PDF, TXT를 읽을 수 있습니다. manga 커뮤니티에서 접하는 모든 형식을 지원합니다. 일본어 스캔 파일은 대개 JPG 이미지가 담긴 CBZ 또는 ZIP으로 배포되는데, BiblioFuse는 별도 변환 없이 바로 지원합니다.

iPhone에 만화 옮기기

대용량 컬렉션에는 Wi-Fi 전송이 가장 빠릅니다. BiblioFuse에서 설정 → Wi-Fi 가져오기를 켜고, 표시된 URL을 Mac 또는 PC의 브라우저에서 열어주세요. manga 폴더를 브라우저 창으로 드래그하면 됩니다. 단권이 아니라 시리즈 전체를 한 번에 옮길 수 있습니다.

단권은 iOS 공유 시트를 이용합니다. 파일 앱에서 CBZ 파일을 길게 누른 뒤 BiblioFuse로 공유하면 됩니다. Mac에서 iCloud Drive → BiblioFuse에 파일을 복사하면 iPhone에 자동으로 동기화됩니다.

파일이 라이브러리에 들어오면 표지를 탭해서 권을 엽니다.

오른쪽에서 왼쪽 읽기 모드

대부분의 리더는 기본적으로 왼쪽에서 오른쪽입니다. 왼쪽으로 스와이프하면 다음 페이지, 오른쪽으로 스와이프하면 이전 페이지입니다. manga는 오른쪽에서 왼쪽으로 읽기 때문에, 이 기본값은 즉시 거꾸로 느껴집니다. 앞으로 가려고 왼쪽으로 스와이프하면 오히려 뒤로 가게 됩니다.

BiblioFuse에서 설정 방법:

  1. manga 권 열기
  2. 화면 중앙을 탭해 컨트롤 표시
  3. 읽기 방향 아이콘(화살표 두 개) 탭
  4. 오른쪽에서 왼쪽 선택

이제 오른쪽으로 스와이프하면 다음 페이지, 왼쪽으로 스와이프하면 이전 페이지가 됩니다. 눈이 컷을 따라가는 방향과 일치합니다.

BiblioFuse는 이 설정을 권별로 저장하기 때문에, 같은 manga를 열 때마다 다시 설정할 필요가 없습니다. 왼쪽에서 오른쪽으로 읽는 작품(대부분의 서양 코믹스, 왼쪽에서 오른쪽으로 번역된 manhwa)은 각자의 설정을 독립적으로 기억합니다.

오른쪽에서 왼쪽 모드의 탭 영역

오른쪽에서 왼쪽 모드가 활성화되면 탭 영역이 반전됩니다.

  • 왼쪽 가장자리 탭 → 다음 페이지
  • 오른쪽 가장자리 탭 → 이전 페이지
  • 중앙 탭 → 컨트롤 표시/숨기기

실물 manga 책을 넘길 때 왼쪽을 잡아당기는 것과 같은 방식입니다.

Webtoon을 위한 세로 스크롤 모드

Webtoon(한국 웹툰과 많은 현대 온라인 manga)은 페이지를 한 장씩 넘기는 것이 아니라 길게 이어진 세로 스트립을 위에서 아래로 스크롤하도록 제작됩니다. webtoon CBZ를 일반 페이지 리더에 불러오면 연속된 세로 이미지 대신 개별 가로 조각으로 표시되어 컷의 흐름을 따라가기가 거의 불가능합니다.

BiblioFuse는 세로 스크롤 모드로 이 문제를 해결합니다.

  1. 권 열기
  2. 중앙을 탭해 컨트롤 표시
  3. 레이아웃 아이콘 탭
  4. 세로 스크롤 선택

페이지들이 세로로 이어져 하나의 긴 스트립이 됩니다. 웹사이트를 읽듯 아래로 스크롤하면 됩니다. 페이지 경계가 없습니다. 작가가 의도한 대로 작화가 이어집니다.

이 모드는 다음에 적합합니다.

  • 한국 webtoon (LINE Webtoon, Naver Comics)
  • 세로 긴 이미지 형식으로 배포된 현대 일본 웹 manga
  • Manhwa
  • 세로 스트립 형식으로 제작된 팬 번역 온라인 만화

페이지 단위로 출판된 전통적인 manga(소년점프 작품, 단행본)는 페이지 넘기기 모드를 그대로 사용하세요.

양면 페이지

많은 manga 단행본에는 전면 스프레드 컷이 포함되어 있습니다. 대향 두 페이지에 걸친 드라마틱한 장면으로, 하나의 넓은 이미지로 그려진 것입니다. iPhone의 세로 모드에서는 이 스프레드가 두 개의 별도 페이지로 표시되어 작화의 시각적 임팩트가 사라집니다.

가로 모드 자동 감지: iPhone을 가로로 돌리기만 하면 됩니다. BiblioFuse가 방향 변경을 감지하고 자동으로 양면 스프레드 뷰로 전환합니다. 두 페이지가 나란히 화면을 가득 채워, 실물 단행본 그대로의 모습으로 보입니다.

오른쪽에서 왼쪽 스프레드: 오른쪽에서 왼쪽 모드가 활성화된 경우, 스프레드 레이아웃도 반전됩니다. 페이지 순서가 읽기 방향과 일치하여 manga 읽기 순서상 '첫 번째' 페이지인 오른쪽 페이지가 화면 오른쪽에 표시됩니다.

세로 모드로 돌아오면 BiblioFuse가 즉시 단일 페이지 모드로 복귀합니다.

스프레드 감지 팁

오래된 스캔 그룹의 CBZ 파일 중에는 두 페이지를 별도 파일로 저장하지 않고 이미 합친 하나의 넓은 이미지로 스프레드를 저장한 것들이 있습니다. BiblioFuse는 두 경우 모두 처리합니다. 가로 모드에서 단일 넓은 이미지는 화면을 꽉 채워 표시됩니다. 별도 설정이 필요 없습니다.

확대/축소 및 탐색

핀치 줌은 모든 페이지에서 작동합니다. BiblioFuse는 페이지를 풀 해상도로 렌더링하므로, 세밀한 선, 작은 글자, 정밀하게 그린 배경을 확대해도 픽셀화가 발생하지 않습니다.

더블 탭으로 맞추기는 확대 후 페이지를 전체 화면으로 되돌립니다.

페이지 스크러버: 화면 하단의 진행 바를 스와이프하면 권 내 임의의 페이지로 바로 이동할 수 있습니다. 다시 읽을 때 특정 화수나 장면으로 건너뛸 때 유용합니다.

챕터 탐색: EPUB manga(일부 출판사에서 사용)는 챕터 목록이 표시됩니다. CBZ/CBR 권은 진행 스크러버가 탐색 도구입니다.

Manga 시리즈 정리하기

같은 시리즈의 여러 권을 가져오면 BiblioFuse가 폴더별로 묶어줍니다. CBZ 파일이 귀멸의 칼날이라는 폴더 안에 있으면 라이브러리에 하나의 시리즈 항목으로 표시됩니다. 시리즈를 탭하면 모든 권이 순서대로 나열됩니다.

권을 길게 누르면:

  • 별점 추가(1~5점)
  • 태그 추가(예: "읽는 중", "완독", "읽을 예정")

라이브러리 상단의 필터 바에서 태그, 형식, 별점으로 필터링할 수 있습니다. 현재 열다섯 개의 시리즈를 보고 있다면 "읽는 중"으로 태그를 달고 필터링해 해당 시리즈만 표시하세요.

읽기 진행 상황 동기화

iPhone과 iPad(또는 Mac) 모두에서 읽는 경우, BiblioFuse는 iCloud를 통해 읽기 진행 상황을 동기화합니다. 출퇴근 중에 iPad로 시리즈 3권을 80페이지까지 읽고, 나중에 iPhone을 들면 진행 상황이 이미 반영되어 있습니다.

이 기능은 iCloud Drive → BiblioFuse에 저장된 파일에만 적용됩니다. iPhone 로컬 저장소의 파일은 동기화되지 않습니다. Wi-Fi 전송으로 가져온 파일은 기기 간 동기화를 원한다면 파일 앱에서 iCloud Drive로 이동해 주세요.

밝기와 눈의 편안함

밝기 조절: 읽는 중에 화면 왼쪽을 위아래로 스와이프하면 리더를 벗어나지 않고 밝기를 조절할 수 있습니다. 밝은 환경에서 어두운 방으로 이동할 때 특히 유용합니다.

iOS 야간 모드 및 True Tone: BiblioFuse는 표준 UIKit 앱이므로 iOS 시스템 수준의 화면 조정(야간 모드, True Tone, 화면 확대)이 정상적으로 적용됩니다.

장시간 독서 시 밝기를 낮추고 야간 모드를 켜면(설정 → 디스플레이 및 밝기 → 야간 모드) 눈의 피로가 크게 줄어듭니다.

문제 해결

페이지 순서가 잘못됩니다. 대부분의 manga CBZ 파일은 파일명에 앞자리 0을 채워(001.jpg, 002.jpg) 정렬이 올바르게 되도록 합니다. 패딩 없이 만들어진 CBZ(1.jpg, 10.jpg, 2.jpg)는 영숫자 정렬로 인해 순서가 잘못됩니다. 해결 방법은 아카이브 내 파일을 이름 바꾸기 하는 것입니다. CBZ를 압축 해제하고, 이미지를 패딩된 번호로 바꾼 뒤 다시 압축하세요. BiblioFuse는 파일명 정렬 순서대로 읽습니다.

새 권에서는 오른쪽에서 왼쪽 모드가 초기화됩니다. BiblioFuse는 읽기 방향을 권별로 저장합니다. 이미 읽고 있는 시리즈의 새 권을 가져오면, 처음 열 때 한 번 방향을 설정해야 합니다. 이후에는 유지됩니다.

세로 스크롤 모드에서 webtoon 페이지가 너무 좁게 보입니다. 매우 길고 좁은 스트립으로 포맷된 webtoon이 있습니다. 핀치 아웃으로 약간 축소해 스트립이 화면 너비를 꽉 채우게 한 후 스크롤하세요. BiblioFuse는 세로 스크롤 모드에서 페이지 간 확대/축소 수준을 기억합니다.

가로 모드에서 스프레드가 제대로 정렬되지 않습니다. CBZ에 스프레드가 하나의 넓은 JPEG로 합쳐져 있다면, 가로로 돌렸을 때 중앙에 표시됩니다. 스프레드가 어긋나 보인다면 원본 CBZ에 두 페이지가 별도 파일로 저장된 것으로, BiblioFuse가 회전 시 이를 합칩니다. 그래도 어긋난다면 스캔 자체의 페이지 순서가 특이한 경우일 수 있습니다.

다음으로 살펴볼 기능

읽기 설정이 완료되었다면, BiblioFuse에는 더 살펴볼 만한 기능들이 있습니다. Mac 홈 라이브러리를 사용하면 Mac의 manga 컬렉션 전체를 Wi-Fi로 iPhone에 스트리밍할 수 있습니다. 기기에 아무것도 복사하지 않아도 됩니다. Mac에 수 TB의 manga가 있어 복제하고 싶지 않다면 이상적입니다. 도구 탭은 압축 기능을 제공합니다. 200MB의 CBZ 권을 눈에 보이는 화질 손실 없이 40~50MB로 줄여 더 많은 시리즈를 위한 저장 공간을 확보할 수 있습니다.

올바른 설정이 독서를 방해하는 리더와 독서 경험에 자연스럽게 녹아드는 리더의 차이를 만들어냅니다. 오른쪽에서 왼쪽으로 manga를 읽고, 가로 모드로 스프레드를 보고, webtoon은 세로 스크롤로 즐기세요. 모든 것이 원래 그래야 하는 방식으로 작동합니다.

자주 묻는 질문

iPhone에서 최고의 망가 리더 앱은 무엇인가요? BiblioFuse는 iPhone 및 iPad에서 망가 전용으로 설계되었습니다. 오른쪽에서 왼쪽 읽기 방향, 웹툰용 세로 스크롤 모드, 가로 방향 자동 양면 페이지, 그리고 시리즈별 읽기 방향 메모리를 지원합니다 — 각 시리즈를 한 번 설정하면 항상 정확하게 유지됩니다.

iPhone에서 망가 오른쪽에서 왼쪽 모드를 어떻게 설정하나요? BiblioFuse에서 망가 볼륨을 열고, 화면 중앙을 탭해 컨트롤을 표시한 다음 읽기 방향 아이콘을 탭하고 오른쪽에서 왼쪽을 선택합니다. 스와이프 방향과 탭 영역 모두 일본어 읽기 순서에 맞게 반전됩니다. BiblioFuse는 이 설정을 볼륨별로 저장합니다.

iPhone에서 웹툰(세로 스크롤 만화)을 읽을 수 있나요? 네. BiblioFuse에서 볼륨을 열고, 레이아웃 아이콘을 탭한 다음 세로 스크롤을 선택합니다. 페이지가 연속 세로 스트립으로 쌓입니다 — 페이지 구분 없음, 수평 분할 없음. 이 모드는 한국 웹툰, 만화, 모든 긴 스트립 형식에 작동합니다.

BiblioFuse가 양면 스프레드를 올바르게 표시하나요? 네. 읽는 중 iPhone을 가로 모드로 돌리면 BiblioFuse가 자동으로 두 페이지 스프레드 보기로 전환됩니다. 오른쪽에서 왼쪽 모드에서는 페이지 순서도 반전되어 스프레드가 올바르게 읽힙니다.

BiblioFuse는 iPhone에서 어떤 망가 형식을 지원하나요? BiblioFuse는 CBZ, CBR, ZIP, RAR — 거의 모든 망가 스캔 릴리즈에서 사용되는 형식 — 과 EPUB, PDF를 읽습니다. 변환이 필요 없으며 형식에 관계없이 파일이 직접 열립니다.
