SmartDecrypt PDF ZIP, 포르투갈어·스페인어·인도네시아어·중국어 지원 추가

SmartDecrypt PDF ZIP CBZ가 포르투갈어, 스페인어, 바하사 인도네시아어, 간체 중국어를 지원합니다. 네 가지 새 언어로 완전한 앱 인터페이스를 제공합니다. iPhone, iPad, Mac, Windows에서 비밀번호로 보호된 PDF, ZIP, 7z 파일을 기기에서 직접 해제하며 파일 업로드가 필요 없습니다. App Store에서 SmartDecrypt PDF ZIP 다운로드하거나, Windows용 Microsoft Store에서 받으세요. SmartDecrypt 페이지에서 전체 기능을 확인하세요.

왜 이 언어들이 중요했는가

이번 업데이트 전까지 SmartDecrypt PDF ZIP은 영어만 지원했습니다. 브라질, 라틴 아메리카, 인도네시아, 중국의 수집가와 전문가들이 자국어가 아닌 언어로 완전히 기능하는 복호화 툴을 사용해야 했습니다. 비밀번호 패턴, 대기열 상태 레이블, 환경설정 패널—모두 영어였습니다.

포르투갈어, 스페인어, 인도네시아어, 중국어 파일명으로 된 수백 개의 보호된 PDF 명세서, 암호화된 ZIP 아카이브, 잠긴 CBZ 만화 파일을 관리하는 사람에게 이 장벽은 실질적인 문제였습니다.

이번 업데이트로 그 장벽을 완전히 제거합니다.

새로운 기능—4개 언어 팩

최신 버전에는 다음 언어의 전체 UI 현지화가 추가되었습니다:

  • 포르투갈어 (Português) — 브라질 및 유럽 포르투갈어
  • 스페인어 (Español) — 라틴 아메리카와 스페인 포함
  • 바하사 인도네시아어 — 인도네시아의 대규모 아카이브 수집 커뮤니티를 위해
  • 간체 중국어 (简体中文) — 만화 독자와 중국 대륙 사용자를 위해

앱의 모든 표시 문자열이 번역되어 있습니다: 메뉴, 툴바 버튼, 파일 대기열 레이블, 상태 배지, 패턴 매칭 대화상자, Pro 업그레이드 안내, 환경설정 패널. 이 네 가지 언어 중 하나로 시스템 언어를 설정하면 영어가 남지 않습니다.

앱은 운영 체제 언어 설정을 자동으로 따릅니다. iOS에서 설정 → 일반 → 언어 및 지역에서 기본 언어를 목록 상단으로 이동합니다. macOS에서는 시스템 설정 → 일반 → 언어 및 지역에서 설정합니다. Windows에서는 설정 → 시간 및 언어 → 언어 및 지역에서 변경합니다. SmartDecrypt를 재시작하면 인터페이스가 즉시 전환됩니다.

SmartDecrypt PDF ZIP이 하는 일

아직 사용해본 적 없다면, SmartDecrypt PDF ZIP이 실제로 무엇을 하는지 소개합니다.

매번 비밀번호를 입력하지 않고 보호된 파일 해제

SmartDecrypt는 Apple 키체인을 사용해 이름 있는 비밀번호 프로필을 저장합니다. 프로필을 한 번 만들고 파일명 패턴(예: 명세서-*.pdf 또는 급여명세-*.zip)에 연결하면 파일을 드롭할 때 자동으로 올바른 비밀번호가 적용됩니다.

지원 파일 형식

  • PDF — 은행, 급여 서비스, 정부 기관의 비밀번호 보호 PDF
  • ZIP — 만화 다운로드, 문서 묶음, 파일 전송에 사용되는 암호화 ZIP
  • 7z — 대용량 아카이브에 일반적인 오픈소스 7-Zip 형식

한 번에 일괄 복호화

SmartDecrypt Pro(iOS와 macOS의 일회성 유니버설 구매)로 여러 파일을 한 번에 불러와 저장된 프로필을 자동으로 순환시킬 수 있습니다. 급여 명세서 20장을 드롭하고 프로필을 적용하면 모든 파일이 기기에서 순서대로 처리됩니다.

키체인에 안전한 비밀번호 저장

iOS와 macOS에서는 Apple 키체인에, Windows에서는 시스템 자격 증명 관리자에 비밀번호가 저장됩니다. SmartDecrypt는 비밀번호를 데이터베이스 파일에 기록하거나 서버로 전송하지 않습니다.

100% 기기 내 처리

모든 복호화는 Rust FFI 코어를 사용해 로컬에서 실행됩니다. 파일이 기기 밖으로 나가지 않습니다.
