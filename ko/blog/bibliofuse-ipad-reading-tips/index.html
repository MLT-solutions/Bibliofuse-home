BiblioFuse iPad 사용 팁: 전체 화면 만화 읽기 가이드

iPhone은 편리하지만, 만화 읽기의 진정한 즐거움은 iPad에서 시작됩니다. 12.9인치 iPad Pro는 iPhone보다 약 4배 큰 화면을 제공해 두 페이지 스프레드를 거의 실제 크기로 표시하고, 긴 독서 세션에서도 눈의 피로를 줄일 수 있습니다. BiblioFuse는 iPad의 이러한 장점을 최대한 활용하도록 설계되었습니다.

올바른 표시 모드 선택

iPad에서 BiblioFuse로 만화를 열면 가장 먼저 페이지가 화면을 채우는 방식을 결정합니다. 설정 → 디스플레이로 이동하세요:

  • 너비 맞춤 — 각 페이지를 수평으로 화면 가장자리까지 확장. 세로 방향 페이지에 적합.
  • 높이 맞춤 — 페이지 전체를 표시하고 양쪽에 여백 표시. 내용이 잘리지 않음.
  • 화면 채우기 — 전체 화면을 채우도록 늘리거나 자름. 고화질 스캔에 최적.

표준 미국 만화를 iPad Pro에서 읽을 때 높이 맞춤 모드가 종이 책과 가장 유사한 경험을 제공합니다.

가로 모드에서 두 페이지 스프레드

두 페이지 스프레드에서 iPad를 가로로 회전하면 — BiblioFuse가 회전을 자동으로 감지하고 인쇄판처럼 두 페이지를 나란히 표시합니다.

설정 → 두 페이지 레이아웃에서 첫 페이지를 단독 표시(표지에 일반적)할지 쌍으로 표시할지, 그리고 페이지 간 간격을 제어할 수 있습니다.

일본식 오른쪽에서 왼쪽으로 읽는 만화는 읽기 방향 → RTL 설정 후 페이지 순서가 자동으로 반전됩니다.

썸네일 스크러버

iPad에서는 하단 스크러버 바가 실용적으로 사용할 수 있을 만큼 넓습니다. 화면 중앙을 탭해 표시하고 드래그해 원하는 위치로 이동합니다. 드래그하는 동안 썸네일이 표시되어 페이지 번호가 아닌 시각적 콘텐츠로 탐색할 수 있습니다.

스크러버를 길게 누르면 현재 위치에 북마크를 설정할 수 있습니다.

iCloud로 기기 간 동기화

iPhone에서도 읽는 경우 BiblioFuse가 iCloud를 통해 자동으로 읽기 위치를 동기화합니다. iPad에서 60페이지를 읽고 앱을 닫으면 iPhone에서 같은 파일을 열었을 때 60페이지부터 이어서 읽을 수 있습니다.

주 라이브러리는 Mac 홈 라이브러리 스트리밍을 활용하세요. Mac에서 BiblioFuse 스트림 서버를 실행하고 iPad에서 Wi-Fi로 연결합니다. Mac 라이브러리 — 수 TB의 만화도 — 를 파일 복사 없이 iPad로 직접 스트리밍합니다.

밝기 및 야간 독서

BiblioFuse 야간 모드 — 리더 툴바의 태양 아이콘을 탭해 활성화합니다. 페이지에 따뜻한 색조 필터를 적용해 블루 라이트를 줄이면서 패널 색상은 그대로 유지합니다.

iPad 라이브러리 정리

BiblioFuse는 iPad에서 더 큰 그리드를 사용하여 시각적 브라우징이 실용적입니다. 시리즈별로 폴더를 만들고 파일 이름을 일관되게 지정하세요 — 예: 시리즈 v01.cbz — 한눈에 파악할 수 있게 됩니다.

대량 가져오기는 Wi-Fi 전송이 최적입니다. 반복적인 AirDrop 전송보다 훨씬 빠릅니다.

Stage Manager와 화면 분할

Stage Manager를 지원하는 iPad Pro 또는 iPad Air에서 BiblioFuse는 크기 조절 가능한 창에서 실행됩니다. 화면 3분의 2에서 만화를 읽으면서 나머지 공간에 Safari를 열 수 있습니다.

외장 키보드의 화살표 키로 페이지를 넘길 수 있어 Magic Keyboard와 함께 사용하면 편리합니다.

핵심 요약

iPad는 만화 읽기를 진정으로 훌륭하게 만들어 줍니다 — 하지만 앱이 화면을 제대로 활용할 때만 가능합니다. 두 페이지 모드, 대형 썸네일 스크러버, iCloud 동기화, Mac 스트리밍이 모두 갖춰진 BiblioFuse iPad 버전은 단순히 큰 iPhone 이상의 경험을 제공합니다.
