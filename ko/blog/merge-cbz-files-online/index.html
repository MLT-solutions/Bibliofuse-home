여러 CBZ 파일을 하나로 합치는 방법 — 무료 온라인 도구

2026년 5월 26일

여러 CBZ 파일을 하나로 합치는 방법 — 무료 온라인 도구

만화 시리즈를 다운로드했더니 각 챕터가 별도의 CBZ 파일로 나눠져 있는 경우가 많습니다. 폴더에 파일이 40개씩 쌓여 있으면, 챕터를 다 읽을 때마다 다음 파일을 수동으로 열어야 합니다. 한 권 전체를 하나의 CBZ로 읽을 수 있다면 훨씬 편하겠죠. 문제는 파일을 합치려면 소프트웨어를 설치하거나 유료 서비스를 써야 할 것 같다는 점입니다.

BiblioFuse 웹 도구는 브라우저에서 바로 CBZ 파일을 무료로 합쳐줍니다. 파일을 서버에 업로드하지 않고, 계정도 필요 없습니다. 합쳐진 파일은 바로 컴퓨터로 다운로드됩니다.

CBZ 파일을 합치려는 이유

챕터별 파일 방식은 만화와 긴 시리즈에서 흔히 볼 수 있습니다. 스캔 그룹은 챕터마다 별도로 배포하고, 디지털 스토어도 시리즈 전체를 구매했더라도 호별로 판매하는 경우가 있습니다. 스마트폰이나 태블릿에서 읽다가 30페이지마다 다음 파일로 넘어가야 한다면 독서 흐름이 완전히 끊깁니다.

한 권 또는 아크 전체를 하나로 합치면 이런 장점이 있습니다.

  • 끊김 없는 읽기 — 리더가 하나의 연속 파일로 처리해 챕터 경계의 더블 페이지도 올바르게 표시됩니다
  • 깔끔한 라이브러리 관리 — 수십 개의 챕터 파일 대신 권당 파일 하나
  • 일관된 진도 관리 — 읽기 비율과 북마크가 챕터마다 초기화되지 않고 전체 권에 걸쳐 작동합니다
  • 간편한 전송 — 여러 파일 대신 하나만 이동하면 됩니다

CBZ 합치기 원리

CBZ는 이미지 파일이 담긴 ZIP 아카이브로, 각 페이지가 순서대로 이름 붙어 있어 리더가 순서를 알 수 있습니다. CBZ 파일 합치기란 이 이미지 시퀀스들을 하나의 ZIP으로 합치고, 페이지 번호를 다시 매겨 하나의 연속된 이야기로 읽히도록 만드는 과정입니다.

BiblioFuse 웹 도구는 번호 재매기기를 자동으로 처리합니다. 파일을 올리고 순서를 지정하면, 도구가 각 CBZ에서 이미지를 추출해 올바르게 배열하고 하나의 CBZ로 묶어줍니다.

CBZ 파일 온라인으로 합치는 방법

1단계: BiblioFuse 웹 도구 열기

데스크톱 브라우저에서 BiblioFuse 웹 도구를 엽니다. 로그인이나 계정이 필요 없습니다. 무료 버전은 한 번에 최대 3개 파일까지 합칠 수 있으며, 더 많은 파일은 업그레이드가 필요합니다.

2단계: CBZ 파일을 순서대로 추가하기

드롭 영역을 클릭하거나 CBZ 파일을 드래그해서 올려놓습니다. 순서가 중요합니다 — 도구는 파일을 추가한 순서대로 합칩니다. 대부분의 브라우저에서 폴더에서 여러 파일을 한 번에 선택할 수 있습니다. 파일 이름이 순서대로 지어져 있다면(Chapter 001.cbz, Chapter 002.cbz 등) 한 번에 선택하면 대개 올바른 순서로 추가됩니다.

추가 후 순서를 바꾸려면 파일 목록에서 드래그해서 위치를 조정할 수 있습니다.

3단계: 합치기 실행

합치기를 클릭합니다. 도구가 모든 CBZ 파일의 이미지를 하나의 아카이브로 합치면서 페이지 번호를 연속되게 다시 매깁니다. 진행 표시줄로 상태를 확인할 수 있습니다. 완료되면 합쳐진 CBZ를 바로 다운로드할 수 있습니다.

출력 파일 이름은 기본적으로 첫 번째 파일 이름으로 설정되며, 저장할 때 변경할 수 있습니다.

다양한 파일 형식 혼합 처리

BiblioFuse 웹 도구는 CBZ 외의 형식도 합칠 수 있습니다. 시리즈 내에 CBZ, CBR, ZIP 등 형식이 섞여 있어도 문제없습니다. 합치기 전에 모두 같은 형식으로 변환할 필요가 없습니다. 도구가 각 형식에서 페이지를 추출해 CBZ로 묶어줍니다.

형식이 통일되지 않은 오래된 스캔 컬렉션에도 유용합니다.

합치기 후 이미지 품질

합치기 과정에서 이미지 품질은 전혀 변하지 않습니다. 도구는 원본 아카이브의 이미지 파일을 재인코딩이나 압축 없이 그대로 결합합니다. 합쳐진 CBZ의 모든 페이지는 원본 파일과 동일합니다.

동시에 파일 크기도 줄이고 싶다면 BiblioFuse 웹 도구의 압축 기능을 합치기 전후에 별도로 실행할 수 있습니다. 하지만 이는 선택 사항이며, 일반 합치기는 모든 것을 그대로 보존합니다.

여러 챕터로 권 단위 파일 만들기

10~12개 챕터를 하나의 권 단위 CBZ로 묶는 것이 대표적인 활용법입니다. 다음과 같이 진행하면 됩니다.

  1. 1권에 해당하는 챕터를 모읍니다 — 대부분의 시리즈에서 110화 또는 112화입니다.
  2. BiblioFuse 웹 도구를 열고 챕터 파일을 순서대로 추가합니다.
  3. 합치기를 실행합니다. 결과물을 다운로드하고 시리즈명 Vol 01.cbz처럼 이름을 바꿉니다.
  4. 다음 권도 같은 방식으로 반복합니다.

각 결과 파일은 리더로 전송하거나, 라이브러리 앱에 등록하거나, 보관할 수 있는 깔끔한 단일 권 파일이 됩니다.

합친 후: 스마트폰으로 파일 전송하기

합쳐진 CBZ를 다운로드한 후 iPhone에 넣는 가장 편한 방법은 BiblioFuse의 Wi-Fi 전송 기능입니다.

iPhone에서 BiblioFuse를 열고 Wi-Fi 전송 버튼을 탭하면 화면에 브라우저 주소가 나타납니다. 컴퓨터 브라우저에서 그 주소를 열고 합쳐진 CBZ 파일을 페이지에 드래그합니다. 같은 로컬 네트워크를 통해 즉시 전송됩니다 — USB, iTunes, 클라우드 계정 모두 필요 없습니다.

다른 방법:

  • iCloud Drive: BiblioFuse와 연결된 iCloud Drive 폴더에 파일 넣기
  • AirDrop: Mac에서 iPhone으로 AirDrop 후 BiblioFuse에서 열기
  • 파일 앱: USB로 연결 후 파일 앱의 BiblioFuse 폴더에 복사

자주 묻는 질문

한 번에 합칠 수 있는 파일은 최대 몇 개인가요?

무료 버전은 한 번에 최대 3개입니다. 더 많은 파일을 합칠 때는 여러 번 나눠서 진행하면 됩니다. 1~3화를 먼저 합치고, 그 결과물에 다음 묶음을 합치는 방식으로 반복하면 됩니다.

챕터가 CBR 형식이면 어떻게 하나요?

CBR은 ZIP이 아닌 RAR 아카이브입니다. BiblioFuse 웹 도구는 CBR도 CBZ, 다른 형식과 함께 처리할 수 있습니다. CBZ와 CBR을 섞어서 추가해도 하나의 CBZ로 올바르게 합쳐집니다.

읽기 순서가 유지되나요?

네. 도구는 파일 목록에 표시된 순서대로 파일을 처리하고, 각 파일 내의 페이지는 기존 순서대로 추가됩니다. 챕터 파일을 올바른 순서로 추가하기만 하면 합쳐진 결과물도 올바르게 읽힙니다.

EPUB 파일도 같은 방식으로 합칠 수 있나요?

BiblioFuse 웹 도구는 EPUB 합치기도 지원합니다. EPUB 합치기는 별도 기능으로 제공됩니다. 스캔된 페이지 형태의 만화 콘텐츠라면 CBZ가 더 적합한 형식입니다.

출력 파일에 원본 메타데이터가 유지되나요?

합쳐진 CBZ는 단순한 이미지 아카이브입니다. 원본의 ComicInfo.xml 메타데이터는 자동으로 합쳐지지 않습니다. 메타데이터를 보존하려면 다운로드 후 수동으로 ComicInfo.xml을 추가하면 됩니다.

번거로움 없이 CBZ 파일 합치기

BiblioFuse 웹 도구는 CBZ 합치기가 원래 이래야 한다고 생각하는 방식으로 작동합니다 — 브라우저에서, 개인 정보 보호, 무료로, 아무것도 설치하지 않고. 파일을 순서대로 올리기만 하면 나머지는 도구가 알아서 처리합니다.

iPhone이나 iPad에서 BiblioFuse를 사용하는 독자라면, 웹 도구와 모바일 앱의 연계로 워크플로우가 매끄럽게 이어집니다. 컴퓨터에서 합치고, Wi-Fi로 전송하고, 읽으면 됩니다. 수십 개의 챕터 파일이 몇 분 만에 깔끔한 권 단위 라이브러리로 정리됩니다.

BiblioFuse 웹 도구를 열고 첫 번째 배치를 합쳐보세요.
