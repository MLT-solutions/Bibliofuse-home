ZIP를 CBZ로 온라인 변환: 몇 초 만에 만화 아카이브 이름 바꾸기

2026년 6월 11일

ZIP를 CBZ로 온라인 변환: 몇 초 만에 만화 아카이브 이름 바꾸기

만화 이미지가 가득한 ZIP 파일이 있는데 리더가 인식하지 못하는 상황입니다. 해결책은 약 3초면 충분합니다. CBZ 파일은 확장자만 다른 ZIP 아카이브입니다. 파일 이름을 바꾸면 리더가 만화로 인식합니다. BiblioFuse 웹 도구는 브라우저에서 이 작업을 처리합니다 — 업로드 없음, 소프트웨어 없음, 계정 없음.

만화 리더가 ZIP 파일을 무시하는 이유

대부분의 만화 리더는 파일 내용이 아닌 확장자로 파일 유형을 식별합니다. chapter-01.zip이라는 파일은 일반 아카이브처럼 보입니다. 동일한 파일을 chapter-01.cbz로 이름 바꾸면 즉시 만화 컨테이너로 인식됩니다. 바이트 내용은 변하지 않고 이름만 바뀝니다.

CBZ는 Comic Book ZIP의 약자입니다. CBZ 형식은 만화 리더에게 인식 가능한 확장자를 제공하기 위해 만들어졌습니다. 내부에는 항상 ZIP이 있으며, 올바른 읽기 순서로 이름이 지정된 이미지 파일이 들어 있습니다. CBR은 ZIP 대신 RAR 아카이브를 사용하는 같은 개념입니다.

방법 1 — 파일 직접 이름 바꾸기

가장 빠른 방법은 그냥 이름을 바꾸는 것입니다. Mac이나 Windows에서는 Finder나 파일 탐색기에서 확장자를 .zip에서 .cbz로 변경합니다. iOS에서는:

  1. 파일 앱을 열고 ZIP 파일로 이동합니다.
  2. 파일을 길게 누름 → 이름 변경.
  3. .zip을 .cbz로 변경하고 완료를 탭합니다.

iOS에서 확장자 변경에 대한 경고가 표시될 수 있습니다. 변경을 확인하세요. 이제 파일은 iPhone용 BiblioFuse를 포함한 모든 만화 리더가 열 수 있는 CBZ입니다.

방법 2 — BiblioFuse 웹 도구 사용

BiblioFuse 웹 도구는 WebAssembly를 사용하여 브라우저에서 완전히 실행됩니다. 파일은 장치를 벗어나지 않습니다 — 서버에 업로드되는 것이 없습니다.

ZIP를 CBZ로 변환하려면:

  1. 모든 최신 브라우저에서 **bibliofuse.com/ko/webapp**을 엽니다.
  2. 변환 도구를 선택합니다.
  3. ZIP 파일을 드래그하거나 클릭해서 선택합니다.
  4. 출력 형식으로 ZIP → CBZ를 선택합니다.
  5. 변환을 클릭하고 결과를 다운로드합니다.

출력 CBZ는 입력 ZIP과 동일한 이미지를 동일한 순서로 포함하며, 재인코딩이나 품질 손실이 없습니다.

일괄 변환

수십 개의 ZIP 파일이 있는 경우 웹 도구는 다중 입력을 지원합니다. 여러 ZIP을 한 번에 드래그하면 각각에 대한 CBZ가 생성됩니다.

CBZ 내부 구조

CBZ는 파일 이름 알파벳 순서로 정렬된 이미지를 포함하는 ZIP입니다. 리더는 그 순서로 표시합니다. 권장 명명 규칙: 0001.jpg, 0002.jpg... (0 패딩 포함). 패딩 없이는 10.jpg가 알파벳 순서에서 2.jpg 앞에 옵니다.

CBR 파일은?

CBR은 ZIP 대신 RAR을 사용합니다. CBR을 CBZ로 변환하려면 RAR을 추출하고 ZIP으로 다시 패키징해야 합니다. BiblioFuse 웹 도구도 이를 처리합니다. CBR → CBZ를 선택하면 이미지를 CBZ 확장자의 ZIP으로 다시 아카이브합니다.

핵심 요점

CBZ는 위장한 ZIP입니다. 대부분의 경우 확장자를 바꾸는 것만으로 충분합니다. 파일을 업로드하지 않고 일괄 변환을 원할 때 BiblioFuse 웹 도구는 몇 초 만에 ZIP을 CBZ로 변환합니다 — 완전히 비공개, 소프트웨어 불필요.
