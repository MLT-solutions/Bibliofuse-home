iPhone에서 웹툰 읽기: BiblioFuse의 무한 세로 스크롤

한국 만화(manhwa), 세로 스크롤 웹코믹, 또는 팬 번역 시리즈를 iPhone에 CBZ나 EPUB으로 다운로드했는데, 시도해 본 모든 코믹 리더가 잘못된 방향으로 표시하며 위에서 아래로 흘러야 할 패널을 분리해버리는 문제를 겪은 적 있으신가요? BiblioFuse는 이 형식을 위해 특별히 제작된 세로 스크롤 모드로 이 문제를 직접 해결합니다.

웹툰이 전통 만화와 다른 이유

전통 만화는 패널을 격자 형태로 배치합니다. 웹툰은 완전히 다릅니다. 모바일 읽기를 위해 설계된 길고 좁은 패널이 세로로 쌓여 하나의 연속된 스트립을 형성합니다. 이 형식은 LINE Webtoon과 Naver 같은 플랫폼에서 시작되어 자체 출판 웹코믹과 스캔 번역 만화로 확산되었습니다.

이 세로 형식은 전통 만화를 위해 설계된 앱에 문제를 일으킵니다. 페이지 넘기기 모드는 흐름을 깨고, 양면 보기는 패널을 작게 만들며, 세로 스크롤을 지원하는 앱도 스와이프 감도나 줌 수준 유지 같은 세부 사항을 놓치는 경우가 많습니다.

BiblioFuse의 세로 스크롤 모드

iPhone과 iPad용 BiblioFuse에는 전용 세로 스크롤 모드가 포함되어 있습니다. 각 이미지를 페이지로 처리하는 대신 연속으로 타일링하여 탭이나 스와이프 없이 전체 챕터를 아래로 스크롤할 수 있습니다. CBZ와 EPUB 웹툰 모두 지원됩니다.

읽기 모드 전환 방법:

  1. BiblioFuse에서 CBZ 또는 EPUB을 엽니다.
  2. 화면 중앙을 탭하여 리더 도구 모음을 표시합니다.
  3. 레이아웃 아이콘(겹쳐진 두 직사각형)을 탭합니다.
  4. 세로 스크롤을 선택합니다.

리더가 즉시 내용을 재구성합니다. 이미지가 화면 너비를 채우고, 아래로 스크롤하면 페이지 구분 없이 진행됩니다.

이미지 간격 설정

일부 웹툰은 챕터당 하나의 연속 이미지로 구성되고, 다른 것들은 개별 이미지로 나뉩니다. BiblioFuse를 통해 이미지 간격을 제어할 수 있습니다:

  • 간격 없음: 이미지가 끊김 없이 이어져 하나의 연속 이미지처럼 보입니다.
  • 최소 간격: 흰색 배경 페이지 사이에 얇은 구분선이 표시됩니다.

대부분의 웹툰에서는 간격 없음 모드가 가장 자연스러운 읽기 경험을 제공합니다.

웹툰 파일 가져오기

CBZ 파일은 스캔 번역에서 가장 일반적인 컨테이너입니다. CBZ는 PNG 또는 JPG 이미지를 포함하는 ZIP 아카이브입니다. BiblioFuse는 iPhone과 iPad에서 CBZ를 기본 지원합니다.

EPUB 파일은 공식 라이선스 웹툰에 일반적입니다. BiblioFuse는 고정 레이아웃 EPUB을 지원합니다.

iPhone으로 파일 전송 방법:

  • Wi-Fi 전송: BiblioFuse를 열고 전송 탭으로 이동하면 로컬 웹 서버가 시작됩니다. 같은 네트워크의 브라우저에서 표시된 주소를 방문하여 파일을 업로드합니다.
  • 파일 앱: iOS 파일 앱에서 CBZ 또는 EPUB을 공유 메뉴를 통해 BiblioFuse로 직접 공유합니다.

줌과 패닝

두 번 탭하면 영역이 확대되고, 다시 두 번 탭하면 전체 너비로 돌아옵니다. 세로 스크롤 줌은 페이지에 걸쳐 유지됩니다. 확대된 상태에서 가로 패닝도 지원됩니다.

긴 시리즈 읽기

일반적인 만화 시리즈는 100~300화 이상입니다. BiblioFuse는 폴더 내 개별 파일로 챕터를 구성합니다. 챕터가 끝나면 라이브러리로 돌아가지 않고 다음 챕터를 열 수 있는 다음 버튼이 나타납니다.

읽기 진행 상황은 여러 Apple 기기에서 iCloud를 통해 동기화됩니다.

핵심 요점

웹툰에는 세로 스트립 형식을 이해하는 리더가 필요합니다. BiblioFuse의 세로 스크롤 모드는 iPhone에서 CBZ와 EPUB 웹툰을 기본 지원하며, 조정 가능한 간격, 지속적인 줌, 부드러운 챕터 탐색을 제공합니다. BiblioFuse를 다운로드하여 지금 바로 시작하세요.
