BiblioFuse 도구 탭 사용 가이드: iPhone에서 압축, 변환, 병합하기

BiblioFuse의 도구 탭을 사용하면 CBZ, EPUB, PDF, RAR 파일을 iPhone 또는 iPad에서 직접 압축, 변환, 병합할 수 있다 — 컴퓨터나 클라우드 업로드가 필요 없다. 압축 도구는 이미지를 WebP로 재인코딩하여 일반적으로 파일 크기를 60~75% 줄인다. 변환은 PDF에서 CBZ, RAR에서 CBZ 변경을 처리하며, 병합은 여러 챕터 파일을 단일 볼륨 아카이브로 결합한다.

점심 휴식 중에 iPhone을 보다 보니 만화 컬렉션이 저장 공간을 잔뜩 차지하고 있습니다. 그 만화 시리즈의 CBZ 파일이 하나에 80–120MB씩이나 됩니다. 압축하고 싶지만 노트북은 집에 있고 오늘 밤까지 기다리고 싶지 않습니다. 아니면 시리즈의 여러 챕터 파일을 하나의 볼륨으로 합쳐서 길게 읽고 싶을 수도 있습니다.

BiblioFuse는 컴퓨터 없이 이 두 가지를 모두 처리합니다. iOS 앱에 내장된 도구 탭을 사용하면 iPhone이나 iPad에서 직접 만화와 전자책을 압축, 변환, 병합할 수 있습니다. 모든 작업은 기기 내에서 실행됩니다. 어떤 파일도 어디에도 업로드되지 않습니다. 데스크톱 앱도 필요 없습니다.

이 가이드는 도구 탭의 모든 기능을 살펴봅니다: 각 기능이 무엇을 하는지, 언제 사용하는지, 어떻게 실행하는지 안내합니다.

도구 탭이 하는 일

대부분의 만화 앱은 읽기 전용입니다. 파일을 가져오고 읽는 것이 전부입니다. 압축과 변환은 별도의 데스크톱 소프트웨어나 웹 도구가 필요합니다.

BiblioFuse는 파일 처리 기능을 리더에 직접 통합합니다. 즉, 라이브러리를 정리하고 표시하는 앱이 그 안의 파일을 변형하는 것도 가능합니다 — iPhone을 벗어나지 않고 파일을 줄이고, 형식을 바꾸고, 결합할 수 있습니다.

도구 탭은 세 가지 주요 기능을 제공합니다:

  • 압축 — CBZ 또는 EPUB 내부의 이미지를 재인코딩하여 파일 크기 줄이기
  • 변환 — 파일을 한 형식에서 다른 형식으로 변환 (예: CBZ에서 EPUB로)
  • 병합 — 여러 파일을 하나의 아카이브로 결합

각 작업은 기본적으로 비파괴적입니다: BiblioFuse는 새 파일을 만들고 원본은 그대로 유지합니다.

압축: 품질 손상 없이 파일 크기 줄이기

압축 도구는 아카이브 내부의 이미지를 재인코딩하여 파일 크기를 줄입니다. 기본적으로 이미지 파일의 ZIP 아카이브인 CBZ와 CBR 파일의 경우 파일 크기를 크게 줄일 수 있습니다.

사용 시기

  • 라이브러리가 저장 한도에 근접해 콘텐츠를 삭제하지 않고 공간을 확보하고 싶을 때
  • 느린 연결로 파일을 전송 중이고 파일을 더 작게 만들고 싶을 때
  • 불필요하게 높은 품질로 인코딩된 스캔을 다운로드했을 때 (예: 화면 독자를 위한 300 DPI)

파일 압축 방법

  1. BiblioFuse를 열고 도구 탭(화면 하단의 렌치 아이콘)으로 이동합니다.
  2. 압축을 탭합니다.
  3. 파일 추가를 탭하고 라이브러리에서 하나 이상의 CBZ, CBR 또는 EPUB 파일을 선택합니다.
  4. 압축 설정을 선택합니다:
    • 품질 — 무손실에서 공격적까지의 슬라이더. 만화와 코믹스의 경우 75–85% 품질은 화면에서 보통 눈에 띄지 않습니다.
    • 형식 — JPEG(더 작음) 또는 WebP(동일 품질에서 더 나은 압축)를 선택합니다.
    • 최대 크기 — 이미지 해상도를 선택적으로 제한합니다. 3000px 너비의 스캔이 휴대폰 화면에서 1600px보다 좋아 보이는 경우는 드뭅니다.
  5. 압축을 탭하여 처리를 시작합니다.

BiblioFuse는 각 파일의 진행 표시줄을 보여줍니다. 완료되면 압축된 버전이 새 파일로 라이브러리에 나타납니다. 원본은 변경되지 않습니다.

얼마나 절약할 수 있나요?

결과는 소스 파일이 어떻게 인코딩되었느냐에 따라 다릅니다. 고해상도 PNG 이미지가 있는 CBZ는 80% 품질의 JPEG로 변환하면 60–75% 줄어드는 경우가 많습니다. 이미 공격적인 JPEG 설정으로 압축된 파일은 크게 줄어들지 않습니다. 앱은 압축 전후 파일 크기를 보여주어 압축된 버전을 유지할지 결정할 수 있습니다.

변환: iPhone에서 파일 형식 변경하기

변환 도구는 파일을 한 형식에서 다른 형식으로 변경합니다. 모든 형식이 모든 상황에 동등하게 맞지는 않으며, 때로는 원하는 것보다 덜 편리한 형식으로 파일을 받거나 다운로드하게 됩니다.

일반적인 변환 시나리오

CBZ에서 EPUB로 — CBZ 파일은 이미지 아카이브입니다. EPUB로 변환하면 파일에 텍스트가 포함된 경우 리플로우 가능한 문서가 생성되거나, 일부 읽기 앱이 다르게 처리하는 EPUB 패키지에 이미지가 포함됩니다.

PDF에서 CBZ로 — PDF로 저장된 스캔 만화(구형 스캔에 일반적)가 있는 경우, CBZ로 변환하면 각 페이지가 개별 이미지 파일로 분리됩니다. 이렇게 하면 리더에서 표시 품질을 더 잘 제어할 수 있습니다.

RAR에서 CBZ로 — RAR은 일부 구형 스캔에서 사용하는 압축 아카이브 형식입니다. CBZ는 더 보편적으로 지원됩니다. RAR에서 CBZ로 변환하면 RAR을 지원하지 않는 앱에서 파일을 읽을 수 있게 됩니다.

파일 변환 방법

  1. 도구 탭에서 변환을 탭합니다.
  2. 파일 추가를 탭하고 변환할 파일을 선택합니다.
  3. 드롭다운에서 대상 형식을 선택합니다.
  4. 변환을 탭합니다.

변환된 파일이 라이브러리에 저장됩니다. 원본은 삭제되지 않습니다.

병합: 여러 파일을 하나로 결합하기

병합 도구는 여러 파일을 하나의 아카이브로 결합합니다. 이는 하나의 볼륨으로 함께 속하는 챕터별 다운로드에 가장 유용합니다.

사용 시기

  • 만화 시리즈의 1–12화를 개별로 다운로드하여 하나의 파일로 읽고 싶을 때
  • 다운로드를 위해 분할된 멀티파트 아카이브가 있어 다시 결합해야 할 때
  • 긴 시리즈를 볼륨으로 통합하여 탐색을 쉽게 하고 싶을 때

파일 병합 방법

  1. 도구 탭에서 병합을 탭합니다.
  2. 파일 추가를 탭하고 병합할 파일을 선택합니다. 필요한 만큼 추가할 수 있습니다.
  3. 순서 정렬 — 목록의 항목을 드래그하여 읽기 순서대로 배치합니다. 이것이 중요합니다: 병합 결과는 이 순서를 사용합니다.
  4. 출력 파일 이름을 지정합니다.
  5. 병합을 탭합니다.

BiblioFuse는 선택한 모든 파일의 이미지 내용을 지정한 순서대로 단일 CBZ 아카이브로 결합합니다. 병합된 파일이 라이브러리에 나타납니다. 원본은 삭제되지 않습니다.

병합 팁

병합 전에 순서를 확인하세요. 7화가 4화보다 앞에 오면 병합된 파일이 잘못되어 작업을 다시 해야 합니다. 파일을 올바르게 배치하는 데 잠깐 시간을 내세요.

일관된 소스 형식을 사용하세요. CBZ 파일을 함께 병합하면 깔끔하게 작동합니다. 형식을 혼합하면(일부 CBZ, 일부 PDF) 예상치 못한 결과가 발생할 수 있습니다. 혼합 형식을 병합해야 한다면 먼저 CBZ로 변환하세요.

병합은 CBZ 출력을 만듭니다. 입력 형식에 관계없이 병합된 결과는 CBZ 아카이브입니다.

기기 내 처리: 실제로 무엇을 의미하는가

세 가지 도구 — 압축, 변환, 병합 — 는 모두 iPhone이나 iPad에서 완전히 실행됩니다. 서버가 관여하지 않습니다. 파일이 기기를 떠나지 않습니다.

이것이 중요한 이유는 두 가지입니다:

프라이버시. 만화와 전자책 컬렉션은 개인적입니다. 더 이상 존재하지 않는 소스에서 합법적으로 구매한 대규모 컬렉션이 있거나, 서드파티 서버를 통해 보내고 싶지 않은 회색 지대의 콘텐츠가 있는 사람들도 있습니다.

신뢰성. 기기 내 처리는 네트워크 연결 없이도 작동합니다. 비행기 안에서, 외딴 곳에서, 또는 안정적인 인터넷이 없는 모든 곳에서 도구 탭은 여전히 작동합니다.

대신 속도가 느릴 수 있습니다. iPhone 프로세서는 강력하지만 50개의 고해상도 CBZ 파일을 압축하는 것은 데스크톱보다 휴대폰에서 더 오래 걸립니다. BiblioFuse는 파일을 백그라운드에서 처리하므로 압축이 실행되는 동안 계속 읽을 수 있습니다.

모두 합치기: 실제 워크플로우

도구 탭이 실용적인 루틴에 어떻게 맞는지 보여드리겠습니다:

새 만화 시리즈의 15개 챕터를 각각 별도의 CBZ 파일로, 각각 약 60MB씩 다운로드합니다. 하고 싶은 것:

  1. 챕터 1–5를 병합하여 1권, 챕터 6–10을 2권, 챕터 11–15를 3권으로 만들기.
  2. 저장 공간 영향을 줄이기 위해 각 권을 압축.
  3. 2권과 3권이 백그라운드에서 압축되는 동안 1권 읽기.

이 모든 것이 컴퓨터 없이, 아무것도 업로드하지 않고 iPhone에서 이루어집니다. 처음부터 끝까지: 파일 크기와 하드웨어 세대에 따라 약 10–15분.

BiblioFuse에서 최대한 활용하기

도구 탭은 BiblioFuse가 하는 일의 일부입니다. 같은 앱이 Wi-Fi 전송(Mac 또는 PC 브라우저에서 파일을 직접 휴대폰으로 드래그), Mac 홈 라이브러리(로컬 Wi-Fi를 통해 Mac의 만화 컬렉션을 iPhone으로 스트리밍), iCloud 동기화, 태그 및 평점, 텍스트 추출을 위한 OCR도 처리합니다.

BiblioFuse를 주로 리더로 사용해왔다면 도구 탭은 탐색할 가치가 있습니다. iPhone을 유능한 파일 처리 스테이션으로 만들어 줍니다 — 데스크톱이 필요 없습니다.

BiblioFuse는 App Store에서 iPhone과 iPad용으로 이용 가능합니다.

자주 묻는 질문

BiblioFuse 압축 도구는 무엇을 하나요? 압축 도구는 CBZ 및 EPUB 파일 내의 이미지를 WebP 형식으로 재인코딩하여 iPhone 또는 iPad 화면에서 눈에 띄는 품질 저하 없이 파일 크기를 일반적으로 60-75% 줄입니다. 처리는 iPhone의 이미지 압축 하드웨어를 사용하여 기기에서 완전히 실행됩니다.

컴퓨터 없이 iPhone에서 만화 파일을 압축할 수 있나요? 네. BiblioFuse 도구 탭은 iOS 17 이상에서 완전히 기기 내에서 실행됩니다. 라이브러리에서 CBZ 또는 EPUB 파일을 선택하고, 도구 대기열로 보내고, 압축을 선택하면 iPhone이 로컬에서 파일을 처리합니다. 데스크톱이 필요 없습니다.

BiblioFuse에서 CBZ 파일 병합은 어떻게 작동하나요? 첫 번째 CBZ 파일을 길게 눌러 선택 모드로 들어가고, 다른 챕터를 탭하여 추가합니다. 하단 바의 도구 버튼을 탭하고 병합을 선택하면 BiblioFuse가 지정한 순서대로 파일을 결합합니다. 결과는 내 iPhone -> BiblioFuse -> export 폴더에 저장됩니다.

BiblioFuse 압축 도구는 어떤 형식을 지원하나요? 압축 도구는 CBZ 및 EPUB 파일을 지원합니다. 변환 도구는 PDF에서 CBZ, RAR에서 CBZ로 변환합니다. 병합 도구는 여러 CBZ를 단일 CBZ 파일로 결합합니다. PDF에서 CBZ로의 변환은 WebP 재인코딩의 혜택을 받는 스캔된 PDF 파일에 특히 유용합니다.

도구 탭 사용 후 출력 파일은 어디로 가나요? iOS에서 처리된 파일은 내 iPhone -> BiblioFuse -> export에 자동으로 나타납니다. 거기서 기기 간 동기화를 위해 iCloud Drive -> BiblioFuse로 이동하거나 다른 앱과 공유할 수 있습니다. Android에서는 도구 실행 시 출력 폴더를 선택합니다.
