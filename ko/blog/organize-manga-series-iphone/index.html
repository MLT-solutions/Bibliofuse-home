iPhone에서 BiblioFuse로 만화를 시리즈별로 정리하는 방법

BiblioFuse는 내장 태그 시스템을 통해 iPhone에서 만화를 시리즈별로 정리할 수 있게 해줍니다. 각 권에 시리즈 이름으로 태그를 붙이고 라이브러리를 필터링하면 모든 권을 읽는 순서대로 확인할 수 있습니다. 이 앱은 iOS 17 이상에서 작동하며 CBZ, CBR, EPUB, PDF, ZIP, RAR, TXT 파일을 지원합니다. 태그와 평점은 iCloud를 통해 iPhone, iPad, Mac 간에 동기화됩니다.

문제: 50권의 만화, 아무런 순서 없이

원피스, 나루토, 진격의 거인, 베르세르크 — 인상적인 만화 컬렉션을 구축했지만, 모든 것이 구조 없이 하나의 평면 라이브러리에 뒤섞여 있습니다. 23권을 다 읽고 24권으로 바로 넘어가려 하면, 다음 권이 어느 것인지 기억하려고 표지 더미를 스크롤해야 합니다. 게다가 여러 시리즈의 권들이 뒤섞여 있어 특정 제목을 찾는 일이 정리되지 않은 책 더미에서 무언가를 찾는 것 같습니다. 좋은 정리 시스템이 바로 이 답답함을 해결해 줍니다.

폴더만으로는 왜 부족한가?

BiblioFuse는 파일 브라우저가 아닙니다. 메타데이터, 스마트 필터링, iCloud 동기화를 갖춘 완전한 독서 라이브러리입니다. 즉, 만화를 정리하는 방식은 파일이 어떤 폴더에 있는지를 훨씬 뛰어넘습니다. Mac의 폴더 구조는 깔끔해 보일 수 있지만, 파일이 BiblioFuse에 들어오면 중요한 것은 각 권에 첨부한 메타데이터입니다. 태그는 평면 라이브러리를 검색 가능하고 필터링 가능한 컬렉션으로 변환하여 읽는 방식 그대로 정리해 줍니다.

시리즈 이름으로 권에 태그를 붙이면 기기에 무관하게 지속되는 컬렉션 뷰가 만들어집니다. iPhone에서 iPad로 전환할 때마다 다시 정렬할 필요가 없습니다. 폴더 경로를 기억할 필요도 없습니다. 시리즈 태그를 탭하면 권들이 순서대로 나타납니다.

BiblioFuse에서 iPhone으로 만화를 시리즈별로 정리하는 방법

만화 라이브러리를 위한 깔끔한 시리즈 기반 정리 시스템을 설정하려면 다음 단계를 따르세요.

1단계: BiblioFuse를 열고 라이브러리로 이동

iPhone에서 BiblioFuse를 실행합니다. 화면 하단의 라이브러리 탭을 탭합니다. 모든 만화 권들이 표지 이미지로 표시됩니다.

2단계: 권에 시리즈 이름으로 태그 붙이기

만화 권을 길게 눌러 컨텍스트 메뉴를 엽니다. 태그를 탭하고 시리즈 이름(예: "원피스")을 입력합니다. 태그를 확인합니다. 이제 해당 권이 시리즈 레이블과 연결되며, 이 연결은 iCloud를 통해 다른 기기에 자동으로 동기화됩니다.

3단계: 시리즈의 나머지 모든 권에 태그 붙이기

시리즈의 모든 권에 태그 달기를 반복합니다. 대규모 컬렉션의 경우 일괄 선택을 사용하세요. 오른쪽 상단의 선택을 탭하고, 같은 시리즈의 모든 권을 선택한 다음 한 번에 태그를 적용합니다. 100권짜리 시리즈를 정리할 때 상당한 시간을 절약할 수 있습니다.

4단계: 시리즈 태그로 라이브러리 필터링하기

모든 권에 태그가 붙으면 라이브러리의 태그 필터 아이콘을 탭합니다. 목록에서 "원피스"를 선택합니다. BiblioFuse는 이제 해당 태그가 붙은 권만 표시합니다 — 원피스 전체가 다른 작품들과 분리되어 나타납니다.

5단계: 파일 이름으로 일관된 읽기 순서 설정하기

BiblioFuse는 태그 뷰 내에서 파일 이름의 알파벳 순서로 권들을 정렬합니다. 올바른 읽기 순서로 표시되도록 라이브러리에 추가하기 전에 파일 이름을 일관되게 지정하세요. 다음과 같은 형식을 사용하세요:

  • One Piece v01.cbz
  • One Piece v02.cbz
  • One Piece v03.cbz

앞에 0을 붙인 번호 매기기(v1, v2, v3 대신 v01, v02, v03)를 사용하면 알파벳 순서 정렬에서 v10이 v2 앞에 오는 문제를 방지할 수 있습니다. 일관된 이름 지정이 모든 시리즈에서 올바른 읽기 순서를 보장하는 가장 효과적인 방법입니다.

6단계: 평점을 사용해 읽기 진행 상황 추적하기

BiblioFuse는 모든 권에 1~5개의 별점 평가를 지원합니다. 평점을 사용해 읽기 진행 상황을 표시하세요. 예를 들어, 다 읽은 권에는 별 5개, 현재 읽고 있는 권에는 별 3개, 아직 읽지 않은 권에는 별 1개를 붙입니다. 평점으로 필터링하면 다음에 읽을 권을 빠르게 찾을 수 있습니다.

실용적인 예: 태그 붙이기, Wi-Fi Transfer, iCloud 동기화

시리즈 완전히 태그 붙이기: 원피스 107권을 모두 가지고 있다면, 일괄 선택을 사용해 전부 "원피스"로 태그를 붙입니다. 태그를 붙인 후 "원피스" 태그로 필터링하면 전체 시리즈가 나타나고, 파일 이름의 알파벳 순서 정렬로 자동으로 권 순서대로 배열됩니다.

Wi-Fi Transfer를 사용해 시리즈별로 권 추가하기: Mac에서 만화를 시리즈별 폴더로 정리합니다. "원피스"라는 폴더 하나, "베르세르크"라는 폴더 하나를 만듭니다. BiblioFuse의 Wi-Fi Transfer 기능(설정에서 찾을 수 있음)을 열고 각 폴더의 권들을 브라우저 업로드 페이지에 드래그합니다. 시리즈 폴더 전체를 한 번의 드래그 작업으로 추가하고 BiblioFuse에서 모든 권에 한 번에 태그를 붙일 수 있습니다.

iCloud 동기화로 일관성 유지하기: iPhone에서 적용한 태그나 평점은 iCloud를 통해 iPad와 Mac에 자동으로 동기화됩니다. 출퇴근 중에 iPhone에서 10권에 태그를 붙이면, 나중에 iPad로 앉을 때 그 태그들이 기다리고 있습니다. 같은 컬렉션을 두 번 정리할 필요가 없습니다.

BiblioFuse는 시리즈 정리를 어떻게 다르게 처리하나?

파일 이름과 폴더만으로 만화를 관리하는 방법은 데스크톱에서는 작동하지만 모바일 독서 앱에서는 한계가 있습니다. BiblioFuse의 태그 시스템은 고정된 폴더 구조의 경직성 없이 메타데이터의 유연성을 제공합니다. 한 권에 여러 태그를 붙일 수 있습니다 — "원피스", "완결", "즐겨찾기" — 그러면 세 가지 필터링 뷰에 동시에 나타납니다. 이는 어떤 폴더 시스템도 복제할 수 없는 것입니다.

검색도 파일 이름과 메타데이터 전체에서 작동하므로, 어떤 태그를 사용했는지 잊어버려도 검색창에 "원피스"를 입력하면 일치하는 모든 권이 즉시 나타납니다.

자주 묻는 질문

BiblioFuse가 만화 시리즈를 자동으로 감지하고 그룹화할 수 있나요? BiblioFuse는 현재 CBZ 메타데이터에서 시리즈를 자동으로 감지하지 않습니다. 시리즈 태그를 수동으로 지정하므로 권 그룹화 방식에 대한 완전한 제어권을 가집니다. 메타데이터만으로는 지원되지 않는 사용자 지정 그룹화도 포함됩니다.

태그가 iPad에 자동으로 동기화되나요? 네. 태그와 평점은 iCloud를 통해 모든 기기 — iPhone, iPad, Mac — 에 동기화됩니다. iCloud Drive가 활성화되어 있고 동일한 Apple ID로 로그인되어 있으면 어디서나 정리 상태가 일치합니다.

하나의 권에 태그를 몇 개까지 추가할 수 있나요? 어떤 권에든 여러 태그를 추가할 수 있습니다. 고정된 제한이 없습니다. 이를 통해 같은 권을 여러 카테고리에 동시에 정리할 수 있습니다 — 예: "원피스", "소년 만화", "읽는 중".

BiblioFuse가 CBZ나 EPUB 파일에 내장된 시리즈 메타데이터를 읽나요? BiblioFuse는 표시 목적으로 파일 내용을 읽습니다. 정리를 위해서는 태그 시스템이 주된 방법입니다. 수동 태그 붙이기는 소스 파일이 어떻게 패키징되었든 그룹화가 항상 정확하게 유지되도록 보장합니다.

시리즈 태그 내에서 사용자 지정 읽기 순서로 권을 정렬할 수 있나요? 태그 뷰 내에서 권은 파일 이름의 알파벳 순서로 정렬됩니다. 읽기 순서를 제어하는 가장 안정적인 방법은 앞에 0을 붙인 일관된 번호 매기기(v01, v02, v03)입니다. 이는 권이 추가된 시간과 무관하게 올바른 순서를 보장합니다.

BiblioFuse가 만화에 지원하는 파일 형식은 무엇인가요? BiblioFuse는 CBZ, CBR, ZIP, RAR, EPUB, PDF, TXT를 지원합니다. CBZ와 CBR은 가장 일반적인 만화 아카이브 형식으로 iOS 17 이상에서 완벽하게 지원됩니다.

오늘부터 만화 컬렉션 정리를 시작하세요

잘 정리된 만화 라이브러리는 좋아하는 독서 경험과 탐색하기 힘든 컬렉션 사이의 차이를 만듭니다. BiblioFuse의 태그 시스템, 일관된 파일 이름 규칙, iCloud 동기화를 통해 컬렉션의 모든 시리즈가 단 한 번의 탭으로 — 스크롤 없이, 추측 없이, 답답함 없이 — 손에 닿습니다.
