Mac에서 중복 CBZ 만화 파일 찾기 및 제거 방법

2026년 5월 30일

Mac에서 중복 CBZ 만화 파일을 찾고 삭제하는 방법

Comic Duplicate Scanner는 지각적 해싱을 사용해 Mac에서 중복된 CBZ, CBR 및 만화 아카이브 파일을 찾아냅니다. 파일 이름 변경, 재압축, 재패키징 후에도 일치하는 파일을 감지합니다. 전체 라이브러리를 스캔하고, 일치하는 파일 쌍을 나란히 검토한 뒤, 원본 파일을 안전하게 보호하면서 중복 파일을 휴지통으로 이동할 수 있습니다.

중복 만화 파일이 생각보다 찾기 어려운 이유

디지털 만화 라이브러리는 파일 이름이나 파일 크기 비교로는 감지할 수 없는 방식으로 중복 파일을 축적합니다. 한 소스에서 CBZ를 다운로드한 후 나중에 다른 릴리즈처럼 보이는 파일을 다운로드하면, 실은 같은 스캔 페이지가 다른 파일 이름으로 재패키징된 것입니다. 팬 번역본이 약간 다른 이미지 인코딩으로 재출시되기도 합니다. 3년 전 백업 드라이브를 메인 라이브러리에 병합하면 같은 파일이 다른 타임스탬프로 들어옵니다.

일반적인 중복 탐지 도구는 파일 해시를 비교합니다. 이름 변경이나 댓글 태그 추가와 같은 사소한 변경만으로도 다른 해시가 생성되어 중복 파일이 감지되지 않습니다. 지각적 해시는 다르게 작동합니다. 아카이브 내 이미지의 시각적 내용을 분석하여 정확한 바이트가 아닌 눈에 보이는 것을 기반으로 지문을 생성합니다.

Comic Duplicate Scanner는 지각적 해싱을 사용하여 다른 어떤 도구도 찾지 못하는 만화 중복 파일을 검출합니다.

만화에 대한 지각적 해싱은 어떻게 작동하나요?

Comic Duplicate Scanner가 CBZ 또는 CBR을 스캔할 때, 아카이브를 열고 내부 이미지를 샘플링하여 각 이미지에 대한 지각적 해시를 계산합니다. 이 해시는 정확한 픽셀 값에 의존하지 않고 형태, 대비 패턴, 레이아웃 등 시각적 구조를 포착합니다.

동일한 페이지가 담긴 두 CBZ 파일은 다음의 경우에도 일치하는 해시를 생성합니다:

  • 파일 이름이 완전히 다른 경우
  • 서로 다른 품질 설정으로 재압축된 경우
  • 한 파일에 다른 파일에 없는 메타데이터 태그가 있는 경우
  • 릴리즈 간에 페이지 순서가 다른 경우

이 덕분에 Comic Duplicate Scanner는 여러 소스에서 수년간 다운로드하며 쌓인 만화 라이브러리의 중복 파일을 처리하기에 적합한 도구입니다.

Mac에서 중복 CBZ 파일 찾는 방법

1단계: Comic Duplicate Scanner 다운로드 및 실행

Comic Duplicate Scanner는 Mac App Store에서 이용할 수 있습니다. 설치 후 응용 프로그램 폴더에서 실행하세요.

2단계: 만화 라이브러리 폴더 추가

만화 폴더를 Comic Duplicate Scanner 창으로 드래그하거나 폴더 추가 버튼을 사용해 탐색하세요. 메인 라이브러리, 외부 드라이브, 백업 위치 등 여러 위치를 동시에 스캔할 수 있습니다.

3단계: 스캔 시작

스캔을 클릭합니다. Comic Duplicate Scanner는 각 CBZ와 CBR을 열어 이미지를 샘플링하고 지각적 해시를 계산합니다. 최신 Mac에서 만화 1,000권 라이브러리는 보통 몇 분 안에 완료됩니다.

4단계: 일치하는 파일 쌍 검토

스캔 후 Comic Duplicate Scanner는 감지된 중복 그룹을 표시합니다. 각 그룹에는 파일 이름, 파일 크기, 수정 날짜, 폴더 경로와 함께 일치하는 아카이브가 나란히 표시됩니다. 표지 이미지를 미리 보거나 아카이브를 열어 확인할 수 있습니다.

5단계: 보관할 복사본 자동 선택

Comic Duplicate Scanner의 자동 선택 기능은 설정된 규칙에 따라 보관할 파일을 선택합니다. 더 큰 파일 보관(더 나은 품질 가능성), 선호 폴더의 파일 보관(메인 라이브러리), 또는 가장 최근에 수정된 복사본 보관 중에서 선택할 수 있습니다. 각 그룹에서 수동으로 보관할 파일을 선택할 수도 있습니다.

6단계: 중복 파일을 휴지통으로 이동

중복 제거(또는 휴지통으로 이동)를 클릭합니다. Comic Duplicate Scanner는 파일을 영구적으로 삭제하지 않으며, 휴지통으로만 이동합니다. 필요한 경우 비우기 전에 검토하세요.

대형 라이브러리를 위한 자동 선택 전략

200개의 중복 그룹을 다룰 때 하나하나 수동으로 검토하는 것은 현실적이지 않습니다. Comic Duplicate Scanner의 자동 선택 전략이 결정을 자동화합니다:

더 큰 파일 보관: 더 큰 CBZ 아카이브는 보통 더 고품질 이미지를 담고 있습니다. 이 전략은 자동으로 더 나은 버전을 보관합니다.

선호 폴더의 파일 보관: 메인 라이브러리가 ~/Comics에 있지만 중복 파일이 백업 드라이브에 있는 경우, Comic Duplicate Scanner가 항상 ~/Comics 복사본을 선호하도록 설정합니다. 백업 복사본은 삭제 대상으로 표시됩니다.

가장 최근에 수정된 파일 보관: 최신 파일이 동일한 스캔 그룹의 수정되거나 업데이트된 릴리즈인 경우에 유용합니다.

자동 선택 실행 후 확정하기 전에 표시된 선택 항목을 한 번 검토하세요. Comic Duplicate Scanner는 명시적인 확인 없이는 절대 삭제하지 않습니다.

퍼지 파일 이름 매칭은 어떻게 도움이 되나요?

지각적 해싱 외에도 Comic Duplicate Scanner는 퍼지 파일 이름 매칭을 사용하여 이미지 분석 전에도 중복 가능성이 있는 파일을 찾아냅니다. One-Punch-Man-v01.cbz와 One_Punch_Man_Vol_1.cbz 같은 파일이 잠재적 일치 항목으로 표시됩니다.

퍼지 파일 이름 매칭과 지각적 해시 확인의 이중 레이어 접근법은 사람에게는 명확하지만 정확한 문자열이나 바이트 비교에 의존하는 도구에는 보이지 않는 중복 파일을 잡아냅니다.

자주 묻는 질문

Mac에서 중복 CBZ 파일을 어떻게 찾나요? Comic Duplicate Scanner를 사용하세요. 만화 폴더를 추가하고 스캔을 클릭하면 앱이 지각적 해싱을 사용해 중복된 CBZ 및 CBR 파일을 식별합니다. 파일 이름 변경이나 재압축 후에도 일치 항목을 감지하고, 중복 파일을 휴지통으로 안전하게 이동할 수 있습니다.

파일 이름이 다른 경우에도 Comic Duplicate Scanner가 중복을 찾을 수 있나요? 네. Comic Duplicate Scanner는 CBZ 아카이브 내 이미지에 지각적 해싱을 사용합니다. 두 아카이브에 같은 스캔 페이지가 있으면 파일 이름, 파일 크기, 메타데이터에 관계없이 중복으로 표시됩니다.

Comic Duplicate Scanner가 파일을 영구적으로 삭제하나요? 아니요. Comic Duplicate Scanner는 파일을 휴지통으로만 이동합니다. 비우기 전에 검토하고 복원할 수 있으므로 어떤 크기의 라이브러리에도 안전하게 사용할 수 있습니다.

Comic Duplicate Scanner는 어떤 만화 형식을 지원하나요? Comic Duplicate Scanner는 CBZ 및 CBR 만화 아카이브를 지원합니다. 사진 파일(JPEG, PNG, HEIC, WebP, RAW)과 표준 ZIP/RAR 아카이브도 스캔합니다.

대형 만화 라이브러리 스캔은 얼마나 걸리나요? 스캔 속도는 라이브러리 크기와 Mac 성능에 따라 다릅니다. 1,000개의 CBZ 파일 라이브러리는 보통 몇 분 안에 완료됩니다. Apple 실리콘 Mac은 대형 컬렉션에서 인텔 Mac보다 훨씬 빠르게 스캔합니다.

Comic Duplicate Scanner는 여러 드라이브에 걸쳐 작동하나요? 네. 마운트된 드라이브나 폴더를 스캔 위치로 추가하면 됩니다. Comic Duplicate Scanner는 여러 드라이브에 걸쳐 파일을 동시에 비교합니다. 메인 라이브러리와 백업 드라이브 간의 중복 파일을 찾는 데 이상적입니다.

하루 오후만에 깔끔한 라이브러리 만들기

중복 만화 파일은 저장 공간을 낭비하고, 라이브러리 보기를 어지럽히며, 시간이 갈수록 정리를 더 어렵게 만듭니다. Comic Duplicate Scanner는 가장 어려운 부분인 파일 이름과 무관하게 시각적으로 동일한 아카이브를 감지하는 작업을 처리하고, 휴지통만을 통한 안전한 정리 방법을 제공합니다. Mac App Store에서 Comic Duplicate Scanner를 찾아 오늘 첫 번째 스캔을 실행하세요. 라이브러리가 정리되면 BiblioFuse를 사용해 iPhone과 Mac에서 만화 컬렉션을 읽고 관리하세요.
