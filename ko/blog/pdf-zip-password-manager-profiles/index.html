CBZ 및 PDF 컬렉션의 비밀번호 저장 — 다시는 입력하지 마세요

SmartDecrypt PDF ZIP CBZ는 iPhone, iPad, Mac에 이름이 있는 비밀번호 프로필을 저장하여, 보호된 PDF·CBZ·ZIP 파일의 비밀번호를 다시 입력할 필요가 없게 해줍니다. 프로필을 한 번만 만들면 SmartDecrypt가 일괄 복호화 시 자동으로 적용하며, 단 한 번의 탭으로 수백 개의 파일을 처리합니다. 모든 과정이 기기 내에서 완결되며 클라우드 업로드는 전혀 없습니다.

만화와 PDF 수집가 모두가 겪는 문제

몇 년에 걸쳐 디지털 라이브러리를 구축해왔다면 익숙한 상황일 것입니다. Kickstarter 캠페인의 만화 아카이브, PDF 아트북, 팬 번역본, 구독 아카이브, 출판사 릴리즈 — 이 중 상당수가 비밀번호로 보호되어 있습니다. 새 묶음이 도착할 때마다 같은 사이클이 반복됩니다. 파일 열기, 비밀번호 입력, 복호화, 반복.

비밀번호 자체가 문제가 아닙니다. 이미 알고 있고, 메모 앱에 저장해뒀을 수도 있습니다. 진짜 문제는 기계적인 반복입니다. 파일 30개면 30번의 비밀번호 입력이 필요합니다. 월간 명세서나 주간 신작처럼 정기적으로 도착하는 컬렉션에서는 이것이 반복적인 잡무가 됩니다.

iPhone의 대부분의 복호화 도구는 파일을 하나씩 처리하며 세션 간에 아무것도 기억하지 못합니다.

SmartDecrypt PDF ZIP CBZ는 비밀번호 프로필로 이 문제를 해결합니다. 정기적으로 사용하는 모든 비밀번호를 위한 이름 있는, 안전한, 기기 내 저장소를 제공합니다. SmartDecrypt로 프로필을 한 번 저장하면, 그 비밀번호는 다시 입력할 필요가 없습니다.

비밀번호 프로필은 어떻게 작동하나요?

SmartDecrypt의 비밀번호 프로필은 비밀번호를 기기에 안전하게 저장하는 이름 있는 항목입니다. "출판사 X 아카이브", "2025년 은행 명세서", "시리즈 ABC 팬섭"처럼 설명적인 이름을 붙여두면, 다음에 그 출처의 파일이 생겼을 때 비밀번호를 입력하는 대신 프로필을 선택하기만 하면 됩니다.

프로필은 기기의 보안 엔클레이브로 보호된 안전한 온디바이스 저장소에 보관됩니다. 어떤 클라우드 서비스와도 동기화되지 않으며, 기기 밖으로 나가지 않고, 언제든지 삭제할 수 있습니다. 컬렉션 수만큼 프로필을 만들 수 있습니다.

비밀번호 프로필 만들기 및 사용 방법

1단계: 프로필 탭 열기

iPhone, iPad 또는 Mac에서 SmartDecrypt를 실행합니다. 하단 툴바(Mac에서는 사이드바)의 프로필 탭으로 이동합니다.

2단계: 새 프로필 추가하기

  • 버튼을 탭합니다(Mac에서는 '프로필 추가' 클릭). 컬렉션에 대한 설명적인 이름을 입력합니다 — 목록에서 고를 때 바로 알아볼 수 있는 이름으로 합니다. 그런 다음 비밀번호를 입력합니다.

3단계: 저장하고 사용하기

저장을 탭하면 프로필 목록에 즉시 나타납니다. 복호화할 파일을 가져올 때 비밀번호를 직접 입력하는 대신 "프로필 선택"을 탭합니다. SmartDecrypt가 저장된 비밀번호를 묶음의 모든 파일에 적용합니다.

4단계: 시간이 지나도 프로필 관리하기

언제든지 프로필 탭으로 돌아와 프로필을 이름 변경, 편집, 또는 삭제할 수 있습니다. 출판사가 아카이브 비밀번호를 변경하면 몇 초 만에 프로필을 업데이트할 수 있으며, 이후의 모든 묶음에 새 비밀번호가 자동으로 적용됩니다.

비밀번호 프로필의 실용적인 활용 사례

월간 금융 문서

은행과 금융 기관은 표준 비밀번호로 PDF를 발송합니다 — 주로 생년월일이나 계좌번호입니다. 매달 그 비밀번호를 찾는 대신, 한 번 프로필로 저장해두세요. 월간 명세서 묶음이 도착하면 프로필을 선택하고 12개 파일을 한 번에 복호화할 수 있습니다.

구독 만화 아카이브

독립 출판사와 팬 그룹은 구독자 비밀번호가 있는 CBZ 아카이브를 배포하는 경우가 많습니다. 새 릴리즈는 주간 또는 월간으로 도착합니다. 저장된 프로필이 있으면 다운로드 후 한 번의 탭으로 전체 묶음이 복호화됩니다. BiblioFuse 라이브러리를 번거로움 없이 항상 최신 상태로 유지할 수 있습니다.

다중 출처 컬렉션

각기 다른 비밀번호를 사용하는 여러 출처의 파일이 있다면 각 출처마다 별도의 프로필을 만드세요. 출처 A의 파일을 선택하고 프로필 A를 적용하고, 출처 B의 파일에는 프로필 B를 적용합니다. 어떤 비밀번호가 어느 묶음에 맞는지 추측할 필요가 없습니다.

Mac에서도 작동하나요?

네. SmartDecrypt PDF ZIP CBZ는 동일한 구매로 iPhone, iPad, Mac에서 모두 사용할 수 있습니다. Mac에서는 비밀번호로 보호된 파일 폴더 전체를 SmartDecrypt로 드래그하고, 저장된 프로필을 선택한 후 한 번에 모두 복호화할 수 있습니다.

Mac 공유 확장 기능은 Finder와 통합됩니다. Finder에서 파일을 선택하고 오른쪽 클릭 후 공유 → SmartDecrypt를 선택하고, 프로필을 선택한 후 복호화합니다. 전체 워크플로는 클릭 세 번이며, iPhone에서 만든 비밀번호 프로필이 항상 그대로 있습니다 — 추가 설정 없이.

자주 묻는 질문

SmartDecrypt에 비밀번호를 저장하는 방법은? SmartDecrypt를 열고 프로필 탭으로 이동한 후, +를 탭하여 새 프로필을 추가합니다. 설명적인 이름과 비밀번호를 입력한 다음 저장을 탭합니다. 프로필은 기기에 안전하게 저장되며, 해당 컬렉션의 파일을 복호화할 때마다 선택할 수 있습니다.

SmartDecrypt에 저장된 비밀번호는 안전한가요? 네. SmartDecrypt는 기기의 보안 엔클레이브로 보호된 안전한 온디바이스 저장소를 사용하여 프로필을 저장합니다. 비밀번호는 업로드되거나 클라우드 서비스와 동기화되거나 기기 밖으로 나가는 일이 없습니다. 오직 사용자만 접근할 수 있습니다.

서로 다른 컬렉션을 위한 여러 개의 비밀번호 프로필을 만들 수 있나요? 네. SmartDecrypt는 저장 프로필 수에 제한이 없습니다. 출판사별, 시리즈별, 문서 출처별로 하나씩 — 필요한 만큼 만들 수 있습니다.

SmartDecrypt가 일치하는 파일에 프로필을 자동으로 적용하나요? SmartDecrypt는 파일 이름으로 프로필을 자동 매칭하지 않습니다 — 복호화 단계에서 직접 어떤 프로필을 사용할지 선택합니다. 이를 통해 의도적인 제어가 가능합니다. 파일 묶음을 선택하고, 프로필을 선택한 다음, 한꺼번에 복호화합니다.

저장된 비밀번호 프로필을 편집하거나 삭제할 수 있나요? 네. 프로필 탭으로 이동하여 변경하려는 프로필을 탭하고, 편집하거나 삭제합니다. 프로필 편집은 이미 복호화된 파일에 영향을 주지 않으면서 향후 사용을 위해 저장된 비밀번호를 업데이트합니다.

비밀번호 프로필이 PDF뿐만 아니라 ZIP과 CBZ 파일에도 적용되나요? 네. SmartDecrypt PDF ZIP CBZ는 세 가지 형식 모두에서 비밀번호 프로필을 지원합니다. 동일한 비밀번호를 공유한다면 PDF, CBZ, ZIP이 혼합된 컬렉션을 하나의 프로필로 일괄 복호화할 수 있습니다.

같은 비밀번호를 두 번 다시 입력하지 마세요

비밀번호 프로필은 SmartDecrypt PDF ZIP CBZ를 한 번 설정하면 그만인 복호화 시스템으로 만들어줍니다. 새 컬렉션이 도착했을 때 한 번 프로필을 만들어두면, 이후 그 출처의 모든 묶음이 한 번의 탭으로 복호화됩니다 — iPhone, iPad, Mac 어디서든, 항상 기기 내에서, 업로드 없이. App Store에서 SmartDecrypt PDF ZIP CBZ를 검색하여 시작하세요.
