Changelog

GrepTag Reader

Release history, known issues, and what's coming next.

2026-07-071.0.5
iOS & macOSFeature

Build 17 / v1.0.5 — Whole-book reader search

Search now scans the entire book you're reading, not just the current chapter, for both TXT and EPUB. Wi-Fi Sync is safer — deletions detected on another device are now suggested rather than applied automatically. Fixed a bug that could create duplicate library entries when a book file was replaced in place. Released on the App Store for iOS and macOS.

2026-07-061.0.4
iOS & macOSFeature

Build 16 / v1.0.4 — Reader themes, typefaces & custom fonts

A much richer reader on Mac and iPhone: new typefaces with live previews — Iowan, Charter, Kaiti (CJK-aware) and more — or import your own .ttf/.otf fonts. New Sepia, Gray, and Black (OLED) themes, brightness dimming, justified text alignment, and bold. Find in book for TXT and EPUB, with a matching find bar and typeface preview on macOS. Released on the App Store for iOS and macOS.

2026-06-281.0.3
iOS & macOSFeature

Build 15 / v1.0.3 — Reader navigation improvements

Smoother page-turn and chapter-navigation behavior in the EPUB and TXT readers, with onboarding refinements. Released on the App Store for iOS and macOS.

2026-06-251.0.1
iOS & macOSFix

Build 10 / v1.0.1 — Wi-Fi sync iCloud path refactor

Rewrote the iCloud snapshot path layout for Wi-Fi sync. Snapshots are now written to both Documents/Library/Sync and Library/Sync paths so iPhone, iPad, and Mac can discover each other reliably across all iCloud Drive layouts. Uses the visible iCloud Drive folder for snapshot publishing. Fixed cases where devices could not see each other's sync state. Released on the App Store for iOS and macOS; iOS is version 1.0.1 build 10 and the Mac release is live.

2026-06-251.0.0
iOS & macOSFix

Build 9 — macOS window-close fix

Fixed macOS window-close behavior that could leave the app in a broken state — closing the window now properly cleans up without forcing a full quit.

2026-06-251.0.0
PC (Windows)Fix

Windows — Wi-Fi sync iCloud path fix & Store localization

Fixed Windows writing sync snapshots to the wrong iCloud path — Mac and iOS apps can now correctly discover PC sync state. Removed a Documents/Sync read path that was never written to, and added the Documents/Library/Sync path that iOS/macOS apps actually write to. Store listing localized across 11 languages for the Microsoft Store release.

2026-06-251.0.0
PC (Windows)Feature

Windows — Initial build: grep, TTS & Wi-Fi sync

WinUI 3 desktop app for TXT and EPUB library management on Windows. Keyword grep profiles with AND/OR/NOT logic. Tag database with JSON export/import for cross-device tag sync with the iOS/macOS app. TTS reading with chapter continuation and Now Playing lock-screen controls. Wi-Fi sync over LAN. Available with a 1-month free trial on the Microsoft Store.

2026-06-241.0.0
iOS & macOSRelease

Build 8 — TTS Now Playing & chapter continuity

TTS Now Playing lock-screen controls on iOS and macOS. TTS sleep timer. Chapter continuation — audio keeps playing as TTS advances through chapters without stopping. Indexing progress shown live in sidebar during folder scan. Cross-device sort consistency using file created/modified and library-added dates.

2026-06-221.0.0
iOS & macOSFeature

Build 7 — Onboarding: Book Covers & cross-profile grep guide

New onboarding page explaining the three ways to add TXT book covers (same-folder images, URL fetch, and the Fetch Covers batch tool). New onboarding page showing how to run a grep search across multiple profiles in one pass. 14 new strings localized in all 11 languages.

2026-06-221.0.0
iOS & macOSFix

Build 6 — Localization fixes for builds 4 & 5

29 UI strings introduced in builds 4 and 5 (grep profiles, book covers, TTS volume, Wi-Fi sync) that were missing translations have been fully localized across all 11 languages.

2026-06-221.0.0
iOS & macOSFeature

Build 5 — Book covers, TTS volume & EPUB SVG

EPUB covers auto-extracted on import (free). TXT books match covers from same-folder images on discover (Pro). Set TXT cover from a URL via context menu (Pro). Batch "Fetch Covers" button in Settings (Pro). TTS volume control. EPUB SVG image rendering support. WKWebView EPUB reader improvements and bug fixes.

2026-06-131.0.0
iOS & macOSFeature

Build 4 — Cross-profile grep & LAN thumbnail sync

Run a single grep search across all saved profiles in one pass. Real-time indexing progress during folder scan. Tag display count badge. EPUB cover extraction free for all users. WKWebView-based EPUB reader replacing UIWebView. LAN thumbnail sync between iPhone, iPad, and Mac (Pro).

2026-05-011.0.0
iOS & macOSFeature

Build 3 — Grep tab redesign & profile editor fixes

Overhauled the Grep tab layout for clearer profile and results separation. Fixed profile dialog field validation and edge-case save errors.

2026-04-151.0.0
iOS & macOSRelease

Build 1–2 — Initial launch & 11-language localization

App Store launch for iOS and macOS. Full-text grep search with AND/OR/NOT Boolean logic. Save named scan profiles and re-run as your library grows. Tag books from scan results. Free tier: 2 profiles, 5 tags, EPUB cover extraction. Pro: unlimited profiles and tags, TXT covers, file organisation, Wi-Fi sync. Build 2 added full 11-language localization and an onboarding language picker.
