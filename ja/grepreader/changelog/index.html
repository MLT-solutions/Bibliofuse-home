GrepTag Reader

Release history, known issues, and what's coming next.

2026-07-071.0.5
iOS & macOSFeature

Build 17 / v1.0.5 — リーダーでの全書検索

検索が現在の章だけでなく、読んでいる本全体を対象にするようになりました（TXT・EPUB 両対応）。Wi-Fi 同期がより安全に — 他のデバイスで検出された削除は、自動的に適用されるのではなく提案されるようになりました。ブックファイルを同じ場所で置き換えた際にライブラリに重複項目が作成される不具合を修正しました。iOS と macOS 向けに App Store で公開済みです。

2026-07-061.0.4
iOS & macOSFeature

Build 16 / v1.0.4 — リーダーのテーマ、フォント、カスタムフォント

Mac と iPhone でリーダーが大幅に進化：ライブプレビュー付きの新しいフォント — Iowan、Charter、楷体（CJK 対応）など — または自分の .ttf/.otf フォントをインポート。新しいセピア・グレー・ブラック（OLED）テーマ、明るさの減光、両端揃えテキスト、太字。TXT と EPUB のブック内検索、macOS では対応する検索バーとフォントプレビューも追加。iOS と macOS 向けに App Store で公開済みです。

2026-06-281.0.3
iOS & macOSFeature

Build 15 / v1.0.3 — リーダーナビゲーションの改善

EPUB・TXT リーダーのページめくりとチャプターナビゲーションがより滑らかになり、オンボーディングも改善されました。iOS と macOS 向けに App Store で公開済みです。

2026-06-251.0.1
iOS & macOSFix

Build 10 / v1.0.1 — Wi-Fi 同期 iCloud パスのリファクタリング

Wi-Fi 同期の iCloud スナップショットパスレイアウトを書き直しました。スナップショットが Documents/Library/Sync と Library/Sync の両方のパスに書き込まれるようになり、すべての iCloud Drive レイアウトで iPhone・iPad・Mac が確実に互いを検出できます。スナップショットの公開に可視 iCloud Drive フォルダを使用。デバイスが互いの同期状態を確認できないケースを修正。iOS と macOS 向けに App Store で公開済みです。iOS は version 1.0.1 build 10、Mac 版も公開済みです。

2026-06-251.0.0
iOS & macOSFix

Build 9 — macOS ウィンドウ閉じる修正

macOS のウィンドウを閉じる際にアプリが壊れた状態になる可能性があった問題を修正しました — ウィンドウを閉じると完全な終了を強制せずに適切にクリーンアップされます。

2026-06-251.0.0
PC (Windows)Fix

Windows — Wi-Fi 同期 iCloud パス修正 & Store ローカライズ

Windows が同期スナップショットを誤った iCloud パスに書き込んでいた問題を修正 — Mac と iOS アプリが PC の同期状態を正しく検出できるようになりました。一度も書き込まれなかった Documents/Sync 読み取りパスを削除し、iOS/macOS アプリが実際に書き込む Documents/Library/Sync パスを追加。Microsoft Store リリースに向けてストアリストを 11 言語でローカライズ。

2026-06-251.0.0
PC (Windows)Feature

Windows — 初期ビルド：grep、TTS & Wi-Fi 同期

Windows 用 WinUI 3 デスクトップアプリ。TXT と EPUB のライブラリ管理。AND/OR/NOT 論理によるキーワード grep プロファイル。JSON エクスポート/インポート対応のタグデータベースで iOS/macOS アプリとクロスデバイスタグ同期。TTS はチャプター連続再生とロック画面の Now Playing コントロールに対応。LAN 経由の Wi-Fi 同期。Microsoft Store にて 1 か月無料トライアルあり。

2026-06-241.0.0
iOS & macOSRelease

Build 8 — TTS「再生中」とチャプター連続再生

iOS と macOS での TTS ロック画面コントロール。TTS スリープタイマー。チャプター連続再生 — TTS がチャプターを進む際にオーディオが途切れず再生を続けます。フォルダスキャン中にサイドバーでインデックス作成の進捗をリアルタイム表示。ファイルの作成日・更新日とライブラリ追加日を使用したデバイス間のソート一貫性。

2026-06-221.0.0
iOS & macOSFeature

Build 7 — オンボーディング：ブックカバーとクロスプロファイル grep ガイド

TXT ブックのカバーを追加する 3 つの方法（同じフォルダの画像、URL 取得、バッチカバー取得ツール）を説明する新しいオンボーディングページ。複数のプロファイルにまたがる grep 検索を一度に実行する方法を示す新しいオンボーディングページ。14 の新しい文字列を全 11 言語でローカライズ。

iOS & macOSFix

Build 6 — Build 4・5 のローカライズ修正

Build 4 と 5 で追加された 29 個の UI 文字列（grep プロファイル、ブックカバー、TTS 音量、Wi-Fi 同期）の翻訳が欠けていた問題を修正し、全 11 言語でローカライズを完了しました。

2026-06-221.0.0
iOS & macOSFeature

Build 5 — ブックカバー、TTS 音量、EPUB SVG

インポート時に EPUB カバーを自動抽出（無料）。TXT ブックは発見時に同じフォルダの画像からカバーを自動マッチング（Pro）。コンテキストメニューで URL から TXT カバーを設定（Pro）。設定の「カバーを一括取得」ボタン（Pro）。TTS 音量コントロール。EPUB SVG 画像レンダリング対応。WKWebView EPUB リーダーの改善とバグ修正。

2026-06-131.0.0
iOS & macOSFeature

Build 4 — クロスプロファイル grep と LAN サムネイル同期

保存済みのすべてのプロファイルに対して一度に grep 検索を実行。フォルダスキャン中のリアルタイムインデックス進捗表示。タグ表示数バッジ。全ユーザー向け EPUB カバー抽出（無料）。UIWebView に代わる WKWebView ベースの EPUB リーダー。iPhone・iPad・Mac 間の LAN サムネイル同期（Pro）。

2026-04-151.0.0
iOS & macOSRelease

Build 1–2 — 初回リリースと 11 言語ローカライズ

iOS と macOS の App Store リリース。AND/OR/NOT ブール論理によるフルテキスト grep 検索。名前付きスキャンプロファイルを保存し、ライブラリが増えたら再実行。スキャン結果からブックにタグ付け。無料：プロファイル 2 件、タグ 5 件、EPUB カバー抽出。Pro：プロファイルとタグは無制限、TXT カバー、ファイル整理、Wi-Fi 同期。Build 2 では 11 言語の完全ローカライズとオンボーディング言語ピッカーを追加。
