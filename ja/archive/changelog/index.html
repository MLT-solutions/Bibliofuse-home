Comic Duplicate Scanner

Release history, known issues, and what's coming next.

2026-06-261.2.0
All platformsRelease

v1.2.0 — オンボーディングのローカライズと Windows 言語サポート

オンボーディング画面が対応するすべての 11 言語で完全にローカライズされました。App Store Connect のメタデータが全 11 地域のローカライズされた説明で更新されました。Windows アプリに言語設定が追加され、すべての UI 文字列が完全にローカライズされました — Mac アプリの 11 言語サポートに対応。

2026-05-061.1.7
macOSRelease

v1.1.7 — macOS

macOS で App Store のローカライズ後に StoreKit が誤った通貨金額を表示していた問題を修正しました。

2026-05-021.1.4
macOSRelease

v1.1.4 — macOS

macOS にフリーミアムアクセスフローを追加しました。Mac App Store ビルドで 7 日間のトライアルゲート、StoreKit でのアンロック、機能ごとのプレミアム制限が有効になりました。

2026-04-221.1.0
macOSFix

v1.1.0–1.1.3 — 削除の信頼性向上

直接削除を send2trash に置き換え、ファイルが永久削除されるのではなく macOS のゴミ箱に移動するようにしました。NAS フォールバックのバグとサイレントな行削除の問題を修正しました。クロスプラットフォームのボタン監査に関する全修正を適用しました。

2026-04-221.0.6
macOSFeature

NAS・ネットワークボリューム対応

NSOpenPanel を介して NAS ドライブやネットワークボリューム上のフォルダをスキャンできるようになりました。初回起動時のキーチェーンプロンプトを 4 回から 1 回に削減。IAP の UX 改善とトライアルロジックの堅牢化。

2026-04-211.0.1
macOSFix

v1.0.1 — macOS

初回起動時に StoreKit の購入が復元されない問題を修正。ヘルプメニューにサポートメールリンクを追加しました。

2026-04-211.0.0
macOSRelease

v1.0.0 — macOS リリース

Mac App Store 初回リリース。漫画アーカイブ（CBZ/CBR/ZIP/RAR）と写真の感知ハッシュスキャン。Unicode 対応のファイル名あいまい一致。スマート自動選択、永続的な無視ペアリスト、瞬時の再スキャン用ハッシュキャッシュ、複数フォルダ比較、HTML/CSV エクスポート。

2026-04-22
PC (Windows)Release

Windows — Microsoft Store への申請

Microsoft Store で利用可能になりました。感知ハッシュスキャン、写真の重複除去、ファイル名あいまい一致、自動選択ストラテジー、ハッシュキャッシュ、複数フォルダ比較、HTML/CSV エクスポートを含みます。7 日間無料トライアル；一回払いの無制限アンロック。
