既知の問題

iOS & macOS

ストリーミングは Tailscale が必要 — LAN のみモードは利用不可

iOS の App Transport Security がローカル IP への平文 HTTP をブロックします。Tailscale は自宅 Wi-Fi とリモートストリーミングの両方に必要です。Tailscale なしの LAN 専用モードは現在ロードマップに含まれていません。

Android

Mac/PC → Android ストリーミングは非対応

Android は iCloud 統合のない別の Flutter コードベースを使用しています。ストリーミングサポートにはほぼ全面的な書き直しが必要で、現在予定はありません。

Android

SMB ネットワークアクセスは未検証

最近のリリースでは、Android の SMB ドライブアクセスが完全にテストされていません。
