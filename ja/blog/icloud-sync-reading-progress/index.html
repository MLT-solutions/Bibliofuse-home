BiblioFuseがiCloudを通じてiPhone・iPad・Mac間で読書進捗を同期する方法

BiblioFuse は EPUB、CBZ、CBR、PDF ファイルの読書進捗・タグ・評価・読み方向を、iPhone・iPad・Mac 間で iCloud を通じて自動同期する。iOS 17 以降の iCloud Drive に保存されたファイルが対象で、進捗はファイル名でなくコンテンツハッシュに紐づくため、リネームやフォルダ移動後も正確に引き継がれる。iCloud 同期はプレミアム不要で利用できる。

通勤中にiPhoneでマンガを読む——80ページまで進んで帰宅後にiPadに切り替えると、どこまで読んだか探さなければならない。あるいはMacで一章読み、翌朝iPhoneを手に取ると第1ページが表示されている。

読書進捗の同期はシンプルに見えるが、多くのリーダーで微妙に失敗している。BiblioFuseはiCloudを使い、ページ数・読み方向・表示設定を含む正確な読書位置を、iPhone・iPad・Mac間で手動操作なしに同期する。

なぜ進捗同期が難しいのか

単純な方法——クラウドドキュメントにページ番号を保存する——はいくつかの点で失敗する：異なるエンコーディングでページ数が異なるボリュームに対応できない、読み方向を同期しない、デバイスをまたいで「読了」を永続的に管理できない。

BiblioFuseは進捗データをファイル名ではなくコンテンツハッシュに紐付けたメタデータとして保存する。CBZをリネームしたりフォルダ間で移動しても、どのデバイスでも正しく進捗が一致する。

iCloud同期の設定

iPhone/iPad：システムの設定 → Apple ID → iCloud → iCloud Driveと進み、BiblioFuseのトグルがオンになっているか確認する。

Mac：システム設定 → Apple ID → iCloudを開き、iCloud Driveを有効にしてBiblioFuseがリストにあることを確認する。

設定後はBiblioFuseがすべて自動で管理する。「今すぐ同期」ボタンはなく、デバイスがオンラインのときに変更がバックグラウンドで伝播する。

同期される内容

読書位置：正確なページ番号。87ページで閉じれば、他のデバイスも87ページで開く。読み方向：右から左に設定したマンガは、その設定が全デバイスに反映される。完了状態：一つのデバイスで読了マークをつければ全デバイスに反映。タグと評価：タグと星評価が全デバイスで同期される。

iCloud Driveでのファイル同期

BiblioFuseはライブラリファイル自体にもiCloud Driveを活用する。Mac側でFinderのiCloud Drive → BiblioFuseフォルダにCBZ・CBR・EPUB・PDFを入れると、数分以内にiPhoneのBiblioFuseに表示される。Wi-Fi転送で追加したファイルも同様にiCloudフォルダに保存され、他のデバイスから利用できる。

大量転送はWi-Fi転送：設定のWi-Fiインポートをオンにして、同一Wi-Fiネットワーク上のブラウザで表示URLを開き、ファイルをドラッグする。ローカルネットワーク速度（50–100 MB/s）で転送される。

Macホームライブラリ：コピー不要で読む

メインコレクションがMacにある場合、Macホームライブラリを使えばホームWi-Fi経由でiPhoneにストリーミングできる。デバイスにもiCloudにもファイルはコピーされない。MacのBiblioFuseで環境設定 → ホームライブラリを有効にし、iPhoneでMacライブラリタブに切り替えるだけ。ストリーミングセッションの進捗もiCloudで同期される。

オフライン読書

オフラインで読んだ場合、BiblioFuseは進捗をローカルに保存し、次の接続時に同期する。2台のデバイスが競合した場合、ページ番号が多い方（より読み進んでいる方）が優先される。

始めよう

BiblioFuseはApp Storeで無料配布（iPhone・iPad対応）。iCloud同期はプレミアムなしで利用できる。デバイス設定でBiblioFuseのiCloud Driveを有効にすれば、すぐに進捗の同期が始まる。

よくある質問

BiblioFuse はどのようにして iPhone と iPad 間で読書進捗を同期しますか？ BiblioFuse は進捗メタデータを iCloud に保存し、ファイル名でなくコンテンツハッシュに紐づけます。そのため、ファイルをリネームや移動しても進捗は正確に引き継がれます。設定 → Apple ID → iCloud で BiblioFuse の iCloud Drive を有効にすると同期が始まります。

BiblioFuse は CBZ やマンガファイルの進捗も同期しますか？ はい。CBZ、CBR、EPUB、PDF で進捗同期に対応しています。CBZ/CBR は正確なページ番号、EPUB はチャプター内の正確な位置が同期されます。読み方向（左→右、右→左）もボリュームごとに同期されます。

BiblioFuse で iCloud 同期を有効にするには何を設定すればよいですか？ 2 つ確認してください。設定 → Apple ID → iCloud で iCloud Drive が有効になっていること、そのリストで BiblioFuse がオンになっていること。両方オンにすれば自動で同期が始まります。「今すぐ同期」ボタンはありません。

オフラインで読んでも iCloud 同期は機能しますか？ はい。インターネットなしで読んだ場合、BiblioFuse は進捗をローカルに保存し、次に iCloud に接続したときに送信します。2 台のデバイスが競合した場合、ページ番号が多い方（より読み進んだ方）が優先されます。

BiblioFuse の iCloud 同期と Mac Home Library の違いは何ですか？ iCloud 同期はファイルと進捗データをクラウドにコピーし、どのデバイスでも利用できるようにします。Mac Home Library は Mac からローカル Wi-Fi 経由でオンデマンドにストリーミングし、デバイスや iCloud には何もコピーされません。オフラインでどこでも読む場合は iCloud、自宅でコレクション全体を閲覧したい場合は Mac Home Library を使いましょう。
