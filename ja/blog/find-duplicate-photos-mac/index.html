Mac で重複写真を見つける方法——HEIC・WebP・RAW 形式にも対応

Comic Duplicate Scanner は、知覚ハッシュ技術を使用して Mac 上の重複写真を検出します。ファイル名・形式・圧縮設定が異なっていても、視覚的に同一の画像を一致させることができます。HEIC、WebP、JPEG、PNG、TIFF、RAW など数十種類の形式に対応しており、Mac と Windows の両方で利用可能です。重複ファイルは完全削除ではなくゴミ箱に安全に移動されます。

際限なく膨らむフォトライブラリの問題とは？

5 年前は管理しやすかった Mac のフォトライブラリが、今では 40,000 個のファイルとなり、ダウンロードフォルダ・バックアップドライブ・3 回分の iCloud エクスポートに散在しています。同じ風景写真が IMG_4521.HEIC、IMG_4521-edited.JPG、landscape-final.webp として保存されているかもしれません。RAW の連写バーストでは、11 枚のほぼ同じフレームが選別後も残っています。そして iPhoto から Apple 写真、新しい Mac への移行のたびに、重複ファイルの層が積み重なってきました。

標準搭載のツールでは対処できません。Apple 写真にも基本的な重複検索機能はありますが、ライブラリ内の写真だけが対象で、Finder 上に散在するファイルは見逃してしまいます。Spotlight は画像内容を比較できません。一般的なファイル重複検索ツールはバイト単位のハッシュ比較を行うため、JPEG 圧縮率が異なったり再エクスポートされたりした時点で機能しなくなります。

なぜ知覚ハッシュはあらゆる形式で重複写真を見つけられるのか？

知覚ハッシュは、バイトデータではなく画像の見た目に着目します。Comic Duplicate Scanner はすべての写真に対してコンパクトな視覚フィンガープリントを算出します。人の目には同じに見える 2 枚の画像は、一方が HEIC でもう一方が JPEG であっても、85% 品質で再エクスポートされていても、ファイル名が変更されていても、同じ（または非常に似た）フィンガープリントを生成します。

これが汎用ファイル重複排除ツールとの根本的な違いです。汎用ツールは SHA-256 ハッシュ——バイト単位で完全一致——のみを比較します。知覚ハッシュは、視覚的には同一でも技術的には異なるファイルを一致させることができます。

Mac で重複写真を見つけるステップごとの手順

  1. Comic Duplicate Scanner をダウンロードして開く——Mac App Store または開発者サイトから入手します。
  2. 写真フォルダを追加する——写真ライブラリのエクスポートフォルダ、ダウンロードフォルダ、スキャンしたい外付けドライブをドラッグして追加します。1 回のスキャンで複数のフォルダを追加できます。
  3. スキャンを実行する——アプリは知覚ハッシュを使用してサポートされるすべての画像のフィンガープリントを生成します。M シリーズ Mac では 10,000 枚の写真ライブラリを通常 2 分以内にスキャンします。
  4. グループを確認する——重複ファイルはグループ単位で表示されます。各グループにはファイル名・形式・ファイルサイズ・解像度・サムネイルが表示され、一致内容を確認できます。
  5. 削除戦略を選択する——Comic Duplicate Scanner は自動選択戦略を提供します：最大ファイルを保持（最高品質）、最古ファイルを保持（オリジナル）、優先フォルダ内のファイルを保持、などです。手動選択も可能です。
  6. ゴミ箱へ移動する——アプリはファイルを完全削除しません。選択された重複ファイルは Mac のゴミ箱に移動されるため、誤って削除したファイルをいつでも復元できます。

対応写真形式は？

Comic Duplicate Scanner は Mac ユーザーが遭遇するあらゆる形式に対応しています：

  • HEIC / HEIF——iPhone のデフォルト形式。HDR およびポートレートモードのファイルも含む
  • JPEG / JPG——再エクスポートや編集版を含む汎用形式
  • WebP——SNS ダウンロードやウェブスクリーンショットで増加中の形式
  • PNG——ロスレスのスクリーンショットやグラフィック
  • TIFF——高品質スキャンやレイヤー付きエクスポート
  • RAW 形式——Canon CR2/CR3、Nikon NEF、Sony ARW、FUJIFILM RAF、Adobe DNG など
  • CBZ / CBR——コミックアーカイブも写真と並んでサポート

ファジーファイル名マッチング機能は、vacation-2023.jpg、vacation-2023(1).jpg、vacation-2023-copy.jpg のようなグループを数秒で検出します。視覚ハッシュ比較が実行される前にも機能します。

3 つの実践的な使用シーン

1. 移行後のクリーンアップ Mac 移行のたびに、Apple の移行アシスタントはファイルをコピーしますが重複排除は行いません。複数回 Mac を乗り換えてきた場合、最も古い写真には 2〜3 枚のコピーが存在する可能性があります。ホームフォルダ全体に Comic Duplicate Scanner を実行してすべて見つけましょう。

2. RAW + JPEG ペアの整理 多くのカメラ——および iPhone の ProRAW モード——は RAW と JPEG を同時に保存します。Lightroom や Capture One での編集後、元の JPEG は不要になることがほとんどです。Comic Duplicate Scanner は RAW ファイルと対応する JPEG をグループ化するため、日付で手動ソートしなくても不要なファイルを削除できます。

4. カラー写真とモノクロエクスポートの比較 カラーで撮影したが、プロジェクト用にモノクロバージョンをエクスポートし、どちらがどちらか忘れてしまった。知覚ハッシュは色の値ではなく輝度構造を比較するため、カラーオリジナルとそのグレースケールエクスポートはほぼ同一のハッシュ値を生成し、重複としてグループ化されます — バイトツールが完全に異なるファイルと見なしても同様です。

3. SNS ダウンロードの蓄積 Instagram のスクリーンショットや保存した写真が何年も積み重なると、同じ画像の異なる解像度のコピーが数十枚になります。ファイル名がランダムな数字の文字列でも、知覚ハッシュがすべて検出します。

よくある質問

Comic Duplicate Scanner は写真を完全削除しますか？

いいえ。アプリはファイルを Mac のゴミ箱に移動します。自分でゴミ箱を空にするまで、何も完全削除されません。誤って削除したファイルを復元できる完全な安全ネットが備わっています。

HEIC と JPEG など異なる形式間で重複を見つけられますか？

はい。知覚ハッシュは視覚的なコンテンツを比較するため、ファイルのバイト単位比較は行いません。HEIC として保存され JPEG として再エクスポートされた写真は、ファイル形式もバイト内容も異なっていても重複として検出されます。

大容量ライブラリのスキャンにどのくらいかかりますか？

Apple M シリーズ搭載 Mac では、10,000 枚の写真ライブラリが通常 2 分以内に完了します。50,000 枚以上の場合は、ドライブ速度やファイル形式にもよりますが 5〜10 分かかります。HDR メタデータを含む HEIC ファイルはデコードに若干時間がかかります。

RAW 連写の近似重複を見つけられますか？

はい。類似度しきい値を高感度に設定すれば可能です。同じシーンの連写写真は見た目がほぼ同じで、非常に似た知覚ハッシュ値を生成します。感度を調整してグループ化し、最もシャープなフレームだけを残せます。

外付けドライブやネットワークストレージにも対応していますか？

対応しています。Finder でアクセス可能な任意のフォルダ——内蔵ドライブ、外付け USB ドライブ、SD カード、ネットワークマウントボリューム——を追加できます。スキャンは Mac 上でローカルに実行され、ファイルがサーバーにアップロードされることはありません。

カラー写真と同じ画像のモノクロバージョンを照合できますか？

はい。dHashアルゴリズムは指紋を計算する前に各画像をグレースケールに変換するため、色の値ではなく輝度パターンを比較します。フルカラー写真とそのモノクロエクスポートはほぼ同一のハッシュ値を生成し、重複としてグループ化されます。

画像の解像度は重複検出に影響しますか？

いいえ。dHashは比較前にすべての画像を小さな固定グリッドにリサイズするため、同じ写真の低解像度ウェブダウンロードと高解像度オリジナルは同じハッシュ値を生成します。ソース間の解像度の違いは完全に無視されます。

Comic Duplicate Scanner に Windows 版はありますか？

あります。Comic Duplicate Scanner は Mac と Windows の両方で利用可能です。Windows 版も同じ知覚ハッシュエンジンを搭載し、同じファイル形式をサポートしています。

ストレージを安全に取り戻す

重複ファイルが 30% 含まれる 40,000 枚のライブラリは、数十 GB のストレージを無駄にしています。Comic Duplicate Scanner は安全なクリーンアップツールを提供します——形式をまたいで視覚的重複を検出する知覚ハッシュ、手間を省く自動選択戦略、そして安全なゴミ箱のみの削除機能。
