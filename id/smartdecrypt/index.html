Buka Otomatis Berbasis Pola

Simpan sekali, buka selamanya

Laporan bank, PDF asuransi, dan slip gaji selalu tiba dengan pola kata sandi yang sama — NRIC Anda, tanggal lahir, atau nomor rekening. Simpan profil sekali dan SmartDecrypt akan memilih kata sandi yang tepat secara otomatis setiap kali jenis file tersebut tiba.

  • Cocokkan berdasarkan pola nama file — misalnya "DBS_Statement_*" → kata sandi NRIC Anda
  • Profil terpisah untuk laporan bank, PDF asuransi, faktur, slip gaji
  • Siklus kata sandi otomatis — mencoba semua profil yang cocok secara berurutan

Format yang Didukung

PDF, ZIP, dan 7z — ketiga-tiganya

Buka PDF terlindungi kata sandi, ekstrak arsip ZIP, dan buka file 7z — termasuk arsip yang tidak terenkripsi yang diekstrak secara normal tanpa meminta kata sandi.

  • PDF — buka dan jaga dokumen tetap utuh
  • ZIP & 7z — ekstrak isi dengan atau tanpa kata sandi
  • Arsip yang tidak terenkripsi diekstrak otomatis — tidak perlu prompt
iOS Share Sheet & Keychain

Terima file dari mana saja. Kata sandi tetap pribadi.

Di iPhone dan iPad, buka file langsung di SmartDecrypt atau kirimkan melalui iOS Share Sheet dari Files, Mail, atau aplikasi lainnya. Kata sandi disimpan secara eksklusif di Keychain sistem — tidak pernah di database, tidak pernah di server.

  • iOS Share Sheet — kirim file dari Files, Mail, Safari, dan lainnya
  • Kata sandi disimpan di Apple Keychain, bukan di aplikasi
  • File diproses secara lokal — tidak ada yang diunggah

FAQ

Pertanyaan yang Sering Diajukan

Format file apa yang didukung SmartDecrypt?

PDF, ZIP, dan 7z. Arsip ZIP dan 7z yang tidak terenkripsi diekstrak secara normal tanpa meminta kata sandi. SmartDecrypt saat ini tidak mendukung RAR, EPUB, atau format lainnya.

Apakah SmartDecrypt melewati kata sandi yang tidak diketahui?

Tidak. SmartDecrypt dirancang untuk file yang sudah Anda miliki izin untuk akses. Ia menggunakan kata sandi yang Anda simpan di profil — tidak membuka paksa atau brute-force kata sandi yang tidak diketahui.

Apakah kata sandi saya aman?

Ya. Kata sandi disimpan secara eksklusif di Keychain sistem di perangkat Anda — penyimpanan aman yang sama yang digunakan oleh Safari dan aplikasi lainnya. SmartDecrypt tidak pernah menyimpan kata sandi di database-nya sendiri, dan tidak ada yang diunggah ke server.

Apa itu pola nama file?

Pola nama file adalah nama parsial atau kata kunci yang mengidentifikasi file berulang. Misalnya, jika bank Anda mengirim laporan bulanan bernama 'BankStatement_2024-01.pdf', buat profil dengan pola 'BankStatement' dan kata sandi. SmartDecrypt akan mencocokkan file di masa depan dengan profil tersebut secara otomatis.

Apakah pembelian Pro dibagikan antara iPhone dan Mac?

Ya. SmartDecrypt Pro adalah pembelian universal sekali. Beli sekali di iOS atau macOS dan buka Pro di kedua platform menggunakan Apple ID yang sama.
