Cara Menggabungkan Beberapa File CBZ Menjadi Satu — Alat Online Gratis

Kamu mengunduh seri manga dan setiap chapter hadir sebagai file CBZ terpisah. Sekarang ada 40 file di satu folder — Chapter 001.cbz hingga Chapter 040.cbz — dan setiap kali selesai membaca satu chapter, kamu harus membuka file berikutnya secara manual. Akan lebih nyaman membaca seluruh volume sebagai satu file CBZ. Masalahnya, menggabungkan file-file itu tampaknya memerlukan instalasi software atau layanan berbayar.

BiblioFuse Web Tool menggabungkan file CBZ langsung di browser, secara gratis. Tidak ada yang diunggah ke server. Tidak perlu akun. File hasil gabungan langsung diunduh ke komputermu.

Mengapa Perlu Menggabungkan File CBZ

Pola satu chapter satu file sangat umum dalam manga dan seri panjang. Grup scan mengemas tiap chapter secara terpisah. Toko digital terkadang menjual isu secara satuan meski kamu sudah membeli satu run lengkap. Saat membaca di ponsel atau tablet, harus berpindah ke file berikutnya setiap 30 halaman akan memutus alur membacamu.

Satu file CBZ gabungan untuk satu volume atau arc penuh memberikanmu:

  • Membaca tanpa gangguan — reader memperlakukannya sebagai satu file berkelanjutan, menangani halaman ganda di batas chapter dengan benar
  • Perpustakaan lebih rapi — satu file per volume, bukan puluhan file chapter terpisah
  • Pelacakan progres yang konsisten — persentase baca dan bookmark bekerja di seluruh volume, tidak reset per chapter
  • Transfer lebih mudah — satu file untuk dipindahkan ke ponsel, bukan banyak file

Cara Kerja Penggabungan CBZ

CBZ adalah arsip ZIP yang berisi file gambar, satu per halaman, dinamai secara berurutan agar reader tahu urutannya. Menggabungkan file CBZ berarti menyatukan rangkaian gambar tersebut menjadi satu ZIP, dengan penomoran ulang agar terbaca sebagai cerita yang berkelanjutan.

BiblioFuse Web Tool menangani penomoran ulang secara otomatis. Kamu menyediakan file dan menentukan urutan; alat ini mengekstrak gambar dari setiap CBZ, mengurutkannya dengan benar, lalu mengemasnya menjadi satu CBZ output.

Cara Menggabungkan File CBZ Secara Online

Langkah 1: Buka BiblioFuse Web Tool

Buka BiblioFuse Web Tool di browser desktop. Tidak perlu akun atau login. Fitur penggabungan tersedia di tier gratis untuk batch hingga tiga file; upgrade untuk batch yang lebih besar.

Langkah 2: Tambahkan file CBZ sesuai urutan

Klik area drop atau seret file CBZ ke sana. Urutan penting — alat ini menggabungkan file sesuai urutan kamu menambahkannya. Sebagian besar browser mengizinkan memilih beberapa file sekaligus dari folder; jika file-filemu sudah diberi nama berurutan (Chapter 001.cbz, Chapter 002.cbz, dst.), memilih semuanya sekaligus biasanya akan menambahkannya dalam urutan yang benar.

Jika perlu mengubah urutan setelah ditambahkan, kamu bisa menyeret file ke posisi yang tepat dalam daftar file alat ini sebelum memulai penggabungan.

Langkah 3: Jalankan penggabungan

Klik Gabungkan. Alat ini menyatukan gambar dari semua file CBZ menjadi satu arsip, menomori ulang halaman untuk membentuk urutan yang berkelanjutan. Indikator progres menunjukkan prosesnya. Setelah selesai, CBZ hasil gabungan siap diunduh.

Nama file output secara default mengikuti nama file pertama dalam batch-mu, yang bisa kamu ganti saat menyimpan.

Menangani Berbagai Jenis File Sekaligus

BiblioFuse Web Tool bisa menggabungkan lebih dari sekadar file CBZ. Jika serimu menggunakan campuran format — beberapa chapter sebagai CBZ, beberapa sebagai CBR, beberapa sebagai ZIP — alat ini menangani semuanya bersama. Kamu tidak perlu mengonversi semuanya ke format yang sama sebelum menggabungkan. Alat ini mengekstrak halaman dari setiap format input dan mengemas hasilnya sebagai CBZ.

Ini berguna untuk koleksi scan lama, di mana konsistensi format jarang menjadi prioritas.

Kualitas Gambar Setelah Penggabungan

Penggabungan tidak memengaruhi kualitas gambar. Alat ini menggabungkan file gambar dari arsip input tanpa mengenkode ulang atau mengompresinya. Setiap halaman dalam CBZ hasil gabungan identik dengan halaman yang sesuai di file aslinya.

Jika kamu juga ingin mengurangi ukuran file sekaligus menggabungkan, BiblioFuse Web Tool juga memiliki langkah kompresi yang bisa dijalankan sebelum atau sesudah penggabungan — mengenkode ulang gambar pada level kualitas tertentu untuk memperkecil file akhir. Tapi itu opsional; penggabungan biasa menjaga segalanya tetap sama persis.

Membuat Volume dari Banyak Chapter

Skenario umum adalah mengemas 10–12 chapter menjadi satu CBZ berukuran volume. Alur kerja ini berjalan dengan baik:

  1. Kumpulkan chapter yang termasuk dalam Volume 1 — untuk kebanyakan seri, itu chapter 1–10 atau 1–12.
  2. Buka BiblioFuse Web Tool dan tambahkan file-file chapter tersebut secara berurutan.
  3. Jalankan Gabungkan. Unduh hasilnya dan ganti namanya menjadi sesuatu seperti Nama Seri Vol 01.cbz.
  4. Ulangi untuk volume berikutnya.

Setiap file hasil adalah volume yang bersih dan mandiri yang bisa kamu transfer ke reader, katalogkan di aplikasi perpustakaan, atau diarsipkan.

Setelah Menggabungkan: Memindahkan File ke Ponsel

Setelah mengunduh CBZ hasil gabungan, cara paling praktis untuk memindahkannya ke iPhone adalah melalui fitur Wi-Fi Transfer BiblioFuse.

Buka BiblioFuse di iPhone-mu, ketuk tombol Wi-Fi Transfer, dan alamat browser akan muncul di layar. Di komputermu, buka alamat itu di browser. Seret file CBZ gabunganmu ke halaman tersebut. File langsung berpindah melalui jaringan lokal — tanpa USB, tanpa iTunes, tanpa akun cloud.

Pilihan lain:

  • iCloud Drive: Letakkan file di folder iCloud Drive yang terhubung ke BiblioFuse
  • AirDrop: Dari Mac, AirDrop file CBZ ke iPhone dan buka di BiblioFuse
  • Aplikasi Files: Hubungkan via USB, salin ke folder BiblioFuse di aplikasi Files

Pertanyaan Umum

Berapa maksimum file yang bisa digabungkan?

Tier gratis menangani hingga tiga file per batch. Untuk penggabungan yang lebih besar, pertimbangkan untuk membagi pekerjaan menjadi beberapa tahap: gabungkan chapter 1–3 menjadi file parsial, lalu gabungkan file parsial itu dengan batch berikutnya, dan seterusnya.

Bagaimana jika chapter-ku dalam format CBR?

CBR adalah arsip RAR, bukan ZIP. BiblioFuse Web Tool menangani CBR bersama dengan CBZ dan format lainnya — kamu bisa menambahkan campuran file CBZ dan CBR, dan semuanya akan digabungkan dengan benar menjadi satu output CBZ.

Apakah alat ini mempertahankan urutan baca?

Ya. Alat ini memproses file sesuai urutan yang tampak dalam daftar file, dan halaman dalam setiap file ditambahkan dalam urutan yang sudah ada. Selama kamu menambahkan file chapter dalam urutan yang tepat, output gabungan akan terbaca dengan benar.

Bisakah saya menggabungkan file EPUB dengan cara yang sama?

BiblioFuse Web Tool juga mendukung penggabungan EPUB, yang ditangani sebagai operasi terpisah. Untuk konten bergaya komik — halaman hasil scan, bukan teks reflowable — CBZ umumnya merupakan format target yang lebih baik.

Apakah file output mempertahankan metadata dari file aslinya?

CBZ hasil gabungan adalah arsip gambar biasa. Metadata ComicInfo.xml dari file aslinya tidak digabungkan secara otomatis. Jika kamu perlu menyertakan metadata dalam file gabungan, kamu bisa menambahkan ComicInfo.xml secara manual setelah mengunduh output.

Menggabungkan File CBZ Tanpa Repot

BiblioFuse Web Tool menangani penggabungan CBZ sebagaimana mestinya — di browser, secara privat, gratis, tanpa perlu menginstal apa pun. Kamu menyediakan file secara berurutan, dan alat ini mengurus sisanya.
