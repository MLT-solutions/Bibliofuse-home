Memulai dengan BiblioFuse

BiblioFuse adalah pembaca komik dan ebook gratis untuk iPhone, iPad, dan Mac yang mendukung file EPUB, CBZ, CBR, PDF, ZIP, RAR, dan TXT di iOS 17 atau lebih baru. File langsung dimuat melalui fitur Wi-Fi Transfer bawaan — tanpa kabel, tanpa iTunes, dan tanpa akun cloud. Tag, penilaian, dan progres membaca disinkronkan secara otomatis di semua perangkat Apple kamu melalui iCloud.

Anda mengunduh aplikasi baca, menyalin folder berisi file CBZ, dan sekarang perpustakaan Anda terlihat seperti lemari arsip setelah diterjang badai. Familiar?

BiblioFuse dibangun di atas ide sederhana: koleksi ebook, komik, dan dokumen Anda seharusnya terasa seperti perpustakaan sungguhan — terorganisir, cepat, dan sepenuhnya milik Anda. Tidak ada unggahan ke cloud, tidak ada langganan, tidak ada iklan. Hanya file Anda, di perangkat Anda.

Panduan ini mencakup semua yang Anda butuhkan untuk membangun perpustakaan yang berfungsi dari nol dalam beberapa menit.

Langkah 1 — Tambahkan Buku Pertama

BiblioFuse mendukung EPUB, PDF, CBZ, CBR, ZIP, RAR, dan TXT. Ada tiga cara untuk mengimpor file:

Berbagi dari aplikasi lain — di iPhone atau iPad, buka file yang kompatibel dari aplikasi File, Mail, atau Safari. Ketuk ikon Bagikan dan pilih Buka di BiblioFuse. File langsung muncul di perpustakaan Anda.

Transfer Wi-Fi — ini adalah cara tercepat untuk mengimpor koleksi besar dari Mac atau PC. Di BiblioFuse, buka Pengaturan → Import Wi-Fi dan aktifkan. URL lokal akan muncul di layar — buka di browser desktop Anda dan seret file ke sana. Tanpa kabel, tanpa iTunes, tanpa cloud.

Streaming Perpustakaan Mac — jika koleksi utama Anda ada di Mac, instal aplikasi BiblioFuse untuk Mac dan aktifkan perpustakaan rumah. iPhone dan iPad Anda akan bisa mengakses seluruh perpustakaan Mac melalui Wi-Fi — atau melalui 5G dengan Tailscale — tanpa menyalin file apa pun ke perangkat.

Langkah 2 — Atur dengan Tag dan Penilaian

Setelah buku diimpor, tekan dan tahan sampul mana pun untuk membuka menu aksi. Dari sini Anda bisa:

  • Menandai buku dengan label kustom — Fiksi Ilmiah, Belum Dibaca, Selesai, apa pun yang Anda inginkan
  • Memberi nilai dari 1 hingga 5 bintang
  • Menghapus dari perangkat tanpa menghapusnya dari perpustakaan Mac

Tag dan penilaian tersinkronisasi di iPhone, iPad, dan Mac melalui iCloud. Bilah filter di bagian atas kisi perpustakaan memungkinkan Anda menyortir dan memfilter berdasarkan tag, penilaian, atau jenis file.

Langkah 3 — Pilih Mode Membaca

Ketuk sampul mana pun untuk membukanya. BiblioFuse mendeteksi jenis file dan menggunakan mode membaca yang sesuai secara otomatis, tetapi Anda bisa menggantinya kapan saja melalui bilah alat dengan memilih Tata Letak.

Peek Zoom berguna di halaman komik yang padat: tekan dan tahan panel mana pun untuk mendapatkan tampilan perbesar tanpa kehilangan posisi baca.

Langkah 4 — Hemat Penyimpanan dengan Kompresi

File CBZ dan EPUB yang besar cepat menumpuk, terutama di iPhone 128 GB. BiblioFuse menyertakan alat kompresi langsung di perangkat — tidak ada yang diunggah.

  1. Tekan dan tahan buku di kisi perpustakaan, lalu ketuk Kompres.
  2. Sesuaikan slider kualitas. Kompresi lebih tinggi = file lebih kecil, lebih rendah = kualitas gambar lebih baik.
  3. Ketuk Kompres. File diganti di tempat ketika selesai.

Untuk menggabungkan seri manga menjadi satu volume, tekan dan tahan beberapa file CBZ untuk memilihnya, lalu ketuk Gabungkan.

Apa yang Perlu Dijelajahi Selanjutnya

  • BiblioFuse Web Tool — kompres dan gabungkan file EPUB, PDF, dan CBZ di browser mana pun, tanpa instalasi.
  • iCloud Bookshelf — aktifkan di Pengaturan untuk menyinkronkan perpustakaan, posisi baca, tag, dan penilaian secara otomatis.
  • Comic Duplicate Scanner — aplikasi pendamping untuk Mac yang menemukan file CBZ dan foto duplikat menggunakan perceptual hashing.

Pertanyaan yang Sering Diajukan

Format apa saja yang didukung BiblioFuse? BiblioFuse membaca EPUB, PDF, CBZ, CBR, ZIP, RAR, dan TXT secara native — tanpa konversi. CBZ dan CBR langsung terbuka tanpa pengaturan tambahan.

Apakah saya perlu akun atau langganan untuk menggunakan BiblioFuse? Tidak. BiblioFuse gratis diunduh dan tidak memerlukan akun, login, atau langganan untuk membaca file. Perpustakaan disimpan secara lokal di perangkat kamu.

Bagaimana cara memindahkan koleksi komik ke BiblioFuse? Metode tercepat adalah Wi-Fi Transfer: buka BiblioFuse, aktifkan Wi-Fi Import di Pengaturan, lalu kunjungi URL yang ditampilkan di browser mana pun di komputer kamu dan seret file ke dalamnya. Koleksi 10 GB dapat ditransfer dalam beberapa menit di jaringan Wi-Fi rumahan.

Apakah progres membaca tersinkronisasi antara iPhone dan iPad? Ya. Posisi membaca, tag, penilaian, dan arah membaca disinkronkan melalui iCloud antara iPhone, iPad, dan Mac. Aktifkan iCloud Drive untuk BiblioFuse di pengaturan perangkat untuk mengaktifkan sinkronisasi.

Bisakah saya membaca manga dari kanan ke kiri di BiblioFuse? Bisa. Ketuk bagian tengah halaman saat membaca untuk menampilkan kontrol, lalu ubah arah membaca ke Kanan ke Kiri. BiblioFuse menyimpan pengaturan ini per volume sehingga hanya perlu diatur sekali per seri.
