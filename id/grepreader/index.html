Tag dan Penemuan

Tag berdasarkan genre. Filter seketika.

Tag berasal dari apa yang sebenarnya ada di dalam buku — bukan dari judul, penulis, atau ISBN yang harus Anda ketik secara manual. Terapkan tag dari hasil pemindaian dengan satu ketukan. Filter perpustakaan menggunakan logika AND untuk menemukan buku yang cocok dengan SEMUA tag yang dipilih sekaligus.

  • 5 tag gratis · Pro membuka tak terbatas
  • Pilih banyak tag dengan logika AND — temukan buku yang cocok dengan SEMUA tag yang dipilih
  • Penandaan satu ketuk langsung dari hasil pencarian — tanpa isian formulir
Sinkronisasi Wi-Fi

Sinkronkan lewat Wi‑Fi lokal, dengan fallback penemuan Apple.

Transfer buku, tag, rating, profil kata kunci, dan struktur folder antara iPhone, iPad, Mac, dan Windows melalui jaringan lokal Anda. Perangkat Apple dapat memakai snapshot metadata iCloud kecil untuk penemuan, tetapi file buku ditransfer melalui Wi‑Fi dan tidak diunggah ke server pengembang.

  • Mac / PC ↔ iPhone / iPad, iPhone ↔ iPad, Mac ↔ Mac, PC ↔ PC
  • Buku, tag, rating, profil kata kunci, dan hierarki folder disinkronkan antar perangkat
  • Fitur Pro — server perangkat sumber harus berjalan selama transfer

FAQ

Pertanyaan yang sering diajukan

Apa itu GrepTag Reader?

GrepTag Reader adalah aplikasi pencarian teks penuh dan penandaan untuk koleksi ebook EPUB dan TXT di iPhone, iPad, Mac, dan Windows. Memungkinkan Anda mencari seluruh perpustakaan untuk kata kunci apa pun menggunakan logika boolean AND / OR / NOT, menandai buku berdasarkan genre, dan menyinkronkan perpustakaan melalui Wi-Fi lokal — tanpa mengunggah file ke server mana pun.

Apa perbedaan antara GrepTag Reader dan BiblioFuse Reader?

BiblioFuse Reader dibuat untuk file komik berukuran besar (CBZ, CBR, PDF) yang kaya gambar. File ini melakukan streaming dari Mac, mengompres arsip, dan mendukung membaca gaya webtoon/manga. GrepTag Reader dibuat untuk perpustakaan teks EPUB dan TXT — menawarkan pencarian grep teks penuh, tag genre, dan sinkronisasi Wi-Fi ke segala arah. Keduanya adalah aplikasi terpisah karena model penyimpanan yang sangat berbeda: komik membutuhkan streaming, novel muat sepenuhnya di perangkat.

Apakah pencarian grep teks penuh gratis?

Ya. Pemindaian grep selalu gratis dan tidak terbatas — Anda dapat mencari seluruh perpustakaan EPUB dan TXT untuk kata kunci apa pun sebanyak yang Anda inginkan. Simpan hingga 2 profil pencarian bernama secara gratis. Pro membuka profil tak terbatas.

Apakah GrepTag Reader mengunggah file saya ke server?

Tidak. GrepTag Reader membaca langsung dari file lokal Anda. Tidak ada yang diunggah ke server mana pun — tanpa indeks cloud, tanpa backend pengembang. File tetap ada di perangkat Anda.

Format file apa yang didukung GrepTag Reader?

GrepTag Reader mendukung file EPUB dan TXT. Dapat mencari dan menandai kedua format, mengekstrak sampul EPUB secara otomatis, dan mencocokkan sampul untuk buku TXT dari gambar di folder yang sama atau URL yang ditempelkan (fitur Pro).

Berapa banyak tag yang bisa saya miliki secara gratis?

Lima tag disertakan dalam tingkatan gratis. Pro membuka tag tak terbatas. Anda dapat menerapkan tag dari hasil pemindaian dengan satu ketukan, atau menandai buku secara manual dan memfilter berdasarkan beberapa tag menggunakan logika AND atau OR.

Bisakah saya menyinkronkan perpustakaan antara iPhone, iPad, Mac, dan PC?

Ya. Wi-Fi Sync mentransfer buku, tag, penilaian, profil kata kunci, dan struktur folder antara kombinasi apa pun dari iPhone, iPad, Mac, dan Windows PC. Semua perangkat Apple dalam sinkronisasi harus masuk ke akun iCloud yang sama — iCloud hanya digunakan untuk penemuan perangkat, bukan untuk menyimpan buku Anda. File berpindah melalui jaringan Wi-Fi lokal Anda dan tidak pernah diunggah ke server pengembang mana pun. Wi-Fi Sync adalah fitur Pro.

Bisakah saya menggunakan lebih dari satu iPhone atau iPad dengan Wi-Fi Sync?

Ya. Beberapa iPhone dan iPad semuanya dapat berpartisipasi dalam sinkronisasi perpustakaan yang sama. Versi iOS GrepTag menyimpan jalur relatif terhadap folder root aplikasi, sehingga struktur jalur yang sama berfungsi di setiap perangkat iOS yang masuk ke akun iCloud yang sama.

Haruskah saya menggunakan Mac atau PC sebagai sumber utama perpustakaan saya?

Kami menyarankan untuk menjaga satu Mac atau satu PC sebagai perangkat perpustakaan utama. Metadata GrepTag melacak tiga jenis jalur terpisah — jalur PC, jalur iOS, dan jalur Mac. iOS bekerja di beberapa iPhone dan iPad karena menggunakan jalur relatif. Mac dan PC masing-masing menyimpan jalur absolut untuk perangkat mereka sendiri, sehingga sinkronisasi paling andal ketika hanya ada satu Mac dan satu PC dalam pengaturan Anda.

Bisakah iPhone saya menyinkronkan buku baru kembali ke Mac atau PC saya?

Ya. Jika Anda menambahkan buku di iPhone ke dalam folder yang belum ada di Mac atau PC Anda, GrepTag akan meminta Anda membuat folder tersebut dan menambahkannya ke daftar perpustakaan di perangkat penerima sebelum menyelesaikan sinkronisasi.

Bagaimana cara mencari kata atau frasa tertentu di dalam file ebook saya?

Buka GrepTag Reader, pergi ke tab Cari, dan ketik kata atau frasa apa pun. Aplikasi membuka setiap file EPUB dan TXT di perpustakaan Anda dan mengembalikan setiap buku yang mengandung istilah pencarian Anda — tanpa indeks yang dibangun sebelumnya, tanpa unggahan. Gabungkan istilah dengan AND, OR, dan NOT untuk hasil yang lebih tepat (misalnya: "sihir AND naga NOT romansa"). Simpan pencarian yang sering digunakan sebagai profil bernama yang dijalankan ulang secara otomatis saat buku baru diimpor. Semua pencarian selalu gratis dan tidak terbatas.

Apakah GrepTag Reader memerlukan koneksi internet untuk bekerja?

Tidak. GrepTag Reader bekerja sepenuhnya secara offline. Semua pencarian, penandaan, dan pengorganisasian terjadi langsung di perangkat Anda — aplikasi membaca file EPUB dan TXT Anda secara lokal tanpa memerlukan server atau koneksi internet. Jaringan hanya digunakan untuk fitur Wi-Fi Sync opsional yang mentransfer buku antara perangkat Anda sendiri melalui jaringan lokal.

Bisakah saya menyesuaikan tampilan dan font pembaca?

Ya. Pilih dari tema Sepia, Abu-abu, dan Hitam (OLED) di iPhone, iPad, dan Mac. Pilih font dengan pratinjau langsung — termasuk Iowan, Charter, Kaiti, dan font serif yang mendukung CJK — atau impor font .ttf/.otf milikmu sendiri. Sesuaikan kecerahan, perataan teks, dan tebal sesuai gaya membacamu.

Bisakah saya mencari di dalam buku yang sedang saya baca?

Ya. Pencarian seluruh buku memindai seluruh file TXT atau EPUB yang sedang kamu baca, bukan hanya bab saat ini. Di Mac, tersedia bilah pencarian khusus dengan pengalaman pratinjau font yang sama seperti di iOS.
