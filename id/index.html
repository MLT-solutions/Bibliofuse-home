Mengapa dua aplikasi?

Perpustakaan berbeda membutuhkan alat berbeda.

BiblioFuse Reader dibuat untuk file komik besar yang penuh gambar — streaming dari Mac atau PC, kompres di aplikasi. GrepTag dibuat untuk file teks kecil yang muat penuh di perangkat — grep, tag, sinkronisasi ke arah mana pun.
