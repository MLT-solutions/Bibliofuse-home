Masalah yang diketahui

iOS & macOS

Streaming memerlukan Tailscale — mode khusus LAN tidak tersedia

App Transport Security iOS memblokir HTTP biasa ke IP lokal. Tailscale diperlukan untuk Wi-Fi rumah dan streaming jarak jauh. Mode khusus LAN tanpa Tailscale saat ini tidak ada dalam roadmap.

Android

Tidak ada streaming Mac/PC → Android

Android menggunakan basis kode Flutter terpisah tanpa integrasi iCloud. Dukungan streaming memerlukan penulisan ulang hampir total dan tidak direncanakan.

Android

Akses jaringan SMB belum diverifikasi

Akses drive SMB di Android belum sepenuhnya diuji pada rilis terbaru.
