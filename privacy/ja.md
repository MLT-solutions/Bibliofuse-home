# **プライバシーポリシー - BiblioFuse Reader**

**発効日:** 2026-01-29

## **1. はじめに**

BiblioFuse Reader(「アプリ」)はModern Logic Tech Solutions(「当社」または「当社の」)によって提供されています。当社はお客様のプライバシーと透明性を優先します。

## **2. データ収集とクラウド同期**

当社は、お客様の個人ファイル、ドキュメント、または書籍を当社自身のサーバーに保存**しません**。

**クラウド同期:**

このアプリは、お客様の個人クラウドストレージアカウントを使用して、お客様のコンテンツ(EPUBs、PDF、コミックブックなど)をデバイス間で同期することを容易にします:

* **iCloud (iOS/macOS):** ファイルはお客様のプライベートiCloudコンテナに保存されます。当社は、アプリの特定のコンテナ外でこれらのファイルにアクセスすることはありません。
* **Google Drive (Android):** ファイルはお客様の個人的なGoogle Driveに保存されます。アプリは、お客様が明示的に選択したファイルまたは同期用に指定したフォルダーにのみアクセスします。

## **3. 支払いとサブスクリプション**

**アプリ内購入:**

このアプリは、1回限りの購入(「ライフタイム」)とサブスクリプション(「月額」、「年額」)を通じて有料機能を提供します。

* **支払い処理:** すべての金融取引は、**Apple App Store**または**Google Play ストア**によって直接処理されます。当社は、お客様のクレジットカード情報や請求詳細を収集、保存、またはアクセスすることはありません。
* **取引データ:** 当社は、お客様の「プレミアム」ステータスを検証するために匿名の取引領収書を受け取る場合があります。このデータには、お客様の個人的なアイデンティティは含まれません。

## **4. サードパーティサービス**

このアプリは、機能分析のためにお客様のデバイスを識別するために使用される情報を収集する可能性のあるサードパーティサービスを使用します:

* **Google Play Services**
* **Apple StoreKit**(購入検証用)

## **5. 権限**

このアプリが機能するには、次の権限が必要です:

* **ストレージアクセス:** 書籍ファイルをローカルで開く、読み取る、および管理するために必要です。
* **ネットワークアクセス:** 選択したクラウドプロバイダー(iCloudまたはGoogle Drive)とファイルを同期し、プレミアムライセンスを検証するために必要です。

## **6\. Windows アプリ (Windows 用 BiblioFuse)**

**Microsoft Store の支払い:**

Windows アプリは、**Microsoft Store** によってのみ処理される 1 回限りのアプリ内購入(「BiblioFuse Pro — 永久ロック解除」)を提供しています。当社は支払い情報を収集、保存、またはアクセスすることはありません。購入検証は Microsoft Store API (`Windows.Services.Store`) を通じて実行され、お客様のデバイスと Microsoft 間で直接通信します。当社は個人識別情報または請求情報は受け取りません。

**権限:**

* **ローカル ファイル アクセス:** デバイスに保存されている電子書籍ファイルを開く、読む、および表示するために必要です。
* **ローカル ネットワーク アクセス:** 同じ Wi-Fi ネットワーク上の iOS デバイスにライブラリをストリーミングするために必要です。ストリーミングは完全にオプションでユーザー開始です。データは当社のサーバーには送信されません — すべてのストリーミングはローカル ネットワーク上のピアツーピアです。

**テレメトリと分析:**

Windows アプリはテレメトリ、クラッシュ レポート、または使用分析を収集しません。お客様の使用に関するデータは当社または第三者に送信されません。
